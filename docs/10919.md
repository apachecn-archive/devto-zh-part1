# 什么是谷歌代码之夏(GSoC)？

> 原文：<https://dev.to/venkat2811/what-is-google-summer-of-code-gsoc--16ig>

[![banner-gsoc2016_2.png](img/d04e4ce8f143066c7c52191d26c06d9b.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--amG4l0He--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn.hashnode.com/res/hashnode/image/upload/v1668357680215/pEMzFrFMy.png)

Google Summer of Code 是一个全球性项目，旨在让更多的学生开发者参与开源软件开发。更重要的是，开源组织发现被 Google 选中成为这个项目的一部分是很有声望的。是啊！！谷歌根据一定的标准选择组织。并非所有开源组织都被选中。组织也发现它是吸引年轻优秀人才的最佳场所。

对于学生来说，参与并成功完成项目，在各方面都会有很大的优势。以下是我的一些经验。

*   暴露程度非常高。学生可以从写提案中学到很多东西。
*   大多数项目涉及大量的编码。学生可以编写大量代码。是的，很多代码！！
*   学生将对他们的项目拥有完全的所有权。他们开始思考设计方面的问题，并做了很多计划。然而，当学生陷入困境时，导师和社区成员会随时伸出援助之手。
*   由于大多数导师和学生将在不同的国家，他们通过 GitHub 进行合作。学生在使用 VCS 的协作环境中获得了非常好的知识。
*   大多数项目旨在解决现实世界的问题，学生编写的大部分代码都将被使用。做出有价值的贡献会有很大的满足感。
*   由于来自不同地方和文化的人将成为组织的一部分，学生们将学会适应并与他们有效地沟通。所以学生的沟通能力会提高很多。
*   学生社团是最好的部分。这非常令人兴奋。来自世界各地的学生在脸书合作。是的，我们在脸书 GSoC 2016、LinkedIn、Telegram 有单独的组。这是非常令人兴奋的部分。学生们分享他们的经验，提供支持和帮助，非常令人鼓舞。与陌生人交流总是很有趣。
*   GSoC 项目将为简历增加巨大的价值。学生们将会编写代码并学到比他们在大学里学到的更多的东西。
*   学生不必是伟大的算法极客。具有良好分析和编程技能的学生可以参与并成功完成该项目。
*   最后但同样重要的是，谷歌总是让你兴奋不已。学生在社区联络期后会得到一个欢迎礼包，包括 GSoC 贴纸、笔、日记和 500 美元。对于学生来说，通过中期评估的人将获得 2250 美元，成功完成项目的人将获得 2750 美元，这是谷歌和 GSoC T 恤的证书。
*   此外，很少有组织坚持每周或每月写关于项目的博客文章。所以学生们有很好的机会开始写博客。
