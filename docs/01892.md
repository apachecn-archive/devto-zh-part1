# 项目就是一切

> 原文:[https://dev.to/theiyd/projects-are-everything-dp9](https://dev.to/theiyd/projects-are-everything-dp9)

> 对于非 wow 项目来说，人生苦短。

回到 2015 年第三季度，在进入 11 年级后，我决定从 HTML、CSS 和 JavaScript 这些网络核心开始。我练习 HTML，研究标签，从 W3Schools 学到了很多东西。感谢网站。

我始终相信，一个人实际上不是靠强记来学习，而是通过做一些小项目来获得更广泛的理解。我开始做一些小项目，并在 Github 上开源其中一些*(我还学习了版本控制的基础知识‘Git’和‘Github’)*。

* * *

### [](#material-loaders)**材料装载机**

材质加载器是一个小的库，包含加载器和旋转器，作为一个专业网站的小元素，具有材质颜色的填充。

[![Material Loaders](../Images/da63b3557eafd928a3cf7d5103161629.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--56YoVWJ0--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/6kfxhaepprw4txk0v0ws.png)

### [](#omkarshindeme)**omkarshinde.me**

为了练习我早期的 HTML 和 CSS 技能，也为了给我的朋友 Omkar Shinde 建立一个作品集，我为他建了一个网站。这个项目让我在这个过程中学到了很多东西。

### [](#href-scraper)**Href 刮刀**

一个很小的 chrome 扩展，它从一个特定的页面中抓取所有的 hrefs，并把它们下载为一个文本文档。

[![Href Scraper](../Images/b7282508f0ff0a49396c770a665c7147.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--geQqa9wt--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/tvz9gwk7f5b0cmxwtua0.png)

### [](#gfontjs)**GFont.js**

GFont.js 是一个小库，它使用 Google Fonts API 获取所有字体并显示在一个`<select>`标签中。这是我第一次使用 API。

[![GFont.js](../Images/d80a8c7bf1a46e75a4cecf006fdd3bb1.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--edSLLNvn--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/p1i6v3t2kf75ctdx6et9.png)

### [](#inspire-me)**启发我**

当我开始从事 web 开发时，我会在网上搜索优秀的前端开发人员&看看他们的作品集，读读他们的故事。但我不会找到太多。所以我实现了一个基于 SQL 和 PHP 的项目，这是一个众包应用程序。

一个 web 应用程序，用户可以在其中添加开发人员的作品集链接，激励该特定领域的初学者。

### [](#fireupload)**火上传**

FireUpload 是一个用 Electron 制作的小型桌面应用程序，主要用于将文件上传到 Firebase，同时检索这些文件并下载到您的机器上。它还有一个**学期成绩跟踪系统**，我可以在那里输入我的学期成绩并保持跟踪。它还会保存我的文件&把它们备份在 Firebase &上，我可以在世界上任何时间任何地点检索这些文件。

[![FireUpload](../Images/653127014b7e4bc6a9e41a2d11ec5c53.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--yji-CIPM--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/cjxrajviwmops20dttf3.png)

### [](#jumble-password)**混杂密码**

一个小的电子工具，用你的名字和出生日期创建独特的密码组合。该应用程序使用的随机数字排列算法被称为费希尔耶茨洗牌算法，简而言之就是打乱一个序列。

[![Jumble Password](../Images/681ed74ee685ad33dc27ee1c3904a175.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--qvNWB-_x--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/xuowihllkf1cfwptq43s.gif)
