# 论坛 Magento 扩展

> 原文:[https://dev.to/fmeaddons/forum-magento-extension](https://dev.to/fmeaddons/forum-magento-extension)

FMEAddons 的论坛 Magento 扩展允许您在 Magento 电子商店上运行自己的论坛。这个插件让你可以完全控制管理所有的东西，如线程，文章，用户，垃圾邮件等。您还可以为产品问题、建议和偏离主题创建不同的部分，以吸引客户。论坛插件还可以帮助您轻松地在论坛中推广您的新产品。

Magento 论坛扩展的主要特性

*   启用/禁用签名
*   谷歌验证码
*   参与社区
*   推广产品
*   免费终身升级
*   100%退款保证
