# 开源贡献是我职业发展的一部分

> 原文：<https://dev.to/tryexceptpass/open-source-contributions-are-part-of-my-professional-development-38l>

### 我们应该有权利贡献开源吗？

“好家伙，我今天花了很多时间来组织我所有的问题”——他说，同时努力不让自己听起来像一个抱怨者——“你知道吗……我昨天发现了这个很好的工具，它可以很容易地与我们的存储库管理软件集成。它增加了项目管理功能，可以节省我们的时间。或许有兴趣看一看？”—他用一个问题结束了他的句子，这是他绝对讨厌做的事情，自从和一个不停这样做的同事相处了很长时间后，他就有了这个症结。他绝对不想在和他的经理谈话时显得不自信。

经理抬起头，离开电脑屏幕，他正努力地将问题重新分配给他们新请求的发布，而不是用令人恐惧的方式回答:“这个工具是免费的吗？下载了吗？”。程序员是一个孤独、孤立的人，只是想让自己的生活变得更轻松，他肯定没有想到会有这样的反应，所以他用未经过滤的事实回答了这个问题(你们中经常与管理层交流的人知道我在说什么)。
