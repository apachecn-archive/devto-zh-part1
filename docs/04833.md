## 我说的全职黑客是什么意思

如果你正在读这篇文章，很有可能你是一名开发人员，即使你不是，你至少应该知道谁是软件开发人员ðÿ˜‰.我在这里的任务是解释什么是黑客，谁是黑客，以及作为一名开发人员，了解他们的操作如何影响我们的决策是多么重要。

在过去，术语“黑客”被用来描述计算机专家。今天，这个术语已经和那些利用他们的技术知识，利用错误或漏洞侵入计算机系统的人联系在一起。

当我说一个开发人员是全职黑客时，我指的是那个开发人员是一个**白帽黑客**，一个主要在他的专业领域内掌握最新安全问题的人。例如，如果我是一名 web 开发人员，我应该知道我编写的每一行代码的安全含义。注意，我只是在支持[白帽黑客](https://www.techopedia.com/definition/10349/white-hat-hacker)，不要试图黑 NASA ðÿ˜'.永远不要使用黑客知识来破坏或非法进入任何系统。我并不是说你必须是一个黑客，我也不是说如果你不了解黑客，你就不是一个开发人员，我是说作为一个开发人员，至少学习一点这方面的知识是明智的，这样你就可以开发良好的编码实践，并把安全性放在第一位。
