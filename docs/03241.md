# woo 商务水印

[![](img/b18aaecea7372a5163cc2174862e6736.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--EpRUwwkE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/q68l5sh1xtcmpsryanqq.png) 
**WooCommerce 水印插件**通过给产品图像添加水印，使其安全可靠。您可以添加文本或上传任何图像来为产品图片添加水印。
**[WooCommerce 水印](https://codecanyon.net/item/woocommerce-product-image-watermark-plugin/20846672)** 插件也节省了你的时间，因为你也可以按产品类别水印。您可以为水印制定规则，也可以从后台设置规则的优先级。当你给你珍贵的产品图片加上水印时，没有人会去复制你的图片。
这个插件的价格只有 39 美元。
