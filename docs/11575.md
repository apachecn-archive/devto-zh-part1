# 加布里埃尔·曼里克斯的书评即时 jqGrid

> 原文:[https://dev . to/robinvanderknaap/book review-instant-jq grid-by-Gabriel-manricks-nl4](https://dev.to/robinvanderknaap/bookreview-instant-jqgrid-by-gabriel-manricks-nl4)

JqGrid *是一个支持 ajax 的 jQuery 插件，用于在网格中显示和编辑数据。我在整个项目中广泛使用 jqGrid，并创建了一个* [*Html 助手*](https://github.com/robinvanderknaap/MvcJqGrid) *来简化 ASP.NET MVC 中的实现。因为我对 jqGrid 的体验，Packt 出版社让我评论他们的一本新书，* [*即时 jqGrid 作者 Gabriel Manricks*](http://bit.ly/1an91tS) *。*

这本书主要关注从 jqGrid 开始的人，有经验的用户可能知道书中写的大部分东西，对他们来说没有任何附加价值。

这本书可以分为两部分。第一部分讲授 jqGrid 的基本概念、安装，并引导您完成创建第一个网格的步骤。第二个也是最大的部分叫做“你需要知道的 7 大特性”,深入探讨了 jqGrid 最重要的特性。

第二部分介绍了格式化程序、添加控件、编辑数据、连接到后端服务器以及 jqGrid API 的一些常见方法和事件的描述。我特别喜欢将 jqGrid 连接到后端的例子。后端代码是用 PHP 编写的，但是应该很容易被其他平台的开发者理解。该示例展示了如何从服务器检索数据，以及在服务器上分页、搜索和编辑数据，这些都是 jqGrid 的核心特性。作者以清晰简单的形式详尽地描述了每一步。

我在书中漏掉的是如何处理自定义造型。提到了可以使用 jQuery UI 的角色，但是没有给出任何信息或者例子来进一步定制网格的样式。我还认为作者在描述寻呼机时没有提到虚拟滚动选项是非常糟糕的，我非常喜欢这个选项，而且它很容易实现。

这本书很容易阅读和理解。在阅读完本书和示例之后，开发人员应该对 jqGrid 有了很好的了解，并且能够根据自己的需要定制网格。我记得当我第一次开始使用 jqGrid 时，浏览在线文档和示例的痛苦，对于想开始使用 jqGrid 的人来说，这本书是一个真正的时间节省器。这本书虽然只有 58 页，但并没有涵盖所有的功能或选项，但我认为它已经足够让您开始探索更多的功能，比如子网格、树网格和数据分组，您可以自己使用在线文档和示例。
