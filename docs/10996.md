## [](#transcri%C3%A7%C3%A3o-livre)自由抄本

> “没人对初学者这么说。我希望有人告诉我。我们所有从事创意工作的人，我们进入这一领域是因为我们有很好的品味。但有一个漏洞。在我们做事情的最初几年，它们并不那么好。她们想做好人，她们有潜力，但她们没有。但你的品味让你加入这场游戏的东西仍然是个杀手 but your good taste，the thing that made you into this game is still a killer .你的品味令你失望。几个人从来没有过这个阶段，他们放弃了。几个我知道从事创造性和有趣工作的人经历了很多年。我们知道我们的工作没有你想要的那种特别的东西。我们都经历过。而且，如果你只是刚开始或仍处于这一阶段，你需要知道这是正常的，你需要做的最重要的事情是大量的工作。为自己定个目标，每周你都会完成一个项目。只有经历了大量的工作，你才会缩小这个差距，你的工作会和你的抱负一样出色。我花了比我见过的任何人都多的时间想出了如何做到这一点。要花点时间。这很正常。你只需与你的道路抗争。”

## [](#an%C3%A1lise-e-introspec%C3%A7%C3%A3o)分析与反思

在我写这篇分析的时候，我听到了一个‘t0’【Spotify 的婊子播放列表】，我建议你一起去听，也许她也会激励和鼓舞你。

你有没有觉得你做的每件事都不如你希望的那样好？20 多岁的人做了你从未想过会有能力做的了不起的事情？不管你学了多少，转了几个小时，永远都不足以达到你认为好的目的？然后你开始崩溃，感觉很糟糕，想放弃一切，尝试别的东西？

正如 Ira Glass 说的那样，这是正常的，你有很好的品味，你知道看着好工作，告诉他们他们有多好。

你所不能做的是把自己的工作和别人的工作做比较，如果减少，每个人都有自己的学习时间，提高时间，更重要的是，都经历了和你所经历的相同的阶段。你要做的就是找出并了解你正在经历的这个阶段，因为当它通过时，你将会超越一切，你的工作将达到你如此渴望的水平。

你可能认为我经历过这一阶段，我感觉很棒，我有合法的工作，我写博客，很多人都知道，我做演讲。但你错了，我每天都觉得我还有很多事要做我已经做和正在做的一切都不值得。

但我终于开始认识到这只是一个阶段。我做了什么才能让自己感觉更好？我开始项目和项目。我已经为他们设定了最后期限，所以我会学到一些我一直想要的东西，如果我没有创造这些挑战，也许我永远不会开始。

这些项目可以是任何东西。你一直想写博客吗？**开始！**一直想在 github 上投稿吗？**开始！**一直想发言，把自己的知识传递给社会吗？**开始！**

你可能会觉得一辈子都没有“足够的知识”，没有“必要的经验”，人们会嘲笑你。或者你可以开始展现自我，开始做你一直想做的事，但我害怕。如果他最初的批评不好，**继续**，没人说会很容易，但也没人说不可能。

有一个丹尼尔的儿子名叫**的帖子烧了他的偶像**。我更喜欢从他的帖子里了解到的是，不是让你停止有你镜像的人，而是他们像你一样的人，他们也犯了很多错误，他们经历了与你相同的阶段。不要以为她们在那里是因为她们比你好得多，没有人知道的足够少。他的偶像可能是 JS 上的天才，几乎不知道 CSS，这并不使她不好。我对你说同样的话，你是刚开始呢，还是已经在这一带呆了很长时间了，不要退缩，尽量把你知道的少花在别人身上，你就会发现你知道的少花在别人身上已经太多了。

几天前，我在 Facebook 的时间表上发布了一段非常有趣的视频，展示了 Node.JS 的创建者在他的第一次演示中。

[https://www.youtube.com/embed/ztspvPYybIY](https://www.youtube.com/embed/ztspvPYybIY)

一个胆小怕事的人嘴里塞着字他根本不知道他那一方的计划能否奏效。而今天，我们所知的只是帮助创造了世界上增长最快的开发工具。

也就是说，如果你害羞，就不好意思自我介绍，就不好意思做什么，从不向别人展示什么。别害怕！在 Github 上投稿，写博客，录制视频，炫耀！你可能是十年来最伟大的开发者。
