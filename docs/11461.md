# 在 OSX 上安装 VirtualBox

> 原文：<https://dev.to/adamkdean/install-virtualbox-on-osx-291e>

我们将使用[自制软件](http://brew.sh/)在 OSX 上安装`VirtualBox`。

如果你还没有的话，[在 OSX](http://adamkdean.co.uk/blog/read/138/install-homebrew-on-osx) 上安装家酿。

一旦你安装了它，像这样安装 VirtualBox:

```
brew install caskroom/cash/brew-cask
brew cask install virtualbox 
```
