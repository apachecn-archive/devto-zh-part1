# 为什么我会成为一名软件工程师

> 原文:[https://dev.to/edemkumodzi/why-i-became-a-software-engineer](https://dev.to/edemkumodzi/why-i-became-a-software-engineer)

我听说过许多软件工程师的故事，他们总是知道他们将与计算机一起工作，因为他们在小时候就开始摆弄它们。或者他们在学校有一个电脑俱乐部。或者他们的父母在他们岁的时候买了一台电脑，他们认为编码很有趣。所有这些对于美国和欧洲的软件工程师来说都很常见，但是对于我们非洲的许多人来说呢？直到高中读到一半，我们才看到电脑。而当我爸终于买了一台 PC，在家里摆好之后，它就不是“修修补补的机器”了。这是为了工作！ðŸ˜‚

我记得看到我的一个叔叔安装了一个操作系统，它看起来与我习惯的 Windows 2000 和 Windows XP 完全不同。我问他是什么，他说“是 Linux”。当时他是一名网络管理员，有一本装满命令的大书，他在工作时必须参考这些命令。对我来说这看起来太可怕了，我就像“不，这太疯狂了，让我去玩些电子游戏吧”。

我直到大学才开始做编程。C++对我来说很无聊。我可以做到，但除了课堂作业之外，我没有什么动力去做。我的第一个学期结束了，我爸爸坚持让我参加一些课外活动，所以他给我报了一个编程班，在那里我们学习了 Visual Basic 和. NET。我真的很喜欢 Visual Studio 和构建 windows 窗体的整个拖放方法。那仍然没有真正激励我。

是回到学校的时候，我才恍然大悟。我们不得不去学校外面的银行交学费，然后去财务室要一张正式收据。然后用那张收据登记我们参加的每门课程。在每一步，你都要排队等候 2-3 个小时。实际完成整个注册过程需要 3 到 5 天。我很不耐烦，开始问自己:“没有更好的办法吗？为什么我必须一遍又一遍地提供相同的信息，去多个地方完成我的注册？”。我开始研究 web 开发、数据库、windows 应用程序开发，并告诉自己我可以做这件事。不是为了转卖，只是为了证明一个观点。

所以一些让我沮丧的事情让我有了动力。我就是这样开始认真学习如何编码的。因为我已经熟悉了 VB，所以我想我可以用 VB 建立学生信息系统作为一个 Windows 应用程序。我从设计带有字段的表单开始，做了所有的验证，然后我陷入了这样的困境:“我实际上如何保存它？我需要做些什么才能存储这些信息并在以后进行查询？”

我用 VB 停顿了一下，然后跳到如何使用 Microsoft SQL Server 和 SQL 语言本身。一旦我对此感到满意，我就回到我的应用程序，让它连接到数据库服务器并执行基本的 CRUD 操作。此时，我有了一个管理学生和课程的工作系统。现在我如何让学生在线注册他们的课程？我不能构建另一个 windows 应用程序并期望他们下载它。

这也是我学习使用 ASP.net 进行 web 开发的原因。因为我那时不知道 HTML，所以我从它开始。然后学了 CSS。这很痛苦，因为没有像 bootstrap 这样的 CSS 框架。但是我一直坚持，因为我真的很想建造这个东西。

当我最终建成它时，我真的很高兴。我觉得很有成就感！从那以后，我总是根据我想要实现的目标来学习新的工具、框架和技术。

所以，不，我成为软件工程师并不是因为我喜欢编程，或者是在计算机或类似的环境中长大的。我成为一名软件工程师，因为我关心解决的问题需要我学习如何编码。
