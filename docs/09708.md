# 嗨，我是迈克·埃尔文

> 原文:[https://dev.to/dangercobram/hi-im-mike-erwin](https://dev.to/dangercobram/hi-im-mike-erwin)

35 年来，我已经写了 22 年的代码。

你可以在 GitHub 上找到我的名字[有效位](https://github.com/significant-bit)，在 Twitter 上找到我的名字[危险程序](https://twitter.com/DangerCobraM)。

2015 年搬到了北卡罗来纳州 Apex(罗利附近)。之前在波士顿呆了几年。每年还是会回波士顿一次&去纽约两次参加民主党人的聚会，去看朋友，任何借口都可以！

我在 Blender 上工作，这是一个免费的开源 3D 图形套件。主要是通过外部公司的自由职业者/合同/赞助，因为 Blender 项目本身没有资源来支持全职开发人员。我以前做过 OpenGL 驱动和其他令人讨厌的系统。

我用 C 和 Python 编程，最近在意识到 C++11 把它变成了一种非常好的语言之后，我又回到了 C++上进行个人项目。

我目前正在学习更多关于 Python、Vulkan、Unreal Engine 的知识，并指导其他开发人员。

一个显著的奇怪之处(或超级大国？到目前为止还没有回报)是我对新旧技术同样感兴趣。我会问这样的问题:“为什么这在 Windows 2000 上不起作用？是什么阻止了我们？”或者“我怎样才能在 G3 iMac 上适应 Rage 128 的现代图形技术？”。我不认为一种解决方案应该适用于所有情况。但我确实喜欢了解任何技术的优势和局限性，并揭示“为什么”。
