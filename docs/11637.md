# 发行版选择器:选择您的 Linux 发行版

> 原文：<https://dev.to/rndmh3ro/distro-picker-chooses-your-linux-distribution-58aa>

Pacman 开发者 Allan McRae 发现了一个有趣的小工具，叫做[发行版选择器](https://linuxformat.com/tuxradarchive/content/distro-picker-0)。这个工具根据您喜欢的桌面环境、软件包管理器和其他设置来查找 Linux 发行版。

当我试用它时，它给我提供了我当前的发行版: [Arch Linux](https://www.archlinux.org/) ，有趣的是，它运行得非常好！

我一定要看看其他发行版。
