# 对专业知识的剖析

> 原文:[https://dev.to/ben/the-anatomy-of-an-expertise](https://dev.to/ben/the-anatomy-of-an-expertise)

不久前，我还没怎么考虑过内容交付网络。在那之前的一段时间，我甚至从未听说过这个短语。内容交付网络(简称 CDN)被定义为“部署在多个数据中心的代理服务器的全球分布式网络”，但这个技术定义对我来说没有我对它的概念理解有用，因为它是一种减少网络延迟的技术。我开始了解 cdn，因为我对 web 性能感到好奇，并确定它们是我以前认为理所当然的关键部分。

令我着迷的是，随着我了解得越来越多，越来越感兴趣，我发现自己对这种技术的障碍做了额外的研究，学习高频交易者如何将这些概念发挥到极致，阅读第一部电报等网络历史，并拼凑出这一领域的专业知识。这个词让我感到紧张，因为我仍然处于这个过程的早期，太早了，不知道我是否会保持动力去更深入，或者最终失去兴趣并对其他概念着迷。我还不是专家，差得远，但我正在逐渐成为一名专家。我正在利用我对这一主题的兴趣，更深入地研究其他技术概念，如点对点网络，并以一种没有大量专业知识积累的方式思考这项技术。

我不知道我的未来会怎样，但是随着我了解的越来越多，我对网络的可能性和可能的创新有了自己的看法。在我目前的阶段，我不知道如何区分清晰和天真，我需要在这些固化之前做更多的研究。当我花一点时间欣赏我现在的智慧，并与我的过去和潜在的未来进行对比时，它让我惊叹不已。我的过去对我在这里描述的一切完全不感兴趣，然而现在它却让我着迷。我知道整个情况的连续性，这让我理解了这一切，但从根本上说，新的知识如何打破精神障碍是不可思议的。

一个过去的自己也会直接跳到这个等式的“制造”部分。然而，我已经成熟到了这样一个程度，我宁愿用新的信息来解决我当前的问题，也不介意让未来以一种更温和的方式展现。我已经有很多事情要做，我不能一头扎进网络技术的汪洋大海。但我可以继续学习，继续着迷，以允许我在未来再次像这样反思的方式增长我的知识。
