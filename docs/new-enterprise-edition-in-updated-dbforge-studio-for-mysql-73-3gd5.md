# 更新的 dbForge Studio for MySQL 7.3 中的新企业版

> 原文：<https://dev.to/devartsoftware/new-enterprise-edition-in-updated-dbforge-studio-for-mysql-73-3gd5>

数据库工具和数据连接解决方案的捷克软件供应商 Devart 发布了 dbForge Studio for MySQL v7.3，这是 MySQL 和 MariaDB GUI 工具的新版本。除了新的企业版，dbForge Studio for MySQL 还更新了数据导入/导出、数据比较、数据库备份和数据导出、SQL 文档等改进功能，并引入了新的连接可能性、试用产品激活等等。

该工具的新企业版为用户提供了以下独有功能:

*   数据生成器允许生成测试数据，并立即用它填充数据库。
*   数据库文档管理器是一个生成 HTML、PDF 和 Markdown 文件格式的数据库文档的工具。
*   使用复制数据库，用户可以通过生成和运行脚本来复制一组数据库。

    新的连接可能性

*   支持 MariaDB 10.1，MariaDB 10.2，MariaDB Galera Cluster 10.0 系列，谷歌云平台，阿里云，Azure MySQL。

*   测试连接时显示服务器版本

    支持新的表格引擎

支持 TokuDB、NDBCLUSTER、RocksDB

dbForge Studio for MySQL 是一个用于数据库管理、管理和开发的通用 MySQL 和 MariaDB 客户端。在这个智能 MySQL 客户端的帮助下，处理数据和代码变得更加容易和方便。

有关 dbForge Studio for MySQL v.7.3 的更多信息，请访问[https://www.devart.com/dbforge/mysql/studio/](https://www.devart.com/dbforge/mysql/studio/)。
