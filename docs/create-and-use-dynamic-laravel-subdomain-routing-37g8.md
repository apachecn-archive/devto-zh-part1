# 创建和使用动态 Laravel 子域路由

> 原文：<https://dev.to/macnux/create-and-use-dynamic-laravel-subdomain-routing-37g8>

许多网站为用户的个人资料或页面提供了自定义的子域，因此用户可以访问 username.website.com 的个人资料页面，而不是在 website.com/users/50,访问用户的个人资料，这要好得多。在这篇文章中，我们将看到如何轻松地制作动态 Laravel 子域路由。
