其他功能
■保存/打开/清除
“保存”暂时保存正在编辑的数据。“打开”重新显示保存的数据。“清除”删除保存的数据。

■下载正在下载的
文件下载数据

■代码
编辑器颜色选择(Emmet)

■正常
选择引导模板颜色

■实时视图
预览正在编辑的代码(PC、平板电脑、手机、CodeOnry)

■错误指示
显示 JavaScript 错误
