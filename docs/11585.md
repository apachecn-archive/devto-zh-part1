# sudo 命令的正式发音

> 原文:[https://dev.to/shiena/sudo-436d](https://dev.to/shiena/sudo-436d)

关于成为话题的指令的发音，`sudo`好像有官方的发音。
Sudo Main Page 的[FAQ and Troubleshooting Tips](http://www.sudo.ws/sudo/troubleshooting.html) 中有以下常见问题。

> sudo 怎么发音？
> A)正式读音为 soo-doo(为苏“do”)。但是，一个
> 备选读音，即“伪”的谐音，也是常见的。

因为这里是 soo-doo，所以用“怎么样”来决定吧！
