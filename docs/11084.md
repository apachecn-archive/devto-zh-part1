# t 代表过渡

> 原文:[https://dev.to/captainsafia/t-is-for-transitions-4na1](https://dev.to/captainsafia/t-is-for-transitions-4na1)

在过去的一年里，我作为联合组织者与芝加哥皮拉迪斯和芝加哥 T2chick tech 一起积极参与了指导和志愿者活动。参与这些组织的创立和成长是令人兴奋的，但像所有事情一样，是时候继续前进了。

在过去的一年里，我一直致力于一家专业服务初创公司( [dsfa](http://dsfa.io) )，旨在为小型企业和社区组织提供数据科学服务。这是我真正关心并想要推进的事业。但事实是，在过去的一年里，我的业务没有增长，因为我没有花时间去做。创业是一条双行道，你投入多少就能得到多少回报，而我投入的还不够。

因此，我将辞去在 PyLadies Chicago 和 ChickTech Chicago 的领导职务，将更多精力投入到我的创业中。带着更新的精力和更专注的激情，我希望完成我的目标，建立一个有意义的企业。

但是我也不想离开芝加哥技术社区。这是一个很棒的网络，我很自豪能和这么多很棒的人在一起。这就是我组织 PyData 芝加哥社区的原因。这是我关心并希望发展的另一项事业。如你所知，我对让大众更容易接触到数据科学感到非常兴奋。

我期待着未来会给我的企业和芝加哥数据社区带来什么。好事正在酝酿中！

如果你有兴趣与我谈论 dsfa 或 PyData Chicago，请随时通过 Twitter 或 T2 的电子邮件联系我。
