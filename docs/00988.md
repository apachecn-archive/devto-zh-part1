# 工作机会:不要迷失方向

> 原文:[https://dev . to/einen lum/job-offers-don ' t-get-lost-along-way-ab6](https://dev.to/einenlum/job-offers-dont-get-lost-along-the-way-ab6)

亲爱的潜在雇主:

我知道贵公司正在处理紧急问题。我知道目前市场很不景气。我知道优秀的开发人员现在有着高而多样的期望:跟踪这些期望一定不容易。

似乎对你们中的一些人来说，免费咖啡或免费糖果是必须的。办公室里可用的 Nerf 枪可能是你工作邀请中的一条线，它会让你的公司从其他公司中脱颖而出。

也许你是对的。也许我完全低估了这种特性。但是有一件事对我来说很重要，而且你也经常忘记，那就是你想要我处理的技术栈。

我没开玩笑。有些报价甚至没有提到一种编程语言。你可能会认为这是一个例外，我只选择了一些不可靠的小公司。你就错了。

这是一份在 Reddit 工作的[工作邀请](https://boards.greenhouse.io/reddit/jobs/655395?t=onmjps1#.WlRYj_ZG2Hs)。

[![A job offer to work for Reddit](../Images/1fe194ae32ec40002c8ee0e42f1b2c9d.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--sm06zRe8--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/aagddeldcoeem2ssofas.png)

似乎他们想要的就是*干净、可维护、经过良好测试的代码*。你应该知道这意味着什么，你，小小的*后端*开发者。包括 Node 吗？Python？露比？生锈？PHP？去吗？没人会知道。我希望你正在努力熟悉*整个网络堆栈*。
考虑一下我们所知道的关于该报价的所有信息。

*   宠物护理津贴
*   每月手机津贴
*   提供餐饮和小吃

知道这个太好了。但我感觉作者们在这一过程中迷失了方向。请下次不要忘记提及更多关于我将为你做的工作的细节。即使在你的工作邀请的最上面也要写下来。这会节省任何人的时间。
