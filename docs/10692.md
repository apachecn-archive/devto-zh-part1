# [](#the-future-of-machine-learning)机器学习的未来

在过去的几年里，机器学习得到了很多关注。在我们大多数人不知道的情况下，它已经接管了我们的生活。网上商店、社交媒体和我们的手机都以某种方式利用它。听起来很吓人，但是利大于弊吗？

我个人认为确实利大于弊。这是因为机器学习和大数据有助于系统了解你是什么样的人，以及如何尽可能地帮助你。它会更好地知道如何帮助你，适应你，预测你可能感兴趣的东西。允许系统根据你在系统中的行为收集数据，需要对系统有很大的信心。但是，如果做得正确，这是一个非常有价值的“个人助理”。机器学习的一个例子来自沃纳·威格尔(亚马逊首席技术官)在下一届网络大会上的演讲。他提到，像基于你的搜索历史提供建议的电子邮件这样的小事已经收集了大量数据。在亚马逊，他们分析哪些电子邮件和哪些产品被打开、点击或删除。通过这种方式，系统可以了解哪些产品你可能比其他产品更关心。

### [](#will-it-only-be-used-for-advertising)会不会只用于广告？

广告无疑是使用机器学习的一个重要组成部分，但如果你想一想:如果你对某个特定的产品或产品范围真的感兴趣，并且一个系统帮助你找到最可能的解决方案来满足你的需求，这真的是广告吗？的确如此，但这不仅仅是瞎猜，希望有人抓住机会做出回应。这对买卖双方来说都是双赢的局面。卖方有更有信心的机会进行销售，买方找到他或她需要的最好的产品。

这就是为什么我认为机器学习将变得比现在大得多。它不仅用于广告，也用于像网飞这样的服务。建议在一年中的哪个时间或一天中的某个时间观看哪个电影或连续剧。它甚至可以为一种情绪推荐合适的电影。该系统将学习帮助你选择最适合你和你的情况的最佳系列或电影。

### [](#it-will-need-to-be-secure)它需要安全

所有这些数据也带来了很多风险。保持数据安全对于使用数据的系统的完整性非常重要。外界的任何人都可以在没有见过这个人的情况下了解这个人的一切。这是一个可怕的想法。这不仅会导致危险(跟踪)的情况，如果发现一个人对非传统电影、产品或服务有私人兴趣，还会造成私人和职业伤害。这可能会导致个人、团体、企业和社区形象受损。

当机器学习和大数据以正确的方式使用时，它们非常有用。它们可以让各种各样的人生活得更轻松，但也可能是一种威胁。系统将能够给出非常个性化的意见、建议和总体帮助。安全性需要随时保持更新，因为泄露的数据会在许多不同的层面上造成伤害。
