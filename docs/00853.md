# 用终端显示加密货币的行情

> 原文：<https://dev.to/sot528/-7ao>

为了密码货币瘾君子的工程师。
密码货币的行情用[Coinmon](https://github.com/bichenkk/coinmon) 在终端确认吧。

```
 npm install -g coinmon
    coinmon 
```
