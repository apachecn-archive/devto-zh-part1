# 回顾 2017 年

> 原文：<https://dev.to/funkysi1701/looking-back-at-2017-21ee>

随着 2017 年开始接近尾声，让我们来看看过去一年的一些亮点。

[![Edward](img/ae12146c2c4d02bc3ec256c987543bd7.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--jBFeQQH8--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i0.wp.com/www.funkysi1701.com/wp-content/uploads/2015/07/DSC_0376-e1510563039193-225x300.jpg%3Fresize%3D225%252C300%26ssl%3D1)

①爱德华

11 月，我又当了父亲，庆祝爱德华来到这个世界。在出生前，我担心我的第一个儿子詹姆斯会不会适应新来的孩子，但我很高兴地说，我不需要担心，他基本上忽略了新来的孩子。

2)《星际迷航》

9 月，最新的《星际迷航》电视连续剧《发现》抵达网飞。到目前为止，我对这部剧非常积极，我仍然很积极，但是新剧有一些让我烦恼的事情，但是我渴望每周都能看到下一集，所以他们肯定做了一些正确的事情。

3)办公室搬迁

五月份，我帮助我的雇主搬到了全新的办公室。这是一项艰巨的工作，但一切都按计划进行，我们的新家很棒。

4)侧面项目 [![Google Play](img/df089f3525db7408dee34e10b70f1b63.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--hjI9YMyw--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i1.wp.com/www.funkysi1701.com/wp-content/uploads/2017/08/googleplay.jpg%3Fresize%3D300%252C300%26ssl%3D1)

我在二月份写了一篇关于有一个副业项目的博客，但是直到今年晚些时候，这才变成我现在正在做的事情。它开始是一种理解界面的方式，然后导致学习 Xamarin 和创建一个 android 应用程序来帮助推广这个博客。Troy Hunt 扩展了他的 HIBP API，以包括数据泄露中的密码，然后我开始研究 [pwned 密码](https://dev.to/funkysi1701/pwned-pass--available-from-the-play-store-1o8a-temp-slug-8432011)，这是一个简单的 android 前端。

5)学习 R

在二月份，我还尝试了一种新的编程语言。它以一个关于汇率转换的问题开始，以一个执行一些 R 代码的 SQL 存储过程结束，我还想了解更多，但它是一个很好的例子，说明了可以用 R 构建的各种东西。

6)以编程方式进行 DNS

我最成功的博客之一是关于如何在你的应用程序中创建 [dns](https://dev.to/funkysi1701/creating-dns-records-programmatically-1o3c-temp-slug-3062796) 记录，例如在 url 中有你的 name.domain 的站点。这是一个很好的机会来看看 dns 是如何工作的，并了解更多关于 azure 可以做些什么来管理它。我想在此基础上扩展并构建一些东西。

这就是 2017 年，关于 2018 年可能会发生什么的一些想法，请关注我的下一篇博文。
