# 我的代码年

> 原文：<https://dev.to/funkysi1701/my-year-of-code-2iei>

无法回避的事实是，我们现在是在 2015 年。我的新年决心是做更多的编码工作。

圣诞节期间，我想我会去 http://www.codecademy.com 试一试。我完成了 javascript 课程，正在想下一步该做什么时，我看到了一条关于 http://www.freecodecamp.com T2 的推文

从那以后，我一直沉迷于做他们做的运动，几乎每天我都有一点进步。

但是这个网站不仅仅是教你如何编码，它有一个活跃的论坛，在那里你可以提出问题，了解更多关于如何成为一名开发人员。

在我开始这些练习之前，我对 javascript 或 jquery 知之甚少，但是我开始学习一些命令，希望我能在某个时候投入使用。无论我最终做什么开发，我确信在某个时候我会需要 javascript。

我学到的秘诀之一是你需要每天练习。你曾经用抽认卡来帮助通过考试吗？现在有一个电子版本叫做 Anki。只需在你的卡片上添加一对问题/答案，每天复习它们，那些你觉得难的问题你复习的次数要比你学过的多。

希望我能保持下去，在今年晚些时候，我会写很多有用的代码。
