# 如何升级到 Windows 10

> 原文:[https://dev.to/funkysi1701/how-to-upgrade-to-windows-10-eo6](https://dev.to/funkysi1701/how-to-upgrade-to-windows-10-eo6)

微软将于 7 月 29 日发布名为 Windows 10 的新操作系统。

如果你对 Windows 10 有任何疑问，请观看斯科特·汉斯曼的视频。他解释了如何从 Windows 7 和 8 升级到 Windows 10，以及升级后会发生什么。

我给每个人的建议是，在升级之前备份你的设备，这是在你安装任何软件或更新之前的好习惯。

微软将免费提供 Windows 10 一整年，所以不要惊慌。慢慢来，备份你所有的重要文件，当你准备好安装 Windows 10。7 月 29 日之后，如果你的电脑没有最新的操作系统，任何东西都不会停止工作。
