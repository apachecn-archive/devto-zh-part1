理想情况下，在每次微迭代之后(在工作的每个阶段)。

为什么这种方法更好？

1.  别担心:你已经承诺了。

2.  更好的时间旅行可能性。

3.  在一天结束的时候，您可以将所有的微提交压缩成一个大的、有趣的提交，其中包括为实现一个函数所做的每一个更改。

这没什么难的:这只是 git rebase。
