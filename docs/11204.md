# 面向 Web 开发人员的 Github Atom 插件

> 原文:[https://dev . to/shanalikhan/github-atom-plugins-for-web-developers-482 I](https://dev.to/shanalikhan/github-atom-plugins-for-web-developers-482i)

与其他文本编辑器相比，Github 的 Atom 是一个令人惊叹的、强大的现代文本编辑器，支持这个编辑器的一个主要原因是 Github 已经得到了它的支持！如果你是一个网页开发人员，并期待着在新的编辑器中得到你的手脏。Atom 非常适合学习，可以让您的工作效率更上一层楼，但前提是您知道自己需要哪些功能来完成工作。Atom 社区开发了数以千计的插件，这些插件确实有助于提高您的工作效率。这些插件应该在每个开发人员的 Atom 口袋里。

*   原子美化是一个方便的工具，在编码时只需轻轻一点就能让你的代码变得漂亮。它还美化了几乎每种语言的缩小文件。
*   如果你是一个打字爱好者，并且希望在 Visual Studio 之外的其他编辑器上工作。这个插件实现了所有需要的东西，比如实时的类型脚本到 javascript 的转换。
*   emmet-atom Auto complete 由这个插件扩展而来，为你提供更多每个 web 开发者都需要的核心特性。
*   **[高亮-选中](https://github.com/richrace/highlight-selected)** 只需双击一个单词，该插件将高亮显示整个文件中所有出现的单词。非常有帮助
*   **[atom-jshint](https://github.com/sindresorhus/atom-jshint)** 期待在你打字的时候验证你的 javascript！那对你来说是肯定的。强烈推荐。
*   **[项目经理](https://atom.io/packages/project-manager)** 非常有用，可以让你的文件夹成为项目，像 Eclipse 等一样只需点击一下就可以打开- **[小地图](https://atom.io/packages/minimap)** 非常有用，可以在编辑器的右栏或左栏显示你的代码预览。最佳工作方式与[高亮选择](https://atom.io/packages/minimap-highlight-selected)
*   **[Atom-sync-settings](https://github.com/Hackafe/atom-sync-settings)**为您的办公室和家庭使用 Github Atom！期待在您的所有设备上同步您的包。这是最好的！
*   **[预览](https://atom.io/packages/preview)** 在 Jade 等模板引擎上工作，想要查看 HTML 或任何其他语言的实时预览。它来了！

这些插件应该安装在您的 Atom 中。相信我，这真的会提高你的工作效率。如果你知道其他最好的插件，请在下面评论。
