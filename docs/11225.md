# 志愿加入技术团队

> 原文:[https://dev . to/funky si1701/volunteering-for-a-technical-team-le0](https://dev.to/funkysi1701/volunteering-for-a-technical-team-le0)

在过去的几年里，我在当地教堂的技术团队中做志愿者。

为了教会的良好运作，它需要各种角色的志愿者，人们欢迎游客，人们祈祷，人们演奏音乐，人们照看孩子和许多其他角色。

我天生的能力更多地取决于技术，而不是人，所以当我决定更多地参与时，我自愿加入了技术团队。技术团队分为两个主要部分，公共广播团队负责确保每个人都能听到所有声音，放映团队负责放映所有内容。我不知道什么听起来好，什么不好，所以我加入并参与了这个项目团队。

教堂有三台投影仪，一台大的，两台稍小的用于侧通道，还有一台位于教堂后面的电脑，放映员用它来控制放映的内容。有许多东西被投影，歌词，PowerPoint 演示文稿和偶尔的视频，为了使事情变得简单，一个软件被用来控制所有这些。

easy feast 包含一个巨大的歌词数据库，所以你可以快速选择正在唱的歌曲，并将歌词显示在屏幕上。它还允许添加 PowerPoint 文件、图像或视频文件，甚至网页，并且它们可以被投影。easy feast 允许不同的事情按顺序排队，这个顺序可以保存为一个时间表。

放映员的工作是按照服务的顺序，在正确的时间把东西放映到屏幕上。简单！不，不是真的。崇拜领袖将跳过歌曲，以不同的顺序播放歌曲，添加新歌，视频将无法播放，简易崇拜将冻结或崩溃，或者牧师想要的礼拜仪式不在简易崇拜数据库中。

作为一名放映员，你需要时刻保持警惕，但是这些年来，我已经成为了一名优秀的放映员，并且非常享受这样做的时光。你可以看到我与音乐和崇拜的头在右边笑。

最近，我在投影团队担任了一个更重要的角色。我一直在协调放映队志愿者的值班表。这给了我新的挑战，但与技术经理和最近的音乐和崇拜主管密切合作并更好地了解整个团队是很棒的。这个团队最大的问题是志愿者不足，当我加入这个团队时就有这个问题，今天也有这个问题。如果你正在读这篇文章，考虑一下这是不是你可以参与的事情？这个团队有一些非常努力工作的人，他们每周都做得非常出色，只是希望我们有更多的人来减轻每个人的工作负担。

由于我妻子将在几天后生下我的第一个孩子，我将离开放映队一段时间。我度过了一段美好的时光，我和一些很棒的人一起工作过，在很大程度上，我会怀念志愿工作。当然也有不太好的事情，志愿者离开了团队，教堂发生了变化，很多清晨我宁愿躺在床上，但我很高兴我参与了进来。
