万圣节快乐！如果你错过了其中的任何一个，这里有一些过去一周最值得注意的链接。

Eve 被称为“为人类设计的编程”,它是一种编程语言和 IDE，当然是华而不实的。是否如此还有待观察。夏娃背后的想法当然是两极化的。Eve 旨在使开发体验更加精简和直观，但很容易被怀疑。

[Git 从里到外](https://codewords.recurse.com/issues/two/git-from-the-inside-out)这是一篇关于 Git 基础的全面文章，有助于建立心智模型。

苹果发布了新的 MacBook Pro，这是本周最受欢迎的开发机器之一。来自我们社区的评论最多是褒贬不一。当然有[的担忧](https://blog.devteam.space/new-macbook-pro-is-not-a-laptop-for-developers-anymore-d0d4b1b8b7de#.tpt1vssco)苹果正在逐渐远离软件开发的可用机器。

[Next.js](https://zeit.co/blog/next) 一个小型的 JavaScript 框架，用于服务器渲染的通用 JavaScript webapps，构建在 React、Webpack 和 Babel 之上。

[快乐是一个无聊的堆栈](http://www.expatsoftware.com/Articles/happiness-is-a-boring-stack.html)“我无法告诉你在一个无聊的堆栈上生产软件有多好。它给了你做其他事情的自由。”

[UTF-8 无处不在宣言](http://utf8everywhere.org/)一份促进使用和支持 UTF-8 编码的文件，旨在说服我们，它应该是在内存或磁盘上存储文本字符串、用于通信和所有其他用途的默认编码选择。

[应用 Linus Torvalds 的“好品味”编码需求](https://medium.com/@bartobri/applying-the-linus-tarvolds-good-taste-coding-requirement-99749f37684a#.7rlswmjhs)理解 Linus Torvalds 所说的有“好品味”的开发人员是什么意思这些开发人员如何概念化任务并定义正确的界限？
