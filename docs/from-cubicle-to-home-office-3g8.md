# 从隔间到家庭办公室

> 原文：<https://dev.to/_patrickgod/from-cubicle-to-home-office-3g8>

嘈杂的办公室不时让我心烦。耳机甚至紧闭的门都无济于事。其他部门的同事会走进房间，站在你的办公桌前，开始交谈。他们不懂耳机的意义。即使你在用 Skype 通话，他们似乎也不在乎。

他们需要你的帮助，这是一种恭维(如果他们真的需要你的帮助，有时他们只是想闲聊或告诉你，他们已经给你发了一封电子邮件，但那是另一个话题)，但尽管如此，当你几乎总是被打断时，真的很难进入状态，即使是来自应该更了解这一点的开发同事。

有什么解决办法？远程工作。这是有可能的，即使你通常在小隔间里工作。让我告诉你怎么做。事实上，只有两件事你必须坚持做。

在大多数公司，特殊情况下可以选择在家工作。也许你希望在家里送货，你有一个医生的预约或其他任何事情，在这种情况下，你可以在家工作，例如通过 VPN 登录到你公司的网络。

但是，如果你只是需要一个平静而熟悉的环境，让你感到非常舒适，也就是你的家，那该怎么办？

嗯，你要做的第一件事是:问你的主管。

你在家工作的原因是:你有一堆任务，你需要一些独处的时间，这样你可以更好地集中精力，妥善完成任务。而这句话已经包含了很重要的一点:**完成很多东西**。

我猜你的目标是能够多次在家工作。也许一周一次或两次，甚至更频繁。

现在想象你是你公司的老板。我知道我们生活在一个数字化的世界，但可悲的是，许多高管仍然想控制他们的员工，并认为你只有在办公室时才工作。

我知道这毫无意义。尤其是在软件开发领域。你必须签入你的代码，完成任务，你所做的几乎每一步都可以在 git(或者任何其他版本化工具)的历史中找到。如果你决定一整天都在网上冲浪，你也可以在办公室进行。你不必为此呆在家里。

尽管如此，人们还是希望在办公室见到你，因为他们害怕你不在家工作。要说服他们，一定要提高你在家的表现，比在办公室完成更多的工作。当你这样做的时候(他们可以看到，因为你签入了你的代码)，你将获得信任，如果你有时想在家工作，你的老板最终不会有任何顾虑。

在家工作的另一个要点是:**随时可用**。

为了获得更多的信任，并表明你真的在完成你的任务，你应该在任何时候都可以放松、打 Skype 电话或做任何其他事情。这并不意味着每当有人给你发信息或打电话时，你就不得不中断工作。简单地回复一条短信，说你必须先写这一行代码，你会在十分钟左右给他们打电话。

你可以随时打电话和聊天，这也表明你在家的时候工作有多出色。你甚至可以在通过 Skype 或 Google Hangout 进行日常脱口秀时打开网络摄像头。

已经这样了。完成大量的任务，并且有空。当谈到家庭办公室时，这些是对我来说最重要的因素。

我没必要告诉你所有的好处。你不需要通勤，这样可以减少压力，在一天结束时你会感觉更好。这里很安静，没有打扰，比如同事走进房间，告诉你他们给你发了一封邮件。你可能已经有了更多的梦想。本质上，你会更快乐，你会喜欢你的工作。
