# 宣布 Clarifai 的新的和改进的 v2 API 视频识别功能

> 原文:[https://dev . to/clari fai _ team/announcing-clari fais-new-and-improved-v2-API-video-recognition-feature](https://dev.to/clarifai_team/announcing-clarifais-new-and-improved-v2-api-video-recognition-feature)

*我们很高兴地宣布，Clarifai 的 v1 API 中的视频识别功能现已从测试版升级到我们的 v2 API，正式上市！现在，所有的 [Clarifai](https://developer.clarifai.com/signup/) 用户将能够通过我们 API 的最新和最棒的功能和改进来了解他们的视频中发生了什么。*

[给我看文档](https://developer.clarifai.com/guide/predict#videos)

作为首批推出人工智能视频识别解决方案的公司之一，我们很高兴地宣布，我们的 v2 API 现在支持 v1 的所有视频识别功能，甚至更多！在测试阶段，我们收到了许多关于该视频产品的积极反馈，在对该产品进行了一些增强后，我们将向所有开发人员和企业提供该产品，以用于调节和分析视频内容，从而改善工作流程和用户体验。

[![image1](img/f23740984890d1bd6b9f5f32c7dd23b8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--qLYyuqSV--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://blog.clarifai.com/wp-content/uploads/2017/07/image1-1.png)

用户将能够对以下[公共模型](https://developer.clarifai.com/models)进行预测操作调用:

*   一般
*   衣服
*   食物
*   适度
*   NSFW
*   旅行
*   婚礼

与之前的 v1 API 视频产品相比，我们提供了更多更精确的模型，并且我们还扩展了基础设施容量，以更快地响应您的请求。我们决定将视频功能转移到 v2 API 中，因为我们为视频计划了许多其他令人兴奋的功能，您会喜欢的(敬请关注！).

我们的 v2 API 中的视频以每秒 1 帧的速度分析和处理输入，这意味着您将收到视频每秒的预测结果列表。类似于我们对图像识别的定价，1 帧视频将等于 1 次操作。这意味着一个 2 分钟的视频将是 120 帧，你将为 120 次操作付费。

我们的[文档指南](https://developer.clarifai.com/guide/predict#videos)详细介绍了如何将视频产品集成到您的应用中，您可以在我们的[网络演示中试用新的 API](https://demo.clarifai.com/)！
