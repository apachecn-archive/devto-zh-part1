# 如果你在纽约的话，明天和我一起研究一些长生不老药

> 原文:[https://dev . to/Ben/hack-on-some-elixir-on-some-my-tomorrow-if-you-in-new York](https://dev.to/ben/hack-on-some-elixir-with-me-tomorrow-if-youre-in-new-york)

这可能有点仓促，但我明天将在 Thoughtworks 参加“改变现状”工作会议，如果你有空，我推荐你参加。 [Make A Diff](https://twitter.com/makeadiff_nyc) 是我在上个月帮助创立的一个组织，目的是帮助我们这些技术人员一起完成政治和公民参与的任务。在我们的国家选出了一个橙色的煽动者之后，我们开始了这个项目，我们中的一些人感到特别有动力去使用我们的神奇天赋去更好地入侵主机。这不是一个政治党派聚会，但是，它基本上是关于使用我们的技术技能做出积极的改变。

明天我们将为一些 rad 开源项目做出贡献。详细信息在上面的链接中。我特别兴奋能够接触到 Elixir，这是一种非常有趣的函数式编程语言，我可以看到自己非常喜欢它。

有三个非常有趣的项目，包括一个假新闻检测器，但我最兴奋的是为代码团队开发 [Elixir/Phoenix API 的想法，因为它已经在我的学习清单上有一段时间了。Elixir 看起来是一种非常友好和强大的语言，Phoenix 看起来是一个很好的组合框架，吸引了我对 Ruby on Rails 的喜爱。](https://github.com/code-corps/code-corps-api)

明天应该是在协作、GitHub 和开源方面获得一些好经验的好机会。欢迎你一整天或一天中的一部分时间过来。欢迎所有经验水平。如果你确实打算来，请回复。也欢迎你来帮忙，即使你没有技术经验，也有很多方法可以帮忙。但如果你对代码感兴趣，我真的认为这些项目从技术角度来看有很多令人兴奋的特性，这将是非常有趣的一天。我希望在那里见到你。

如果你有朋友想来，就告诉他们。
