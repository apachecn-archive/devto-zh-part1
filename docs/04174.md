# 空虚地死去

> 原文:[https://dev.to/tedhagos/die-empty-4hif](https://dev.to/tedhagos/die-empty-4hif)

我从托德·亨利的书《空虚地死去》中得到的启示；

有三种工作；*映射*、*啮合*和*制作*。

1.  **规划** -对目标进行优先排序、规划和设定
2.  **啮合** -磨尖锯。这是长期成功的重要决定因素。我很难证明这一点，因为它与结果没有直接联系
3.  **做**——做实际上划掉你待办事项列表中项目的工作。这就是“做工作”。你有一个工作产品。生产有价值的东西

你需要有目的地从事这三种工作。为了成功，你需要做到这三点。

还有，永远记住；

*   你的日子屈指可数。总有一天它们会耗尽
*   不是为了习惯安逸。偶尔这样很好，但不要让它成为一种生活方式
*   今天播种，以后收获
*   你的作品应该反映出对你来说什么是重要的
*   平庸不是突然发生的，是日积月累发展出来的；你可能太舒服了，太自满了。你认为没有什么会出错，你可以永远停留在“那一刻”。这可能是享受当前成功的结果，例如一份稳定的工作(这听起来像是矛盾的说法，没有工作是稳定的)。从这个角度来看，没有人天生平庸，每个人都梦想并计划摇滚明星级别的成功，然后生活发生了
*   明智地定义你的战斗，并围绕赢得战斗来构建你的生活
