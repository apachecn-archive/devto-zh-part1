# 如何穿着上班

> 原文:[https://dev.to/_patrickgod/how-to-dress-for-work-5616](https://dev.to/_patrickgod/how-to-dress-for-work-5616)

我知道这是一个非常敏感的话题。在我 30 岁之前，我不想听到任何相关的事情。但是相信我，你在办公室的穿着的确很重要。

在我的职业生涯中，我见过很多不同的事情。在游戏公司，人们可能会赤脚在办公室里走来走去。我在银行的时候，我的主管也不穿鞋。幸运的是，他穿了袜子。IT 部门的一名“普通”员工每天都穿西装，尽管他并不需要这样做。最终，他被选入工会，每个人都尊敬他。

作为一名软件开发人员，绝对没有必要每天都穿西装。John Sonmez 建议穿比你现在职位高两级的衣服。在我看来，这有点太多了，尤其是当你在一家小型初创公司，两级以上可能是公司的负责人，他可能确实每天都穿西装。

尽管如此，有两个主要原因让你应该关心你早上的穿着。

第一:你的同事。你想看他们都光着脚跑来跑去吗？你想看凉鞋吗？至少把你的运动鞋配一条牛仔裤(或者短裤，如果太热的话)和一件 t 恤。这就引出了另一个话题，我想简单谈一下:卫生。早上或者至少前一天晚上洗澡，做头发，使用除臭剂。令人愉快的服装与良好的卫生习惯密切相关。在炎热的夏季光脚穿凉鞋会导致没人想在办公室里闻到的气味。相信我，**你的同事可能永远不会谈论它，但他们会注意到**。

一个市场部的家伙(我知道，他们是邪恶的，但仍然…)时不时地告诉我这些，虽然这不是很好的行为，但不幸的是，我不得不承认他是对的。这意味着他不是唯一注意到这一点的人。拜托各位，让我们一劳永逸地消除对又脏又臭的开发商的刻板印象。

第二:尊重。当你穿衬衫或打扮得体时，人们会对你另眼相看。我知道你内心的叛逆不喜欢这个事实，但这是绝对的事实。我也喜欢我的塞尔达 t 恤，当然，当同事第一次认出它时，这很棒，但对 t 恤的兴奋只持续了几秒钟。反过来，对一个穿得很漂亮并且知道如何编写软件的人的尊重会持续更长时间，这可能会带来新的机会——如果你想要新的机会的话。

但是如果你不在乎这些，那就不要做。做真实的自己，用余生继续编码，没有错。不要误解我。我也想一天到晚编码。这是我工作中最喜欢的部分。但是我也喜欢被看作是一个专业的软件开发人员，知道如何穿着，并且有一些很好的社交技巧。你可以成为规则的例外，这感觉很好。此外，你正在习惯这种衣服，它在你生活中的其他场合也很方便。
