# 我会想念 Windows Phone 的

> 原文：<https://dev.to/stipegrbic/i-ll-miss-windows-phone-58pn>

时间过得很快，好像昨天我拿到了我的第一部 Windows phone。在厌倦了 iOS 和 Android 之后，拥有一个大胆的新平台是如此令人兴奋。四年多后，Windows Phone 受到冷落，自生自灭。

### 第三平台

Windows Phone 刚推出的时候我最喜欢的就是它的与众不同。它有很多其他更好的平台没有的东西。我很欣赏整个平台是为智能手机而设计的。大的动态磁贴和小的静态图标，大字体而不是小字体，滑动动作而不是按钮，浏览器地址栏在底部而不是顶部。还有像 hub 这样的概念，它真的可以帮助你比在其他平台上更快地完成想要的动作。你可以查看 windows phone 视频，看看这是怎么回事。现在是微软尝试将 Windows Phone 设备交到人们手中的时候了。

### 如何(不)增长市场份额

许多人要么立即喜欢上了所有这些不同的东西，要么他们不理解，也不想费心去学习，这就是我认为(修复)问题开始的时候。微软希望保持这个平台的现状，与众不同，并让用户学习它。当时，诺基亚在创新和推广 Lumia 系列手机方面做得非常好。但事实证明，人们不愿意学习新平台，他们中的大多数人都买了 Windows Phone，因为它价格实惠，而且在当时比廉价的 Android 手机性能更好。虽然微软确实在这个平台上工作，但它并没有改变这个平台的外观和感觉。用户看到的只是 WP7 没有升级，然后在 8.1 中引入 Cortana 和其他 PC 共享功能。在此期间，廉价的 Android 手机变得更好，不关心应用程序的用户没有理由选择 Windows Phone，市场份额甚至在起飞前就停滞不前了。

### Windows 10

发生了很多影响 Windows Phone 命运的事情，诺基亚收购，微软 CEO 变更，Windows 10。在 Windows Phone 起飞的关键时期，所有这些都是相当大的负担。微软厌倦了这种追赶游戏，开始更多地关注个人电脑，这是它最有影响力的领域。他们现在想为所有有兴趣运行它的设备开发一个操作系统。手机现在变得不那么重要了，他们不想销售硬件，而是想全部留给原始设备制造商。现在，在 Windows 10 发布近一年后，它仍然不太适合在手机上使用，它根本不是一个手机操作系统，与 iOS，Android 甚至 Windows Phone 都没有竞争力。他们希望明年能做得更好，但这是微软再次改变主意的一段时间。Terry Myerson 说，Windows Phone 由于用户基数小，目前还不是做酷的新东西的平台，这基本上概括了一切。这里有一篇非常好的[文章](http://www.theregister.co.uk/2016/05/27/windows_phone_history_special_report)促使我写了这篇文章。

### 接下来是什么

对于 Windows Phone 来说，被抛弃是一个漫长而悲伤的过程。对于 Windows 10，我们希望它最终能成为智能手机的良好平台。但是既然微软希望明年开始着手这项工作，那么谁知道这何时会发生，是否会发生。对我来说，要么是 iOS，要么是 Android。我以前使用过这两个平台，我不喜欢回到其中一个，但目前我们没有像五年前使用 Windows Phone 那样的第三个平台。
