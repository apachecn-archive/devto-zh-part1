# 降低噪音

> 原文：<https://dev.to/designfrontier/reducing-noise-2642>

噪音在我们的生活中无处不在。从商业广告，到广告牌，到一个新的啼哭的婴儿，它几乎无处不在。有些是好的噪音。其中一些会分散我们对想要关注的事情的注意力。

我正在做几件事来减少我生活和工作中被打扰的次数，我想我会在这里为子孙后代分享其中的一些。

### 再见脸书

开玩笑的。我不会离开脸书的。它解决了我生活中的一个重要问题，使我能够组织与朋友和家人的聚会，并以最小的摩擦与他们分享我的生活。这也让我了解他们生活中正在做的事情。

但是，它在我的手机上已经没有位置了。它只存在于我的平板电脑上。这极大地减少了我在脸书屏幕前的时间，让我有时间做其他事情。我只是觉得现在没有必要经常打开它。

### 停留聚焦

StayFocusd 是一个方便的小浏览器扩展，可以让我在浏览器中屏蔽所有与工作无关的网站。更好的是，它让我每天总共有 10 分钟的时间在上面。有足够的时间进行快速的精神休息，但还不足以让我频繁地击打它们作为一种回避行为。

我在工作中更加专注，完成更多工作。唯一的缺点是我在新闻上落后了几个小时。

不过说真的，我们过去总是等到第二天才看到新闻。没什么大不了的。非常值得交易。

### 对 OS X 的通报

能够在一段时间内全局关闭通知是 OS X 最棒的事情之一。当我需要真正专注于一些代码时，我经常打开开关，这很有帮助。我可以在处理完任何需要集中注意力的事情后，赶上任何突然出现的火灾。

### 关闭松弛/推特/消息

有时隐藏通知是不够的。当应用程序让我分心时，我会完全关闭它们，以此来克服我对错过的恐惧。所以有时候我会关闭 Slack (*gasp*)和 Twitter，甚至是 messages，这样我就可以只工作或者只和家人在一起。

### 改变我们的夜晚

我们在晚上少点 Netflix/Hulu，多点艺术/音乐。我的妻子是一位才华横溢的音乐家，我们都喜欢涉猎艺术。所以我们正在努力将晚上的时间转移到创作美丽的东西上，而不是仅仅消耗所有的时间。这很难。但是我们真的很享受我们成功的夜晚。

### 追上去

我喜欢生产东西...我从来没有足够的时间去创造所有我想做的事情。希望这些变化和其他变化能帮助我腾出时间去创造它们。更重要的是，这些变化将为我赢得更多的时间，和我的妻子和女儿共度更多美好时光。这确实是我能做的最重要的事情。
