# 网络活动

> 原文：<https://dev.to/funkysi1701/networking-event-45b2>

我最近参加了一个由 Core Com Consulting 组织的名为 ITBoss 的活动。真的很好，部分得益于免费的酒吧和美味的食物。

活动在利兹市中心的一家酒吧举办，所以我下班后赶到火车站。是的，我是第一个到达的(为什么我总是这样？).该活动面向 IT 领导和经理，人数限制在 20 人左右。

一开始，我只是和主持人以及其他人聊天，等待每个人的到来。一旦我们开始了，我们有一个关于战略需要的简短介绍。有趣的是，第二天我建议和我的主管开一个战略会议，所以一定有什么事情发生了。

该演讲的主旨是，IT 领导需要从日常工作中走出来，看看他们要去哪里，这应该与公司的总体方向相匹配。我同意这一点，最近我一直在思考我们的基础设施需要朝什么方向发展，目前我只是在想，但至少这是一个开始。然而，我们公司的董事们不善于分享他们的长期愿景，所以我很难知道公司的发展方向。我需要站出来挑战这种行为，这样我就可以做出必要的改变。这并不容易，但这正是最好的 IT 领导者正在做的事情。

在这之后，我们停下来吃东西，这很好。我和几个人聊了聊。我觉得有趣的是，一位先生对我想从事开发感到惊讶。我想我明白为什么房间里的大多数人都是高级经理，虽然我分享他们的头衔，但我不分享他们的薪水。他认为发展是降职，我认为是晋升。

吃完饭后，我们讨论了一些问题。我的第一个问题是，如何让别人相信改变的必要性？提出了一些有趣的想法，但需要与他人密切合作，找出他们的需求，这是困难的部分，而不是任何技术问题。讨论的其他话题涉及到招聘你不了解的技能人员，以及当今时代的安全和隐私话题。两者都非常有趣，表达了许多观点。

我很享受这个晚上，希望有一天能去另一个地方。我以前参加过的活动都是技术性的，但缺乏这次活动所具备的人脉。虽然我不太擅长交际，但在这次活动中，我很容易与人交谈。
