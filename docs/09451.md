# 网页设计师和网页开发者的区别

如果你曾经在互联网上工作过，你肯定听说过“网页设计师”和“网页开发者”这两个术语。这两个短语经常被不熟悉这个行业的人交替使用。我的意思是——设计师和开发者——他们是一回事，对吗？不对！！。

事实是，这个术语指的是网站开发的两个根本不同的方面，需要两种不同的独特技能。

让我们面对现实吧，如果你委婉地使用了“网络”这个词，你可能已经被来自两方面的问题所困扰。设计师会听到类似“我们怎样才能在网站上添加一个数据库？”，而开发人员却在努力回答关于布局、颜色选择、图像放置和字体样式的问题。

**仔细看看网页设计**

在我们继续之前，一个网页设计师的一个重要方面是理解客户网站的目标，然后其他一切都随之而来。最好的网页设计师是有创造力的类型。他们有进入客户头脑并实现客户愿景的诀窍。

设计师的最大目标是选择正确的颜色、布局、字体和图像，最终创建网站的整体个性。除了用户友好的网页，设计师还致力于网站的可用性。网页设计者创建目标市场能与之相关的网页。

例如，如果你为托儿所的孩子们创建一个网站，在大字体、明亮的颜色、图形、较少的文本和图像的帮助下进行设计。这将使网站具有吸引力，并抓住目标受众的注意力。

因此，他们的工作依赖于信息流、图形设计和色彩计划，他们的工作工具包括使用 HTML、CSS 和 Javascript 等语言。

#### 那么 web 开发呢……

网络开发人员本质上通常更注重技术。

他们使用设计团队创建的设计，并在后端工作，以创建一个功能齐全的网站。当用户与网站交互时，这些人是网页上每个元素平稳运行的幕后推手。他们负责一个网站的所有幕后工作。

这些开发人员中的大多数会使用某些编码语言来创建供您使用的网站，例如，使用 HTML 作为前端设计语言，而使用 PHP 和 MySQL 等动态且强大的服务器脚本语言作为网站的后端。
