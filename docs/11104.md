# 直面你的恐惧

> 原文:[https://dev.to/craigbuchek/face-your-fears-2lpp](https://dev.to/craigbuchek/face-your-fears-2lpp)

我一直都是直面恐惧的人。

我有中度的蜘蛛恐惧症。当我看到一只蜘蛛时，我不会逃跑，但是当一只蜘蛛在我身上爬行时，它会让我毛骨悚然。当我在大学的时候，我决定买一只狼蛛来克服我对蜘蛛的恐惧。我以为随着时间的推移，我会越来越适应狼蛛，最终让它爬到我的胳膊上。没用。尽管我确实发现我对狼蛛的恐惧是有道理的——我只是因为触摸了掉在它吸水海绵里的令人发痒的毛发就患上了严重的荨麻疹。

上周，我在墨西哥度假。我们参加的一次[短途旅行](http://www.xenotes.com/)包括多次跳入水中。我不擅长游泳——主要是因为我很难闭上鼻子；我跳进去的时候捏着鼻子。我们在水里滑了很多次。在某一点上，有一个悬崖可以从那里跳入水中。它高出水面约 15 英尺。跳下去之前没看那么远。但是我第一次从上面跳下来的时候，感觉真的很远。对我来说很可怕。所以我又做了一次。没有任何同龄人的压力让我跳第二次。我特意跳了第二次**，因为**我很害怕。

恐惧是一种奇怪的东西。恐惧会保护我们。但它在那里保护我们免受非洲大草原的威胁。我们的恐惧保护我们免受的大多数事情在我们的日常现代生活中并不存在。因此，也许我们应该努力更好地理解我们的恐惧是如何产生的，弄清楚什么时候关注它们，什么时候忽略它们。

幸运的是，我们的大脑有一个很好的机制来帮助我们做到这一点。我们的大脑基本上有 [2 个主处理系统](https://en.wikipedia.org/wiki/Thinking,_Fast_and_Slow#Two_systems)。第一个是快速反应。这一个包括几乎是反射的事物。恐惧存在于这个系统中。第二个系统是我们的分析推理系统。这个系统需要更长的处理时间，但是能够接收更多的信息。

只要情况允许我们有时间让两个系统都工作，我们就需要倾听它们。我们需要倾听我们的恐惧，因为它们的存在是有原因的。但是这个原因可能不适合我们的情况。所以我们需要意识到这一点，让缓慢的分析系统来决定我们是否应该忽略我们的恐惧。

如果我们不让两个系统都工作，我们就没有充分利用我们的大脑；我们没有充分利用生活呈现给我们的情况。
