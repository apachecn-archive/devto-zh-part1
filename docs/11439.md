# 表单输入类型颜色元素

> 原文:[https://dev.to/mandarbadve/form-input-type-color-element 数控](https://dev.to/mandarbadve/form-input-type-color-element--nc)

现在，颜色选择器不再需要插件和库。HTML 5 给你输入元素来选择颜色。

让我们看看语法。

```
<input type="color" name="mycolor" /> 
```

只需将类型的值写入颜色，浏览器就会向您显示颜色选择器。
