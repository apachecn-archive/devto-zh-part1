# 别问了，开始做吧

> 原文:[https://dev.to/_patrickgod/stop-asking-start-doing](https://dev.to/_patrickgod/stop-asking-start-doing)

你有很多很多的想法，想知道如果你开始在生活中实施它们会发生什么？这不仅仅是关于软件开发，也是关于你生活的想法，比如冥想，早晨的例行公事，健身或者开始一个新的副业项目(好吧，你懂我的意思，这将是与开发相关的事情)。

我注意到，当我有这样的想法时，我几乎总是会问其他人对此有何看法。与市场调查类似，我会问朋友和家人对去健身房或变得更擅长烹饪有什么看法。有时我会想，为什么我要请求别人允许我做一些新的或不同的事情呢？这太疯狂了！这并不是说我想创造一个新产品，首先要测试是否有我可以向其销售的受众。大多数时候都是关于能让我的生活变得更好的事情。

或许你可以认同以下的观点。过去，我有许多新产品或项目的想法，甚至是我想融入日常生活的新习惯。有时，我甚至开始实施这些想法，但没过多久，我又停下来了。我并没有真正承诺过这些事情。但是为什么呢？

当你开始询问别人的意见时，他们可能会告诉你你想听的，即你的想法听起来很棒。但往往你不会得到真正的支持，这完全没关系！别人怎么看你的想法并不重要。想想历史上和我们这个时代所有伟大的创新者。埃隆·马斯克不在乎别人对他改变世界的想法有什么看法。现在他制造电动汽车和火箭。火箭！

所以尝试新的东西。测试你的想法。根据我自己的经验，我可以说，有时事情会很顺利。一天，我在 codeproject.com 上发布了一个 Windows 8 应用的想法。结果呢？他们很喜欢，还送了我一台笔记本电脑作为奖励。

我们确实不时地在我们的想法上取得成功。但是为什么我们仍然停止做事呢？

我认为这是关于承认。人们不认可或欣赏我们的工作。由于我们似乎依赖于它，并希望其他人告诉我们，我们的想法和我们的工作是伟大的，一旦他们不再做我们的工作，我们就停止工作。

尽量不要在意别人对你的想法或工作的评价。你得自己测试它们，看看结果，得出正确的结论。我们经常停止甚至不开始项目，因为“其他人说这不值得或者愚蠢。我不知道人们为什么会这么说，但大多数时候**并不是因为他们有经验判断你的意图是否成功**。我知道这是一个刻薄的说法，但有时人们不希望你成功，因为他们自己也不觉得成功，这就是为什么他们试图拉你下来——即使这只是潜意识的。

所以请继续创造和实现你的想法。还有最重要的:**要一致**。只有当你坚持你的计划，在最初的几周甚至几个月后不放弃，你才会知道你是否在正确的轨道上。

另一个可能阻碍你的方面是恐惧。你只是害怕结果。这并不一定意味着你害怕失败。有时我们害怕成功。这听起来可能很奇怪，但却是真的。如果你的新产品很棒，而你正在围绕它建立一个观众或社区，会发生什么？如果你定期锻炼，看起来越来越好，会发生什么？

一方面，你可能会有更多的工作。不过，也许有些工作是你想做的。另一方面，人们可能会嫉妒你。甚至可能是远离你的朋友。

在这些情况下，为自己的决定负责是有帮助的。抓住它们，相信自己。然后**别人的意见不会影响你**你可以轻松地追求你喜欢和想做的事情。
