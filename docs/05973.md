# 新职位:开发商代言人

> 原文:[https://dev.to/wiredferret/new-job-title-developer-advocate](https://dev.to/wiredferret/new-job-title-developer-advocate)

在我的职业生涯中，我有过很多头衔:

*   技术写作实习生
*   文档女王(那是 2000 年，好吗？)
*   技术作家一、二、三
*   技术通信员
*   高级技术作家
*   技术写作顾问
*   文档架构师
*   文件雇佣军

你可能会注意到一个主题。我为许多不同的公司做过技术写作，因为这是我的职业、专长和热情所在。我想利用技术的一切优点，让它更容易使用，更透明，更周到，更人性化。

最近，我很难单独用写作来描述我正在做的事情。连续两次工作面试，我的面试官不再问我资格方面的问题，这样他们就可以记下我对他们产品的想法。我的会议演讲名义上是关于写作的，但实际上是关于我在世界和技术中注意到的模式。我热爱写作，我永远不会放弃它，但它也…不再是一个很好的选择了。

通过全女性科技人脉的力量，我得到了一个开发商代言人职位的面试机会。我自己绝不会申请这个职位——这远远超出了我认为自己的能力范围。但在讨论和面试中，我真的开始相信，这不仅仅是一家我可以愉快工作的公司，一种我认为有用且无毒的产品，而是一个让我能够出去思考、帮助和解决我喜欢的问题的职位。

图片来源:科技聊天中的色女

开发者倡导者实际上是一个非常广泛的职位，但我们对它的解释基本上是我继续做我现在正在做的所有事情:会议发言、博客、倾听和注意。只是现在我将做所有这些事情并从中获得报酬，而不是把它作为我咨询的赔本赚吆喝。我开始走向世界，找出开发者和用户需要帮助的地方，并找出如何为他们提供帮助。我们真的处于“掐我一下，我一定是在做梦”的兴奋水平。我甚至可以继续写一点，尽管我可能已经达到了我的个人职业目标:不写发布说明。

[![](img/10369d35cebe2326367d0731bc032107.png)T4】](https://res.cloudinary.com/practicaldev/image/fetch/s--MX3eJD1t--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://www.heidiwaterhouse.com/wp-content/uploads/2017/07/img_0744.jpg)

是的，我故意对我的新雇主含糊其辞。它应该有自己的位置。我只想说，我认为我们会相处得很好，他们说我会继续做一个粉红色头发的怪人，我会为这个产品感到骄傲。

[![](img/aca4ac8cbb0b937d7df0e1b35400b47e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ub1n9uZq--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://www.heidiwaterhouse.com/wp-content/uploads/2017/07/img_0743.jpg)

老实说，我觉得改变我的工作头衔就像你有了一双新鞋，却意识到自己已经穿不下那双旧鞋了。

> 哦！这太舒服了。
