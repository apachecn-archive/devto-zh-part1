# o 表示开放。在浏览器中打开存储库的简单 CLI 工具。

> 原文：<https://dev.to/plutov/o-means-open-simple-cli-tool-to-open-repository-in-browser>

下面是我的小 bash 函数！当您从终端运行它时，它会在浏览器中打开 GitHub/BitBucket/GitLab 页面，显示您当前所在的 git 存储库。它有一个简短的名字`o`。我发现自己做了很多这样的事情，因为我同时处理多个存储库，切换到浏览器，搜索正确的链接等等。

只需输入`o`:)

### 如何安装

```
curl https://raw.githubusercontent.com/plutov/o/master/install.sh | sh 
```
