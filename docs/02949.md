<ruby>标签？那是什么？
第一次看到，可能不知道怎么用。</ruby>

<ruby>标签主要用于指示如何阅读汉字。
比如，</ruby>

```
<ruby><rb>花</rb><rp>(</rp><rt>はな</rt><rp>)</rp></ruby> 
```
