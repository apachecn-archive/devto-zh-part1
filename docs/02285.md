#### [](#top-5-things-most-junior-developers-forget-before-applying-a%C2%A0job)**初级开发人员在求职前最容易忘记的 5 件事！**

1.  收集代码示例，将它们放在您的文件夹中！
2.  从你以前的项目中取出你引以为豪的代码片段！
3.  想想他们为什么要选你的几个理由！或者你为什么想为他们工作！
4.  仔细阅读职位描述！他们在问什么？你认为自己处于那个位置吗？
5.  做你自己！不要试图展示你比实际更好的一面！真相总会找到你！
