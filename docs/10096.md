# 矢量手表销售

> 原文：<https://dev.to/horia141/vector-watch-sale-28l5>

罗马尼亚一家生产智能手表的初创公司 Vector Watch 最近被 T2 FitBit 收购。这个消息现在已经很老了，所有能说的都已经说了。但这并不是不写这篇博文的正当理由。

不用说，出口并不是很好。但这仍然是一个令人印象深刻的壮举。总的来说，制造硬件是一件困难的事情。制造消费类硬件更加困难。在罗马尼亚建造它是自找麻烦。作为参考，罗马尼亚还有另外一家[公司](http://allview.ro/)生产消费手机，尽管他们看起来做得不错。

有趣的是，没有其他公司做得这么好。事实上，据说市场在 2016 年第三季度下降了[51.6%。这不是好消息。但这也不是 T2 可怕的新闻。至少对那些能撑一段时间的公司来说不是。这意味着像苹果，三星，索尼等。但不是独立生产商，比如](https://www.idc.com/getdoc.jsp?containerId=prUS41875116) [Pebble(被 Fibit 收购)](https://www.pebble.com/)、 [Garmin](https://buy.garmin.com/en-US/US/wearables/c10002-p1.html) ，甚至不是传统手表厂商的智能产品【2】。

这不是可怕的消息的原因是我们仍处于游戏的早期。智能手表的 iPhone 还没发明出来。也没有规定说现在的玩家将是传递它的人。但市场潜力巨大，尤其是在医疗保健和支付领域。

我还觉得产品类别本身被不公平地设置为与智能手机/手机相对抗。这是一个难以置信的计算最佳点。要打败它，还需要其他大的技术进步。无论是什么打败了它，智能手表、虚拟现实和其他一系列“辅助”技术都可能受到同样的影响。

看看“杀手版”是什么样子会很有趣。它会被科技进步所带来吗？还是用目前玩家还没有想到的方式来建造？
