# 新世界

> 原文:[https://dev.to/cyberomin/the-new-world](https://dev.to/cyberomin/the-new-world)

> “这条线是画出来的，诅咒是铸出来的，现在慢的会很快，因为现在会很快过去，秩序正在迅速消失，现在的第一个会持续到他们改变的时代”——他们改变的时代，鲍勃·迪伦

前阵子，故事是这样的；你学习一种特定语言的字母，比如说英语，然后你学习把字母串在一起组成单词，这就成功地学会了造一个完整的句子。当你习惯这样做时，你就学会了语法、标点符号等规则，现在你可以开始写和说一种语言了。在大多数情况下，这是你被称为有文化或精通该语言的时候。

我上面描述的场景已经有一个多世纪了，直到今天仍然适用。我认为这种情况不会很快改变。你的读写能力让你比那些不具备这些技能的人领先一步。

今年是 2016 年，时代在变。正如我们所知，生命在进化，人类在适应。这一演变的核心是技术，如果我们今天看到的趋势是可以遵循的，这将持续很长时间，也许再过一个世纪。

我今年的部分目标是学习驾驶，感谢上帝我实现了。但是这是一次有趣的经历吗？不，一点也不好玩。拉各斯不是那种你可以学习驾驶的城市。其他道路使用者可能会对你宽容，但黄色巴士司机不会。充其量，他们会辱骂你，但那是以后的事了。

当我反思这次经历时，我问自己，我的孙子们会学到驾驶这门美丽的艺术吗？我不这么认为，不是因为他们不会像爷爷那样聪明或勇敢，而是因为这没有必要。在这个世界上，自动驾驶汽车将像我们呼吸的空气一样普遍，为什么还要麻烦呢？让机器人做他们的事情。他们会做得更好。

无人驾驶汽车、太空探索只是新世界的一部分。而我们今天看到的只是冰山一角。机器人已经来了，他们不会来了。他们在这里，我们只是不想承认这个事实。它们无处不在，它们在我们的手机、电脑、电视上，无处不在。我们大多数人每天都在使用它们并与之互动，但我们只是不知道而已。

谷歌，通过它的 Allo 即时通讯服务，给我们带来了名为谷歌助手的神奇功能。不，说真的，这种技术是可怕的，如果有的话，我已经看到城市和导游的工作受到严重威胁。优步、特斯拉、谷歌在自动驾驶汽车上投入了数十亿美元，这个靠开车为生的人应该担心了。众所周知，亚马逊(Amazon)无视华尔街，不畏艰难，按自己的方式做事，该公司已经在履行中心部署了机器人，除了机器人，它还在测试无人机送货。我们都应该担心。

所有这些趋势指向什么？还在想？当你是它的时候，不要搞错，技术和技术革命就在这里。在这个新世界中，统一的东西将是代码。编码是对*新的读写*。你控制和操纵电脑的能力将和学习阅读和写作一样重要。

作为今天的年轻人，为什么不为这个新世界做准备呢？我所说的这个新世界是不可避免的。做好准备…
