# 弹出 Magento 扩展

> 原文:[https://dev.to/paulsimmons/popup-magento-extension](https://dev.to/paulsimmons/popup-magento-extension)

当你的用户访问你的网站时，你可以向他们显示重要的信息。您可以创建仅在特定产品页面上显示的特定于产品的弹出窗口，而全局弹出窗口则显示在网站的所有页面上。使用弹出扩展，您还可以向特定的客户组显示弹出窗口。

Magento 弹出式插件的主要特性:

*   弹出窗口的静态块
*   在弹出窗口中添加醒目的横幅、链接、图像和文本
*   在 CMS、主页、类别和产品页面上显示弹出窗口
*   弹出窗口有响应
*   设置弹出窗口的大小
*   100%退款保证
*   免费终身支持
