# convol PHP 1 . 1 . 0 发布

> 原文:[https://dev . to/jaysonjphillips/convole-PHP-1-1-0-released-45hb](https://dev.to/jaysonjphillips/convore-php-1-1-0-released-45hb)

今天早上我在 GitHub 上发布了一个新版本的 Convore PHP API 库。该类现在拥有自上次提交以来所有新添加的 API 方法，包括:

`account/mentions.json`
`groups/:group_id/request.json`
`groups/:group_id/track.json`
`topics/:topic_id/track.json`
`users/username/:username.json`
`messages.json`
`messages/:user_id.json`
`messages/:user_id/create.json`

`messages/message/:message_id/delete.json`
`groups/discover/trending.json`

你可以在回购时把它拉下来:[https://github.com/jaysonjphillips/convore-php](https://github.com/jaysonjphillips/convore-php/releases/tag/1.0.1)
