# 为谷歌网站管理员工具设置首选域名

> 原文：<https://dev.to/adamkdean/set-preferred-domain-for-google-webmaster-tools-3p0c>

将这个网站添加到 Google 网站管理员工具中只是给我带来了一个小而恼人的问题，即无法将首选域名设置为包含`www.`而不仅仅是`adamkdean.co.uk`。我收到一条错误消息，但没有如何操作的说明:

> 设置首选域名过程的一部分是验证您是否拥有[http://adamkdean.co.uk/](http://adamkdean.co.uk/)。请核实 http://adamkdean.co.uk/的[。](http://adamkdean.co.uk/)

我试图使用 no `www.`而不是 dice 来访问验证文件。我也尝试添加网站作为根域名，`adamkdean.co.uk`，但仍然没有运气。

结果是你必须添加网站*两次*，有没有`www`子域都要添加，所以你的网站管理员工具里有两个条目，两个验证步骤。

本可以扔给我一根骨头，谷歌！
