# Govhack 冒险

> 原文:[https://dev.to/diskdrive/govhack-adventures-40pb](https://dev.to/diskdrive/govhack-adventures-40pb)

几周前，我参加了我的第一次 GovHack。在这个活动中，一群人摇滚起来，分成几个小组，利用澳大利亚政府发布的数据集在 48 小时内开发出一些有用的东西。

这是我第一次参加黑客马拉松，所以我真的不知道会发生什么。但现在经历过了，我可以有把握地说，这是一次紧张、疲惫但最终非常有益的经历。

首先，像这样的事件表明，在大多数情况下，人们确实真诚地想要有所作为。我的意思是，虽然许多人喜欢他们的工作，但我们大多数人都用他们来支付账单，在很多方面，我们对自己为社会做出的贡献没有太多发言权。

但是如果你不用付账单，你会做什么？你还会免费做你正在做的事情吗？我不能说我们大多数人会。然而，考虑到 govHack(竞争者和志愿者都一样)的参与情况和该活动的精神，它表明，如果我们不需要照顾自己，我们大多数人仍然会选择照顾他人，试图以我们自己选择的方式做出贡献。这是一件很好的事情，这是一种很好的氛围。

govHack 的另一个特点是——你非常忙。你编码，你睡觉，你编码，你睡觉…偶尔他们给你食物(是的食物和饮料是免费提供的！).但是和正常工作很不一样。没有规格或代码审查或利益相关者会议。不是说这些事情不好或不必要——只是说它不会在黑客马拉松中发生——它基本上只是一场与时间的赛跑——并且你关心交付高于一切。

所以这是一种调整——你发现自己在想“嗯，我可以智能地构建这段代码，使它更容易阅读和维护……或者我可以只写这段代码”,你发现自己在做后者。你觉得有点脏，但你还是做了，因为这样更快。

但是结果是你确实很快地开发了一些东西——它可能不完美，但是它展示了一个概念。我想重点是，如果它被证明是有潜力的，你回去，然后把任何铃铛和口哨。如果没有-那就只有 48 小时了。

我和我的同事 Jonathan Moffatt 一起工作，我们决定使用电器及其能源评级的数据集来构建一个能源计算器。我们使用了[Meteor](http://meteor.com/)——这是一个我们已经熟悉的 JavaScript 框架，在今年年初用它开发了 [WebStandup](http://www.webstandup.com/) 。

[![](img/ca498a68783ce134762c75a97d8e66f9.png)T2】](http://3.bp.blogspot.com/-aQK6vplFgyA/U9EKJL8aY-I/AAAAAAAAAS0/5Uf_hbI9_tc/s1600/Screen+Shot+2014-07-24+at+11.28.34+pm.png)

我承认我们的想法并不特别吸引人——其他团队有更高的计划——使用各种 API 提供的艺术作品和图像或地理空间数据来混合更动态和可视化的产品。但是我们相信它有用处。你告诉我们你家有哪些型号的电器，我们会帮你算算，告诉你如果升级的话能省多少钱。

[![](img/0f923f97c2109088b7c4ee5fe3f2dfed.png)](http://i.telegraph.co.uk/multimedia/archive/02983/Ferrari-250-GTO-Be_2983288b.jpg) 
更重要的是，我们相信基于我们现有的技能，我们可以在 48 小时内生产出一个工作模型。实现这个目标对我们来说非常重要。有很多工作要做——首先是计算公式(仍然不太明白对数是如何工作的),但是编码就像开着法拉利在城市中穿行一样。当它好的时候，它是甜蜜的，你可以尽情享受，根本不需要时间。但是无论你在高峰时间开什么车，你都不会很快到达任何地方。当你编码的时候，你会碰到一些看起来不容易做的事情——但事实并非如此——而且你会花很长时间去做这件小事。这就是软件开发成本如此难以估算的原因——因为从理论上讲，它不应该需要那么长时间。最费时间的是那些意想不到的问题。我们最终完成了我们的产品，并在周日还剩一个小时的时候制作了我们的视频。

因此，GovHack 对我来说是一次非常棒的经历。它让我走出了自己的舒适区，并教会我快速而肮脏地建造东西实际上也是一项重要的技能。能够定义一个 [MVP](http://en.wikipedia.org/wiki/Minimum_viable_product) (最小可行产品)来测试一个想法是否有价值，并且能够快速构建它——这并不容易。我想我还需要再做几次黑客马拉松。

不管怎样，这是我们的 Youtube 视频，这是我们的 T2 能量计算器。让我知道你的想法(哦，你可以在这里为我们[投票](http://hackerspace.govhack.org/content/energy-calculator-and-comparison-tool))。除了 GovHack，我们正在考虑在某个时候，我们可能会完善网站，也许会添加一些我们无法获得的功能(例如，为用户提供更高效的电器产品推荐)。
