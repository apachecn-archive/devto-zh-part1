# 我是如何让开发人员失望的

> 原文:[https://dev.to/episod/how-i-fail-developers-2io2](https://dev.to/episod/how-i-fail-developers-2io2)

我不告诉你未知的事。即使我知道。

我不会告诉你什么事情不能做。虽然我知道你想做什么。

**我不告诉你这些怪癖。**我不想相信他们。

我不清楚这件事和那件事有什么不同。我让你来解决这个问题。

我不会告诉你为什么会这样。不关你的事。

我不告诉你我不知道的事。虽然我知道我不知道。

我没有发现我不知道的东西。我不够好奇。

我没有修改旧的东西。虽然我知道它发霉了。

谁编辑的？

我错了。花了太长时间。我用了太多的词。我没用够。

我忘了附上一张照片。老实说，我都忘了怎么做了。

我的代码样本只是假设的。

(括号太多)

我没有寻求帮助。

我没有测试它。

我没有改。

我无法测试它。

我无法改变它。

我关于它如何会不同的论点没有获胜。

我只是写文件。
