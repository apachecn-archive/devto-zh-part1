# 好聚会的技术笔记

昨晚我去纽约参加了一个进步技术专家的聚会，讨论对最近选举的回应。人们分享了想法、感受和前进的方法，我愿意分享它们。在结论中，我将试着总结一下我所听到的。

#### 这些是我以最快的速度写的笔记，所以不是所有的引用都是准确的，我让每个人匿名。

一个人谈到对周二投票的选民感到沮丧和愤怒，并满足于保持愤怒。她说“愤怒可以是美丽的，强大的。它不会让你停滞不前。”

另一个人谈到他是如何震惊的。“超现实。好像这不是真的一样。你知道这个世界已经破碎，但你不知道它有多破碎。”

另一个:“我们并不像自己认为的那样了解这个国家。我想质疑我的偏见。”

另一个:“这个国家我一半都不懂。我们所有人都没有。”他问房间里的人，如果有一天我们来上班时发现所有员工都被解雇了，取而代之的是海外的 UX 设计师，而且价格只有一半，我们会作何感想。我们会有什么感觉？在这种情况下，如果这意味着解决这个问题，也许你会愿意忽略你的候选人的罪过。“不要与你的同胞作战；找到一个目标，并为之奋斗。”

一位女士说“我们作为一个群体(技术社区)拥有比我们意识到的多得多的力量。我们对思想的影响比我们意识到的要大。这还没完。”她讨论了过去 8 年是进步的想法是多么容易变得自满。

另一个人引用了一句话:“在文化战争中引入事实就像在枪战中引入刀子。”他推荐了一些注重隐私的人应该采取的措施:

1) [获取 Tor](https://www.torproject.org/)

2) [启用 2 因素认证](https://en.wikipedia.org/wiki/Multi-factor_authentication)随处可用

3)获取[信号](https://whispersystems.org/)

4)获取 [VPN](https://en.wikipedia.org/wiki/Virtual_private_network)

另一个:“我刚刚意识到，有一半的美国人投票支持性别歧视和种族歧视。”她听到很多人鼓励人们思想开放，理解，以及另一方有他们投票的理由。她的观点是，所有支持性别歧视和种族主义的人都被证实了；那些观点现在是主流，好吧。这个观点是首先需要解决的问题。

一名妇女提出了这样一个事实，即特朗普以自我为中心的事实可能对进步人士有利。他可能会被引诱。

其他人对其他发言者的惊讶表示惊讶。她提到了英国退出欧盟。俄罗斯、土耳其和中国。她谈到我们如何不需要 7 万多个应用程序；是一个房主在密歇根的水中发现了铅。像这样的信息需要被收集和分享。

一名移民在选举后的一天里在国家历史博物馆四处走动，心想“怎么会有人想要破坏这个呢？”他鼓励人们走出自己的社区，去教书，以此来增强人们的能力。帮助那些最需要帮助的人。

最近一个从硅谷移植过来的人提出了一个问题:为什么美国允许选举归结为克林顿和特朗普之间的选择？他说，人们投票给特朗普是因为他告诉了他们想听的话。美国人不想听到叙利亚移民。她没有解决她的钱的问题。作为一名印度移民，他谈到了他的梦想，那就是来到美国将会带来巨大的改变。“我来到这个国家不是为了绝望。我想专注于希望。”

一个单身母亲的儿子，第一代美国人谈到了过去 48 小时的挫败感。不知道如何处理自己的情绪。对不可避免的种族歧视感到恐惧。

一名男子很兴奋地讨论激进主义，作为一种应对机制。寻找想法，想办法离开纽约的泡沫。“我们能做些什么来确保技术不会蒙蔽我们？而不是把人变成漫画？”

一位妇女谈到她对没有做更多的事情感到内疚。并且对认为技术可以改变很多感到内疚。

一名男子对特朗普深感担忧，无论是国内还是国外。他想建立一种更强的个人联系感，并想办法鼓励同情和团结。

有人建议我们去另一边的地方，比如布莱巴特，参与进来，开始评论。即使别人不同意你的政治观点，也要大胆地说出来。

一个男人恳求每个人“摆脱这个惊喜”。世界就是这样。”他谈到了他是如何把自杀热线的电话号码给那些承受着极度压力和焦虑的人们的。“这就是我们国家的所在。这是它一直在的地方。一个人基本上建立了一个我们相信并赢得的东西的模拟像。”他说，就像女性和少数族裔在科技领域存在多样性差距一样，也存在政治多样性差距，“这是由我们坚持公开羞辱和驱逐那些持反对立场的人造成的，这给那些为了维持关系而宁愿不说话的人灌输了一种恐惧。”这使得社区中的人们无法理解另一方是如何投票的。“他们也害怕和我们说话。”

一个男人谈到社区里的人们有很多想法。大量的能源，参与开源。“我认识到有很多机会去做我们最擅长的事情。为了彼此，为了世界上的其他人。”

一名女性谈到自己是一名技术官僚，并感到隐隐的愤怒。她谈到她的南方家庭认为她生活在一个幻想的国度。投票给特朗普就像向她的窗户扔砖头。“你现在在看吗？”她反问。她谈到技术不能解决一切，谈到她渴望走出泡沫，但不知道如何走出泡沫。她恳求人们见面，而不是在网上。

一名妇女谈到她如何“不在一个充满希望的地方。”关于数据是如何失败的。她说“白人辜负了我们。我只关注黑人和同性恋群体。我没有试图改变白人。作为程序员，我们知道错误(也就是问题)在哪里。”

一名男子谈到了长期的重要性，但短期的事情更重要，如解决即将到来的欺凌，白人权力圣歌，以及更糟糕的事情。以及让受害者知道他们并不孤独是多么重要，即使他们生活在一个非常保守的地方。

一位移民讨论了利用工作作为治疗，并鼓励人们作为技术专家在活动中提供帮助，并在元层面上确定哪些方法是最成功的。

一位产品经理鼓励人们像对待客户一样对待选民(有时客户可能会非常令人沮丧，但却是有效应对的必要条件)。

一名男子指出，这个房间“把另一边称为‘另一半’”。只是想指出不到一半的人口投了票。所以重要的是要明白全国只有四分之一的人投票给这个人。重要的是要理解为什么没有投票的那一半人没有投票。"

一名妇女讲述了她是如何被收养的，因为她出生在一个因为她是女孩而拒绝她的国家。看到第一位女性候选人被拒绝让她心碎。她讲述了她堂兄的小女孩现在是如何害怕去上学的。让孩子们感到没有归属感是多么不合理。

#### 此时，谈话转移到要求以行动为导向的想法上来。

一名 CS 学生谈到了美国的数字服务，以及技术人员如何兴奋地在奥巴马手下工作，但却与特朗普背道而驰。以及那里可能有机会，尽管这感觉像是对立面。

一名男子建议更新应用程序的算法，以建议更多来自你的泡泡之外的东西。也许周期性的看到你不认同的故事。

另一个人建议支持你支持的特定组织。作为“技术专家，他们可能有很高的薪水，他们可能会在接下来的四年里得到大幅度的减税，我们为什么不捐出额外的钱，并成为志愿者呢？”

有人建议使用数据来找出谁在以前的选举中投票，谁在 2016 年没有投票。找到那些认为不重要的人并倾听。

一位女士建议学习茶党的方法，特别是他们早期对小型和地方性倡议的关注。她提到，她认识的人对地方一级的工作方式知之甚少。她还建议建立一个邮件列表来记录最初的 100 天。

另一名妇女也讨论了茶党，以及不同的团体如何能够选择一个原因并关注它。相反，左边是如何断裂的；“我收到了五封不同的环保组织的邮件”。

一名男子反驳了戳破他的泡沫的想法:“我在一个泡沫里。我是同性恋，很难处理脸书的歧视性问题。在成长过程中，我将许多同性恋歧视内化了。所以我们也需要考虑我们自己。我们不必假装自己完全不同，因为我不认为我们完全不同。”

一名男子谈到了对选民的压制，以及动员人们参加未来选举的方法。他谈到下一次期中考试将是一场艰苦的战斗，以及“事情肯定会变得更糟”

一个男人说:“我从 1995 年就开始用 PGP 了。界面和那时一样。他妈的怎么回事？不行！”他恳求人们从 UX 的角度出发，让加密文本变得更容易。我们应该让加密成为一个标准。

一位来自塔拉哈西的 UX 教师刚刚购买了她的第一份报纸。她谈到“我们已经习惯了免费获取媒体，因为我们是廉价的千禧一代。但是我们需要支持那些做真正跑腿工作的机构。我们已经得到承诺，第一修正案将在未来四年受到攻击。”特朗普的支持者不应该被拒之门外，我们可以向右翼学习，学习他们如何自下而上地推动立法。

一名男子谈到他不认识一个特朗普的支持者，他觉得他周围的一些人很可能是不愿意直言不讳的支持者。就像深红州的进步支持者可能也不愿意直言不讳。

一位女士谈到人们不了解他们的国会议员，另一位男士谈到让投票变得更容易，就像在你的 Xbox 上投票一样容易。并鼓励人们建立一些东西，让人们更有效地沟通。

一位女士推荐了亚历克西斯·齐格勒写的《综合行动主义》这本书。

一个人推荐[代码军团。](https://www.codecorps.org/)

一名妇女谈到民权运动如何将资源从歧视性企业转移出去。她恳求人们利用他们的特权来帮助那些不太幸运的贫困社区。

一位女士说“如果房间里有来自另一边的人，那会很酷。去理解他们在想什么。”

一个人谈到我们如何需要“保护自己和反击”。特朗普心目中的统治者是普京。他恳求每个人学会如何在监视状态下组织起来。很难想象这种事情会发生在美国，但特朗普将控制一切:行政部门、立法部门和最高法院。除了我们，没有什么能阻止他。

#### 外卖

出现了一些共同的主题:

1.  人们绞尽脑汁想如何应对:与另一方接触，试图理解他们的观点，尽管你可能会觉得他们的观点令人反感？或者在沙子里画一条线，说这个(种族主义，性别歧视)不可以？
2.  很多很多人谈到，由于生活在泡沫中，他们被蒙在鼓里。很多人不知道如何解决这个问题。
3.  对于一群技术专家来说，有相当多的共识认为技术不是唯一的解决方案；让人们亲自参与是解决方案的一部分。
4.  围绕地方和州级政治展开了讨论，并研究了茶党是如何掌权的。
5.  人们谈到用时间和金钱来支持你所信任的特定组织。尤其是通过订阅来支持新闻业，因为这些组织已经在财务上苦苦挣扎，未来可能会受到政治压力。

此外，概括一位与会者建议的防范监控和黑客攻击的措施:

1.  [Get Tor](https://www.torproject.org/) 。Tor 是旨在保护您隐私的软件。
2.  [随时随地启用双因素身份验证](https://en.wikipedia.org/wiki/Multi-factor_authentication)。2FA 让人们更难侵入你的邮箱和其他敏感信息。
3.  得到[信号](https://whispersystems.org/)开始在那里发短信。Signal 是一款用于 iOS 和 Android 的私人消息应用程序，可以加密您的消息，防止它们被拦截。
4.  弄个 [VPN](https://en.wikipedia.org/wiki/Virtual_private_network) 。VPN 将加密你的数据，并增加隐私。有免费的选择，但你应该找到一个付费的 VPN 提供商。[这里有一份名单](http://www.pcmag.com/article2/0,2817,2403388,00.asp)。如果您有其他建议，请发表评论。
