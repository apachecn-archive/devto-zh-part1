# 在 OSX 安装 fleetctl

> 原文:[https://dev.to/adamkdean/install-fleetctl-on-osx-3gnk](https://dev.to/adamkdean/install-fleetctl-on-osx-3gnk)

在 OSX 上安装`fleetctl`最简单的方法就是用[自制](http://brew.sh/)。

如果你还没有的话，[在 OSX](http://adamkdean.co.uk/blog/read/138/install-homebrew-on-osx) 上安装家酿。

安装好之后，像这样安装 fleetctl:

```
brew install fleetctl 
```
