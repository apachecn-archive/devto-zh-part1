### 用于基于 Linux 的系统

第一步:打开终端
第二步:打开主机文件`sudo vi /etc/hosts`
第三步:添加您想要阻止的网站并保存

```
127.0.0.1 facebook.com
127.0.0.1 www.facebook.com
127.0.0.1 web.facebook.com 
```
