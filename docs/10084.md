# 收回我的话

> 原文:[https://dev.to/kaseycodes/eating-my-words](https://dev.to/kaseycodes/eating-my-words)

本周，我不得不与我的一名团队成员(我的日常工作是管理)坐下来讨论他是如何“努力把事情做好，而不是努力把事情做好”。他理解我的观察，并同意我提出的一些修改建议。

三天过去了，现在我坐在这里，意识到我今天已经浪费了 6 个小时在我的编码项目上。我发现自己在努力把事情做好，而不是努力把事情搞清楚并学习。

我最近已经从训练营最初的“通过信息超载学习”部分转移到完成我的第一个只有方向的项目，而不是方向。差别很大。但我仍在尝试“继续前进，以后会有意义”的方法，这种方法帮助我生存至今。它不再帮助我了。

所以现在我坐在这里，收回我说的话，忍不住笑了笑…还哭了一会儿。
