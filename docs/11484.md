# 在 Ubuntu 上安装 Sublime 文本

> 原文：<https://dev.to/adamkdean/install-sublime-text-on-ubuntu-1723>

这与在 LTS Ubuntu 14.04 上安装 Sublime Text 有关。

对于 Sublime Text 2，运行以下命令:

```
$ sudo add-apt-repository ppa:webupd8team/sublime-text-2
$ sudo apt-get update
$ sudo apt-get install sublime-text 
```

Enter fullscreen mode Exit fullscreen mode

对于 Sublime Text 3，运行以下命令:

```
$ sudo add-apt-repository ppa:webupd8team/sublime-text-3
$ sudo apt-get update
$ sudo apt-get install sublime-text 
```

Enter fullscreen mode Exit fullscreen mode

注意，这只是将`sublime-text-2`改为`sublime-text-3`的情况。
