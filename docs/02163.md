作为该领域的新开发人员，您可能已经花时间为您的编码训练营或其他程序构建了项目。你可能和其他同学一起做过结对编程。也许你在数字海洋和 Github 的 [Hacktoberfest](https://hacktoberfest.digitalocean.com/) 到*期间采取了主动，轻轻的*参与一些开源项目，提交几个拉请求。然后你决定参加你的第一次全天黑客马拉松，并深入一个组织的代码库。这就是上周日发生在我身上的事情，令人振奋。

我带着信心离开了这一天，我可以为一个我从未见过的项目克隆一个代码库，花一些时间仔细查看它，并继续为这项工作做出有价值的贡献。在黑客马拉松之前，我当然知道这是可能的，但是现在我知道这是 T2 确定的。

在 [Progressive HackDay](https://www.eventbrite.com/e/progressive-hackday-tickets-40110680124) 活动中，我和另一名开发人员一起为纽约市的[新避难所联盟编写了一个 Ruby on Rails web 应用程序。我的合作伙伴是一名经验丰富的优秀开发人员，但对 Ruby 生态系统来说是新手。我是一名新的开发人员，但对 Ruby 非常熟悉。我们一起在工作中互相称赞，并在手头的任务上取得了很大进展。我相信这也有助于打破高级开发人员和初级开发人员之间的错误划分。初级和高级开发人员都在不断学习，并且都有知识和技能可以与他人分享。](http://www.newsanctuarynyc.org/)

*我从黑客马拉松中得到了什么？*

这是可能的。是的，你可以！

如前所述，当我走进一个我从未见过的项目时，我会结束之前的工作。从我以前的职业生涯中，我了解了该组织的关键工作，但从未偷看过这个代码库。在不到一个小时的时间里，我从下载存储库的本地副本开始着手解决一个问题。这是非常肯定的。这是可能的。这是可以实现的。

这一阶段最可怕的是遇到了我以前不知道的代码。这个项目背后的开发人员构建了一个漂亮优雅的应用程序。ActiveRecord 中有一些我不熟悉的关键字，如果我知道的话，这些关键字会在我自己的项目中节省我的时间！例如，`:dependent`允许您指定在删除主记录时应删除的关联记录。这是保持代码干爽并同时完成多项任务的好方法。

**结对编程最好**

[![](../Images/aadbf39562905f121245966764c0e207.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--xalXIlFN--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://cdn-images-1.medium.com/max/500/1%2AvGiHsn7rCNg72WQOAQj5MQ.gif)

在黑客马拉松之前，我没有花太多时间进行结对编程。一整天我都在与另一名开发人员合作，感觉棒极了。这让我想起了我在神学院度过的岁月，那里的教育方法是每天大部分时间两人一组学习(希伯来语为*havruta[*חברותא*】*)。和另一个人一起学习有一种动力。与你面前的另一个活生生的人一起思考的机会是无与伦比的。对我来说，同样令人振奋的经历发生在昨天的结对编程中。

我们应该这样构建方法吗？在数据库中查询该项目的最佳方法是什么？这块宝石能做我们需要它做的事情吗？我们如何为此构建一些自定义的错误处理呢？

当你独自编程时，你是在和自己就这些问题进行对话。你尝试一下，看看会发生什么，然后继续前进。有时(*或经常*)你甚至可以冒险栈溢出，看看社区里的人写了些什么。结对编程时，你可以大声问这些问题，另一个人会实时回答。你的想法相互碰撞，不断重复，并因此获得更好的结果。

如果你来自不同的技能水平，这也没多大关系。你的合作伙伴可能是一个在 Python 方面经验丰富的初级开发人员，而你可能是一个对 Java 了如指掌的高级开发人员，但是如果项目是用 Python 开发的，那么初级开发人员将能够分享很多关于如何前进的信息。与其说是你带来的不同技能水平，不如说是你愿意不断地与对方交流以写出漂亮的代码。

人们经常谈论编码可以完成的所有不可思议的事情。世界依靠科技运转，这种稳步前进的数字化进程不会很快逆转。精通技术语言意味着在这个快速发展的世界里成为一个真正的善的代理人。这就是修辞。在这次活动中，我亲身经历了这一切。

我没有为这个项目贡献任何开创性的代码。我主要致力于几个突出的问题，并与我的结对编程伙伴一起提供解决方案。然而，我们不可能忽视更大的背景。我们正在开发一个网络应用程序，使移民和志愿者之间的联系更容易，以帮助他们处理大量的法庭诉讼和文书工作。作为难民的孙子，我不能不感受到这项工作中责任的重量。

那天早上，我醒来，执行我的常规周日清晨的例行公事。我给自己冲了杯咖啡，看了看报纸。周末我会买一份纸质报纸，因为我仍然喜欢手指夹着报纸的感觉。当我浏览内容时，书页的皱折和折叠的动作。与大多数周末我阅读报纸，消化最新消息，然后继续前进不同，在过去的这个周末，我能够在这个时刻在我们国家和世界的一个至关重要的领域做出一点小小的改变。那是因为我在熨斗学校学习了技术语言，并且我有特权能够应用它。

我会参加另一个黑客马拉松吗？绝对的！谁知道下一次会有什么好事发生？
