# 重新成为开发者

> 原文:[https://dev.to/tomlous/re-becoming-a-developer](https://dev.to/tomlous/re-becoming-a-developer)

从童年开始，我就一直是一名程序员。我开始学一些 GW BASIC，但是在我高中的时候很快就转向了 C 和 C++，尽管我从来没有真正考虑过这是一个可能的未来职业。这更像是一个有趣的爱好，因为我的朋友们也这样做，这看起来并不奇怪或特别。有些人喜欢足球，有些人喜欢画画，我喜欢编程。

事实上:我一直想成为一名兽医。编程和黑客对我来说更像是一种爱好，也因为它来自并结束于玩游戏(例如，试图在基恩指挥官中获得无限弹药，在模拟城市中获得无限资金)。

所以计算机编程似乎很有趣，但我会成为一名兽医，仅此而已。直到有人向我介绍了在阿姆斯特丹 VU 计算机科学学院学习人工智能的可能性。突然间，所有的事情都井井有条了，我意识到这是一个适合我所有兴趣的研究领域。充满活力和热情的我在阿姆斯特丹开始了我的大学生活。

然而，事实是，虽然我可能已经为人工智能做好了准备，但我肯定没有为大学做好准备。我不习惯在课堂或学习上真正付出努力。因此，对于一个从未真正学会如何学习的人来说，高中的学习曲线是毕业，大学的学习曲线太陡了。我在课堂上大失所望，发现教材太枯燥，不实用。同样在我的同龄人之外，没有人理解我在做什么。

你看，这是第二个冬天过后的几天。而艾的名声并不太好。课堂上充斥着大量的理论，但实际的解决方案要么遥不可及，要么令人印象深刻。

我确实喜欢实际作业，这重新点燃了我对开发的热爱，现在是 Java、Prolog 和一些 PHP 类的形式。但即使我真的想相信不是这样，我在 VU 作为人工智能学生的生涯已经结束了。

对我的学术冒险的最后一击是当我创建了自己的网络开发公司，并开始在很酷的新网站上工作。我合作开发了许多网站，包括当地的汽车经销商、出租车公司、理发店，甚至还有一家铝厂。那是 2001 年初:没有人有网站，每个人都想要一个。

不幸的是，我的第一家公司没有成功。我不知道做生意意味着什么。我擅长做开发人员。我也没想到会成为一名会计、销售员、业主、设计师和营销人员。因为我没有做任何收购或客户保留，钱不再来了，最后我不得不离开我的公司，选择成为一名受薪的专业网络开发人员/软件工程师。

从那以后，我做过几份工作。每次学习越来越多关于 web 开发和随之而来的一切。PHP 5 出现了，MySQL 成为了一个严肃的数据库，CSS & JavaScript 框架爆炸了，我经历了宏媒体 Flash *(地狱耶)的兴衰。* Linux 可能是我开发生涯中最永恒的东西。

尽管你永远也学不完，尤其是在 IT 领域，甚至变化的速度每年都在加快，但我觉得经过 7 年多的发展，我已经达到了饱和点。实际上，我想要更多编程本身无法提供的东西。我身上的技术得以延续。我想拥有更大的影响力，分享我的愿景，制定战略，简而言之:… **我成为了一名经理。**

所以我辞了工作，当了经理。不是传统的那种，更像是以身作则的开发经理；Scrum 大师消除了障碍，并以洞察力和多年的知识和经验帮助其他开发人员。当经理很有趣。我带领一群热切的开发人员从多个部门的瀑布项目进入到基于多学科团队的 scrum 环境中，等等。

但是(并非如此)暗地里:我还是热爱编程的。这是一种永远不会消失的瘾。

然而，我确实认为这一章已经结束。没有一个头脑正常的人会放弃经理的职位。你爬上了顶峰，而不是后退。所以我根本没考虑过。我换了工作，所以我可以更专注于管理大型软件项目，而不是许多较小的网站和工具。但我过去和现在都是 IT /运营经理。

## [](#and-then-big-data-happened)然后大数据发生了。

起初，这感觉像是又一次炒作。这是 Web 2.0 的又一次重演*(好吧，好吧，这不是炒作，而是对许多正在进行的开发来说非常恼人的行话 IMHO)* 。不过，当时我对这个叫做“安卓”的小东西也有同样的想法，所以，嘿，我可能错了。

天哪，我错了！

虽然我承认“大数据”确实是一个时髦词，但它抓住了当今这场革命的本质。分布式计算、函数式语言和通过 scala 开源解决方案实现的近实时流的力量，让 2000 年大学里那些不切实际的例子突然变得更加可行。

**我对人工智能和机器学习的热爱被重新点燃**我对 Hadoop、Spark、Solr、Kafka、R、Scala 和其他工具、框架和语言的兴趣每天都在驱使我学习越来越多的东西*(向 [*Coursera*](https://www.coursera.org/) 致敬，它不仅教会了我许多技能，还教会了我学习)*。我已经吸收了所有这些知识，现在是我行动的时候了。

我很幸运能为这家伟大的公司工作，这家公司将自己定位为餐饮专业人士的合作伙伴*。但 Datlinq 也是一家数据公司，它存储、清理、匹配公共和私人数据，并以一种易于理解的方式将其提供给客户。Datlinq 在这些数据上构建的工具是它的产品，但 Datlinq 跳动的心脏毫无疑问是**数据**。*

由于没有比现在更好的时机，我们将从我们的组织内部开始这个新的精益创业公司/部门(DataLabs ),以建立一个新的和改进的数据管道，准备好交付这一大数据承诺，并持续接收和处理来自互联网上的大量数据，从而将 Datlinq 及其产品堆栈投射到未来。

所以这是我作为运营/ IT 经理的一章结束的地方，我将翻开新的一页，重新成为一名开发人员。

我会随时向你通报进展情况；-)
