啊，工作的地方，我们认识新朋友的地方，有些是我们喜欢的，有些是我们不喜欢的，还有为我们赚钱的地方。但是在工作场所，我经常看到人们忘记一件事。这就是你的快乐有时在工作场所，它会让我们因为许多原因而感到痛苦。但是你不必忍受呆在一个让你感到悲伤的地方，一个你不该呆的地方。

有几个我工作过的地方和团队让我感到痛苦，但我太害怕离开。我不想离开，因为我觉得这会让我的历史看起来很糟糕，从新雇主的角度来看也不太好。但最终结果却不同，在我最近的一次采访中，他们问我为什么离开其中一个团队，我告诉他们发生了什么，他们只是同意一个人不应该在那种环境中。

# 如果你在一个让你不舒服的地方会发生什么？

我通常会尝试做几件事。最初，如果有一个团队领导与他们交谈，提出你的问题，并从那里开始。毕竟，团队领导的工作之一就是确保团队成员快乐。现在这种情况在我身上发生过几次，每次组长总是给我和其他人整理。但如果它没有发生，这并不意味着你应该留在那里，这是你的责任，如果它让你不舒服，不高兴，你有权离开它。

# 离开你不开心的地方有不好的一面吗？

是也不是。一般来说，这取决于你离开的原因。离职的原因有很多，这取决于你的雇主或团队领导如何看待这件事。离开后进入一个新的地方，他们可能会联系你的前雇主，询问你为什么离开，大多数时候我发现他们同意你离开的原因。我总是告诉很多人“我们永远不知道未来会给我们带来什么”未来充满了惊喜，我们要随遇而安。

# 最终消息

这没什么丢人的，一点也没。在一天结束的时候，你必须做你需要做的事情来让自己快乐。如果其他人会抱怨，那好吧，那是他们的问题，不是你的问题。你的幸福永远是关键。
