# 这就是我所做的，我想继续做下去，因为我真的很享受

> [https://dev . to/highercomve/这——这就是我所做的——我想继续做——因为——事实上——我享受——31l](https://dev.to/highercomve/esto-es-lo-que-hago-y-quiero-seguir-haciendo-porque-en-verdad-lo-disfruto-31l)

六年来，我一直致力于网络/移动发展领域的教育，作为一个日常的培训者，我发现当代人比我本人更优秀，他们能够以比我们更快的速度处理信息。

但同样大量的知识和经过处理的信息并不总是能够找到当前企业行业和网络开发领域创业所需要的良好做法和方法的正确途径，因此，如果找不到一个足够好的企业来理解，这个年轻人可能会很快在工作中被录用

这就是为什么我们要启动一个由两个关键组成部分组成的项目，第一个是通过教育途径培养这些年轻人的聪明才智，使他们走上未来成为行业领导者的道路，同时利用现有行业开发人员的短缺和快速增长作为工具而且因为这是我们的工具，因为我们会利用这一需求，让我们的学生在保持大脑不断学习行业最佳技术和做法的同时，能够进行真正的项目。

我们会吗？三个月的强化训练，是另一门课程！∞天啊！这是一场每天八小时的战争训练你唯一的目标就是改变你的生活。在最好的创业空间之一，在那里你还会与行业直接接触，在你学习的同时，它会在你身边。另一方面，企业可以开发它们希望实施的新项目，同时通过网络系统跟踪和控制它们的产品，使学生能够了解跟踪和构建网络项目的过程和方法。

你资助他吗？工业界，与我们的天才学生一起开发项目的公司，通过支付这些项目的开发费用来支付这些学生的教育费用。

什么不收对学习感兴趣的学生的费用？，因为这些天才年轻人通常只能通过互联网、进入社区、youtube 以及互联网上免费提供的所有信息来学习。为了不改变他们的思维模式，他们将继续在良好做法的道路上消费免费但重点突出、有条不紊和直接的内容。

这是我们提出的继续做我们热爱的事情的价值主张，同时我们加快培育出年轻而聪明的头脑，让他们走出困境。这是进一步黑进 aweb 学校的下一步。

愿原力与他们同在！
