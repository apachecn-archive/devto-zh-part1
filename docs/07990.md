# Magento 脸书登录扩展

> 原文:[https://dev.to/fmeaddons/magento-facebook-login-extension](https://dev.to/fmeaddons/magento-facebook-login-extension)

如果你想让注册过程对你的客户来说更容易，那么 FMEAddons 的 Magento 脸书登录扩展是一个完美的工具。这个扩展启用了注册页面上的四个社交按钮，用户可以通过选择任何帐户登录。四个社交账户是 Gmail、脸书、Twitter 和雅虎，因此用户只需点击几下鼠标就可以登录。客户不需要输入任何数据，因为它会自动更新他们的帐户信息。

Magento 的脸书登录扩展的特性

*   它支持 Twitter 登录功能
*   使用电子邮件和密码手动登录
*   它启用了三个按钮，如注册、登录和忘记
*   后台办公室的社交网络设置
*   100%开源
*   免费终身升级
