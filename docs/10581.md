# 打造自己的电脑

构建自己的计算机？如果你可以在商店里买一个，你为什么要这么做呢？你会立即准备使用它，你知道它会工作。但是你知道吗？那将是简单的出路，不是吗？理解计算机是如何制造的，它是如何工作的，不同的组件有什么用途难道不是更有趣吗？拥有一台专门为你所需要的用途而制造的计算机，不多不少，难道不是更有用吗？难道你不想能够控制计算机本身的每一个方面，甚至外观、成本和额外的吗？这正是为什么你应该建立自己的电脑。

### 理解

理解计算机是如何工作和制造的既有趣又非常有用。找出不同部分如何协同工作，哪些部分必须兼容，以及不同部分组合的影响可以是一个有趣的研究项目或实验。但是除了有趣之外，它在组件损坏的情况下也非常有用。然后，您将能够更具体地找到如何解决问题，或者如何找出是哪个部件导致了问题，或者是哪个部件坏了，需要更换。确切地知道您在计算机中安装了哪些部件有助于诊断问题和寻找解决问题的方法。

### 应用

知道你可以把哪些部分组合起来实现某个目标或得到某个结果不仅是一个有趣的实验，而且它可以对你的新电脑执行某些任务的方式产生很大的影响。例如，如果你想做网页开发，就没有必要买一个又快又贵的显卡。Web 开发需要大量的硬盘、内存和 CPU 资源。大量数据将需要保存在硬盘上，也需要检索。在保存或检索数据时，也需要处理这些数据。这意味着用于这一特定任务的计算机将需要快速处理器、相当数量的 RAM 内存(至少 4 到 8GB)和快速硬盘驱动器，如 SSD(固态驱动器)或 M.2 驱动器。但是，举例来说，如果你想要一台电脑玩视频游戏，你至少需要一个高速显卡。每一帧都需要渲染到屏幕上，没有帧延迟。这意味着一个快速的图形卡，但也是一个良好的处理器和 RAM，以在后台进行计算，并确保任务得到正确执行。就游戏电脑而言，硬盘就不那么重要了。你仍然需要一个大空间的电脑来安装你想要的所有游戏。你可以在固态硬盘上安装一到两个以获得最佳性能，但你真的不会注意到大量的额外平滑度。

一个真正需要所有最佳组件组合的应用程序是视频编辑和渲染计算机。您需要一个快速显卡来渲染视频的所有帧，需要大量 RAM 来处理您将保存到快速硬盘上的所有信息，还需要一个快速处理器来管理即将播放的所有不同任务。这可能是上述三个选项中最昂贵的一个。

当然，你可能想为更多的应用构建一台计算机。也许你只是想要一台非常简单的电脑进行文本编辑。在这种情况下，您可以轻松地处理所有部件，只处理 OS(操作系统)正常运行所需的最低要求。这就是构建自己的计算机的另一个优势。你可以把它变得便宜或昂贵。你不需要根据选择来调整预算，但是你需要根据预算来调整你的选择。如果你这么说，例如，你想花 500 欧元/美元买一台电脑，但你想在电脑上毫无问题地玩视频游戏。好吧，你开始选择一个显卡来运行你玩的或者计划玩的每一个游戏。在你弄清楚哪种显卡适合你的需求后，你可以选择一个处理器，你认为你需要的内存量(最近请至少选择 8GB)，然后选择一个将这些部件完美连接在一起的主板。你可以买便宜的硬盘，但请不要在电源上太便宜。高质量的电源是你最好的朋友，会让你的电脑保持快乐。争取获得 80+青铜级或更高质量的复选标记。你甚至可以选择你想要的情况。这真的可以是任何一种方式，一个非常便宜的情况，或者一个非常花哨，但很贵的情况。只要确保你挑选的主板，以及所有其他组件，将适合你选择的情况。通常这是由主板尺寸(ATX，迷你 ATX，微型 ATX 等)来标志的。).

### 反射

如你所见，可能性是无限的。即使您决定改变对计算机用途的想法，升级也很容易。只需添加更快的处理器、更快的显卡、SSD 或 M.2 驱动器或任何其他您可能需要的东西，就可以获得您想要的机器。因为你首先自己制造了你的电脑，你会确切地知道哪些部分是兼容的，或者至少你可以通过谷歌搜索找到答案。所以，下次你想买一台新电脑，但你不想走捷径，或者有非常具体的需求或预算，考虑自己做一台电脑。它可以是很多乐趣，一个伟大的学习时刻，或者只是一个有趣的经历。
