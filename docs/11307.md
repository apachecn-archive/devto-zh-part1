# 牛逼的 Umbraco

> 原文:[https://dev.to/mozzydev/awesome-umbraco-2c98](https://dev.to/mozzydev/awesome-umbraco-2c98)

你们中的许多人可能都知道，Umbraco 的一大优点是它易于扩展，可用于各种灵活的解决方案。这在很大程度上要归功于社区和人们，他们不断地在坚实的基础上进行建设。

所有这些伟大的工作都被忽视了，这将是一种耻辱，所以 [Lee Kelleher](https://twitter.com/leekelleher) 创建了一个存储库，展示其中的一些包，并将其分组使用。

> 一个令人敬畏的 Umbraco 7 包，资源和闪亮的东西的集合。
