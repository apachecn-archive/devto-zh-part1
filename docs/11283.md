# 试用 windows 10

> 原文:[https://dev.to/funkysi1701/trying-windows-10-3k37](https://dev.to/funkysi1701/trying-windows-10-3k37)

我的笔记本电脑被东西弄得有点乱，所以这个周末我把它重置了一下。

由于我使用的是 windows 8，这比我在以前的操作系统中多次进行的格式化重装要容易得多。打开设置并选择恢复，大约 4 小时后，一切都恢复到出厂设置。

然后我安装了一些更新，然后安装了 windows 10 预览版。windows 10 的要求之一是 Windows 8 更新(8.1 之后的版本)，它需要几次重启和更新，直到我安装了它。

我已经购买了 pro pack，它给你提供了与 windows 8 pro 相同的功能，但我直到安装了 10 之后才注意到它还没有安装。我需要回滚到 windows 8 应用更新，然后重新安装 10。我真的不想这么做，所以现在我将没有 HyperV。事实上，我花了很多时间坐在那里等待安装完成。

与 Windows 8 相比，我更喜欢 Windows 10，早期的预览版在我的第二个屏幕上出现了驱动程序问题，但现在似乎已经解决了。开始菜单回来了，我更喜欢它，而不是被带到 windows 8 的额外选项屏幕。

与 windows 10 版本相比，我更喜欢 windows 8 的电子邮件客户端，因为我的工作邮件无法使用它。我认为这是由于我的 exchange 服务器的配置，也许以后的预览版可以让我的电子邮件正常工作，否则我将不得不正确修复工作中的电子邮件。

现在所有的应用程序都可以在 windows 中打开，这要好得多。Cortana 非常类似于我手机上的 Google Home，我可以在那里聊天和问问题。但是除了测试之外，我从来没有使用过这种特性。

新的网络浏览器有一些不错的功能，比如可以在网页上乱涂乱画，但没什么特别的，我想我会把我的默认浏览器从 Chrome 换下来。看看它在浏览器统计中的表现会很有趣。

我正在安装 Visual Studio 2015 RC，但我认为这将是一个单独的帖子。
