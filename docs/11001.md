# 创建一个动态的转到顶部按钮

> 原文:[https://dev . to/mariordev/create-a-dynamic-go-to-top-button-215 a](https://dev.to/mariordev/create-a-dynamic-go-to-top-button-215a)

还记得过去在页面内容之间散布的“返回页首”链接吗？是的，那些…这里有一个快速的方法来创建一个现代的替代物，一个浮动的动态“转到顶部”按钮。该按钮仅在用户向下滚动页面后出现，一旦用户向上滚动，该按钮就会消失。我发现转到顶部按钮非常有用，尤其是在移动设备上，内容被压缩在一个较小的屏幕上，这使得用户向上滚动到顶部变得更高、更繁琐。
