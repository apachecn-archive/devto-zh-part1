# D3 的 SVG 基础知识

当你开始学习或使用 d3 时，你会遇到 SVG。这是大多数人在用 d3 创建可视化时使用的。这是更容易的选择，也是更有据可查的选择。所以你需要了解 SVG 的几个基本概念，才能更好地利用 d3，学得更快。

在这篇短文中，我将向您展示 SVG 中值得了解的三个元素。

## 组

您可能遇到过这样一种情况，人们在他们的 svg 中使用了一个`g`元素。

`g`代表组，是构建 svg 的一种简单方法。特别是如果您有任何类型的交互或者想要更新图表，将 svg 元素分组会更容易。您可以轻松地选择它们，并只向组中添加一个类。当你有组的时候，设计它们也很容易。

## 路径元素

d3 最常用的元素是 path 元素。这也是最灵活的元素。你可以用它创造几乎任何形状。

我们可以用路径的`d`属性来实现。好的一面是，你不需要知道它是如何工作的。您只需要知道它的存在，并且您将使用它和 d3 来创建定制元素。对于映射，我们通常只使用路径元素。

## 翻译

这是很多人纠结的地方。老实说，我并没有花太多时间去理解它，只是像其他人一样使用它。

但其实挺简单的。这里有一个简短的解释:

用`transform="translate(20,20)"`你将元素向右移动 20 像素，向下移动 20 像素。为什么？因为在浏览器中坐标系从元素的左上角开始。

为什么有用？

因为现在可以移动没有 x，y 属性的元素了。比如一个团体。
