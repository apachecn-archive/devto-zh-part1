# 🛠为自己建造了一些东西

> 原文：<https://dev.to/shaunchurch/build-something-for-yourself-fc3>

*最初发布于 2014 年*

这种为自己制造东西的想法有些道理。我从来没有像现在这样有动力去做一个项目，也从来没有像现在这样对一个[产品](https://play.google.com/store/apps/details?id=com.shaunchurch.songwriter.app)的用户体验有更清晰的认识。

我确切地知道问题是什么，因为我自己经历过。这凸显了我在没有真正理解的情况下建造东西是多么的盲目。让最终用户参与原型设计和测试想法的重要性无与伦比。如果你*不*，就不容易模仿真正需要使用的东西。

我喜欢建造东西。我享受挑战，享受解决问题和创造新事物带来的满足感。与一个才华横溢的团队一起开发产品是每个人贡献的总和，这种经历会带来很多回报。

但是建造一些你自己需要的对别人也有用的东西——即使是很小的规模——是很特别的事情。

如果我们不使用我们建造的东西，我们怎么知道它们是否能发挥它们应有的作用呢？我们得问一个知道的人。

所以给自己造点什么吧。

做批评家，做创造者。
