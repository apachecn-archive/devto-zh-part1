# 为您的用户提供智能服务

> 原文:[https://dev . to/devtouser 432/be-intelligent-for-your-users-6d 0](https://dev.to/devtouser432/be-intelligent-for-your-users-6d0)

昨天，我和一个朋友谈论了我对苹果 AirPods 的体验，以及尽管价格昂贵，它们是我用过的最神奇的技术之一。你真的不会指望一副耳机能以这种方式取悦你。

更多的是一种感觉，无法完美描述。但这是迄今为止我最未来派的自我。它改善了你的一天，让你想象不到的事情变得无比简单。但令人惊讶的是，神奇之处不仅仅在于硬件，还在于内容本身。感觉就像原始信息通过以太无线进入你的耳朵，通过某种无缝的魔法。我将在公交车上或在早晨的通勤路上行走，一股知识流(通过播客)将直接进入我的耳朵。感觉不太像是涉及到技术。就像一股风吹进来一些高保真的智慧。所以才神奇。

但是，还有一些小事。如果你在听什么，看到旁边有人想说什么，你会很自然地把其中一个头从耳朵上拿开，说“对不起？”。嗯，我第一次这么做的时候，音乐自己立刻就停了。当我把盒子放回耳朵时，音乐又开始了。

太神奇了。我向我的朋友反映，本质上，这些东西没有任何按钮，所以它们被迫代表我们做决定，并假设我们的意图。通过移除按钮，它们被迫为我们变得智能。iPhone X 是这一点的绝佳体现。这是迄今为止我使用过的最神奇的 iPhone 设备，它在很大程度上与没有 home 键有关。

我真的很佩服苹果能把自己困在一个角落里，拿出看似神奇的解决方案。我不知道还有哪家这样规模的公司能够不断取得这样的成就。苹果从本质上消除了环境中的变量——请注意，这些变量是数百万人所依赖的——并问，现在会是什么样子？而解决方案往往是惊人的。

这里最重要的一课是，如果你被困在一个角落里，你将如何被迫创新？你能为你的产品创造什么样的约束来迫使你为你的用户变得聪明？关键似乎是将决策权从用户手中拿走，并将其转移到智能假设上。我的朋友指出，在聪明的假设和令人讨厌的，甚至是侵犯性的强迫之间很容易越界。的确是这样，但是我想大多数产品都充满了决策点，这条线看不到了。

真的，看起来苹果产品如此频繁地创造的那种*神奇*的感觉是基于他们对我想如何使用他们的产品的正确假设，这让我节省了自己找出答案的时间。
