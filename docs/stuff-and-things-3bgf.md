所以我们做了很多。网络发展最近在老商店。我现在通过了 MS SQL 2008 开发认证，接下来是。网！:)

我一直在努力学习更多关于 api 和 ajaxy 类型的东西——尽管它已经存在了很长时间，但我在工作和学校接触到的东西让我没有太多机会学习新的旧的东西。

我目前正在开发一个 API，它将有助于 MMORPG 最终幻想 14 中未受破坏的节点跟踪。到目前为止，我已经将游戏中的时钟时间计算融入到 API 中。

学习日程的下一步是弄清楚 api 的认证和授权。我注意到使用。net 有新的内置函数来使它更容易，但对我来说仍然是一个学习过程。一旦我在 API 上设置了授权，我就必须弄清楚如何从一个单独的子域上托管的网站进行身份验证。

说到域和子域，我已经从 GoDaddy.com 购买了一个域，目的是尝试获得一个 SSL 证书，以便加密数据，供 web api 使用。
