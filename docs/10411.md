# 我的介绍/我的编程之旅！

> 原文:[https://dev.to/gabedunn/introduction-to-me-我的编程-旅程-](https://dev.to/gabedunn/introduction-to-me--my-programming-journey-)

嘿，伙计们。我叫加布。我已经决定，我将为编程社区做更多的贡献和互动，写博客将是我将要做的方式之一，我将开始努力记住每月至少发布一次，也许会从这一点向前迈进。我也将尝试为开源项目做贡献，看看效果如何。好了，现在稍微介绍一下我...

我第一次真正对编程感兴趣(哪怕是一点点)是在我 13 岁左右的时候，当时我和几个朋友一起玩《我的世界》，我决定要托管自己的服务器，使用定制插件。在做了一些研究后，我发现要获得我们自己的插件，我们必须付钱给一个开发者来为我们定制它们，我们中没有人能负担得起，所以我们研究了如何制作我们自己的插件。原来有人需要学习 Java，然后是 bukkit API，我是计算机方面的佼佼者，所以我研究了一下。我，没有任何编程背景，直接研究如何开发插件。当然，我对这些几乎一无所知，因为任何事情都需要至少 Java 的基础知识，所以在几个星期什么也没学到之后，我放弃了，但仍然对编程保持着相当大的兴趣。

真正让我开始的是一个学校项目。在我的学校，八年级的每个班级都有机会参与他们感兴趣的项目。出于我不打算说的原因，我们班一直等到九年级才真正开始这样做。我意识到，这是学习如何编程和组装电脑的绝佳机会。一开始是看 Udemy 上一些关于 Java 的免费教程。我开始还不错，每周看几部，但几周后，我就觉得很无聊了。我一点也不喜欢它，这是这个项目的主要目标之一。老师一点也不投入，我也不喜欢他的教学风格。我需要别的东西。这时候我发现了 Codecademy。我开始学习他们的 Python 课程，这比我之前尝试学习的 Udemy 课程有趣多了。我完成了这个项目，很快就必须找到一个能展示我所学的最终项目。我做了一个简单的程序，可以与 Twitter API 交互，发布、删除和阅读最近的推文。我对此感到非常高兴，并被吸引住了。

在我的项目成功后，我继续 Codecademy 推荐路径上的下一门课程:web 开发(HTML、CSS、JS、jQuery、PHP 和 Bootstrap)。我吹通了这一点，非常喜欢它。我做了自己的网站，也想给别人做网站。我的一个朋友真的让我做了他的网站，我对结果很满意。

接下来，我想学习 Ruby，主要是为了它的 Rails web 框架。我参加并完成了 Codecademy 的课程，比我想象的更喜欢这门语言，并决定在这方面工作一段时间。那太有趣了。

在玩了一段时间 Ruby 之后，我意识到你真的不能用它来做一个完整的应用程序。我尝试了鞋子框架，喜欢它的简单，但我希望有更多的可定制性和功能。

我开始对 C 和 C++感兴趣。我了解了一些关于这些语言的有用的东西，但是还在寻找其他的东西。我记得很多人告诉我，C#是现代应用程序开发的“主流”,所以我在 Udemy 上找到了一门很好的课程，并从那里开始学习。从我开始学习这门课程到现在已经有 3 天了，我已经完成了差不多 110 节课，并且做了一个比以前更好的版本。我仍然计划不断地改进它，并将继续向你们更新我在编程领域的进展！

如果有人在这里坚持到了最后，我真的很感激。我知道这是一个很长的帖子，希望通过把它放在那里，人们可以看到我是如何进入编程的，以及为什么我如此喜欢它。我希望记得每月至少在这里发布一次，所以你应该会很快收到我的消息！感谢您的阅读，祝您度过愉快的一天。

~加布(redxtech)。
