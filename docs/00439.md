# [](#tracing-my-technology-roots)追溯我的技术根源

欢迎来到我的旅程。在接下来的几段中，我将努力找出使我成为技术爱好者、计算机维修技师、网络工程师和软件开发员的精确时刻。基本上，我生命中成就今天的我的时刻。

# [](#chapter-1%E2%80%8A%E2%80%8Aatari-2600)第 1 章—雅达利 2600

我的第一个记忆要追溯到雅达利，具体来说是动视的一款游戏，叫做“拳击”。那是大约 80 年代末，我大约 7 岁，像其他任何一天一样，我的母亲决定去看望她的表妹，我的心思快要被吹离宇宙了。突然，她的丈夫决定向我们展示这个新的“玩具”。我可以用一个控制器在屏幕上移动这个神奇的角色，不仅如此，我还可以揍我的朋友！

[![International Soccer (Atari 2600)](../Images/635bceee0ead0dc51bc8f0c74dc7ef78.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--0WaYeZhn--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2AyoPnoHDn57RYRpMsd4zuYw.png)

此后不久，我父亲决定买一台雅达利游戏机。我记得有两个游戏一直在玩:一个丛林人必须跳到短吻鳄的身上，并且可以钻到地下试图躲避蝎子。游戏的名字？陷阱！

[https://www.youtube.com/embed/QMd-976C24o](https://www.youtube.com/embed/QMd-976C24o)

第二场比赛是足球比赛。我喜欢足球，现在我可以在屏幕上控制足球运动员了。我还是个孩子的时候，在我的客厅里探索一种新型技术。我想知道这对我开始以不同的方式看待事物有什么影响，不可能的事情是如何成为可能的，毕竟，这是不可思议的。

# [](#chapter-2%E2%80%8A%E2%80%8Apc)第二章——个人电脑

[https://www.youtube.com/embed/5mFRGnug-j8](https://www.youtube.com/embed/5mFRGnug-j8)

游戏似乎对我的余生产生了重大影响，因为我对科技的第二个记忆(不包括计算器、录像机等)也与游戏有关。让我们快进到 1993 年左右。这时，我的表弟已经决定买一台个人电脑。我们过去经常去他们家，圣诞节，新年，现在我有理由去得更多了。“妈妈，我们去他们家吧！”我会说。我能记得的就是玩这个叫“街棍”的游戏。别让我开始玩这个游戏，它真的改变了我的生活！这不仅仅是一个赛车游戏:你实际上可以成为一名机械师，修理你的车，卖掉它，购买新零件，并让它跑向顶峰！我喜欢这些细节，你实际上必须如何去加油，你如何给你的车上漆，在自动或手动变速器之间选择，等等。这绝对是一个超越时代的游戏。你不仅可以被动地享受它，还可以让你边玩边创作。我仍然记得我的“银弹”，一辆 1955 年的双门雪佛兰 Bel-Air。

我很幸运，我的母亲意识到我喜欢这些新机器的一些东西，她没有给我买一台，而是给我报名参加了意大利协会的计算机课程(这是在阿根廷，我知道我忘了提到它，我来自南美，阿根廷，布宜诺斯艾利斯，圣马丁)。现在，我不知道她是如何做到这一点的，因为我与意大利大使馆没有任何“联系”，我的大多数祖先来自德国和西班牙。此外，这不是其他人都去的“高级”地方，“阿根廷计算机”(我不记得名字了！)，这是我最想去的地方，但现在回想起来，它要好得多。这个意大利的地方有更多的自由，它不是那么“商业”，有更多的关怀，尽管这也有助于我是一个 11 岁的孩子，与 20 至 50 岁的人一起上课。此外，正如我生活中真正关心的其他事情一样，我非常兴奋和高兴地了解到这些新机器。

[![Italian Society, San Martin, Buenos Aires](../Images/df671fe6de1e268bb5bb217941e81074.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--W_Uidr2r--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2AJ0HwcNF7gQNLylquFIWfww.png)

让我告诉你一个秘密:如果你对你的新冒险充满热情，人们会尽力帮助你；如果你真的对此感到兴奋，它会表现出来，人们也能看出来。尽你最大的努力，你会得到回报的。

现在，这是它变得令人兴奋的时候，也是我不断发生的事情(甚至今天)。几乎每次我开始一个新的职业，一个新的认证，一个新的课程，我都不会从头开始，我要么在末尾开始，要么在中间开始，但从来不会在开头，你猜怎么着？它对我有用。老实说，我不是故意的，但它就是发生了，让我解释一下。回到 1993 年，当我开始我的第一个计算机认证时，我晚了一个学期，而其他人都已经完成了 DOS 3.1 课程。我从微软的 word 开始，我发现了这一点，因为有几个 DOS 命令你可以从 Word(这是在 Windows 之前)中运行，比如“Del”，来删除一个文件。

如果您在“文件”下查找，您可以在“退出”之前看到“Dos 命令”:

# [](#chapter-3-internet)第三章——互联网

“你的意思是我可以免费看报纸？”

让我们快进到 1998 年，我的国家(阿根廷)已经走下坡路了。我父亲决定移民到美国。当时，我只有 15 岁，我不知道发生了什么，我只是很高兴我们搬到了美国。这意味着我有几个月没有电脑。

我们到达美国大约 3 个月后，我的父母交了一个朋友，里卡多·佩雷拉。里卡多有一台电脑，他告诉我们他可以免费看报纸。最初，我很震惊。

— *“你说可以免费看报纸是什么意思？”*

——*“是的！有 t 这个东西，叫互联网"*

我的天啊。这改变了我的生活，现在我可以免费阅读新闻，查阅阴谋论…和人们聊天！

[![AOL](../Images/27e68700936250534d4471250242ec35.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--gMx80N_O--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2Acht4S_E7Ftlp9MiQvohACA.jpeg)

[![mIRC](../Images/70e24edbfafc6e007b7271cc4c547c64.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--la7dnNYD--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2AEWSD_L1a9NqEeOKR62eI0Q.png)

美国在线、mIRC、ICQ、MSN Messenger 等等。太有趣了！

这一切都始于美国在线，尤其是在聊天室。我和一个熟悉电脑的家伙成了朋友，他开始教我一些技巧，比如如何进入别人的账户。

您知道吗，只需同时使用旧版和新版软件，就可以登录两个不同的 AOL 账户。

这是学习的时间，我学到了很多！我开始使用一款名为 Sub-7 的软件来访问别人的电脑、文件、打印机或相机。当时，这一切都很有趣，我没有想到“侵犯隐私”，我在想“哇，我真的可以这样做吗？我想知道我还能做什么”。这都是在 1999-2000 年左右。我想当我能够窃取信用卡信息时，我达到了我的目标，就在那时，我意识到有些事情不对劲，我并不喜欢这样做。我很幸运，从来没有真正使用过我非法获得的任何信息，现在我想起来，就此打住的决定是迄今为止我最美好的时刻之一。

# [](#chapter-4-turning-professional)第 4 章——转向专业

到目前为止，这一切都很有趣，我通过为别人开发网站赚了一些钱，但不多。然后，我开始修理自己的电脑，更换自己的硬件。就在那时，我得到了去一家电脑公司面试的机会。我只是重复了一个朋友告诉我的关键词，就诚实地得到了这份工作。

*就说:IP，TCP/IP，ping，IP 打印等*

一旦我得到了这份工作，我又开始学习了。我大概可以说，到那年年底，我能够解决任何计算机问题:硬件、软件、网络，任何东西。你说出它的名字，我能修理它。

我一直喜欢创造东西，但现在我意识到我 8 年的职业生涯是修理东西，而不是创造。

就在那时，我成了一名软件开发人员
