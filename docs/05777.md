事实是，它不是。这可能是一件重要的事情。

因为他们哪儿也不去。

这些 1 和 0 存储在一个存储介质上，在接下来的对话中，这个存储介质就是一块白板。不同的程序在白板上书写，当他们需要写更多的时候，他们会擦除一部分。

有控制；你不想重写别人写的东西。我想起了学生时代教室黑板一角写着的**法国俱乐部周二下午 6 点在 Rm 220 -SAVE-** 聚会。剩下的等下课了再做，下一节课的下一个老师可以擦掉再往前走。

就像白板笔记在课程结束时不会消失一样，1 和 0 也不会因为被删除而消失。他们在白板上，没有上下文，直到更多的文字覆盖顶部。文件或程序同样位于存储设备上，直到被重写。

这意味着有工具可以在你点击删除后恢复删除的文件和程序，如果你在沮丧中销毁了你的论文笔记并需要取回它们，这可能是好的，但如果你放弃了一台旧电脑，不想让其他人查看你的照片，或者更糟，这可能是坏的。

还有白板擦，它用 1 和 0 来写存储中的每一位，以确保没有留下任何东西，这将妨碍上述工具，但从理论上讲，不是一个非常坚定的人可以使用电子显微镜。因此，在某些计算领域，他们不认为硬盘可以在没有物理破坏的情况下安全处理，用[粉碎机](https://www.youtube.com/watch?v=kCtwsD47nK8)或[铝热剂](https://www.youtube.com/watch?v=k-ckechIqW0)。

所以，这不仅不是一个愚蠢的问题，它还引出了用铝热剂摧毁东西的视频。那有多好？
