## [](#concept)概念

我认为今天的 web 应用程序开发需要 GUI 编辑器本身，并具有有效的语义。

不是`Home Page Builder`。

是的，它非常粗糙，但很有效。

这个工具生成的 css 是这样的。

```
.gridContainer {
  width: 100%;
  height: 100%;
  display: grid;
  grid-template-columns: 120px 4fr 1fr;
  grid-template-rows: 60px 1fr 40px;
  grid-template-areas: 'header header header' 'left content right' 'footer footer footer';
}
.headerArea {
  grid-area: header;
}
.leftArea {
  grid-area: left;
}
.contentArea {
  grid-area: content;
}
.rightArea {
  grid-area: right;
}
.footerArea {
  grid-area: footer;
} 
```
