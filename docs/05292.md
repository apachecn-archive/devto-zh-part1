# 2017 年你应该知道的 JavaScript 框架

JavaScript 框架因其效率、安全性和成本等优势而在开发人员中广受欢迎。每个开发平台的框架种类繁多。不可能全部描述出来，也没有必要。让我们缩小选择范围。由于几乎每个公司都倾向于拥有一个网站或至少一个登录页面，所以回顾一下最流行的 JavaScript 框架会很有好处。

### 9 大 JavaScript 框架列表

[**Angular.js**](https://www.google.es/url?sa=t&rct=j&q=&esrc=s&source=web&cd=1&cad=rja&uact=8&ved=0ahUKEwiOqIrTne_VAhVDvBQKHTdgB6UQFggmMAA&url=https%3A%2F%2Fangularjs.org%2F&usg=AFQjCNFqlo950zrbVuOv_hTZHgDD2fck9A)

这一框架于 2009 年实现。只要它是用于构建单页面应用程序的最受欢迎和最常用的 JavaScript 框架之一，您就很有可能听说过或使用过它。它是一个开源解决方案，由 Google 提供支持。后者是一个令人信服的论点，有利于 Angular 成为一个 web 应用程序的前端解决方案。Angular JS 是混合移动应用程序开发的首选。这个框架的特点是具有覆盖图、侧边栏、开关和绝对定位导航条等组件，将动态功能融入到应用程序中。

[**Backbone.js**](http://backbonejs.org/)

Backbone 是 JavaScript 开发人员最流行的 web 开发框架之一，部分原因有两个——易于理解可用性模块，以及非常简单的学习曲线。Backbone 使用最传统的 web 开发库创建单页面应用程序。它的构建思想是，所有服务器端功能都应该通过一个 API，从而最大限度地减少需要编写的代码量，以实现更复杂的 web 框架的相同复杂功能。

[**react . js**T3】](https://facebook.github.io/react/)

React.js 由脸书开发者团队创建，于 2013 年问世。与其他 JavaScript 框架相比，它是相当新的，因此很有把握赢得它的位置。如果你喜欢脸书和 Instagram 的用户界面，那么你可能也会喜欢 React.js。这个框架是这两个社交巨头的前端场景的背后。其他公司也从使用它中受益:Atlassian、BBC、Coursera、Dropbox、Flipboard、网飞、Paypal、WhatsApp。

[**ember . js**T3】](https://www.emberjs.com/)

这个开源的 JavaScript 框架于 2011 年推出，被誉为 2015 年 web 应用开发的最佳 JavaScript 框架。其最接近的竞争对手是角和反应。Ember 可能是一个很好的解决方案，适合一家初创公司或一家希望构建复杂 web 应用的企业。它有一个巨大的在线社区，定期更新和 JavaScript 的广泛应用，旨在提供一个引人入胜的用户体验。该框架被广泛接受用于创建功能丰富的 web 应用程序。使用该框架设计的最常见的网站是 LinkedIn、Kickstarter、Blue 围裙和许多其他网站。

[**Aurelia.js**](http://aurelia.io/)

作为一个自称的 web 开发框架，Aurelia 使网站开发的过程成为一个创造性的过程。Aurelia 采用最新的 JS 标准 ES6 设计，能够在所有现代浏览器上运行。Aurelia 设计在一个类似模块的框架上，有各种大小不同的库，可以组合使用，也可以单独使用，这取决于所设计的应用程序的种类。

[**Meteor.js**](https://www.meteor.com/)

凭借后端、前端开发和数据库管理的各种特性，Meteor 成为最受欢迎的 JavaScript 框架。自 2012 年问世以来，该框架在性能、UI 设计等方面都有所发展。MeteorJS 涵盖了软件开发周期的所有阶段，并负责诸如链接、文件连接等令人沮丧的过程。它目前用于马自达、宜家、霍尼韦尔等商业公司的实时应用程序开发。

[**vista . js**](https://vuejs.org/)

2016 年推出的 Vue 有一个更容易的学习曲线。该框架提供双向数据绑定、服务器端渲染、Vue-CLI 和可选的 JSX 支持。它是迄今为止发展最快的框架之一，是快速跨平台解决方案的首选。

[**高分子**](https://www.polymer-project.org/)

Polymer 是 Google 的另一个 JavaScript 框架。凭借其材料 web 设计和现代观点，Polymer 是一个令人瞠目结舌的 JavaScript 框架，具有创建和重用 web 组件的能力。它具有聚合物应用程序工具箱，允许您创建和交付尖端的渐进式网络应用程序。该框架是完全可互操作的，为速度而构建，并在所有现代浏览器上提供支持。

**[插座](https://socket.io/)**

Socket 在实时开发者社区获得了很大的发展势头。使用 Socket，您可以在客户端和服务器之间享受全功能的实时通信。Socket 分为两个不同的部分——第一部分是从浏览器运行的客户端库，而服务器库构建在 Node.js 之上。两个库共享一个非常相似的 API，Socket 也是事件驱动的；很像 Node.js。借助 Socket，您可以实现实时二进制流、即时消息平台、交互式文档协作、应用和项目的实时统计(分析)等等。

请记住，选择框架的过程不仅取决于它的功能，还取决于它如何在您自己的项目中使用。
