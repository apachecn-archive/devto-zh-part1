# MaxCDN PHP 0.5.0 发布

> 原文：<https://dev.to/jaysonjphillips/maxcdn-php-0-5-0-released-1lio>

现在升级到版本 0.5.1，PHP API 现在实现了 MaxCDN 的 XML-RPC 接口的所有方法。唯一剩下的事情就是完成所有的单元测试，创建一个自动化的构建系统(为自己)，加入一个轻量级的教程，开始为各种框架(WordPress、Drupal、CodeIgniter、CakePHP 等)做扩展。

如果你想现在就获得它，请前往 [GitHub](https://github.com/jaysonjphillips/maxcdn-php-api/releases/tag/v0.5.1) 查看最新的提交。你可以在你的项目中随意使用它，因为它是麻省理工学院许可的。黑客攻击，报告 GitHub 项目的任何错误(是的，我使用 GitHub Issues 作为它的问题跟踪器),让我知道你对此的任何其他评论。
