# 让 Visual Studio 2012 看起来更好

> 原文:[https://dev . to/adamkdean/making-visual-studio-2012-look-nice-4e 78](https://dev.to/adamkdean/making-visual-studio-2012-look-nice-4e78)

因此，根据我 2013 年的新年决心，我要伸出手指，成为我应该成为的开发人员，我已经检查了一些新的。现在到处都是网上的东西。我喜欢它的大部分，一些我已经知道并使用的 C# 3.0 和 4.0 的特性，但是其中一些真的很棒。

Visual Studio 2012(Visual Studio 11)并不出色。

[![Visual Studio 11 is ugly.](../Images/1773503b5053dd0a97a0edfb0b3de465.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--2sklh4xT--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://i.imgur.com/0zyvP5c.jpg)

我从 10 岁开始使用 VS，我有 Visual Basic 6.0。然后我去了 Visual Studio。NET 引入了这个新世界。NET 应用程序。VB6 搬到了 VB.NET，人们一直在谈论一种叫做 C#的东西。网，还有 ASP.NET，所有这些点网。一开始我有些怀疑，但很快就接受了，到 2006 年，我开始学习 C#并使用 Visual Studio 2005。在某个时候，我来到了 2008 年，这看起来就应该是这样。Visual Studio 的一贯风格。

但是后来我尝试了 VS2010。我的上帝，黑暗的颜色，变化，你的眼睛只关注编辑，其余的都是黑暗的。一开始，很可怕。但是慢慢地你会觉得其实也没那么糟糕，然后你喜欢它，然后你爱它。Visual Studio 2010 看起来棒极了。

然后 Visual Studio 2012 上市了。击中，打滑，错过了目标。

我还没有发现我喜欢它的一点。我不喜欢它的样子。我不喜欢这个图标。我不喜欢这些帽子。我不喜欢这些颜色。它丑陋，块状，在进化中倒退了一千步。

但是有一个解决办法，我对它的解决方式很满意。

**1。你要做的第一件事就是安装 Visual Studio 2012 颜色主题编辑器，在这里找到:**
[http://Visual Studio gallery . msdn . Microsoft . com/366 ad 100-0003-4c9a-81 A8-337d 4 e 7 ace 05](http://visualstudiogallery.msdn.microsoft.com/366ad100-0003-4c9a-81a8-337d4e7ace05)

安装该扩展并重新启动 visual studio。它会提示你选择一个主题，看看它们，但我选择了蓝色 VS2010 的主题。

更新:我已经删除了 NiceVS 扩展的链接，因为它只会导致崩溃，还有图标，嗯，我想我已经习惯了紫色的那个。

恭喜你，你现在应该有一个不会让你眼睛出血的 Visual Studio 2012 安装了...太多了。
