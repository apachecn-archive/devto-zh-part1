# 我现在是一名开发商

> 原文:[https://dev.to/funkysi1701/im-a-developer-now-4jf0](https://dev.to/funkysi1701/im-a-developer-now-4jf0)

有一段时间，我一直站在把自己描述成一名开发者的门口，但是我想我终于走过了那扇门。

这完全是一种心态，与拥有多少知识无关，只是说“是的，我现在是一名开发人员。”

推动我走进那扇门的一件事，是一份申请表，在申请表上我勾选了软件开发人员，而不是 IT 经理。

另一件事是我最近做的一些任务。我们的一个数据库运行着一个 Web 服务，为一个网站提供一些数据，我们最近发现，由于这个网站的变化，这需要改变。我们的承包商报价给我们做这项工作，我想是 8 小时。我的老板说你为什么不试着去做，这样可以节省我们的钱。

所以我试了一下，心想这至少要花我几天时间，因为我必须弄清楚代码是如何工作的，最终却无法理解它。但是没有，我在比承包商报价更短的时间内完成了变更。走吧我。我要说的是，这些变化还没有得到充分的测试，所以我可能会在这上面花更多的时间。
