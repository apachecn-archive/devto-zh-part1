如果你错过了其中的任何一个，这里有一些过去一周最值得注意的链接。

[RethinkDB 正在关闭](https://rethinkdb.com/blog/rethinkdb-shutdown/)关闭的是公司，而不是开源项目。RethinkDB 的结构类似于 MongoDB，是一个由面向服务的业务模型支持的开源数据库。本周，我们发现该公司正式认为他们的努力是不可持续的。RethinkDB 团队将加入 [Stripe](https://stripe.com/) ，但是公司将会关闭。在评估以这种方式构建的未来项目时，这是一个有趣的参考点。

[雅虎秘密扫描客户电子邮件获取美国情报](http://news.trust.org/item/20161004170601-99f8c)雅虎秘密开发了一个软件，在所有客户的电子邮件中搜索美国情报官员提供的特定信息。

在与胆管癌进行了长期斗争后，这位开源传奇人物本周选择了安乐死。欣金斯为[写了一份死亡协议](http://hintjens.com/blog:115)，并在[的播客中多次亮相](https://dev.to/pod?q=peter+hintjens)。在他的整个职业生涯中，他的思想和工作都令人着迷。

[JavaScript 的现状](http://stateofjs.com/)JavaScript 社区成员的广泛问卷调查结果。

[2016 年学习 JavaScript 的感觉如何](https://hackernoon.com/how-it-feels-to-learn-javascript-in-2016-d3a717dd577f#.7c4hijjdx)本周，一篇关于 JavaScript 现状的幽默社论引发了争论和回应，比如[2016 年编写 JavaScript 的实际感觉如何](https://medium.com/@kitze/how-it-actually-feels-to-write-javascript-in-2016-46b5dda17bb5#.n6km7etue)

只有六个字符的 JavaScript 之旅对 Javascript 和计算基础的有趣探索。
