一个由这些人组成的小组和我们所有人提出的一些问题；这就是调用编码，所以我们做了编码。观众向小组提出的 4 个有趣问题。

问题 1:你是如何把学校教育和学习如何编码结合起来的？

> 不要让你的成绩受到影响，利用现有的教程和这种技术聚会。

在所有这些教程之前，meet up、库、框架和稳定的互联网都不容易访问。至少在网吧通宵浏览是一种趋势，那时我们开发者有三个名字:当他们想尊重你的时候:软件人，当你想让他们付钱给你的时候:网页设计师，最后他们不理解你做什么，他们觉得你在无缘无故地赚钱:雅虎男孩。

问题 2:如何从学习代码迁移到专家？

> 作为一个开发者，你总是在学习，试着发布一些东西给人们使用，等待苛刻的反馈，编程并不容易

问题 3:只能用 php 搭建 web 端吗？现在这个问题出现在这里，因为它暴露了一些黑暗的秘密，一些人正在密谋暗杀曾经热爱的 PHP。

> 这取决于你想用你的 web 应用程序做什么，虽然 php 只限于服务器端，你可能在某个时候需要客户端的 Javascript，但是你可以用 php 建立一个网站。

问题 4:不先学习香草 javascript 或者 jQuery 可以学习 VueJs 吗？

> 对于任何你想学的语言，你最好先了解基础知识，然后再学习它可能附带的库或框架。
