如果你错过了其中的任何一个，这里有一些过去一周最值得注意的链接。

[大规模 DDoS 攻击影响 Twitter、亚马逊、其他](https://krebsonsecurity.com/2016/10/hacked-cameras-dvrs-powered-todays-massive-internet-outage/)对 Dyn DNS 服务的攻击是通过感染恶意软件的物联网设备(如数码相机)进行的。这次袭击并不意外。事实上，Cloudflare 在 10 月初的一篇博客文章中描述了这种攻击。这是弹幕上的 [Dyn postmortum](http://dyn.com/blog/dyn-statement-on-10212016-ddos-attack/) 。

[2017 Rust 路线图](https://github.com/aturon/rfcs/blob/roadmap-2017/text/0000-roadmap-2017.md)Rust 项目一年后的愿景。重点是提高 Rust 的生产率，同时保持其对快速、可靠代码的重视。

[为什么我们选择 Vue.js](https://about.gitlab.com/2016/10/20/why-we-chose-vue/) 一个 GitLab 喜欢视图库的简单概述 [Vue.js](https://vuejs.org/) 。这篇文章并不那么有趣，但它是开源 web 应用程序中一个重要参与者的认可。Vue 很受欢迎，当然值得在这个领域考虑。

[现在生产的所有特斯拉汽车都有完全的自动驾驶硬件](https://www.tesla.com/blog/all-tesla-cars-being-produced-now-have-full-self-driving-hardware)这代表着向软件驱动方向迈出的具体一步。了解内情的人对特斯拉的选择做了很多[自行车运动](https://news.ycombinator.com/item?id=12752750)，但很难否认这对交通的未来是一件大事。

迪士尼已经建立了一个开源项目这是一个有趣的举动，因为该公司是全球版权法收紧的最具代表性的公司，通常被视为滥用法律。这似乎是朝着正确方向迈出的一步，但人们很容易对此持怀疑态度。

[成为一名开发人员的曲折道路](https://medium.com/@mogest/the-winding-path-of-being-a-developer-196b4074f236#.e8bhl9185)这是对大多数开发人员在职业生涯成长过程中所面临的困难的精彩总结。

[谷歌云存储的新存储类别](https://cloudplatform.googleblog.com/2016/10/introducing-Coldline-and-a-unified-platform-for-data-storage.html)谷歌云存储现在比亚马逊 S3 更便宜、更快、更可用。

[网络如何变得不可读](https://backchannel.com/how-the-web-became-unreadable-a781ddc711b6#.53hcsosef)为了可读性和可用性，网页设计的发展已经背离了印刷的印刷原则。这是一篇写得很好的恳求设计师抛弃一些现代时尚的文章。
