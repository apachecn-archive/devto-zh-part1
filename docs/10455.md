# 快速提示:暂时禁用 ReSharper 的快捷方式

> 原文:[https://dev . to/seankilleen/quick-tip-a-shortcut-to-temporary-disable-resharper-m94](https://dev.to/seankilleen/quick-tip-a-shortcut-to-temporarily-disable-resharper-m94)

## [](#problem)问题

我喜欢 ReSharper，但是它对性能有一些影响，有时我需要它离开几分钟，特别是当我处理大型代码库时。

## [](#solution)解

您可以创建一个快捷键来切换 ReSharper。

*   打开 Visual Studio
*   前往`Tools`–>`Options`–>–`Keyboard`
*   设置`ReSharper_ToggleSuspended`的快捷方式(可以在菜单中搜索)
    *   我用了`CTRL` + `ALT` + `R`

现在，你应该可以在需要的时候切换 ReSharper 了。
