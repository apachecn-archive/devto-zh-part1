嘿，戴夫！我是 Gildásio，到目前为止，我已经和 Flutter 一起工作了一年！这是一次神奇的旅程，终于可以一边编码一边享受乐趣了。我最初是从 Python 和 Django 开始的，但主要专注于移动开发，在那里我爱上了 Android，但对持续了近三年的 SDK 并不满意。Kotlin 帮了我很多，我仍然怀念那里的一些东西，但主要是不用等几分钟就能看到显著的 UI 变化。

我在 2018 年 3 月/3 月开始学习 Flutter，比他们宣布 alpha 版本早一点，因为我想为我的毕业设计做一个 iOS 应用程序(仅限 Android)，但学习原生端并不容易，我有点厌倦了尝试。在习惯了声明式方法以及一切与原生 Android 端的不同工作方式后，我爱上了当你觉得自己可以控制一切时，开发工作会有多么令人惊叹。

2018 年 10 月晚些时候，我获得了我的第一份自由职业者工作(与 Flutter 一起)，这要归功于我在 [Medium](https://medium.com/flutter-community/flutter-adding-sign-in-with-google-and-phone-authentication-to-your-app-69f681518f9b) 上写的一篇文章，到目前为止，我一直在两个应用程序中工作。

这是一个更新的介绍，我很高兴来到这里！
