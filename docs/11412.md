# 处于最佳状态

> 原文：<https://dev.to/funkysi1701/in-the-zone-4bkj>

周五晚上，我有启动 Visual Studio 的冲动，明天(周一)我就要搬家了。

那是怎么回事？当我有箱子要打包，有架子要拆的时候，为什么我会有编程的心情？

在过去的几周里，我一直忙于其他事情，没有时间做太多的编程，或者当我有时间的时候，我更喜欢放松，而不是学习任何新的东西。

我如何进入编程区，这个特殊的周末是什么让我想要？仅仅是因为我没有时间专注于它，还是有一些更有用的因素控制着我对编码的渴望？

我能想到几个我应该注意的因素。

1.  和平与安静——周五晚上我有自己的地方，所以我可以集中精力做我想做的事情。
2.  想法——在我开始之前，我有一些想法，为什么我不尝试做 x。一旦我有了一个编码想法，我需要调查它，看看它会去哪里。
3.  打破常规——周五晚上是一周不用工作的开始，这让我有更多的时间忘记办公室的日常压力，思考其他事情。
4.  目标——周五的工作让我心烦，如果我想实现我的目标，那我就需要努力。

我不知道如何将这些付诸实践，这样我就可以花更多的时间在开发上，但是我们会看到的。我想在我的新家实现的一件事是一个安静的地方，集中精力，不受干扰地思考，我有几个选择，我将在未来几周内调查。

总之，我要回去打包行李了。
