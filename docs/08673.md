### [](#the-usual-suspects)平常的疑点

这是一个非常有效的 Erlang 模块:

```
 -module(my_mod).
    -export([whoami/0]).

    -spec whoami() -> atom().
    whoami() -> my_mod. 
```

### [](#the-cool-guy)酷家伙

但是……那里的 **spec** 属性没有括号。我们还可以去掉其他属性的括号吗？T2 的医生对此不太清楚。它们显示了以下属性的通用语法:

```
-Tag(Value). 
```

但是还有像*规格*、*类型*或*回调*这样的属性，显示时没有括号。那么，为什么不试试呢？如果括号周围没有圆括号，导出会好得多，对吗？

```
 -module my_mod.
    -export [whoami/0].

    -spec whoami() -> atom().
    whoami() -> my_mod. 
```
