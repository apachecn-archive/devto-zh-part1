# 90 天内的世界级会议，第 1 部分-演讲人

> 原文:[https://dev . to/react ive conf/90 天后世界级会议-演讲者](https://dev.to/reactiveconf/world-class-conference-in-90-days--speakers)

当我们开始组织第一年的 [ReactiveConf](https://reactiveconf.com/) 时，我们没有可以遵循的指南。今天，我们即将完成第二年的工作，该活动将于 10 月 26 日至 28 日在斯洛伐克的布拉迪斯拉发举行，我感到很兴奋，也有点如释重负，因为很多人问我，我们是如何在如此短的时间内举办如此出色的活动的。

ReactiveConf 的想法并不是来自我，而是来自我的商业伙伴 Matej Ftacnik，去年我们一起参加了巴黎的 ReactEurope。就在我们回国后，Matej 说:“我们必须把它带到布拉迪斯拉发，让它成为中欧和东欧最好的东西！”

没有人告诉他，没有任何经验，不可能在 90 天内做出世界级的应用程序开发会议。所以我们就这么做了！我希望下面的话能帮助你，激励你，让你在成为下一个世界上最好的会议的创始人的旅途中避免不必要的错误！

组织一次会议是一件复杂的事情，所以我决定把这篇文章分成几个部分。在这一篇中，我将试着概述让演讲者参加你的活动的所有错综复杂的事情。

# 天时地利人和

让我们明确一点:没有伟大的演讲者就没有伟大的会议。我宁愿在旧车库里听一场鼓舞人心的演讲，也不愿在五星级酒店的会议厅里听一场普通的谈话。所以给自己找一个有激情的团队，一个有共鸣的名字，建立一个有基本信息的网站。寻找演讲者是你将要面临的最大挑战——不仅是第一次，而是每年。

一个好的起点是世界各地的其他会议。在我们的例子中，这些是[React euteria](https://www.react-europe.org)、 [React Rally](http://www.reactrally.com) 和 [official React.js Conf](http://conf.reactjs.com) ，我们邀请了几位演讲者。寻找出类拔萃的人，与你的关注点相符的人，以及在你的领域中被有影响力的人追随的人。

不要羞于查看他们的推特账户。去年，我浏览了以下名单:丹·阿布拉莫夫、T2、大卫·诺伦和 T4，他们都是真正的 React 梦想家。这帮助我找到并最终得到了优秀的演讲者，主要是那些小众的。

# 酷因子

虽然看起来可能不是这样，但当谈到你的会议阵容时，“小众”是一个重要的词。这样的人通常不为主流观众所知，他们经常做一些太新或太晦涩的东西，但是他们会刷新你的日程表并吸引许多有影响力的人。

像尼基塔·普罗科波夫、T2 和斯塔尔茨、T4 和维克多·格里先科这样的演讲者在我们的议程中都是非常陌生的名字，但正是他们引发了社区对 ReactiveConf 的第一次兴趣。毕竟，多亏了阿布拉莫夫的推特账户上的[“我的反应列表”](https://medium.com/@dan_abramov/my-react-list-862227952a8c#.v2nf1fncl)，我才找到了它们。这是一个悖论，但这些“不寻常的名人”最终说服了 React 中许多最热门的名字加入我们。

此外，一旦你进入了影响者的意识，询问个人最喜欢的。去年，ReactJS 的策划人和我们的一位主演 Brent Vatne 建议, [Daniel Woelfel](https://twitter.com/DanielWoelfel) 和 [Krzysztof Magiera](https://twitter.com/kzzzf) ，他们都是各自领域的专家，所以我们邀请他们今年出场。他们都来了——和瓦特一起！

# 与众不同

许多演讲者，主要是观众中最受欢迎的，每个月都会收到大量的请求和邀请。自然，你的任务是说服他们来参加你的大会。但是如何实现这一点呢？嗯，你必须拿出一个令人信服的故事来——我们的故事相当可靠！

ReactJS 是一个闪亮的新事物，我们正在做世界上第二大的 React 会议。2015 年，每个人都说我们是一个大社区，Ember、Clojure、Elm 和其他前端人员应该相互学习。

这一信息被反复提及，但我们是唯一一次邀请来自各种环境的大量人员见面并发言的会议。最后，这样的行为受到了广泛的赞扬——我总是为这样一个事实感到自豪，即使是克里斯托弗·切多也在推特上说我们有最好的演讲者阵容。

# 永不放弃

邀请演讲者仍然是你清单上最困难的项目之一，尤其是当你刚刚开始的时候。因此，无论你要联系谁，给他们发一封电子邮件，解释你是谁，你做了什么(和已经做了什么)，你为什么想要他们，你想通过你的会议达到什么目的，以及你为什么想要他们。

要坚持，不要放弃。他们还没有答复吗？再试一次！去推特、脸书或者 Github，和他们交流。并愿意满足发言者可能有的任何特殊条件和要求。

然而，如果他们对你的提议说“不”,不要惊讶。对每个演讲者来说，每次会议都是一个巨大的承诺——他们花很长时间旅行，为他们的演讲做准备。此外，他们的老板可能不会因为连续几天失去一名富有成效的团队成员而感到兴奋。

在我们的例子中，说服大多数演讲者上司的论点是，斯洛伐克是一个招聘人才的好国家。是的，像脸书和谷歌这样的公司知道这一点，近年来他们在这里招募了许多有才华和创造力的程序员，但我们仍然是一个相对较新的、大部分未被发现的领域。

当然，每个国家都是不同的，所以想出一些原创的适合你的情况的东西。

# 自我介绍

无论你在哪里召开会议，你都不能忘记使用你的网络。去年，我打电话给 Elm 语言的开发者 Evan Czaplicki，但是他不能来，所以他推荐了 T2 的 Richard Feldman，他在 T4 的同事 NoRedInk，也是 Elm 和 React 的爱好者。不用说，他今年又要来 ReactiveConf 了！

请你的朋友介绍你。如果有人能把你推荐给你渴望带领阵容的演讲者，马上联系他们。当说话者拒绝时呢？请他们向他们的朋友和关系人提议你的项目；他们通常很乐意这么做。

此外，请记住，我们是一个相互关联的社区，所以许多演讲者可能会在开始时犹豫不决，但即使是最好的演讲者也可以被说服，当你开始在你的网站上添加演讲者(即使是相对不知名的)。如果你对你的事情很认真，并且想和任何人约会，那么一个充满名字、谈话和概要的页面——用两个词来说:伟大的内容——是必须的。

# 建立你的信誉

但不要让今年是大会的第一年这一事实对你不利。当然，一流的演讲者想要参加一流的活动，所以当你刚开始演讲并且没有历史背景时，你能做些什么来得到他们呢？

人数和会期长短至关重要。为一个有 500 人参加的为期三天的项目找到你想要的演讲者，比一天一百人的活动要简单得多。走向国际，不要本地化。

你可能没有想到这一点，但票价也起着巨大的作用。不要以每件 50 欧元的价格推销自己——你已经花了很多精力将“精英”带到你的会议上，而且你即将推出下一个最好的技术活动，所以让世界知道你的价值。毕竟，你可能试图说服酷人来到一个酷的地方，和其他酷人一起玩。

# 乐趣是第一位的——但也有问题

然而，你必须时刻提醒自己，无论是一个完全的新人还是马克·扎克伯格来参加你的会议都没有关系。所有的演讲者都是你的客人、朋友和会议的超级明星，你必须以同样的尊重和友善对待他们。

让他们乐在其中。告诉他们带上他们的伴侣和家人，安排和支付他们的旅行和住宿，给他们分配一个私人助理来实现他们的所有愿望。派辆出租车送他们往返机场。不仅如此:我们组织特别旅行，举办仅限演讲者的晚宴。去年，我们在布拉迪斯拉发城堡举办了一场宴会，那是整个城市最高档的地方。把你的客人当成贵宾需要花费金钱、时间和精力，所以你最好事先考虑到这些。

尽管你尽了最大努力，但总会有一些演讲者取消的风险，你必须找到合适的替代者。在这种情况下，请发言人向他们的网络传播关于你的会议的消息，让他们尝试寻找替代方案。不要对整个情况耿耿于怀，你的恶劣态度不会帮助任何人。尽可能保持积极和灵活。

# 打造自己的明星

认识新的人，倾听新的想法，从新的概念中感到兴奋，这就是为什么我们首先要做 ReactiveConf。这也是我们开始做闪电谈话的原因，这是一种比赛，我们给伟大的新思想一个机会，与观众分享他们的创新思想。

任何人都可以在 ReactiveConf 申请他们的五分钟成名——他们只需要给我们发送一个启发性的推介！我们收集所有的参赛作品，然后让与会者在会前通过 Twitter 和其他社交媒体选择他们最想听到的想法。

通过这种方式，我们在去年发现了 Ossi Hanhinen，一位榆树和长生不老药专家，他开始在赫尔辛基共同组织自己的榆树聚会，并在著名的黑客杂志上发表了他的文章。

我甚至无法表达我有多自豪，我们的会议不仅是一个蓬勃发展的业务和令人兴奋的事件，而且是一个社区可以见面、分享和成长的平台。
