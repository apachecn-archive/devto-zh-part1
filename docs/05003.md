# 慈善专业是我最喜欢的科技演讲者

> 原文:[https://dev.to/ben/charity-majors-is-my-spirit-animal](https://dev.to/ben/charity-majors-is-my-spirit-animal)

大约一年前，我在参加的一次会议上听到了慈善专业人士的演讲，从那以后，我一直关注着她的演讲。每一个都很棒。她在我们行业的监控/运营/系统部门工作，这是一个充斥着废话和流行词汇的空间*。没有人能像她一样幽默和诚实。*

她是 Honeycomb.io 的创始人，我从来没有用过这个网站，但是看起来很光滑，而且做得很好。

我留给你一些你可能想看的演讲/演讲。我不确定这些是不是好的。无论哪种方式，跟随她的谈话，跟上她的工作。这会让你成为更好的工程师。
