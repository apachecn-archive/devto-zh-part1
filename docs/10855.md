# 我在什么行业？

> 原文:[https://dev.to/ben/what-industry-am-i-in](https://dev.to/ben/what-industry-am-i-in)

我很犹豫地说，我在“软件行业”。我觉得我可以和硬件开发人员谈论工作，或者潜在地过渡到一个以硬件为中心的角色，而不会觉得我已经转行了。然而，“编码行业”作为一个标题似乎太不正式。“编程”太模糊；“科技”、“数字”，方式太模糊；“发展”，过于具体；“web 开发”，太狭隘；“工程”，太宽泛，“计算机科学”，ðÿ™„.我知道我的行业是一个抽象的概念，但也许我们会说给事物命名很难，因为我们的行业就是不擅长给事物命名。
