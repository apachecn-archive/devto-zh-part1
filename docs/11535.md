# 鹈鹕支持的新博客

> 原文：<https://dev.to/rndmh3ro/new-blog-powered-by-pelican-10id>

我把我的博客平台从 [Wordpress](http://wordpress.org) 改成了 [Pelican](http://blog.getpelican.com/) 。Pelican 是一个用 Python 编写的静态 html 生成器。我还改变了整个设计。这是一个自举主题，有一些小的修改。

我改变 Wordpress 的原因是为了维护一个安全和最新的安装，而不是静态的 html 文件，只有最少的 Javascript。我也没有必要使用像 Wordpress 这样成熟的 CMS，因为我只想写一些关于我喜欢的技术的博客文章。也许我还会描述一下从 Wordpress 到 Pelican 的过程。

仍然有[皮维克](http://piwik.org/)在后台运行，收集一些关于我写的帖子的统计数据。如果您不想被跟踪，您可以选择退出此处[]。

这个评论系统叫做 [isso](http://posativ.org/isso/) ，是由 Python 驱动的“轻量级 disqus 替代品”。

所以…如果有一些小故障或事情不像预期的那样工作，留下评论或给我发电子邮件！
