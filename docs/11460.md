# 在 OSX 安装流浪者

> 原文:[https://dev.to/adamkdean/install-vagrant-on-osx-30b](https://dev.to/adamkdean/install-vagrant-on-osx-30b)

我们将使用[自制软件](http://brew.sh/)在 OSX 上安装`Vagrant`。

如果你还没有的话，[在 OSX](http://adamkdean.co.uk/blog/read/138/install-homebrew-on-osx) 上安装家酿。

一旦你安装了它，安装流浪者像这样:

```
brew install caskroom/cash/brew-cask
brew cask install vagrant 
```
