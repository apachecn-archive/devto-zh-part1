# 2。承诺:

一个承诺是一个价值的代理，这个价值在承诺产生时不一定是已知的。它允许您将处理程序与异步操作的最终成功值或失败原因相关联。这使得异步方法像同步方法一样返回值:异步方法不是立即返回最终值，而是返回一个在未来某个时间提供该值的承诺。
