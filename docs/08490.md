Python 是一种容易学习的语言，可以用于很多事情。Python 很酷的一点是已经开发了许多有趣的东西。这方面的一个例子是 [Tweepy](http://www.tweepy.org) ，这是一个易于使用的 Python 库，用于访问 Twitter API。

我真的很喜欢尝试整合不同的技术，所以我给自己一个任务，尝试用 Tweepy 建立一个 Twitter 机器人。结果:我创建了一个机器人，它每天在世界协调时 11:00(墨西哥时间 6:00)生成一个随机的复数，然后在一个函数中迭代它以生成一个 Julia 集，最后这个机器人发布生成的分形图。可以在 [@JuliaSetBot](https://twitter.com/JuliaSetBot) 查看。
