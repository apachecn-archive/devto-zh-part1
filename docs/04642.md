# 如何找到一位新的伟大导师，与之共事，并脱颖而出

> 原文:[https://dev . to/Kyle Galbraith/how-to-find-work with-and-emerge-as-a-new-great-mentor](https://dev.to/kylegalbraith/how-to-find-work-with-and-emerge-as-a-new-great-mentor)

前几天晚上，我参加了一个 Twitter 聊天。讨论都是关于导师，如何接近他们，问他们什么，以及拥有优秀导师的价值。这让我开始思考导师对我来说有多重要。他们在塑造我的职业发展和个人动力中所扮演的角色。

我最近也从被辅导者变成了导师，也从那次经历中学到了很多。通过我的经历，我收集了一些关于导师的意见。如何找到他们。与他们合作的最佳方式。那成为其中一员意味着什么？

### 是什么造就了伟大的导师

这是因人而异的看法。每个人都会被我们人类同胞的不同特征所吸引。然而，我要大胆地说，伟大的导师都有一些总体特征。

1.  他们平易近人。
2.  他们很有耐心。
3.  他们激励人心。
4.  他们渴望学习并帮助你学习。
5.  你的成功就是他们的成功。

当我看着生活中的导师时，我脑海中浮现出这五条原则。我有很多了不起的人指导过我。通过我的计算机科学学位，成为一名开发人员，甚至渴望有一天能经营自己的公司。所有人都有这些特征。

### 与伟大的导师一起工作

导师是一种资源，可以把你的学习和理解从 0-60。要做到这一点，你必须最大限度地利用你和他们在一起的时间。现实是每个人的时间都是有限的。我们都有最后期限、要完成的项目和其他承诺。这绝不意味着你的导师没有时间陪你。

这意味着你应该理解约束。在与我的导师一起工作的过程中，我发现以下几点可以让我的钱发挥最大的作用。

1.  阐明你的问题/想法/事情。没有人会期望你已经调查了所有的事情。但是尽你所能清楚地表达你在寻求什么帮助。这也可以归结为成为一个有效的沟通者。如果你能够交流，你将会得到导师能提供的最好的建议。
2.  当你知道你需要帮助的时候要理解。如果你觉得你的头撞到了墙上，你就是。这个我继续纠结。很多时候我觉得我应该“知道这些”。这不是一件坏事。事实上，经历这个过程实际上可以帮助你学到更多。也就是说，在某个点上，你做的弊大于利。记住这一点，并找到自己的位置。一旦你这样做了，你现在知道在哪里向你的导师提问了。
3.  解释一下到目前为止你做了什么。上下文切换是真实存在的。你如何走到这一步的过程对你来说很棒(即橡皮鸭调试)。这也让你的导师能够赶上你，并切换到你的背景。

伟大的导师可以提升你的学习和理解。通过留意这三个步骤来帮助他们。

### 从学员到导师

现实是，一个人拥有的经验年限都是扯淡。没有任何意义。我们都在学习，因为我们必须学习。我希望这是因为你也想，否则，你是在错误的游戏。没有人能学到所有的东西并成为国王的导师，这是不可能的。但是我想多了解一点这个或那个？太好了。动手吧。但是你也知道其他最有可能帮助别人的事情！

当你处于职业生涯的早期，你可能会认为这是几年以后的事。你错了。这一天就在眼前，或者明天，如果你想的话。每一天你学到一些东西并向前迈一步，就会有一个人回到你之前的那一步。

我一直想成为某人的导师。我想教别人我知道的东西，因为这是你能做的最有意义的事情。所以我等着有人给我这个机会。我等待着。我等待着。然后我就醒了。

没人会把它交给你。如果你想指导他人或与世界分享你的知识，现在就去做吧。不要像我一样等着。

我可能帮助了 2000 人解决这个问题。要知道，做导师就是要平易近人，有耐心，有动力，渴望学习，并希望他人成功。这可以是你想要的任何形式。你需要采取一个可以帮助别人的行动。
