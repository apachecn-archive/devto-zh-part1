# 为什么感恩让你成为更好的程序员

> 原文:[https://dev . to/_ Patrick god/why-感恩让你成为更好的程序员-233f](https://dev.to/_patrickgod/why-gratitude-makes-you-a-better-programmer-233f)

这应该是一个简短的提醒。因为我相信你知道接下来要讲的所有东西，但有时我们只需要有意识地提醒自己我们拥有的所有令人惊奇的东西。

问题如下:作为一名软件开发人员，有些任务你必须完成，但你不想去做，因为它们看起来太大或太无聊，你说吧。现在这样想:你被允许这样做。你能够征服这些任务。有人告诉你去做，因为有人认为你有能力编写代码并完成这些任务。

所以，让我们想一想，为了完成你的工作，你可以感激的许多事情。

很有可能，你正坐在一个温暖的房间里，一把舒适的椅子上，盯着一台或多或少现代的屏幕和一台昂贵的电脑。这难道不令人惊讶吗？

你可以上网，对吧？我猜你不是线下看这篇文章。所以你的办公桌上有世界上所有的知识。我打赌你有时会利用这些知识来完成你的工作。所以你有很多选择来完成你的任务。疯狂。

你被允许在工作时戴耳机吗？我敢打赌。比方说，你可以把耳机插到电脑上，然后打开亚马逊音乐、Spotify、YouTube 等等，边编码边听你喜欢的音乐。对我来说，没有比这更好的进入“区域”并专注于编程的方法了。

你的午休时间呢？首先，你要吃午饭，对吗？另一个并不总是显而易见的感恩理由。我知道我有点夸张，但从微波炉或街角的小吃店得到一顿热饭，时不时地喝点咖啡，真的是值得你感激的事情。

我最喜欢的是我舒适的床。每晚都能躺在自己温暖舒适的床上，早上起床前五分钟就能享受床单的温暖，这是一个奇迹。

记住所有这些，你会更快乐，你会抱怨得更少，也许你会更有动力去征服你每天的程序员生活，实现你一天的所有目标——也许在假期里更容易忍受家人。；)

所以下次你接到一个糟糕的任务时，面带微笑去做。

圣诞快乐！
