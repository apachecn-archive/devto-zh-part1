# 焦点水蛭的攻击

> 原文:[https://dev.to/codemouse92/attack-of-the-focus-leeches-1mjg](https://dev.to/codemouse92/attack-of-the-focus-leeches-1mjg)

对我来说，最难控制的就是我的注意力。我开始怀疑，在我高中头部受伤后，我可能已经捡起了 ADD 的碎片，或者至少是功能上类似的东西，以及其他新的挑战。只要一个随机的想法出现在我的脑海中，我就会立即感到有必要去追踪它。

我已经隔离了几个在工作时间吸干我生产力的主要焦点水蛭:

*   无限分散注意力的网站，如社交媒体:对我来说，那就是 Twitter、YouTube 和 GoComics。

*   无关任何事的随机 IRC 讨论。

*   琐事潜水:阅读各种电影和书籍的情节笔记，了解谁发明了弹珠，或找到巴西的国民生产总值。

控制这些干扰对我来说是一场持续的艰苦战斗，但我发现了一些帮助控制这些干扰的策略。诀窍，也许是 2018 年一个很好的个人目标，是*实施*它们。

# [](#actively-block-distractions)主动阻挡杂念

第一步也是最容易的:使访问时间链接变得困难。对我来说，这意味着安装干扰屏蔽插件(在 Firefox 上安装 [Leechblock NG](https://addons.mozilla.org/en-US/firefox/addon/leechblock-ng/) ，在 Chrome 浏览器上安装 [Stayfocusd](https://chrome.google.com/webstore/detail/stayfocusd/laankejkbhbdhmipfmgcngdelahlfoji) )。

这两个插件都允许我在网站上设置特定的屏蔽时间和时间限制，甚至是在工作时间让我分心的子网站。Twitter、YouTube 和所有我喜欢的漫画都在这个列表上。

当然，问题是，从专业的角度来看，我有时需要 Twitter 和 YouTube。这就是为什么我有时间限制是很重要的，给我 15 分钟的时间在工作时间浏览所有黑名单上的网站。

我也只屏蔽包含`/watch/`的 YouTube URLs，从而允许我无限制地访问我自己的控制面板，以及搜索索引本身(为其他人寻找链接)。在 Twitter 上，我同样将`api.`子域列入了白名单，因为我需要它工作。

我确实有其他方法进入任何网站。从技术上来说，在我的两个主要浏览器(Firefox 和 Vivaldi)之间，我总共有 30 分钟。我也可以通过 Chromium 或 Opera 免费访问，这是我为网站管理工作保留的，或者通过 Tor。然而，我发现这些可供选择的路线对我个人来说不会损害这个策略:

*   我不认为故意绕过我的分心过滤器是一个可行的选择。感觉就是欺骗，欺骗自己。

*   我从来不登录 Chromium、Opera 或 Tor 上的任何东西，因此仍然会屏蔽社交媒体。

*   我发现我的自制力至少足以让我不作弊。这也是为什么我让 Leechblock 的控件保持解锁，以防我需要收紧控件或修复错误。

分散注意力屏蔽器对我也有帮助，当我真的需要集中注意力时，我就在两者上开启“核模式”,立即屏蔽我黑名单上的所有东西。

# [](#cut-down-on-bookmarks)剪下书签

不幸的是，有些事情我不能添加到我的黑名单。为了工作，我经常需要无限制地访问维基百科、CodinGame、StackOverflow 和 dev.to()等网站。其中，维基百科是最糟糕的冒犯者，助长了许多“琐事潜水”。

同样，那些必要的 15 分钟访问我黑名单上的网站甚至可以成为转移注意力的跳板。Twitter 是一个名副其实的分散注意力的外部链接的雨林。我不能合理地勾选这个框来阻止黑名单网站的“外部链接”，因为有时我会用 Twitter 来查找我工作需要的链接！

事实证明，这个难题的解决方法非常简单。我不得不更新我的书签。每个人的书签习惯不一样。有些人让标签页开着，有些人在口袋或侧边栏上整理列表。对我来说，我使用书签栏，省略了大部分作品，仅仅依靠网站图标和 Unicode 符号。

看看我的维瓦尔第书签栏。(习惯上，我工作的时候用火狐。)

[![A cluttered bookmarks bar](../Images/f38d806a2fd63d95762e94b98842d02d.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--lidYUdtl--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/cifjqug0he7cymev5etj.png)

上个月，当我习惯性地在一天内点击 Twitter 大约二十次，并到达我已经点击过无数次的*同一个*“网站被封锁”页面时，我意识到我的问题:**我的注意力太容易被视觉上的东西分散了**。每当我在打开一个新标签页时感到漫无目的，我就会点击中间的某个东西，那通常是 Twitter。

如果不是 Twitter，那就是我钉在书签栏上的其他十几个焦点水蛭中的任何一个。(那个笑脸文件夹是我收藏的漫画列表，也是我的一个主要点击孔。)

因此，当我为新的工作年设置 Firefox Quantum 时，我决定限制我会带哪些书签。

[![A controlled bookmarks bar](../Images/a90b8dcae55d24072bfd1083383158e7.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--CDXTHaih--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/aqptxd6bhe1t2i7l7bo5.png)

我现在发现我只是故意访问 Twitter，可能一天两三次。我不得不开始输入“twitter.com”，而不是点击书签，这种额外的努力延长了我的选择时间:我经常停下来问自己，“为什么我会走进一个已知的分心点？”

此外，请注意在修剪过的书签栏上的*是什么:只有工作用的东西！DuckDuckGo，Archive.org，StackOverflow，CPlusPlus.com，dev.to()，GitHub，Authorea，我们公司工作网站，然后是我要处理的东西的文件夹(！！！)、文档和工具。我已经戒掉了所有的游戏，所有的社交媒体，以及其他任何可能让我在工作中分心的东西。*

是的，我仍然在 Vivaldi 上保留这些东西，这基本上是我的“休闲”浏览器。我以后可能会剪掉，也可能不会。按照习惯，我不会在工作时间使用维瓦尔第，除非技术需要。

# [](#unplug)拔下

我再也没有智能手机了。我做过一次，它吸引了我很多注意力。我不得不使用一些技巧来控制它:

*   如果我不需要接电话或发短信，我会关掉智能手机，把它收起来。

*   尽管我很喜欢《愤怒的小鸟》( Angry Birds )( T1 ),但有一段时间我确实删除了它和许多其他游戏。我确实保留了一些游戏，比如《T2:愤怒的小鸟》!，因为微交易在一定的时间或生命后屏蔽了我的游戏。

*   我把社交媒体应用从主屏幕上移除了。寻找图标的努力增加了我选择的机会，就像书签一样。我甚至把 YouTube 完全卸载了。

*   虽然我个人从来没有这样做过，但是如果你发现自己经常在手机上分心，并且你不能把它收起来，考虑购买并安装一个分心阻止器，比如 T2 自由 T3。

# [](#remind-yourself-of-work)提醒自己工作

这仍然留下了一些焦点:StackOverflow 和 dev.to。我工作需要这些，但是我也浪费了很多时间去浏览它们。

除此之外，我还有 IRC 要对付。我在 IRC 上生活、娱乐和工作。对于我的工作流程来说，它就像咖啡一样重要。简单地断开连接不是一个选项。

为了让自己专注于工作，我必须使用更积极的方法:提醒自己工作。对我来说，这是以[仓鼠](https://github.com/projecthamster/hamster)的形式出现的，这是一个桌面时间跟踪工具，它还可以定期闪现通知，提醒我当前的任务。

我发现，当仓鼠跑步时，我的注意力至少是平时的两倍。如果我发现自己陷入了精彩的 IRC 对话，或者阅读了太多的 dev.to()文章，及时的“您正在编写代码”通知将我带回现实。

# [](#schedule-curiosity)日程好奇心

也许最阴险的焦点水蛭是我邪恶的好奇心。当我开始心流的时候，它会偷偷溜到我身后，要求知道医生有多少个同伴，直到满意为止。我称之为“琐事潜水”,因为它们不会因为一个问题而停止。一提到唐娜，我就会想重新观看她击败达雷克斯的史诗般的时刻，这将导致对达雷克斯第一次出现的好奇，这将导致想知道还有什么其他著名的角色被扮演最初的演员扮演达夫罗斯...........

(我有两种访问 YouTube 的方式，而且没有办法堵住这个缺口，这没什么帮助。我不是在描述另一种方法，以免把别人引入歧途。)

我真的摆脱不了那种好奇心。这是我的一部分，也是我职业发展的驱动力之一。相反，我必须控制它。

其中一个最有效的方法是把笔记本放在手边，尽管这个方法没有被充分利用。每当我的好奇心要求知道最大的亚马逊蜘蛛有多大时，我就可以草草记下，稍后再去查这个事实。

我也有特定的自由时间来锻炼我那贪婪的好奇心。这是我每个工作日早上 6 点起床的一个原因。我发现，如果我在开始工作之前把好奇心从我的系统中清除出去，那么一整天都有东西可以咀嚼。

换句话说，拥有像我这样的好奇心就像拥有一只非常大的哈士奇。在你工作的时候，防止它破坏你的房子的唯一方法是确保它得到足够的锻炼，并且给它留下可以玩耍的东西。你不能压制哈士奇的能量，你只能重新引导它。

# [](#limit-aimlessness)限制漫无目的

我发现如果我事先不知道我在做什么，我就无法集中注意力，花在努力回忆(或思考)要做什么上的时间是注意力水蛭进入的最佳机会。

在一周的开始，我喜欢写下我想在那一周完成的所有事情。然后，每天早上(或者前一天晚上)，我为这一天设定具体的目标。事实上，我写下的事情比我知道要完成的事情还要多，所以如果某件事情不需要很长时间就能完成，我就不会缺少其他事情去做。

# [](#putting-this-in-use)投入使用

显然，我的焦点问题的确切性质对我来说是独特的。然而，这是我们许多人在不同程度上挣扎的事情。这是我的建议，以一种更通用的形式。

1.  识别无效的干扰，使用干扰阻挡工具减少干扰。根据你的自制力程度来定制他们的严格程度。然而，注意不要把自己锁在工作需要的东西之外！

2.  配置你的书签、图标和快捷方式来增加开始一些分散注意力的事情的时间。这都是为了增加你的选择机会。

3.  移动设备是焦点的主要敌人。在你的设备上使用其他策略，只要有可能就拔掉插头。

4.  为你正在做的事情设置积极的提醒，让你摆脱分心。你可以使用时间追踪工具，闹钟，甚至是敲入休息提醒——任何能经常提醒你正在工作的东西。

5.  驯服你的好奇心。写下一些事情供以后查阅，并设定时间让你的求知欲“跑一跑”。

6.  提前计划你的工作日。不要给自己不知道自己在做什么的机会，因为这是注意力吸血鬼的最佳机会。
