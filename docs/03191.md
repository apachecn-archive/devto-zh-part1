#### 为什么你应该作为初级开发人员开始写博客

在深入了解如何开始写博客之前，了解作为初级开发人员写博客的好处是很重要的。

#### 1 它会显示你学到了什么

博客就是与网络世界分享你要说的话。因此，作为一名开发人员，你或许可以谈谈你学到了什么或发现了什么。这将有助于其他人从你身上获得灵感或学到东西。即使是最小的东西也是有价值的！

你可以开始展示 HTML、CSS、JavaScript 或你正在做的任何东西的技巧和窍门。使用代码示例使它变得更好。因为秀比讲好！

#### 2 它会训练你的大脑去解释

当你开始写你学到的东西时，你的大脑会被训练去解释。你会强迫自己向别人解释事情。你需要为你的题目做好充分的准备来解释它。

越来越多的解释会让你成为一个更好的老师，也许是教一些小事情。多做事最终会让你做得更好！

#### 3 它会显示你的知识和专长

你的博客将成为你在线形象的一部分。如果人们要搜索你的名字，他们肯定会找到你的博客(如果它是公开的)。

当他们这样做时，他们能够发现你的专业水平和你喜欢做的东西！所以要让它有价值！

我的个人博客总是让我的故事在潜在的求职面试中更加有力。因为人们可以看到我所有的博客帖子。通过我的博客帖子，我证明了我知道我在说什么！

一个很好的例子是关于 [Harry Roberts](https://csswizardry.com) ，他是一个著名的前端开发人员。因为写博客，他的事业开始成长。最近，他在追求播客上接受了[杰西卡·罗斯的采访。(顺便说一句，这是一个很好的播客😉)但在那次采访中，他分享了自己关于它的故事，所以绝对值得一探究竟！](https://pursuit.podiant.co)

#### 作为一名初级开发人员，你应该如何开始写博客

来点实际的吧！下面我会给出几个开始写博客的地方。有的洒脱有的付费更高级！

#### 1 & 2 在 WordPress.com 或 Medium.com 免费建博客*(简单&免费)*

没有借口，你买不起任何域名或虚拟主机。因为 WordPress.com 和 Medium.com 都是自由的。只需注册一个帐户，然后开始写作！

在媒体上就是这样。在 WordPress.com 上，你也可以安装插件和改变你的主题。

与 WordPress.com 相比，媒体有一个更广阔的平台来获得更多的读者。但是挑一个你喜欢的！

#### 3 用静态站点生成器建立你的博客 *(easy+ &免费)*

静态站点生成器是一个从 MarkDown 文件生成博客文章的工具。有很多不同的选择。

选择和比较它的好地方[Staticgen.com](https://www.staticgen.com/)

我试过哲基尔和金属史密斯。我知道 Jekyll 可以在 Github 上免费使用，也可以在你自己的虚拟主机上使用！如果你有一个域名，你可以把它连接起来。它花费了一点点的工作，但它的工作很棒！我从 Smashingmagazine 上找到了一个很棒的使用 Github 作为 Jekyll 网站虚拟主机的指南，这值得一试！

Metalsmith 也可以托管在 Github 或者你自己的虚拟主机上。在 [Sitepoint 上，有一个很棒的使用 Metalsmith 的教程！](https://www.sitepoint.com/create-static-site-metalsmith/)

#### 4 托管自己的 WordPress 网站*(易+ &免费)*

WordPress 可以免费使用，它是开源的。有很多免费的插件。但是你也可以选择高级主题和插件。

拥有自己的 WordPress 是我写博客的最爱。我把我的网站放在[webfision，它非常适合各个层次的 Web 开发者](https://www.webfaction.com/?aid=73781)。通过即插即用，你可以安装最新的 WordPress，把它连接到一个数据库，并把它放在你的域名上。

有了你自己的 WordPress 网站，你可以使用高级的 WordPress 主题，但是更好的是建立你自己的 WordPress 主题。最好的开始方式是用[下划线基本主题](https://www.sitepoint.com/create-static-site-metalsmith/)。如果你愿意，你可以只做 CSS 和 HTML。它已经充满了最好的功能。

#### 5 服务器端渲染 NodeJS 应用*(高级，免费)*

这有点意外收获！我刚刚发现了 NextJS，我很喜欢它！

我想推荐两个框架， [NextJS](https://zeit.co/blog/next) ( *在 ReactJS* 之上)和 [NuxtJS](https://nuxtjs.org) *(在 Vue 之上)。它通常不用于写博客，但也用于建立网站。*

它的好处是，你可以使用 API 来显示你的文章和页面。是的，它更先进，但如果你喜欢挑战，这是值得一试！

NodeJS 可以在 Heroku 上免费托管。但是我的虚拟主机 WebFaction 也支持 NodeJS 应用程序。但我更喜欢 Heroku 测试，因为它很容易设置！
