# 知识共享系统

> 原文：<https://dev.to/briannipper/kiss-42mk>

[![](img/c4c427bdb9c11927057dd0c15576f977.png)](http://upload.wikimedia.org/wikipedia/commons/e/e8/Crystal_Clear_app_kedit.svg) 开发中一个永远都不会被说够的原则是“保持简单愚蠢”(或者保持简单愚蠢或者保持超级简单)。在任何情况下，这一开发原则都可以用于生活的许多方面，但显然它可以用于处理日常计算机任务。

大多数人执行的一项任务是记录各种“事情”。有时，当人们开始收到大量的笔记时，他们认为他们应该尝试使用一些花哨的程序或电子表格，甚至是专门的软件来跟踪所有这些笔记。

然而，我发现最简单的解决方案仍然是最好的，记事本。

记事本在 Windows 3.x 时代就已经出现了，我敢肯定，即使在 dos 时代，人们也会使用 EDIT(编辑文件的 DOS 命令)来记笔记。尽管我将越来越多的电子任务转移到智能手机上，但我发现手机自带的简单的“记事本”类应用程序变得非常有价值。

有些人可能会说记事本缺乏功能；然而,“缺乏特色”会确保你专注于任务。您不必花费大量时间来格式化和插入剪贴画，而必须用它来捕捉重要信息...文字。

起初，一个简单的文本应用程序可能看起来“太简单”，但通过一些提示和标记，你可以用这些简单的小笔记捕捉和传达很多信息。

*   您可以使用任何非字母数字字符来表示项目符号/列表。

*   您可以使用下划线来“加下划线”。
