# 感动世界

> 原文：<https://dev.to/designfrontier/moving-the-world-3cbh>

亲爱的美国:

在昨天的选举之后，回想一下我们的第一任总统乔治·华盛顿说过的这些明智的话是明智的。

“我已经向你暗示政党的危险，特别是在地理歧视的基础上建立他们。现在让我从更全面的角度出发，以最严肃的方式警告你们，不要让党派精神普遍产生有害的影响。

不幸的是，这种精神与我们的本性密不可分，它植根于人类心灵中最强烈的激情。它以不同的形式存在于所有的政府中，或多或少受到抑制、控制或压制；但是，在那些流行的形式，它被认为是最大的 rankness，真正是他们最大的敌人。

一个派别对另一个派别的交替统治，被复仇的精神激化，自然党的纠纷，在不同的时代和国家犯下最可怕的暴行，本身就是一个可怕的专制。但这最终导致了更正式和永久的专制。由此产生的混乱和痛苦逐渐使人们倾向于在个人的绝对权力中寻求安全和安宁；或早或晚，一些盛行派系的首席，比他的竞争对手更有能力或更幸运，把这种倾向的目的自己的提升，在公共自由的废墟上。

不期待这种极端(然而不应该完全看不见)，党的精神的常见和持续的恶作剧足以使一个明智的人的利益和义务阻止和限制它。

它总是分散公共委员会和削弱公共管理。它用毫无根据的嫉妒和虚假的警报来煽动社区，煽动一方对另一方的敌意，偶尔煽动暴乱和叛乱。它为外国影响和腐败打开了大门，这种影响和腐败通过政党激情的渠道找到了接近政府本身的便利途径。因此，一个国家的政策和意愿受制于另一个国家的政策和意愿。"

昨晚你的派对是赢是输真的无关紧要。我们的领导人已经选出，现在我们都必须开始工作。我们的国家面临着一些历史上最具挑战性的问题，我们都需要着手解决这些问题。

我们能够解决摆在我们面前的巨大挑战的唯一方法是谦卑地共同努力。所以让我们开始吧。正如约瑟夫·史密斯明智地劝告 12 使徒定额组“围起你的肩膀，忍受这个...在天地之前，在上帝、天使和人类之前”

现在是我们所有人努力解决我们在周围世界看到的问题的时候了。我们不能袖手旁观，等着别人来解决。从小处着手，但是要开始。我们的领导人离不开我们。所以，请今天就去改变你世界的一小部分，我也会这样做，让我们一起推动世界走向更光明的明天。
