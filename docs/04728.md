# 深入学习的下一步需要什么？

> 原文：<https://dev.to/andrewlucker/what-is-necessary-for-next-steps-in-deeper-learning>

你可以在 tech.io 上探索相关的代码片段[。](https://tech.io/playgrounds/2e3ad5751ba8a756462ece41ee0b739d6925/activating-latent-semantic-models-with-natural-language-inference/input-and-output)

深度学习引入了能够从大量输入中提取潜在变量的方法。原始面部图像数据可以快速转换为情感表达、面部取向甚至嫌疑人身份等特征。深度学习同样被证明能够应用于运动控制和近期行动规划。有了这些技术，再加上大量的训练，我们明显更接近科幻级别的人工智能。然而，在*输入*和*输出*应用之间仍然存在很大的差距。在这里，我们提出了一个自然语言模型，它能够在最一般的意义上连接两个应用程序。基本轮廓是自然语言成为识别器的*输出*和规划器的*输入*。该模型还必须考虑各种可用模型的适当使用。

首先，让我们想象一个应用程序，它将通过网络摄像头捕捉到的用户面部情绪映射到普通面部情绪上。这个应用程序可以写成一个简单的快乐/悲伤的映射，用一个数字表示:积极的是快乐，消极的是悲伤。这有点限制性，因为情绪更复杂。具体来说，有四种基本情绪状态可以通过面部表情来表达:快乐、悲伤、愤怒和恐惧。

为了扩大我们的情感范围，我们可以创建一个浮点数结构，每种情感一个浮点数。对于每种情绪，积极代表积极的信心，零代表没有信心，消极代表消极的信心。

如果我们想包括其他信息，如面部方向，这个模型仍然是缺乏的。为了对这些信息进行编码，我们需要另一种结构来存储关于面部方向的可能信息。为了最大限度地提高通用性，我们还应该考虑信息缺失或部分缺失的情况。在这一点上，我们需要考虑一个更正式的语法。忽略这种语法的细节，让我们继续假设这种语法存在(例如，任何人类语言都足以编码这种信息)。

最后一步是实施推理、行动和计划。为了实现这一目标，我们应该反思我们迄今所拥有的一切。当前模型基本上只是具有置信度值的特征。为了完成推理和规划，我们还需要两个与每个特征相关的值:对齐方向和对齐距离。

没有亲身实践的例子，很难理解最后两个值。这两者中最简单的解释就是“调整距离”。创建特征置信度值的过程可长可短。在哲学、数学和其他学科中，人们普遍认为长证明更有可能是错误的，除非它们是精心设计的。由于这个原因，它帮助我们用从很短(原子)到很长(天文)的长度来标记置信值。起初，改变置信度值来解释这种现象似乎是合理的。然而，事实和证明长度通常是独立变化的，因此将它们编码在一起会导致重要信息的丢失。

最后一个特征关联值是对齐方向。这简直就是推理的语义方向。如果特征由于推论应该是真的，那么方向是向前的(从假设到结论)。如果观察到特征为真，则方向是反向的(从结论到假设)。

这就是这个模型所需要的。我将致力于将这些功能集成到我的 OpenAI 机器人中，所以请关注更新。这些技术对于无监督学习是非常必要的，这是最常见的任务，所以有很多值得探索的地方。
