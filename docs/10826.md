# 第二章:这是安德拉。

> 原文：<https://dev.to/cyberomin/chapter-two-this-is-andela>

关于重新开始，这是一份礼物。这是一个将多年积累的经验、知识和教训应用到一个人生活的新旅程、新篇章和新阶段的机会。最重要的是，这是一种祝福。

一个新的开始对不同的人可能意味着不同的东西，对一个艺术家来说，它可能意味着一张空白的画布。新的想法将被画在上面，这些想法可能是在他第二次看了他的最后一件艺术品后成为焦点的。对一个作家来说，这是崭新的一页，没有污点，等待他或她倾诉他或她的想法。写出美丽的故事，那些他或她在最后一部作品中可能遗漏的故事。这些故事将继续激励着尚未出生的几代人，这些故事可能会改变整个历史进程，为人类增添一点情趣。

在过去的几个月里，我试图分享一些对我来说非常珍贵的故事，这些故事大多围绕着我的工作——技术和科技初创公司。有几次，我涉猎了一下[小说](http://cyberomin.github.io/fiction/2016/04/10/the-dinner.html)和[生活](http://cyberomin.github.io/life/2016/08/22/mama.html)。展望未来，我会试着分享我个人感兴趣的其他事情；艺术、生活、工作等。这不是为了自我膨胀。这是留给后代的。我的孩子们总有一天会读到这些故事。这将为他们提供背景。

今天标志着我人生新篇章的开始，一个新的机会，一个新的开始。今天，我变成了安德兰人。我对未来感到兴奋，我迫不及待地想看看它会带来什么。

感谢艾因、纳德、杰瑞米、埃本、德林、纱代、塞尼、利斯比以及所有其他为这个新阶段的诞生做出贡献的人。我期待着我们一起做的令人惊奇的事情。

我们去变魔术吧。这是安德拉。#蒂亚。
