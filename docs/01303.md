# 回顾 dev.to 美好的一年

> 原文:[https://dev . to/Ben/回望美好的一年 devto-kkj](https://dev.to/ben/looking-back-on-a-wonderful-year-for-devto-kkj)

1 月 3 日 [dev.to](https://dev.to/) 两岁。但是到了 2017 年，[杰斯](https://dev.to/jess)和我决定冒险开始全职工作，而[彼得](https://dev.to/peter)作为第三位创始人加入。在过去的一年中，我们看到了前一年的大量辛勤工作在增长和曝光方面表现出来。在回顾伟大的一年时，我想提一件对我们来说很突出的事情。

[问我任何问题](https://dev.to/t/ama)的线程非常有趣。这是一个很好的方式，让我们行业中的一些人做一些有趣的工作。

社区采取建设性评论态度的方式非常好。为了有一个健康的社区，我们需要人性化的对话，我们一直惊讶于社区对行为准则的普遍尊重和对彼此的尊重。我们有几个像这样的[主题](https://dev.to/damcosset/i-think-i-love-you)，人们表达了对环境的感激之情，这温暖了我们的心。很明显，人们带着共同的价值观来到这里。我们仍有工作要做，但如果社区不接受跨所有技能水平、身份和背景的更好合作空间的需求，这一切都不可能实现。

该社区的全球性质是一个惊人的学习机会和许多乐趣。网站的流量来自世界各地，到目前为止，我们已经尽一切努力在技术上和社会上适应这一点。最近日本用户的大量涌入特别令人兴奋，看到来自各种背景和文化的人基于对这门手艺的共同兴趣走到一起，真是太棒了。当然，还有很多工作要做，我们很高兴能在 2018 年继续前进。

我们在国际妇女节那天开始了庆祝活动，来自我们社区的 163 名妇女分享了她们的故事和建议。我们的团队非常努力地让这个活动真实而有影响力。在我们美好社区的帮助下，这是一个巨大的成功。

社区通过口口相传和在平台上传播我们包容性价值观的好消息而成长。这一切都在最近的一条 Twitter 帖子中得到了展示，在这条帖子中， [Dan Abramov](https://dev.to/dan_abramov) 询问他是否应该加入 [dev.to](https://dev.to/) ，社区的反应非常好，说服他注册并提供 React 帮助。我们当然希望他和其他伟大的老师在 2018 年保持下去。

# 踏上新的一年🍾

该平台的健康和成功指标继续保持强劲，我们预计 2018 年将有很大增长。我们在 2018 年初推出了几项关键计划，以继续该平台的发展和可持续性，包括开源代码。我们非常高兴能与大家一起继续这一旅程，并认识到包容性全球软件开发人员社区的惊人但尚未实现的潜力。

对我来说，与整个社区一起学习和分享编码变得更有趣了。我知道早期采用 [dev.to](https://dev.to/) 的人也是这种感觉。我希望随着我们一起改进平台的小事情，我们可以在 2018 年及以后进一步传播这种感觉。

非常感谢你到目前为止参与其中。
