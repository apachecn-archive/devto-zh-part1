# 想成为一名优秀的团队领导吗？“平常的东西”是不够的！

> 原文：<https://dev.to/runtime-revolution/want-to-be-a-great-team-leader-the-usual-stuff-is-not-enough>

* * *

从我的职业生涯开始，成为一名团队领导就成了我职业发展的必经之路。但是说实话，我不知道那是什么意思。在过去的几年里，我有机会担任这个角色。机会可能是一个很强的词，在某个时候，我的团队领导离开了，我是下一个…

我记得调查过我过去的团队领导，并列出每个人的优点和缺点。我已经阅读了很多其他领导人的经验，并将继续阅读。同样，在现场帮助我快速努力地学习。

因此，我觉得这是一个停下来理解我在努力成为一名优秀团队领导时所应用的一些原则的好时机。今天，我知道没有一套特定的规则可以帮助你成为一名优秀的领导者。当然一些好的基础像沟通，公平，能够传播信心会有很大帮助。网络上有很多帖子关注成为一名优秀的团队领导需要具备的一些特质，比如优秀的沟通技巧、自信和有组织性。你需要这些，但它们还不够。

在我过于简单的观点中，我认为领导一个团队更像是每天玩弄一群不同情绪的人，每个人对事物都有不同的看法。像杂耍一样，你需要大量的练习！

所以今天我要讲一些我也用来领导团队的原则，这些原则可能会帮到你。

* * *

# 注意动态变化

一个团队有很多移动的部分，人们加入、离开、改变角色，人们自己也在成长和改变。那是自然的过程；它不是一成不变的。在一群朋友中，当一个新人加入这个群体时，会有一种自然的动力改变这个群体的运作方式。它会适应吸收新成员的个性。同样的事情也发生在职业生活中，但是有点曲折。人们加入公司不是一个自然的过程，这是双方明智的选择(查看我们的博客帖子，了解我们是如何招聘的)。
需要注意的是，无论是好是坏，每个变化都会给团队带来混乱。我们应该为此做好准备，并在它发生之前进行调整。例如，我们知道个性很强的人会加入，这可能会导致一些冲突(这可能是好事也可能是坏事)。此外，你团队中的一些成员可能无法利用/处理强势的个性。在这种情况下，你可以采取一些预防措施，比如之前与双方交谈，以减少影响，并尝试想象这些不同的个性如何能够一起工作。
最终，每一个变化都会产生影响，但目标是确保平衡，保持或增加生产力和团队快乐。

# 注意细节

“预防胜于治疗。”德西德里乌斯·伊拉斯谟

如果你在领导一个团队，你应该知道他们身上发生的一切。对我来说，注意周围每个人的行为模式有助于我为事情发生变化的那一天做好准备。如果一个人总是快乐的，那么在他们不快乐的那一天可能会发生一些事情。理解他们舒服时和不舒服时的行为。当他们的眼睛说他们需要帮助时，理解他们。我积极地试图意识到有些事情已经改变了，并且尽我所能去帮助别人。试图理解人们的行为模式有助于我积极地为最坏的情况做准备，并在问题变得严重之前阻止它们。

没有人喜欢回头看，看到自己还在一年前的同一个“地方”。所以我把每个人都应该在我的领导下进步作为我的个人使命。进化成更好的版本，这可能不一定包括更好地开发软件，至少是直接地。我试着传递如何沟通的知识。向他们表明，当他们看到糟糕的代码时，他们不能对其他开发人员的质量做出判断，因为他们不知道完成代码时的所有环境。教导他们有时必须积极，有时必须有耐心。我的最终目标是让他们不需要我。

* * *

# 帮助他们自助

谁没见过同事一次又一次拐错弯？这没有任何问题。我坚信我们应该让人们失败。人们会记住这些经历和感受，希望不会重复同样的过程。

有时人们不明白还有另一条路可能更容易，或者更好。我经常看到的一个例子是，当某人在一项任务上花了一段时间后，开始迷失在细节中。这很正常，可能发生在每个人身上。你必须记住着眼于大局，最终目标，并理解你和你的团队所做的事情是否继续朝着主要目标的方向发展。当我注意到某件事可能会让某人慢下来时，我会主动去和我的同事谈谈他们在做什么。然后我听他们解释他们在做什么。但在某些时候，我会问他们最终想要达到什么目标，是否有其他方法可以实现。经常向别人解释问题和解决方案有助于引发更好的解决方法。有多少次在向别人寻求帮助时，你突然知道了答案，甚至在他们开始说话之前？

# 每个人都很重要，但没有人是不可或缺的

一个团队会有不同优点和不同缺点的人。有些人会比其他人写出更好的代码。但这就意味着他们比那些能力不同的人更重要吗？就我对事情的理解，没有人比团队本身更重要。如果一个专家开发人员不能教他周围的人，为什么有人会喜欢他呢？对于一个团队来说，让某人为自己储存信息有什么意义呢？团队成员希望其他人知道他们所知道的。他们想要传播知识。他们会希望自己是可有可无的(这最终会让他们变得更不可或缺)。
当然，当有人离开的时候，团队的某个部分也跟着那个人走了。但是一个运转良好的团队可以在有人离开后生存下来，并进行调整。当我们有一个好的团队时，每个人都变得不可或缺。

* * *

这是一个没有对错之分的话题。一切都取决于上下文和周围的环境。这些不是完全可靠的原则。一个好的团队领导的定义远不止这些。对我来说，我每天都努力运用这些原则。这只是个开始，但很有帮助。
