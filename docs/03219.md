# SemaphoreCI 团队正在寻找一名高级 Rails 开发人员

SemaphoreCI 是一种建立在友好环境中的托管连续交付服务。我们的使命是通过为软件交付提供一个健壮的、可扩展的平台，帮助团队更快、更有信心地交付他们的软件。

### [](#examples-of-problems-youll-be-solving)你将要解决的问题的例子

*   发布令人惊叹的信号量功能
*   维护和改进 Semaphore 的内部和外部 API
*   自动化发布过程、监控和解决可伸缩性问题
*   针对延迟和吞吐量优化组件性能
*   帮助我们的客户改进和自动化他们的软件交付管道
*   对开源软件做出贡献并进行合作

### [](#requirements)要求

*   你有 5 年以上的软件工程经验，并且能够自如地用 Ruby 编写代码(以及使用 Rails)。
*   您拥有高度发达的调试技能和解决生产系统问题的能力
*   您关心构建可维护的、高效的、干净的代码。
*   你有出色的书面沟通技巧，包括公认的远程工作能力。
*   你能自我管理，并能做出有效的决策。
*   你喜欢帮助其他开发人员更灵活地工作。
*   你能够负责项目从构思到完成的整个过程。
*   您可以使用与中欧时区重叠的四个小时
*   你所在公司的其他人希望你参与他们的项目。

### [](#benefits)好处

*   在全球市场竞争的产品上工作的影响。
*   加入一个由大约 20 名热爱自己工作的全职人员组成的小团队。
*   健康的 40 小时工作周，友好和支持的工作环境。
*   有竞争力的薪水。
*   公司撤退。
*   不断学习的空间，为您的工作选择工具和设备
*   付费参加你选择的会议和书籍。
*   与使用 Semaphore 的开发人员交流，讨论开发和发布软件的最新和最好的方法。
*   你选择的健身俱乐部的付费会员
