# 鼓励

> 原文：<https://dev.to/craigbuchek/encouragement-42al>

上周我在墨西哥的科苏梅尔度假。一天，我们参加了一个名为 Xenotes 的短途旅行。洞穴是一个充满淡水的落水洞。(“X”是为了给它起一个更像玛雅人的商标名。)我们游泳、划皮划艇和滑索玩得很开心。一位双语导游带领我们的团队，这个团队由来自美国和南美洲的不同年龄和体能的人组成。

我们的导游很有趣。他讲笑话，给我们讲纪念碑，并带领我们参加活动。但他也鼓励我们。当有人害怕做某事时，他会给予支持。他告诉我们没问题，我们可以做到。但是我们也觉得失败也没什么，如果我们真的不能的话。感觉他的鼓励真的是在创造勇气。

真正棒的是，尽管有语言障碍，其他人也都支持和鼓励我。在有人尝试一些困难的事情之前，每个人都因受到鼓励而欢呼。一旦有人完成了对他们的能力来说很难的事情，我们就会大声欢呼。

感受到如此的支持是一种很棒的感觉。这让我觉得我在一个安全的地方，在那里我可以尝试一些超出我舒适区的新事物。我可以把滑索倒过来。我从 15 英尺高的悬崖上跳入水中。我甚至第二次跳下悬崖，尽管我有点害怕。

回到度假村后，我们在游泳池里打了几场排球。这是一个类似的情况，不同年龄和能力的球员。同样，我们试图帮助较弱的球员感到舒适，这样他们就可以提高而不会感到被评判或自我意识。它帮助每个人都玩得很开心。在这样一个支持性的环境中，一切都变得更加有趣，无论我是一个技术较高的人(排球)，还是一个技术较低的人(跳跃或滑入水中)。我们都能够事半功倍。

这些经验提供了一个很好的教训，可以在许多地方应用。这样一个支持性的环境将有助于任何关系，任何团队。我参加过几个非常好的软件开发团队，但是我不认为他们中的任何一个像这两组陌生人一样支持我。

我已经决定这应该是我作为团队领导的目标之一。我想为整个团队创造一个鼓励的环境。我想确保每个人都足够舒适，他们觉得他们可以尝试困难的事情，即使他们可能会失败(只要没有人受伤)。

如果一群陌生人能做到这一点，你的团队也能。你和你的另一半也可以。我们需要每天工作来确保我们互相支持。这是让每个人取得更多成就的最佳方式。
