# 正如我们应该认为的

> 原文:[https://dev.to/rfunk82/as-we-should-think](https://dev.to/rfunk82/as-we-should-think)

道格拉斯·恩格尔巴特是早期的计算机先驱。他的灵感来自于 Vannevar Bush 在 1948 年写的一篇文章《T2》。他们使用电脑的想法不是为了让生活变得更容易，而是为了提高我们的智力，让我们变得更好，改进我们所拥有的工具，让我们作为人类变得更好。遗憾的是，电脑对人类产生了不同的影响，我们喜欢分享我们宠物的照片，我们喜欢在网上看有趣的视频，仅此而已。我想知道我们是否能改变我们使用为我们制造的伟大工具的方式。

我们再也没有借口了！世界的知识都是你可以得到的。有些人可能会责怪社交网络、游戏等，但问题不在于车，问题在于司机！内容是任何人都可以得到的，特别是现在，你可以免费找到任何信息..在家里..在你自己的时间里。基本上，如果科技让我们变得愚蠢，那不是科技的错，而是我们的错。
