# 使用数组资源管理器理解 JavaScript 数组方法

> 原文:[https://dev . to/pauljacobson/making-sense-of-JavaScript-array-methods-with-array-explorer-123h](https://dev.to/pauljacobson/making-sense-of-javascript-array-methods-with-array-explorer-123h)

前几天，莎拉·德拉斯纳在推特上分享了她令人敬畏的[阵列探测器](https://sdras.github.io/array-explorer/)工具。它的设计非常简单，但却是学习 JavaScript 数组的强大方法。你要做的是从下拉列表中选择几个选项，找到你需要的项目的数组方法。

我仍然觉得数组很有挑战性，但是学习如何在 JavaScript 中使用它们是非常重要的。如果你还在琢磨这些东西，一定要看看 Array Explorer，把它加入书签，然后使用它。

Sarah 甚至在 GitHub 上发布了 Array Explorer 的代码,这样你就可以看到她是如何将这些代码组合在一起的。
