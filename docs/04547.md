# 人工智能之旅

> 原文：<https://dev.to/iamjoross/journey-to-artificial-inteligence-2ba>

有人可能会说，让自己学习人工智能是非常令人生畏的。嗯，事实上是的。人工智能是如此先进和多样化的知识流，一旦你遇到这个词本身，你就会被焦虑的想法淹没。

对我来说，掌握人工智能知识的可能性是人工的。我从来没有想过穿过这条路去学习它。想象一下你需要沉浸其中的算法、数学和科学。在我看来，只有出现奇迹，我才敢开始尝试。这实在是太专业了，超出了我的知识范围。

我想多了。焦虑和恐惧占据了我。学习新东西真的会让人害怕，尤其是如果你对这个话题一无所知的话。然后我意识到，如果我继续让我的不安全感占上风，我怎么能成为一个更好的程序员和更好的计算机科学家呢？

最终，当我报名攻读计算机科学硕士学位时，我受到了激励。当学习人工智能时，你可能到达的地方的可能性是指数级的。在我开始阅读 Peter Norvig 和 Stuart J. Russell 关于[人工智能:一种现代方法](https://www.google.com.ph/url?sa=t&rct=j&q=&esrc=s&source=web&cd=3&cad=rja&uact=8&ved=0ahUKEwjZ7eC4qsXWAhUEx7wKHXqwADoQFgg2MAI&url=https%3A%2F%2Fwww.amazon.com%2FArtificial-Intelligence-Modern-Approach-3rd%2Fdp%2F0136042597&usg=AFQjCNHvNYlS6mDJvewRpMnxOnVDtrztGg)的书后，我变得更加感兴趣。从那以后，我无法阻止自己学习。我真的很投入。

随着我在人工知识方面的知识的进步，我想与你分享我可能能够解释和教授的东西。我将发表一系列关于人工智能的话题。让我们一起学习，探索人工智能的世界。

这是我的人工智能之旅，它才刚刚开始。我对这次旅行充满渴望和兴奋！
