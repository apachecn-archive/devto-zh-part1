# 人文科学的计算机科学

> 原文：<https://dev.to/lumochift/computer-science-for-humanities-f9f>

> 计算机科学与计算机无关，正如天文学与望远镜无关，生物学与显微镜无关，化学与烧杯和试管无关。科学不是工具。这是关于我们如何使用它们，以及当我们使用它们时会发现什么。— **迈克尔·费罗斯和伊恩·帕贝里**

想象一下，如果今天没有计算机技术，也许我们仍然以手动方式做一切事情，到处都是纸，没有小工具，互联网连接，智能设备和许多东西。我们生活在电脑时代，有了新的生活方式，电脑对人类真的很有帮助。但是如果你听到计算机科学学位，你首先会想到什么？在我大学的第一个学期，我想象如果我将学习软件像 photoshop，flash，修理计算机硬件，和游戏。我错了，因为我遇到数学，编程，基础自然科学，和其他意想不到的科目。

> 计算机科学是一门文科。它是每个人都应该知道如何在生活中使用和驾驭的东西。— **史蒂夫·乔布斯**

我的心刚刚反叛，为什么我得到了这一切。为了什么？为什么我一定要学生物、物理、化学、微积分、线性代数、数值方法、运筹学、统计学、概率定理、离散数学和与计算机科学毫无关系的数学东西。缓慢但肯定的是，我终于知道计算机科学是否只是连接到其他学科的工具。在进入另一个学科领域之前，我想分享一个关于计算机科学中基础数学的故事。我太天真了，一开始不喜欢数学，后来我意识到每一门计算机科学都是以数学为基础的。

为什么要学微积分？因为要提前计算题目，在 CS 你会需要它们，比如`digital image processing`你会面临积分，求导。你必须学习微积分和数值方法，因为计算机是有限的，不是连续的。你必须将计算从连续转换为离散，这样计算机才能在计算中帮助你。

如果我们想创造电脑游戏，我们必须了解精神，艺术，历史，数学和最优化。就像愤怒的小鸟游戏一样，它将恼人的抛物线定理转化为美丽而有趣的电脑游戏。

生物信息学、医疗记录、信息系统，一切都可以与计算机科学联系起来，甚至像使用`artificial intelligence`预测股票这样的经济领域。在这之前，我们必须学习自然科学、人工智能、信息系统、小会计或支持你工作的每一门学科。

所以计算机科学不会以计算或计算机本身而告终，而是遍布不同的领域，我们可以想象无限的场景。如果你已经投入到计算机科学的世界中，我希望我们能互相支持，让计算机科学变得容易理解和有趣，因为计算机科学是为人类服务的。
