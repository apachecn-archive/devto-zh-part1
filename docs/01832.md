# 圣诞夫人拯救了圣诞节

圣诞老奶奶知道她必须小心向圣诞老人提起这件事。“你的书越来越大了，亲爱的，”她说。

他把它像一盒纸巾一样捡起来。“不，这很好。”他忽略了装订完全破损的事实，因为这本书现在已经超过一百万页了。

他们都知道记录哪个孩子收到了什么礼物是他们工作的核心。他们目前的系统已经不起作用了。但是圣诞老人有点勒德分子。他固执己见，拒绝改变。作为一个传统的人，他总是把智能手机放在家里，从来没有真正了解过电子邮件是什么。

圣诞夫人大部分非假日时间都在和 KWS 一起工作(Kringle Web Services，由圣诞精神驱动)。她热爱技术，甚至从远程网络开发训练营毕业。她站在将应用带到北极和简化流程运动的最前沿。

当她向圣诞老人提出重要的工作时，他总是伸出一只手，善意地说:“呸！”他喜欢现在的生活。他把注意力集中在玩具和欢乐上，把其他的留给其他人。

这意味着圣诞夫人不得不花大量的时间来管理玩具生产中的小精灵。这一年的大部分时间还不算太糟，但最后三个月，事情变得异常繁忙。这意味着她最重要的应用程序的目标受到了影响。

圣诞老人不知道的是，圣诞老人制作了一个礼物追踪应用程序来代替他的书。精灵们不太喜欢这个名字，说听起来她在追踪 gif，而不是礼物。但是圣诞夫人坚持说这是一个时髦的名字，就像现在所有的孩子一样。

GifTracker 使用 GPS 跟踪圣诞老人的位置，并自动在屏幕上显示附近的孩子和他们应该收到的玩具。克劳斯夫人有一个 QA 团队，在应用程序投入生产之前，他们要对其进行两周的测试。但是玩具生产中的所有工作都不允许这种情况发生，因为这意味着圣诞老人直到圣诞节前两天才得到她的最终代码。

“圣诞老人，我给你做了一个应用程序来代替那本书，”圣诞老奶奶在平安夜前夜告诉他。"它使用简单，会让你的圣诞节更加愉快."

圣诞老人对她扬起眉毛，显然没有被说服。

“唯一的问题是，它还没有经过测试，”她悲伤地告诉他。

圣诞老人并不知道那是什么意思，只知道他可以利用它。“那我今年还可以用这本书！我们可以在明年推出这款应用。”他拍了拍她的背。“我一点也不介意。”

克劳斯夫人叹了口气，充满了疑虑。

那天晚上，精灵们开始装载圣诞老人的雪橇。有很多事情要做，他们在到处堆积的玩具中跑来跑去，忙得不可开交。这就是为什么他们没有注意到当玩具被搬动时，圣诞老人的书一直在慢慢靠近炉火。这就是为什么他们错过了跳到书页上的余烬，并点燃了它。

只有当火焰到达天花板时，精灵们才开始尖叫。他们把热巧克力扔进火里去灭火，但是太晚了。这本书被毁了。

精灵的声音吵醒了克劳斯夫人。她徘徊在楼梯顶端，向下凝视着混乱的人群。一个精灵看到了她，叫道:“这本书毁了，伊莲！”(圣诞夫人的名字是伊莲。)

克劳斯夫人突然行动起来，冲下楼梯。“圣诞老人的平板电脑在哪里？”她问道。

精灵们交换了困惑的眼神。“什么平板？”

"他用在雪橇上的那个"

精灵们把圣诞老人的最爱佩妮推到了前面。“跟我来，”她告诉圣诞夫人。“有些东西你得看看。”

她把她领到雪橇前，给她看放在前座上的一台沉重的黑色笔记本电脑。“这是什么？”克劳斯夫人问。

“等着吧，”佩妮告诉她，她点燃了蜡烛。

当它启动时，克劳斯夫人听到一个熟悉的声音。“哦，不。那是——”

“是的，”佩妮说。“他运行的是 Windows 95。而且他还在用 ie 浏览器。如果他上网的话。他大部分时间都在玩单人纸牌游戏。”

“哦，我的天，”圣诞老奶奶说，她踉跄着向后倒去。一个聪明的精灵预见到了她的反应，把她抓到了椅子上。

“他一生中从未用过触摸屏，”佩妮继续说道。“不过别担心！你的应用甚至不需要交互。我们可以在这里安装一个平板电脑，屏幕会显示他需要的信息。我们成功了…怎么说呢…”

“白痴证明，”圣诞老奶奶为她完成。“但我们要在圣诞节推出一款未经测试的应用？”

“最重要的是我们要说服他戴上与之相配的手镯，”佩妮说。“我们知道这部分是可行的。”

“但现在一切都要运转起来。我们知道这本书可能是一个退路，所以我们做了这个。”

佩妮耸耸肩。“我们没有其他选择。我认为最好是在剩下的几个小时里，用 QA 解决一些挥之不去的问题。我来替你管理精灵。”

克劳斯夫人深吸了一口气。“这能行。它必须发挥作用。”

“就是这种精神！”

第二天，圣诞老奶奶将她的应用程序加载到平板电脑上，并安装到圣诞老人的雪橇上。然后她去给圣诞老人传递关于他的书的消息。

“它不见了！?"圣诞老人怒喝道。

“这是烤面包，”她告诉他。“但好消息是，这款应用非常好用。它已经装在你的雪橇上，随时可以出发。你所要做的就是戴上这个手镯。”她在他的手腕上戴了一个橡胶手镯。

"我想我在圣诞节使用电脑只是时间问题。"

她拍拍他的手臂。“没错。我想你会喜欢它的。”

“我们走着瞧吧，”圣诞老人一边爬上雪橇一边说道。

当他飞上夜空时，圣诞老人挥手致意。他一消失，佩妮就骑着驯鹿跑了过来。一个精灵递给她一个背包，她背上。“谢谢你帮我照看他，”圣诞老奶奶告诉她。

“总得有人去，”佩妮说，然后她就飞到了空中，在安全的距离外跟着圣诞老人。

圣诞老人知道他的雪橇，就像圣诞节的许多魔法一样，是由圣诞精神驱动的。他知道，随着人们说“节日快乐”越来越多，“圣诞快乐”越来越少，这种精神正在消失。但他没有注意到这也极大地依赖于他自己的健康。

这就是为什么圣诞老人夫人设计了一个可以跟踪圣诞老人血糖的腕带。他的医生警告他说，他是糖尿病前期，必须注意饮食，但圣诞老人不把他当回事。圣诞夫人已经用健康的替代品取代了北极的大部分食物，这些替代品尝起来与圣诞老人期待的甜食非常相似。但是对于他在圣诞节晚上将要拜访的无数家庭来说，情况就不一样了。

这就是为什么佩妮总是吃着美味的无糖高蛋白饼干。当圣诞老人的糖含量上升时，她会通过 GifTracker 得到警告，她会冲过去用她健康的替代品代替给他留的含糖饼干。圣诞老人一点也不知道，圣诞节就有救了！

晚上结束时，圣诞老人回到北极，爬上床，躺在圣诞老奶奶身边。“又是一个成功的圣诞节，”他打着哈欠宣布道。

“对你有好处，亲爱的，”圣诞老奶奶说。然后她在黑暗中对自己微笑。哦，要是他知道就好了。
