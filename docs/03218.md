SemaphoreCI 是一种建立在友好环境中的托管连续交付服务。我们的使命是通过为软件交付提供一个健壮的、可扩展的平台，帮助团队更快、更有信心地交付他们的软件。

### [](#what-you-will-do)你会做什么

*   计划、构建、测试和交付每月一次的 Semaphore Linux 和 Docker 平台更新，比如这个。
*   通过可重用的代码和工具开发和维护基础设施。
*   使用度量和监控系统来确保和提高性能和稳定性。
*   必要时通过我们的支持渠道与其他团队(技术和非技术)、管理层和信号量用户沟通。
*   维护和改进帮助信号量用户使用平台的公共文档。

作为一名 DevOps 工程师，您将负责构建和支持一个基于 Linux 的环境，在 Semaphore 上运行 CI/CD 作业。这是一个很好的机会，加入一个高技能的团队，扮演一个非常明显的角色，并对我们的用户群产生重大影响。

这是一个远程位置，你应该在异步通信中茁壮成长。我们希望你有很强的书面沟通技巧，能够与全球各地的同事发展工作关系。

作为一名 DevOps 工程师，您还将在帮助客户利用 Semaphore Linux 和 Docker 平台方面发挥重要作用，帮助他们处理技术支持请求。你总是会被挑战去解决有趣和新奇的问题，这些问题对世界交付软件的方式有真正的影响。

### [](#skill-amp-qualifications)技能&资历

*   从高层次和低层次对 Linux 操作系统有深刻的理解。
*   能够快速调试和解决配置、性能和可靠性问题；如果存在直接影响产品用户的问题，有时会面临压力。
*   标准方法的知识:配置管理、监控、警报、指标。
*   能够使用 Ruby 和 Bash 等高级语言工作。
*   优秀的书面和口头沟通技巧，包括与团队远程有效工作的能力。
*   自我管理，能够在快速变化的环境中做出有效的决策。
