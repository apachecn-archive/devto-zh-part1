# RubyConf 不错

> 原文:[https://dev.to/yechielk/rubyconf-is-nice-9l1](https://dev.to/yechielk/rubyconf-is-nice-9l1)

### [](#reflections-on-rubyconf2017)关于 RubyConf2017 的思考

当我写这篇文章的时候，我还在从上周的高潮中走下来。

那些关注我的人知道，我被选中获得了今年在新奥尔良参加 RubyConf 的奖学金(你可以在这里阅读相关内容)。

这次经历绝对没有辜负我的期望，我学到了很多，收获了很多，遇到了很多友好的人，总体来说玩得很开心。

以下是我的一些想法和亮点，排名不分先后:

#### [](#0-rubyconf-is%C2%A0nice)0。RubyConf 不错

我知道这已经是一个被过度使用的比喻了，Rubyists 是多么的好；就好像他们是编程世界的加拿大人，但是我的天啊，我从来没有见过这么友好的社区。我进来时几乎一个人都不认识，我遇到的几乎每个人都主动带我参观，把我介绍给他们认为有价值的人，并尽最大努力让我的第一次会议体验变得很棒。

#### [](#0-ruby-is%C2%A0nice)0。露比很好

尽管我是从学习 Ruby 开始的，但我最近的大部分采访和大部分工作都是用 JavaScript 完成的。我差点忘了 Ruby 编程能给你带来多少快乐。这次会议让我想起了当初为什么会爱上 Ruby 和编程。我当然希望我的第一个开发角色是 Ruby！

#### [](#0-matz-is%C2%A0nice)0。马茨很好

我知道我说过卢布主义者有多好，但我们都知道我们欠谁的。会议的亮点之一是有机会走到 Matz 面前，亲自感谢他编写了让我最初爱上编程的语言。自拍是纯利润。

#### [](#0-dying-is-nice-or-at-least-not-the-end-of-the%C2%A0world)0。死亡很美好？(或者至少不是世界末日)

出于某种原因，这是今年 RubyConf 的一个中心主题。我猜你必须在那里(或者至少在视频出来的时候看一下 [@chadfowler](https://twitter.com/chadfowler) 的主题演讲)。

#### [](#0-all-dressed-up-and-everywhere-to%C2%A0go)0。盛装打扮，无处不在

我不知道我为什么要费心打包衣服。当所有的 t 恤和袜子都被分发出去的时候，我本可以背着这件衬衫出现(然后把它扔在门口)。见鬼，仅仅是贴纸就足以让我穿得体面些了:)

#### [](#0-flatiron%C2%A0pride)0。熨斗骄傲

来自一个非传统的编码背景，你不能帮助那些自我怀疑的时刻。我做了正确的决定吗？我真的能成为一名“真正的”程序员吗？在会议上遇到熨斗校友真的是信心的助推器。事实上，他们中的三个人在那里发表演讲(不包括我自己的闪电演讲)，有助于永久搁置这些疑虑。

#### [](#0-flatiron%C2%A0buds)0。扁铁芽

说到熨斗，在熨斗的这一年里，我交了很多好朋友。这些朋友中的大多数只不过是我脑海中松散的用户名和模糊的个人资料图片。在 RubyConf，我终于有机会见到他们中的一些人。我们一起出去玩，一起吃馅饼，其中一个甚至主动提出打开她的房子让我住(谢谢，乔安娜！)终于有机会把人脸和用户名对上了，这真是太棒了。

#### [](#0-putting-a-face-to-the-personality)0。把一张脸放在人格上

说到人脸识别，在 RubyConf 的三天里，我认识并在过去两年中学会喜欢的许多名字都出现在我的生活中。从像 [@avdi](http://twitter.com/avdi) 、 [@sarahmei](http://twitter.com/sarahmei) 、 [@tenderlove](http://twitter.com/tenderlove) 和 [@eileencodes](http://twitter.com/eileencodes) 这样的 Twitter 手柄，到像 Sandi Metz 和 Noel Rappin 这样的每天都在写书塑造我的代码的作者，到 Ruby Yukihiro Matsumoto (Matz)的作者本人。所有这些名字原来都是人，而且是非常平易近人和友好的人！

#### [](#0-all%C2%A0talk)0。所有的谈话

万一参加一个有 800 个陌生人参加的会议还不足以突破我舒适区的极限，我自愿做一个闪电演讲！对于那些不知道的人来说，闪电谈话是一个半小时的会议，任何人都可以站在几百人面前，就任何他们想谈的话题谈 5 分钟。从我(自称的)Rails 拉比的头衔中汲取灵感，我发表了题为“程序员的塔木德真理”的演讲这里可以看[。](https://dev.to/yechielk/talmudic-truths-for-programmers--the-lightning-edition-62)

#### [](#0-and-all%C2%A0play)0。所有游戏

如果你认为 Ruby 会议都是无聊的演讲，那你就大错特错了！没有人像 Rubyists 一样知道如何找乐子。如果是在水族馆的深夜派对，我第一次抚摸黄貂鱼，随着 DJ 在屏幕上现场编码他的 Sonic PI 跳舞，穿过法国区街道的 Heroku 主题游行，卡拉 ok 之夜，或者只是玩棋盘游戏的安静夜晚，从来没有沉闷的时刻。

说了这么多，做了这么多，毫无疑问，最后一周对我的程序员生涯产生了变革性的影响。会谈、人们和社区一起提供了一个令人惊奇的体验。我等不及要回来了。谁知道呢？也许明年我会有机会把它作为一个刚开始踏上人生道路的学者的向导？
