# 30 秒后颤动

> 原文:[https://dev.to/nitya/flutter-in-30-seconds-1nk1](https://dev.to/nitya/flutter-in-30-seconds-1nk1)

[Flutter](https://flutter.io/) 是来自谷歌的*开源* SDK，帮助你从单一代码库中为现代移动平台(目前是 Android & iOS) *创建原生应用，具有广泛的*工具和部件支持*用于快速原型制作。*

与其他相关解决方案(如 React Native、PhoneGap、Cordova)相比，Flutter 首先关注的是**性能**，它直接编译成本机(ARM)代码，并通过使用与 GPU 协同工作的 [UI 架构](https://docs.google.com/presentation/d/1cw7A4HbvM_Abv320rVgPVGiUP2msVs7tfGbkgdrTy0I/edit)直接渲染到 canvas，消除了上下文切换到 OEM 小部件的开销。

Flutter 的[分层架构](https://docs.google.com/presentation/d/1cw7A4HbvM_Abv320rVgPVGiUP2msVs7tfGbkgdrTy0I/edit#slide=id.gbb3c3233b_0_162)加上它的开源性质，使得开发者可以**定制框架内的一切**，同时依靠久经考验的引擎获得优化的文本(来自 Chrome)、图形( [Skia](https://skia.org/) ，用于 Android)和框架(Dart，用于谷歌的生产)支持。

Flutter 目前在 alpha 版本中。你可以在我于 2018 年 1 月在芝加哥#WindyCityDevFest 举办的讲座中获得更多资源的快速介绍和链接，或者在[尝试 codelab](https://codelabs.developers.google.com/codelabs/flutter/#0) 进行实践学习。
