# 电线之间是什么

> 原文：<https://dev.to/betweenthewires/what-is-between-the-wires>

**在电线之间**是一个采访系列，以那些正在构建开发人员产品的人为特色。在这里，我们希望传播思想，鼓励创新，并提供一个幕后的开发人员产品的创作一瞥。

Preethi Kasireddy 目前是 T2 比特币基地 T3 公司的一名软件工程师，尽管她的发展道路有些曲折。在大学学习工业和系统工程后，Preethi 进入高盛投资银行，后来成为 Andreessen Horowitz 的风险投资家。在那里，她与数千名开发人员和创始人就一系列产品进行了交流。她对他们所做的以及他们是如何做到的非常着迷。在看到他们的产品改变了世界后，她开始研究代码，这样她也可以创建一些重要的东西。起初只是一种消遣，但很快就变成了对编程的热爱。不久之后，她[离开了安德森·霍洛维茨](https://medium.com/swlh/why-i-left-the-best-job-in-the-world-3689a5a4649a#.j5fumemjh)成为一名全职程序员，并为成为一名创造者打下基础。线之间是她对创造者的迷恋的自然延伸，并提供了一个收集她最初发现如此鼓舞人心的一些开发者故事的地方。

## **我们为什么在乎**

我们觉得这个社区缺少的是原始的细节和内部的独家新闻，关于如何成为一个制造者并从头开始创造一个有价值的产品。我们关心这样的问题:

*   是什么促使创客打造开发者产品？
*   是什么让他们走上了今天的道路？
*   他们对编码的第一次体验是什么？
*   获得项目资金的各种途径是什么？
*   有没有可能以经济上可持续的方式从事开源项目？
*   鉴于开发者生态系统的快速创新，他们如何确保一个产品在多年内保持相关性和有用性？

此外，我们还想关注一些更难的问题，如:

*   开发人员在运营业务时面临的最大挑战是什么？
*   他们害怕失败吗？他们是如何克服的？

带着这些问题，我们开始了对创客们的一系列采访。我们这些采访的目的是分享通常隐藏在产品或 Github 回购背后的经验，以便每个人，包括有抱负的制造商，都可以从中学习。

我们将发布我们项目的第一集，从 zeit.co[的首席执行官](http://zeit.co/) [Guillermo Rauch](https://twitter.com/rauchg) 的故事开始。

我们希望这个项目能激励那些想做点什么的开发者。我们希望提供一个透明的视角，让人们了解这些制造商为了他们的梦想所面临的各种挑战和困难。我们希望它能给其他人勇气和动力去做同样的事情。
