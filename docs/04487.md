# Magento 客户属性扩展

> 原文:[https://dev . to/simonwalkerfme/magento-customer-attributes-extension-a0e](https://dev.to/simonwalkerfme/magento-customer-attributes-extension-a0e)

Magento 客户属性扩展允许你在用户注册表单中添加新的字段。有了这样的自定义属性，您可以以一种简单的方式直接从客户那里获得必要的数据。这个扩展允许您添加多种类型字段，设置它们的默认值，并使它们成为可选的或强制的。

使用此 Magento 添加客户属性扩展，您可以添加以下自定义字段:

*   文件
*   图像
*   文本字段
*   文本区域
*   日期
*   仅消息
*   掉下
*   多重选择
*   是/否
*   单选按钮
*   复选框字段

支持的输入验证

*   小数
*   整数
*   电子邮件地址
*   网站 URL 地址
*   仅字母
*   字母和/或数字
