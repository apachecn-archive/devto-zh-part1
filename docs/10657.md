如果你错过了其中的任何一个，这里有一些我们行业过去一周最值得关注的链接。

视频压缩编码标准 H.264 令人着迷。这篇文章对 H.264 和一般压缩有很高的见解。

[DeepMind 和暴雪将发布星际争霸 2 作为人工智能研究环境](https://deepmind.com/blog/deepmind-and-blizzard-release-starcraft-ii-ai-research-environment/)星际争霸 2 将向全球的人工智能和机器学习研究人员开放。这对任何人工智能程序来说都是一个巨大的挑战，并将成为展示该领域令人难以置信的进步的一个伟大平台。这项研究的结果将被广泛讨论，就像 [AlphaGo 挑战赛](https://en.wikipedia.org/wiki/AlphaGo_versus_Lee_Sedol)一样。

[网页字体，孩子，我不知道](http://meowni.ca/posts/web-fonts/)关于自定义网页字体如何影响网页上的绘画性能以及实现它们时需要考虑的事情的批判性阅读。

生产中的 Docker:一段失败的历史一名开发人员写下了自采用 Docker 以来他们所面临的一系列令人沮丧的问题。文章的主题颇有争议，值得一读页面上的评论和黑客新闻上的

[我不喜欢电脑](https://www.happyassassin.net/2016/11/04/i-dont-like-computers/)一个开发人员与他们的工作和我们生活中无处不在的技术的关系

Pornhub 通过 WebSockets 绕过广告拦截器分析该公司如何巧妙地避开广告拦截器检测。帖子已经更新，显示几个广告拦截插件已经发布了解决方法来防止这种情况。猫捉老鼠的游戏还在继续。

[我为什么成为一名软件工程师](https://dev.to/edemkumodzi/why-i-became-a-software-engineer)一名软件开发人员讨论他们进入这一领域的动机。这篇文章结合了独特的个人轶事和个人动机的相关问题，非常值得一读。

[Cylance 披露投票机漏洞](https://blog.cylance.com/cylance-discloses-voting-machine-vulnerability)这是一个演示投票机漏洞的概念验证视频。明天是选举日，我得说这很令人担忧。ðŸ˜

[Jasonette——只用 JSON 创建原生 iOS 应用程序](https://github.com/Jasonette/JASONETTE-iOS)这是原生应用程序开发的一个迷人方法，通过 JSON 使用大量配置来编译一个令人印象深刻的高性能原生体验。这是一个两极分化的想法，非常有趣。
