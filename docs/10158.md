# 我是《榆树在行动》的作者。问我什么都行！

> 原文:[https://dev . to/rtfeldman/I-am-the-author-of-elm-in-action-ask-me-any 的作者](https://dev.to/rtfeldman/i-am-the-author-of-elm-in-action-ask-me-anything)

我是理查德·费尔德曼，[榆树在行动](https://www.manning.com/books/elm-in-action?a_aid=elm_in_action&a_bid=b15edc5c)的作者和[前端大师榆树工作坊](https://frontendmasters.com/workshops/elm/)的讲师。我现在从事的主要开源项目是 [`elm-test`](http://package.elm-lang.org/packages/elm-community/elm-test/latest) 和 [`elm-css`](http://package.elm-lang.org/packages/rtfeldman/elm-css/latest) 。

我在诺丁克公司工作，那里有 80000 行 T2 榆树代码在生产。我们在 2015 年引入了 [Elm](http://elm-lang.org/) ，从那以后我们的生产 Elm 代码总共抛出了零个运行时异常。[我们在 2015 年聘请了 Elm 的创造者 Evan Czaplicki](http://tech.noredink.com/post/136615783598/welcome-evan) 继续开发这种语言，并且[我们正在招聘！](http://noredink.com/jobs)

我会在这里直到太平洋时间中午。问我什么都行！
