# 没有自我的自信

> 原文：<https://dev.to/designfrontier/confidence-without-ego-j3l>

当我年轻的时候，我认为谦逊是把自己放在别人之下。相信他们比我优秀。屈从于他们的意见和要求。我想到了在别人面前谦卑地低头的典型僧侣。以免与他人竞争。他按照指示做自己的事情，不为任何人着想。我认为在某些方面，谦逊是自信的对立面。

我现在没有那种感觉了。我相信谦逊是没有自我的自信。它是可教的。这是一种学术理想，充满自信地为自己的案例辩护，但在被证明是错误的时候愿意靠边站。重视真理，接受它，即使它与以前的观点相左。这使得它成为一个成功团队所有成员的基本属性。

这也是为什么体育运动中的全明星队经常会纠结的原因。这就是为什么 rockstar 的开发者经常离开，也是为什么杰斯·沃伊特继续给我留下深刻印象。如果你不知道谁是 Jens，我很抱歉...去了解一下他吧。他是一名世界级的运动员，他放弃了获胜的机会，去支持他的团队中其他成员的事业和抱负。这也是自行车爱好者喜欢他的原因，甚至超过了他的天赋。

找到自信和消除自我之间的平衡很难，这需要我们付出努力。这既是非常个人化的，也是我们团队、家庭和自我运作的基础。我不知道做这件事的确切方法，但我知道这是可能发生的。如果你想让你的团队以你从未想过的方式成功，那它就必须发生。

但这确实发生了。想想迈克尔·乔丹。

不久前，我在一个团队中开发一个 web 应用程序。它有一些我们从未解决的新的技术挑战，我们刚刚开始形成一个团队。我们也都是年轻、聪明、有才华、自信的开发人员。我仍然认为团队的其他成员是我合作过的最好的开发人员。但是我们都不在乎，结果很神奇。

举个例子，我为一个宗教非营利组织工作。在事后分析中，我们听到我们的开发主管分享了他在我们工作时的神圣和个人印象。他谈到外部灵感如何帮助他解决问题。然后，许多团队成员用他们自己的例子附和道。我们都有过。我们都认识到，是我们自身之外的东西促成了我们的产品。部分原因是团队为产品而工作，而不是为了获得认可。另一部分是我们正在做的。

无论你对上帝有什么感觉，这里的教训是认识到我们的才能和工作的外部来源是很重要的。它转移了我们的注意力，让我们能够体贴他人。因为如果灵感来自我们，那么它就是关于我们的。但是如果它来自我们之外，那么我们就能够把自我放在一边。

这个概念是旧的...事实上，天才曾经在我们之外。我们谈论拥有天才而不是成为天才。接受这一点让我们做决定，同时很容易重新评估我们已经做出的决定。布琳·布朗[比我在这里](http://www.ted.com/talks/brene_brown_on_vulnerability?language=en)谈论这个更好。

总结一下，一些快速的想法:

1.  这不是你的问题，这是整个团队的问题
2.  这不是正确与否的问题，而是做正确的事情
3.  唯一重要的是找到正确的东西
4.  不同意是可以的，在共识和数据面前不同意是炫耀
5.  先听听
