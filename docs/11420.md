# 把它给我

> 原文:[https://dev.to/emmanuelgautier/undo-a-git-add-3fh6](https://dev.to/emmanuelgautier/undo-a-git-add-3fh6)

可能会发生不必要的“git add”。这也是为什么会有重做这个“git add”的命令。下面是执行此操作的命令行。

```
git rm --cached <file> 
```
