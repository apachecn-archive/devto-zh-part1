# 成为高一学生第一部分——书籍和数据

> 原文：<https://dev.to/designfrontier/being-senior-part-1--the-books-and-data-db8>

这只是一个快速的帖子，为今天参加我在犹他州 2016 年 JS 大会上的演讲的人提供我引用的书籍和数据的链接。我把我的想法写在一个长的帖子里，希望能在接下来的几天里发布。

所以，我们不再赘述……以下是本次调查中的匿名数据。较长的表格将更多地介绍我用来评估信息的方法，以及其中的所有颜色和数字的含义。目前...好好享受。

我提到的书籍主要是关于工艺和职业道德的发展，它们是:

*   马修·克劳福德的《作为灵魂工艺的商店阶级:对工作价值的探讨》

*   彼得·科恩(Peter Korn)的《我们为什么制造东西以及为什么它很重要:一个工匠的教育》

*   沃尔特·伊萨克森著

*   马尔孔·格拉德威尔的《眨眼:不假思索的思考的力量》

我没有提到的一个很好的三部曲是罗伯特·m·皮尔西格的《禅宗和摩托车维修艺术:对价值观的探究》。

这些不是编程书籍...但是它们会帮助你成为更好的程序员。
