# 如果你是一个初级前端开发人员，如何制作作品集？

> 原文:[https://dev . to/kozorezv/how-to-make-portfolio-if-you-a-beginner-frontend-developer](https://dev.to/kozorezv/how-to-make-portfolio-if-you-are-a-beginner-frontend-developer)

要得到第一份工作，你必须有一个文件夹。但是...如果你还没有真正工作，你怎么能做一个投资组合呢？！如果你学习前端，它会变得更加复杂。为什么？因为当你是设计师时，你只需在 Behance 上注册并上传图片。换句话说，如果你是开发者，你可以在 github 上展示你的代码。但我们是前端开发者，这意味着图片和代码对我们来说都很重要。我也面临这个挑战，下面是解决方案。

它叫做 [Github 页面](https://pages.github.com/)。正如你可能理解的那样，这是 Github 的一项服务。它允许你从你的仓库创建一个站点！是的，就是这么简单！你只需上传你的目录，经过几个简单的动作后，你就会得到一个免费的托管网站，可以展示你的作品和它的代码。仅此而已。希望，我的这一小段经历能帮助到某个人。祝你的第一个真正的项目好运！永远记得和没有经验的人分享知识。
