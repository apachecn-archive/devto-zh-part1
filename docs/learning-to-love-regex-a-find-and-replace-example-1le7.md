# 学会喜欢正则表达式:查找和替换示例

# 问题

我的博客发布渠道中的某个人——Github、Jekyll 等。–更改了标题的格式。我不得不指定`# Heading`，而不是`#Heading`。这是有道理的，我刚刚从其他地方转换了这个博客，起初它并不是我优先考虑的事情。但是突然事情就坏了，我不得不赶紧改了很多标题。

# 解

我需要找到任何`##[Any Character]`的实例，并在它们之间添加一个空格，跨越我所有的博客文件。

这种事情总会在我脑子里掀起一个正则表达式的灯泡。老实说，从任何意义上来说，我都不是正则表达式的专家，但是通过一些谷歌搜索，我能够找到它。

## 创建正则表达式

要求是找到两个英镑符号`##`后跟任何字母(大写或小写)。

这个的正则表达式是`##[A-Za-z]`。

## 添加捕获组

为了让 Notepad++能够替换正则表达式(而不仅仅是找到它)，我们必须创建一个捕获组来捕获第一个字母(这样`A`就变成了带空格的`A`)

为此，我们将第一个字符部分用括号括起来，得到`##([A-Za-z])`。这将创建一个捕获组注意，Notepad++可以使用`\1`引用它。

## 替换文本

*   在 Notepad++中，我们打开“查找和替换”窗口。
*   我们搜索`##([A-Za-z])`
*   我们将其替换为`## \1`(原始标题，加一个空格，加第一个字符)
*   我们确保搜索模式是正则表达式
*   我们对所有文件执行查找和替换。

这给我带来了巨大的变化，可能节省了我一两个小时的时间。真的让人体会到 RegEx 的价值。
