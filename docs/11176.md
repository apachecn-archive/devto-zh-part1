# 生物学和 CSS

> 原文：<https://dev.to/designfrontier/biology-and-css-3o4l>

我是我爷爷那一代人的平均身高。我有一头棕色的头发，即使到了 31 岁的高龄，头发也在迅速变白。我有蓝色的眼睛和橄榄色的皮肤。

既然你对我的外貌了解得比你想的还要多，那就让我们言归正传吧。

导致我有这些身体特征的遗传模型很复杂。在遗传层面上，在我母亲怀孕期间，隐性和显性特征参与了复杂的计算，以决定我会有什么样的身体特征。蓝色的眼睛胜过了我哥哥淡褐色的眼睛，和我姐姐棕色的眼睛。早期的花白基因和棕色头发击败了我妹妹的红色头发。

此时，你可能想知道你是否无意中进入了错误的博客。因为这跟编程和 CSS 有什么关系？

事实证明 CSS 使用了基本相同的继承模型。就像你的父母为你贡献了特质一样。他们生物遗传的不同部分会影响你的身体属性。DOM 元素经历同样的事情。它们从它们的父母那里继承了不同的属性，并且基于显性计算(特异性！)它们的外形就成形了。

我发现这有助于简化对 CSS 继承的思考。这不是你在其他语言中看到的 CS 继承。正如你的身体所知，这是遗传。这是现存的最动态和最强大的继承模型之一。好好享受吧！当然，它有时看起来像魔术，但任何足够先进的技术也是如此。
