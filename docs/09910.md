# 将 Docker 用于微服务

> 原文:[https://dev . to/pavan _ belagatti/using-docker-for-micro services](https://dev.to/pavan_belagatti/using-docker-for-microservices)

Docker 是目前全球最热门的软件驱动型组织平台。Docker 引起了很大的轰动，因为它有一些令人难以置信的功能，与目前存在的其他软件工具不同。

全世界的开发人员和系统管理员现在都知道 Docker 有多让人头疼了。它的使用将极大地减少许多平凡的任务，并加快软件开发过程。随着集装箱化和微服务的引入，Docker 的使用呈指数级增长。

Docker 是一个开发、发布和运行应用程序的开放平台。您可以非常快速地开发应用程序并快速部署它们。使用 Docker，可以很容易地单独创建所需的服务，并将其作为微服务进行管理，而不会影响其他服务。

这是软件行业中最大的进步之一，在这个行业中，我们曾经拥有庞大、单一的代码。现在，最大的一块可以分成单独的服务，并使用 Docker 轻松管理。
