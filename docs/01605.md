当您创建 PHP 应用程序或库时，您通常有 3 种依赖关系:

*   严格约束:您的应用程序/库需要什么才能运行
*   可选约束:例如，您的 PHP 库可以为不同的框架提供适配器/桥
*   开发约束:调试、测试工具...
