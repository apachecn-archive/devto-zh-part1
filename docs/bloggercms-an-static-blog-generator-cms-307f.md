# 一个静态博客生成器

> 原文：<https://dev.to/sarfraznawaz2005/bloggercms-an-static-blog-generator-cms-307f>

在[的上一篇文章](https://codeinphp.github.io/post/welcome.html)中，我提到过 BloggerCMS，它就是用来建立这个博客的。BloggerCMS 是非常简单的 CMS，可以用来创建全功能的静态博客。尽管[有相当多的静态站点生成器](https://github.com/pinceladasdaweb/Static-Site-Generators#php)，但我还是创建了 BloggerCMS，因为我在这些静态站点生成器中缺少这些特性:

*   Web 界面允许我创建博客文章，添加页面等等，而不是在一堆不同的文本文件中手动编写博客文章
*   具有实时预览功能的 Markdown 编辑器，这样我就不用自己输入 Markdown 语法了
*   我需要一个真正的博客生成器，而不是通用/先进的网站生成器

这三件事是我创建 BloggerCMS 的动力。以下是截至目前为止 BloggerCMS 的完整功能列表:

*   **图形用户界面** -通过浏览器管理您的网站
*   **基于文件的** -不需要数据库
*   真正静态的 -不需要服务器端语言
*   **便携性** -轻松复制、移动、备份或版本控制您的网站
*   **集成降价编辑器** -无需了解降价
*   **内置图像管理器**
*   **问题评论系统**
*   **搜索帖子功能**
*   **分页**
*   **响应式布局** -使用附带的响应式布局轻松入门
*   **RSS 支持**
*   **网站地图支持**
*   **谷歌分析**
*   **代码高亮支持**
*   **标签云支持**
*   **社会追随章节**
*   **类别部分**
*   **档案科**

要了解更多关于 BloggerCMS 的信息，请前往 [it's blog](https://bloggercms.github.io) 或查看[的介绍文章](https://bloggercms.github.io/post/introducing-bloggercms.html)。
