# 微软商业应用平台简介

> 原文:[https://dev . to/fabiofranzini/微软商务应用平台简介-1f3j](https://dev.to/fabiofranzini/introduction-to-the-microsoft-business-application-platform-1f3j)

[![](../Images/79273de8ad3bd6973cead5c0d87d4836.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--SeVN9vXF--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1000/1%2A09HT37HYzS6IEDd0L7bcQA.png)

在一个越来越快的世界中，公司需要工具来实现他们的要求，快速降低成本。

与 Dynamics 365 一起，微软发布了他们的 [**业务应用平台**](https://businessplatform.microsoft.com) ，该平台可用于使用 [**PowerApps**](https://powerapps.microsoft.com) 构建复杂的业务应用(业务线)，这些应用可使用 [**Flow**](https://flow.microsoft.com) 自动化，而数据可使用通用数据服务(CDS)和使用 [**PowerBI**](https://powerbi.microsoft.com) 存储和管理，您可以查看和分析收集的数据，无需编码。

[![](../Images/30f56f0c0806f0cbcb12d0c105de11cc.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--z7KFcRyA--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/533/1%2AVBbDw1LRMYjGlElkLzwYsQ.jpeg)T3】微软商业应用平台

这些服务背后的理念是让了解其业务的超级用户能够快速构建和交付解决方案，而无需编写代码，让 T2 开发者能够轻松地为用户扩展功能，并利用一套丰富的服务，充分利用 Azure 平台的力量。

#### [](#what-is-powerapps)什么是 PowerApps？

> “PowerApps 是一项用于构建和使用自定义业务应用程序的服务，这些应用程序可连接到您的数据并在 web 和移动设备上工作，无需定制软件开发的时间和费用”

PowerApps 附带了一个可视化设计器(两个版本:基于 Web 的和桌面的)，可以轻松地为您的移动或平板设备创建应用程序。如果你熟悉 PowerPoint 和 Excel，你可以用 PowerApp 建立一个移动应用程序。

[![](../Images/d71190a9f62378cfacca883ffd6fb86f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--SpW9dfXP--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/610/1%2Aqu26yMyGEq4pMx6KOXIemQ.png)

借助 Microsoft PowerApps，您可以:

*   使用适用于云和本地(使用本地数据网关)服务的 CRUD APIs 快速创建移动应用和 Web 应用
*   为您的企业轻松构建移动应用和 Web，将单点登录(SSO)集成到企业数据和服务中
*   与组织内的其他人共享应用程序的简单方法

#### [](#what-is-microsoft-flow)什么是微软流量？

> “Microsoft Flow 是一项服务，用于在业务用户依赖的越来越多的应用和 SaaS 服务之间实现工作流自动化。”

Microsoft Flow 是一个 iSaaS(集成软件即服务)产品，市民集成商可以在多个应用和服务之间设置自动化工作流。

Flow 附带了一个 Web 可视化设计器，其中有许多用于常见场景的模板，并且 flow 团队会继续构建新的模板。

您还可以从头开始创建连接多个服务的流，例如 Common Data Service、SharePoint、Dynamics 365、Slack 和 Twitter 等等。

[![](../Images/0ef8ebd9592f1414f62860d36e877f7e.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--Y8H4FYR0--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/622/1%2AEDIyuodnA9KfaNY2qnq4-Q.png) 

<figcaption>微软流量</figcaption>

借助 Microsoft Flow，您可以:

*   将重复性任务转变为多步工作流程
*   在您的工作流程中制定决策，例如仅在满足特定条件时运行操作
*   安全地连接到本地数据(使用本地数据网关)和基于云的服务
*   使用内置或定制的数据丢失防护策略，防止敏感数据离开您的公司

#### [](#what-is-powerbi)什么是 PowerBI？

> “Power BI 是微软提供的商业分析服务。它提供了具有自助业务智能功能的交互式可视化，最终用户可以自己创建报告和仪表板，而不必依赖任何信息技术人员或数据库管理员”

[![](../Images/483c86099edb97e4846f4f9bedff684e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--O3HU5WA7--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/1%2AmpnDtUdMJO_2B3S9qUAnRg.png)

借助 PowerBI，您可以:

*   在一个仪表板上创建整个业务的可视化
*   创建令人惊叹的交互式报告
*   创建令人惊叹的交互式报告
*   轻松将商业智能和分析嵌入您的应用

#### [](#what-is-common-data-service)什么是常用数据服务？

> “通用数据服务允许您在一组标准和自定义实体中安全地存储和管理数据。实体是一组用于存储数据的字段，类似于数据库中的表

通用数据服务允许您在一组标准和自定义实体中安全地存储和管理数据。

实体是一组用于存储数据的字段，类似于数据库中的表。

存储数据后，您可以使用 PowerApps 和 Flow 来使用您的数据构建丰富的应用程序。

借助通用数据服务，您可以:

*   创建自定义实体
*   将数据导入标准或自定义实体
*   向标准实体添加自定义字段
*   在作为数据源开发的应用程序中使用标准和自定义实体
*   使用基于角色的安全性保护您组织内的数据
