# 鲁布罗斯

> 原文:[https://dev.to/michaelmior/rubbos-22o](https://dev.to/michaelmior/rubbos-22o)

在写硕士论文时，我需要一个可靠的数据库基准。RUBBoS 是该领域中相当常见的基准，但不幸的是，它已经过时几年了。虽然设置起来非常简单，但我确实遇到了一些与 PHP 和 Java 新版本相关的问题以及一些小错误。

由于基准是完全开源的，我决定发布我的修改。你可以通过 [GitHub](https://github.com/michaelmior/RUBBoS) 将这些修改下载到 RUBBoS。安装说明和一些附加说明也可以在那里找到。需要注意的最重要的一点是，我只测试了应用程序的 LAMP 版本。

希望有人觉得这有用！
