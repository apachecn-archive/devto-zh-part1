ðÿ'代理(开发者预览版)

因此，这不是一个数据代理，而是一个人的代理，当你给优步司机或送货员打电话时，你希望能够用同样的号码给他们回电话(和发短信)。如果这个号码是本地的，并且是你能识别的区号，那也很好，这样你就知道要接电话了。问题是，很难通过编程来找出所有这些因素，而且使用个人电话号码并不能保护任何一方当事人的隐私。Proxy 处理通信逻辑，为两个人之间的文本和语音对话设置中间电话号码，因此个人电话号码和用户 id 是保密的，甚至可以通过配置过滤和编辑消息的规则来监控双方之间的文本对话的内容。

如果您正在为 X 构建下一个优步，您需要使用代理。

对于 Twilio 来说，这是一个相当疯狂的举动，但它非常有意义。因此，他们发布了“功能”，这是一种在云中运行 Node 的方法，可以让它按需执行和扩展。作为亚马逊的 AWS Lambda 和谷歌的 Firebase 的竞争对手，Twilio 的明显优势是你可以在一个屋檐下创建你的 Twilio 函数。基本上使他们成为您的通信计算的一站式商店。我在会议期间用它为我的 Twilio SIM 卡设置了一个网络挂钩，非常简单。只要这项服务的认知度提高，我肯定能看到人们使用它。他们提供的令人敬畏的教程应该开始包含函数，这是默认的，人们会使用它。
