### **为 2017 年底假期准备一个副业，增加你的技能吧**

#### **为什么？**

*   当你设定需求时，做一些你喜欢的酷的或有趣的事情
*   你可以学到新的东西！
*   选择你一直想学的框架或库，或者用普通的 JavaScript 来学！

#### **灵感**

*   用 movie db([https://www.themoviedb.org](https://www.themoviedb.org))制作一个网飞克隆
*   用天气 API([https://openweathermap.org/api](https://openweathermap.org/api))构建一个天气应用
*   用 WordPress JSON API 创建博客，你可以使用 Mr Frontend WordPress API([http://blog.mrfrontend.org/wp-json/wp/v2/posts/](http://blog.mrfrontend.org/wp-json/wp/v2/posts/))
*   用中型 API([https://github.com/Medium/medium-api-docs#33-posts](https://github.com/Medium/medium-api-docs#33-posts))建立一个博客
*   用 Spotify 搜索 API([https://developer.spotify.com/web-api/search-item/](https://developer.spotify.com/web-api/search-item/))搜索你最喜欢的艺术家
*   用随机用户 API([https://Random User . me](https://randomuser.me))构建一个用户管理系统
*   使用 QR 码生成器 API([http://goqr.me/api/](http://goqr.me/api/))创建自定义 QR 码
*   用 Unsplash API([https://unsplash.com/developers](https://unsplash.com/developers))构建一个壁纸应用

#### **寻找很酷的 API？**

*   检查:[https://github.com/toddmotto/public-apis](https://github.com/toddmotto/public-apis)
