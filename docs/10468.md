# 用 netrw 在 vim 中移动文件

> 原文：<https://dev.to/nflamel/move-files-in-vim-with-netrw-189n>

在 emacs 上呆了一段时间后，我又回到了 Vim。这一次，我使用 Neovim 来看看它能提供什么。

几天前，我发现自己正在尝试做的一件事是将一组文件从 netrw(这是 Vim 的文件资源管理器)中移到一个不同的文件夹中。逐个重命名每个文件是不可行的，所以我在 Vim 的帮助下挖掘了一下，看看我能找到什么。

原来 netrw 有一个`move`命令，您可以通过使用`mm`在一组选定的文件上调用它...问题是它需要相当多的设置才能工作，我花了一些时间才意识到如何去做:

1.  在 netrw 上找到目标文件夹并用`mt`标记它。这将把它设置为移动的目标。
2.  找到您想要移动的文件，并用`mf`标记每个文件。
3.  用`:cd`告诉 netrw 原始文件的文件夹是当前文件夹
4.  在 netrw 中，在任何标记的文件中使用`mm`。

那会把文件移动到你想要的地方。您可以通过`:h netrw-mm`查看 vim 本身的帮助，但是它没有告诉您关于`:cd`的部分，这是我所缺少的部分，并且失败时会显示一条奇怪的消息，提示无法移动文件。

老实说，太难了，一点都不舒服。我也不认为这适用于不同文件夹中的文件。
