# 软件工程师，大数据

**New York, NY**

你喜欢编码吗？想要处理大数据？那么我们有适合你的工作！在 JW Player，我们一天处理超过 3.5 TB 的数据——这些数据令人着迷！我们已经掌握了数十亿人在数千个网站上的观看习惯——从愚蠢的猫咪视频到犀利的新闻报道，无所不包。

对实时处理感兴趣？批量处理？我们两样都做！JW Player 采用了 Spark、Kafka、Flink 和 Storm 等尖端技术，以物理学允许的最快速度提供高质量的分析。

如果您有大数据方面的经验，我们很乐意与您交流！但即使你不愿意，我们还是很想和你谈谈。我们正在寻找一个为生产环境编写高质量代码的人。如果你是一名优秀的开发者，你在 JW Player 有一席之地。

## 要求:

*   强大的编程基础。你知道你的算法，你知道 OOP，你知道调试器。

*   为生产环境编写代码的良好记录。在整个开发生命周期中支持该代码的历史。

*   **用数据体验。**不一定要大数据，但是你的 SQL 要强，要扎实掌握数据建模原理。

*   对测试的痴迷。有 TDD 经验者优先。

*   3 年以上行业经验

*   计算机科学或相关学科的学位

## 我们的技术堆栈:

*   **代码:**大部分是 Python 加一些 Java

*   **实时处理:**卡夫卡，弗林克，风暴

*   **批处理:** Spark，Hadoop

*   **储存:T1】 S3、postgres、Redis**

*   **云:**所有 AWS，所有时间！
