# 远程工作？如何做到令人敬畏！

> 原文：<https://dev.to/stefanbc/working-remotely-how-to-be-awesome-at-it-dk4>

[![Working remotely? How to be awesome at it!](img/311680da0afa9d2f4fb553d13f5c60bb.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--1baSMfQ8--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://stefancosma.xyz/conteimg/2015/07/photo-1428342319217-5fdaf6d7898e-1.jpg)

你可能会发现自己在职业生涯的某个阶段有机会为一家公司工作，这家公司的总部不在你所在的城市或地区，而是位于国外。所以，现在你要远程工作了。这意味着，你和你的团队不会在同一个物理办公空间，而是各自在自己的“办公室”里。

现在我们都知道远程工作的好处，我不打算在这篇文章中概述它们，而是列出一些可以帮助你在远程工作时取得成功的事情。

无论你是在家工作还是在一个共同工作的空间工作，你都可以做一些事情，让整个体验更加愉快，并确保你和你的团队尽可能高效。

### Timezones

确保你知道所有的时区，每个队友都在那里工作。使用像 [time.is](http://time.is/) 这样的应用程序，你可以确保你不会在你的同事们进入**睡眠时间**时 ping 他们。当然，这不适用于绝对紧急的情况，如部署失败或巨大的合并冲突，需要不同时区同事的协助。

### 日程安排

既然我们在时间的话题上，你最好确保保持一个非常透明的时间表。如果你的同事知道你每天的工作安排，那么你就不会有任何问题。这真的很容易实现，只要和你的团队建立一个公开的谷歌日历，你就可以开始了。如果你保持一个非常紧凑和整洁的时间表，那么事情就不会乱了套，以防不可预见的紧急情况。

在同一个话题上，值得一提的是，你应该区分你的工作日程和非工作日程。

当你不工作的时候，一定要出去走走，体验这个世界。这会让你的创造力源源不断。不管你是否有一份高强度的创造性工作，出去走走会让你的大脑重新定位，当你回到工作中时会更有效率。

永远不要强迫自己工作到很晚，并且要保证充足的睡眠。正如我上面已经提到的，你应该有一个清晰的时间表，列出你的工作时间和非工作时间。当然，这只适用于没有紧急情况，需要你注意的情况。

有很多研究解释了睡眠的好处。当然我们不知道我们需要睡眠的确切原因，但你应该意识到你的昼夜节律，并对你的睡眠模式有一个清晰的想法。这个我就不赘述了，不过我推荐一篇关于整个事情的非常[有趣的文章](http://www.highexistence.com/alternate-sleep-cycles/)。

### 办公室

无论你是在家工作还是在一个共同工作的空间工作，拥有一个合适的工作空间都是很重要的。现在，有很多选择。以下是其中的一些:

*   标准书桌和符合人体工程学的椅子。就我而言，我更喜欢宜家的这把椅子和任何有足够腿部空间的桌子。
*   立式书桌。这可以是任何普通的桌子，你可以在上面放一些东西来保持你的电脑在上升，你只需要扔掉椅子。
*   标准桌椅组合和立式书桌的组合。更具体的说，我说的是[这个](https://www.kickstarter.com/projects/1370698497/a-gravity-the-modern-workstation)
*   [跑步机办公桌？嗯，你需要进行日常锻炼，对吗？你知道。还有什么更好的方式来实现这一点，然后在同一时间运行和工作。就我个人而言，我不推荐这种方法，因为你的大脑在工作时无法集中精力做那么多事情，而且你会很快失去注意力。](http://www.amazon.com/TrekDesk-Treadmill-Desks-TD-01-Desk/dp/B002IYRBI0)

当长时间伏案工作时，另一件要记住的重要事情是始终保持水分。有人建议，在任何一个工作日，你都应该喝一定量的液体。就我个人而言，我认为你应该喝你身体需要的量。说到饮用容器，我绝对喜欢这个水壶。

### 保持联系

我们一定会在某个时候谈到这个话题。现在，我确信每个人都有自己喜欢的社交网络来与朋友和家人保持联系，但是如何与同事保持联系呢？

有很多好的应用程序，它们提供了很多与团队保持联系的功能。最受欢迎的是 Slack 和 Hipchat。但是其中一个比另一个有一点点优势，正如我推荐你阅读的这篇 Medium [文章](https://medium.com/@awilkinson/slack-s-2-8-billion-dollar-secret-sauce-5c5ec7117908)中所概述的。

每周和你的团队开一次团队会议总是好的。这样，你可以口头概述你当前的任务和你打算做什么。与整个团队进行一个小时的简短对话会让你对正在完成的事情有一个清晰的认识。你可以使用像 [Zoom](http://zoom.us) 或[Uber conference](https://www.uberconference.com)这样的应用来主持团队会议。

我不建议使用 Skype 或 Google Hangouts 进行团队会议，因为这些视频/音频的质量很差，掉线率也很高(至少从我的经验来看是这样)。

远程工作的另一个重要方面是使用一个好的 bug /任务追踪应用程序。我知道每个公司都有自己的做事系统。无论你使用的是内部构建应用程序还是任何人都可以使用的公共应用程序，重要的是要及时更新任务和错误。确保你的团队知道你在做什么，你也需要一个简单的方法来报告错误并讨论它们。

我没有任何喜欢的应用程序，但我可以推荐几个不错的应用程序，我已经遇到过了。你可以试试 [Asana](https://asana.com) 、 [Pivotal Tracker](http://www.pivotaltracker.com) 、 [Trello](https://trello.com) 、[吉拉](https://www.atlassian.com/software/jira)还有别忘了[螳螂](https://www.mantisbt.org)。所有这些都是多用途的，因此您可以使用它们来跟踪 bug 或跟踪任务。

### 推荐

在结束之前，我有一些额外的建议。

确保在为你远程工作的公司工作的某个时候，你安排了一次与你的团队面对面或 IRL 的会面。没有什么比亲自见人更好的了。当然，你通过视频看到他们，但当你真正见到一个人时，情况就不同了。你观察他们的肢体语言，这样你就能在工作时更好地理解他们。

最后但同样重要的是，尝试使用社交媒体保持联系。你会发现，在工作领域之外与你的团队交流将有益于你的工作关系。

### 结论

为了在远程工作中表现出色，你需要确保你有一个完善的时间表，并且你的办公空间被适当地组织起来。我不是说你应该遵循我的建议，这是你的生活，你可以做任何你想做的事情，这些只是我在远程工作时遇到的事情，对我有用。

作为额外的奖励，我强烈推荐 Lifehack 上的这篇关于充分利用远程工作的文章。
