嗨，我的名字是奥马尔，我是一个有点书呆子。当我第一次知道什么是 linux 并把它安装在一台旧电脑上时，我在中学就开始接触技术。

从那以后，不知何故，我来到了印第安纳大学，对学习技术感兴趣，但害怕尝试计算机科学。因此，我选择了信息学。没听说过，我也没听说过，但听起来不错。

当我刚到印第安纳大学时，我害怕编码，认为自己不够聪明，不能从事计算机科学，现在我即将从印第安纳大学信息科学专业毕业，辅修计算机科学。我已经达到了一个点，我真的很喜欢编码，虽然我还有很多要学，但我敢说在过去的几年里我已经做得很好了。

毕业后，我将前往明尼阿波利斯，成为塔吉特科技领导项目的一员。生活真好！

你可以在 GitHub 上找到我，名字是 [omawhite](https://github.com/omawhite) ，上面还没有太多，但我仍在努力。我主要用 python 编程，最近我学习了一些 php，我还想学习更多的 javascript。最终，我想学习更多的 Java、C#或 C++语言，但还不确定是哪一种。我曾经在 github 上用 C 语言做过一个项目，我很自豪。

简而言之，这就是我。我是一个过去害怕编码的书呆子，现在却喜欢花时间在 python 上。我很兴奋能在月底毕业，更兴奋的是不久后就能搬到明尼阿波利斯。
