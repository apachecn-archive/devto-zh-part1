# pi Power——我如何制作电池供电的 USB 集线器

> 原文:[https://dev . to/lad vien/pi-power-how-I-made-a-battery-powered-USB-hub-5e1p](https://dev.to/ladvien/pi-power-how-i-made-a-battery-powered-usb-hub-5e1p)

[https://www.youtube.com/embed/nxfNJJ34a7Q](https://www.youtube.com/embed/nxfNJJ34a7Q)

当我准备开始给我的 Pi 机器人添加外围设备时，我想确保 Pi 有大约 700ma 的功率预算。在寻找一个便宜的电池供电的 USB 集线器后，发现很少，我决定砍下一些便宜的(有点)零件，自己做一个。

1.  USB 集线器:1.39 美元
2.  5000 毫安时电池:17.93 美元
3.  DC-DC 转换器:2.76 美元

总计:22.08 美元
