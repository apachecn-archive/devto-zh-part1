# CSV 挑战

> 原文：<https://dev.to/jorinvo/csv-challenge-1al>

你得到了一些从社交网络泄露的数据，你想帮助穷人。

幸运的是，你知道一项政府服务可以自动屏蔽一系列信用卡。

这项服务有点老派，你必须上传一个完全相同格式的 CSV 文件。如果 CSV 文件包含无效数据，上传将失败。

CSV 文件应该有两列，**姓名**和**信用卡**。此外，它必须以下列模式命名:

**`YYYYMMDD`。csv** 。

泄露的数据没有每个用户的信用卡详细信息，你只需要选择受影响的用户。
