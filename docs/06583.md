### [](#pwacamp-session-2-programming)#PWACamp 第二场:编程

理解概念很重要，拥有工具和库来促进开发也很关键——但是作为一名开发人员，*如何开始呢？*这是我们夏令营第二部分的重点，也是我们希望在此详细阐述的内容。

在高层次上，我们希望支持两种学习行为:

*   *广度优先。*这是为那些对用代码探索这些概念感兴趣的开发人员准备的，但他们不一定局限于特定的框架或构建过程。在这种情况下，不用担心构建过程、部署选项和开发环境设置，用代码探索想法的最简单方法是什么？
*   *深度优先。*这是面向使用特定前端技术的开发人员，他们有兴趣了解如何将 PWA 功能集成到他们的应用中，或审核现有应用的合规性。在这种情况下，真正的重点是获得特定于前端的库/工具和工作流的实践经验。

2 小时的*编程*部分有效地允许您选择自己的冒险——您将在*讲师指导的*部分听到两种可能的选择，然后在我们现场导师的帮助下，通过*自学*部分的实践代码实验室探索这些(或其他选择)。这部分的两个讲座是:

### [](#building-with%C2%A0glitch)有故障的建筑

[Glitch](https://glitch.com/about/) 是一个*友好社区，在这里你将构建你梦想中的应用*。Glitch 是由 FogCreek 的出色人员创建的，它实际上是一个基于云的“沙箱”,在这个沙箱中，您可以在一个平台上编辑、构建和部署您的 web 应用程序，只需通过浏览器即可。

更重要的是，您可以通过重新组合来学习——简单地寻找提供特定功能或利用特定 API 的示例应用程序，然后重新组合它来创建一个自定义版本，以帮助您更好地理解底层概念。在此了解他们的愿景。

### [](#building-pwa-with%C2%A0angular)建筑 PWA 有棱角

[Angular](https://angular.io/) 是第一批采用以可组合性、可测试性和性能为核心原则的*声明式*web 应用编码的框架之一。在 Angular 2.x 中，[渐进式 web 应用是跨平台支持的核心](https://angular.io/features)，这使得它成为当今最受欢迎的 PWA 就绪 Web 应用选项。

> GDG 纽黑文组织者&全栈 Dev 的 Michael Solati ，将带我们在#PWACamp 构建一个带 Angular 的 PWA。

想了解更多关于如何使用 Angular 构建出色的 PWA 体验吗？查看一下 Google IO 的这个演讲，然后带上你的笔记本电脑，准备好跟随 Michael，或者做一个自我指导的代码实验室。
