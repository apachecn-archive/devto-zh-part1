# 印刷媒体 2.0 中的弹出窗口

好吧，我们兜了一圈。今天早上，我拿着[日报地铁](http://journalmetro.com/)(在离开[地铁](http://stm.info/)时)发现一个有进取心的营销人员在我的印刷媒体上贴了一个弹出广告。

[![](../Images/6c4960636aa8bc55fbfdf5690d58350b.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--V1Z5RKHt--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/630/0%2Ac2j8Q3jfOLyQMnNy.jpg)

我是平面媒体的粉丝。我更喜欢阅读印刷版的《世界报》(Le Monde Diplomatique)或[《单片眼镜》(Monocle)](http://monocle.com/)(即使考虑到平板电脑)。我也认为传统媒体可以向新媒体学习。但是，说真的，让弹出窗口和爆米花早就死掉吧。
