# 我们做的小事在与人交往时会产生巨大的影响

> 原文:[https://dev . to/max cell/the-small-things-we-do-that-make-the-ship-that-impact-when-meeting-people](https://dev.to/maxcell/the-small-things-we-do-that-make-big-impact-when-meeting-people)

去年，我参加了在科罗拉多州埃斯蒂斯帕克举行的名为 [Hackcon](https://hackcon.mlh.io) 的大学黑客马拉松组织者的精彩会议。我利用这次活动走出自己的舒适区，结识新朋友。不是几个，不是一些，而是全部:我试图见到并记住会议上的每一个人。大约有 400 多人参加了这次活动。

我在这里面临的挑战是在 2.5 天内会见比我一个月时间还要多的人。我不指望每个人都像我一样，试图满足大量的人。此外，我们应该永远记住，重要的是我们建立的关系的质量，而不是数量。然而，我在 HackCon 上结交朋友的意图实际上提供了许多关于如何有效会见和记住人们的重要课程。这里有一些提示，你可以在遇见某人时使用。

# [](#1-smile)1。微笑

> “行动胜于雄辩。”

你可以用一个微笑做这么多事情。你和收到微笑的人都会感觉很棒。这并不意味着你要对每个人都露出一个极端的微笑。让它成为有机的。让它成为你从他们那里得到的积极的结果，或者他们能让你感觉到什么。当你对某人微笑时，它是有感染力的。

很多人会说他们不能“只是”微笑。这并不容易；微笑使用你脸上的肌肉，就像任何肌肉一样，你必须锻炼它。一个帮助我的技巧是思考我真正喜欢的事情，只是被动地微笑，而不是对着某人。你认为童菲·史密特是怎么做到的？！

# [](#2-ask-them-their-name-and-say-it-back)2。问他们的名字，然后说出来

> 记住，在任何语言中，一个人的名字对他来说都是最甜美、最重要的声音
> 
> 戴尔·卡耐基

名字中蕴含着力量。当你说出某人的名字时，这表明你确实记得他们是谁。当我们学习新的材料时，我们必须不断地复习以记住它，对于名字也是如此。记住你在哪里遇见某人是很容易的，但是当我们不经常使用一个名字时，这个名字就很有挑战性了。

有一些小技巧可以帮助你记住名字。有些人发现在脑海中重复这个人的名字几次有助于建立联想。我试着倾听他们对什么充满热情，并从中找出一些巧妙的东西，将其与他们的名字联系起来。试试人们做的有趣的破冰练习，你必须说一个描述你的词，这个词必须以你名字的第一个字母开头。对我来说几乎每次都有效。想想看，“风度翩翩的王子。你想要什么？

把这想象成去你最喜欢的咖啡店或餐馆，那里的服务员记得你的名字和订单。有人在一天中遇到的人群中认出了你，这让你感觉很棒。然而，我们中的许多人在另一端从来不问服务器的名称。这不是很奇怪吗？当我们的世界以我们为中心，人们注意到我们时，我们往往真的很快乐，但我们必须回报。与此相关的一个好作品是戴维·福斯特·华莱士的毕业演讲[《这是水》](https://www.youtube.com/watch?v=8CrOL-ydFMI)。表现出你真的关心并想承认这个人。

# [](#3-listen)3。听

> 大多数人没有带着理解的意图去听；他们听着，意图回答。”
> 
> 史蒂文·柯维

在所有三条建议中，这一条对人们来说是最难做到的。我们有两种类型的倾听:“被动”和“主动”区别在于你有多投入。

被动倾听是指我们听到别人说的话。想象一下，在课堂上，你的老师正在进行考试复习。被动倾听时，你除了接受信息什么也不做。积极倾听是指你实际上内化了老师所说的内容，而不是仅仅接受老师的观点。你向老师提问，重申他们提出的观点，并试图获得更深层次的含义。当你来参加考试时，你会发现你对积极倾听的对话记忆更深，并且可以很容易地将事情与你的经历联系起来。

积极倾听的重要性同样适用于与人交谈。你需要把别人说的话放在他们的参照系中理解。倾听他们对什么充满热情，并就此提出更多问题，让他们分享更多信息。你还必须表现出你对它的真正兴趣。他们不会只和那些不愿意听他们说话的人说话。你永远不知道这个人的见解什么时候会派上用场，发现这些见解从简单的倾听开始。当你积极倾听时，你会对某人了解更多，你会了解到你们的共同点。你永远不必担心忘记一个人是谁，因为你对他了解太多了。

我的一些建议听起来可能有点奇怪，但我向你保证它们会有所帮助。当我遇到新朋友时，我会先问他们的名字，然后让他们告诉我他们对什么感兴趣。这样，我倾向于记住某人对 X 充满热情，所以他们的名字是某某。有些人甚至写下这些人的名字和他们的激情来帮助他们记忆。另一个有用的提示是，你交流中的非语言部分有助于你倾听。有两件事非常有效，那就是微微前倾，并与他们进行眼神交流。你的身体实际上会处于倾听和参与的状态。当你的身体感到忙碌时，你的思想也会如此。当我们对某人持积极态度时，这种情况几乎会不由自主地发生，研究表明，当这种情况发生时，我们也会说得更多。

当这三个小行动付诸实施时，你将在结识和记住他人方面取得巨大成功。我发现这些是真正让各方记住彼此的行为。我现在向你发出挑战，在你已经接触过的人身上实施这些小行动，看看你的经历有多大的不同。意识到并有意识地关注这些将有助于你成长。我希望你用好这些，我的学徒！如果你想了解更多关于交朋友的惊人的事情，看看非常有用的书，比如戴尔·卡内基的《如何赢得朋友&影响他人》。
