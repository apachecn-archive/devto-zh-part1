# 会议和学者

> 原文:[https://dev.to/yechielk/of-conferences-and-scholars-6mn](https://dev.to/yechielk/of-conferences-and-scholars-6mn)

参加一个科技会议已经在我的遗愿清单上有一段时间了。当我谈到计时攻击、嵌套表单或 TDD 的最佳实践时，我觉得能够花一些时间和那些目光不会呆滞的人在一起，这对我的编码技能会产生奇迹。能够向如此多的聪明人学习，并在技术社区中建立联系将是一个惊人的机会。

问题是事情没那么简单。技术会议的门票高达数百美元，这还不包括机票和酒店费用。再加上休假 3-4 天，事情会很快增加。

[![RubyConf logo](../Images/4da6361cc0e5c1ece68ac6e2247c5d29.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--w7RvYSsk--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/790/1%2A0Ie9T5JlucseM2kPJzoUlw.png)

但是一个年轻的程序员可以梦想，所以几个月前我申请了参加 2017 年 RubyConf 的奖学金，这是世界上最大的 Ruby 程序员会议，今年将在新奥尔良举行。

我并没有抱太大的希望，毕竟，成百上千的人和我一样申请了，当我收到电子邮件通知我已经被列入候选名单时，我几乎忘记了这件事。

想象一下，几周前，当我收到组织者的电子邮件，说有一个职位空缺，问我是否还有兴趣时，我有多兴奋？

[![Are You Kidding Me? gif](../Images/a1044a101f0ae7cdefd1af5a6f613bc1.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--KdFDgXZD--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://cdn-images-1.medium.com/max/480/0%2AyAnHztzQcCk5XRu-.gif)

显然，我非常感兴趣，问题是我的钱包是否允许。奖学金涵盖了 400 美元的会议门票，但旅行和住宿是另外几百美元。谢天谢地，我们在[谷歌云平台](https://cloud.google.com/)的朋友同意为我们报销部分费用，所以虽然事情仍然紧张，但我认为这是值得的，在与我非常支持的妻子进行了简短的磋商后，我通知组织者我们开始了！

奖学金是一个极好的机会。不仅包括你的入场，而且在整个会议期间，奖学金获得者(称为学者)与导游配对；经验丰富的与会者，他们的工作是陪同我们，向我们介绍人，指出哪些会谈值得参加，并通常使我们的第一次会议体验尽可能愉快。

#### [](#goals)目标:

这是我的第一次会议，我试着以开放的心态参加，但我认为有一个宽松的目标列表会帮助我最大化体验(并让我脚踏实地)。所以不分先后:

*   在会议期间，我想建立至少 3 个会后继续的联系。
*   我想了解至少一种足够有趣的新技术，以便在会议结束后进行研究，并将其纳入项目中。
*   写一篇关于上述新技术的博文。
*   额外加分:建立一个能让你在技术领域找到工作的关系:)

在整个会议期间，我希望更新我的[关注者](https://twitter.com/yechielk)关于我将学到的所有东西和我将遇到的人，所以一定要保持警惕，如果任何读到这篇文章的人会在 RubyConf 打电话给我，我很乐意聊天！
