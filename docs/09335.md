## 我开始编码是因为...

在我就读的女子大学(大约是 20 世纪 80 年代初)，一门“基础”课听起来很吸引人。我发现我有天生的诀窍。

## 我目前正在入侵...

探索微软数据访问平台的最新版本，以帮助他人学习如何使用它。最有趣的是在 Windows 和 Mac 之间来回构建包含这些 API 的应用程序。

## 我很兴奋...

在被锁定在 Windows 上几十年后，能够在各种平台上编码。

## 我对其他代码女性的建议是...

不要让别人引导你远离你想做的事情！
