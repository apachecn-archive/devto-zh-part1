# 请参加我们本周六在纽约举办的安全研讨会

> 原文：<https://dev.to/ben/come-to-our-security-workshop-this-saturday-in-nyc>

除了我们参与的非营利组织[Make Diff](http://www.makeadiff.io/)之外，[杰斯](https://dev.to/jess)和我正在帮助举办一场非常精彩的安全研讨会。该活动将于本**1 月 28 日星期六在纽约市的微软办公室**举行。这将是一个令人难以置信的教育日，但作为奖金，所有收益将捐赠给 EFF。

你将有机会全天参加四个不同的长达一小时的研讨会。我们将涉及加密、在线隐私、开发者安全、PGP/身份、保护您的电子邮件和密码管理器等主题。

我将帮助协调开发者安全研讨会，该研讨会由经验丰富的 Stephen Kapp 领导。这个周六填补一些知识空白真的很兴奋。
