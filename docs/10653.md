# 通信网络

> 原文:[https://dev.to/horia141/communication-networks-4936](https://dev.to/horia141/communication-networks-4936)

这里有一个高度简化的人们建立的主要电信网络的列表，按历史顺序排列:邮件服务，电报，电话，广播，电视，互联网。

令人惊讶的是，物理邮件服务和互联网有一些共同之处，这使它们有别于所有其他“先进”的网络。他们不知道自己在运输什么。电报只能传送电报，无线电只能发送无线电信号，而互联网尤其适用于发送几乎任何东西——邮件、文件、声音、电视等等。它唯一不能运输的是实物，这就是邮件服务的用武之地。

这也证明了通信的力量，像电话或电视这样的大规模基础设施项目的建设，基本上是为了发送一种类型的信号，实现一种功能。然而，随着互联网的发展，为特定类型的消息建立专用网络越来越没有意义，而将它们覆盖在通用通信网络上则越来越有意义。

事实上，Skype &大多数通讯应用，网飞&互联网视频网站，基本上都是这么做的，但来自应用程序端，而不是底层。还有更通用的方法，如 VoIP、IPTV 等。
