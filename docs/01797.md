## [](#whats-markdown)什么是降价？

> Markdown 是一种具有纯文本格式语法的轻量级标记语言。它被设计成可以使用同名工具转换成 HTML 和许多其他格式。[8] Markdown 通常用于格式化自述文件，用于在在线论坛上撰写消息，以及使用纯文本编辑器创建富文本。由于对 Markdown 的最初描述包含含糊不清和未回答的问题，多年来出现了许多 Markdown 的实现和扩展来回答这些问题。参考文献:[来自维基百科](https://en.wikipedia.org/wiki/Markdown)

Markdown 主要是作为一种轻松编写 HTML 的手段。
Medium 也有 Markdown 符号。
比如你想加粗文本的某些部分，写

`<strong> bold </ strong>`

为 HTML，

`** bold **`

为 Markdown。

Boostnote 是一款可以用这种降价方式写笔记的软件。

# boost note 的特性

## [](#sharing-function-via-dropbox)通过 DropBox 分享功能

Boostnote 允许您通过在线存储 DropBox 与其他设备共享笔记。

但是，要使用这个功能，我很遗憾，我必须在我的电脑上安装“DropBox”。

## [](#todo-list-is-easy-to-make-amp-easy-to-use)待办事宜易于制作&易于使用

在 Boostnote 中，您可以通过写“- [ ]”来创建一个复选框，以创建一个待办事项列表。

此外，在创建的 ToDo 列表中还生成了一个完成率的度量，您可以一眼就确认任务现在完成了多长时间。

## [](#bad-feature)不良特征

难以与他人分享
由于分享是通过 DropBox 进行的，所以当你想与他人分享笔记时就不方便了。
