# 你把你的代码放在哪里？

> 原文:[https://dev.to/seanxe/where-do-you-keep-your-code-6an](https://dev.to/seanxe/where-do-you-keep-your-code-6an)

你如何在你的机器上组织你的代码？我不是说 github/bitbucket，实际上是在你的硬盘上？

我总是在我的用户名下创建一个名为“项目”的目录:

`/Users/seanxe/Projects/`

在里面，我把它们分成不同的文件夹:

```
 ├── github.com/
  |   ├── username/
  |   ├── username/
  ├── project1/
  ├── project2/
  ├── playground/
  |   ├── js-test-1 
```
