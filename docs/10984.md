起初，我对这种编程语言有点不知所措，直到有几次我收到了 Android 开发人员的工作邀请，要求我学习这种编程语言。除了知道一些同事已经在专业地使用这种语言之外，他们还使我对这种编程语言有了更多的了解。

#### 学习如何与科特林编程？

除了以上列出的原因以及在网站[本身上列出的原因【https://kotlinlang.org/](https://kotlinlang.org/) (如其易于学习、与 Java 的互操作性、其多功能性等)，我想补充几个原因: ***脑设备落后于此次任务所以，当你学习到它，你会意识到它极大地方便了你的工作，帮助我们解决日常生活中的各种情况。***

那么多了解它也没关系。
