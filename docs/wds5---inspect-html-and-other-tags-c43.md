### HTML 标题

我们已经看到由标签 **< h1 >** 到 **< h6 >** 表示的 HTML 标题，其中 **< h1 >** 是最重要的标题，而 **< h6 >** 是最不重要的标题。现在我提到一些你在使用 HTML 标题时应该遵循的要点:

*   搜索引擎使用标题来索引网页的结构和内容。这很重要，因为你应该始终遵循标题标签的顺序。 **< h1 >** 要用于主标题， **< h2 >** 用于不太重要的副标题，以此类推。事实上，通常遵循的系统是在文档中只使用一个 **< h1 >** 标签来表示文档的标题，而不在其他地方。
*   标题标签应该只用于标题，而不是使文本变大或加粗。不同的标签被用于这个操作，我们将在以后的文章中研究它们。

### HTML 横线

水平线元素用于在网页上显示水平线。它用于表示网页上的断点。它由标签 **< hr >表示。**

### 最后但不是最不重要的

你一定想知道在你访问的漂亮网站背后有什么 HTML 代码。所以你可以很容易地看到任何网站背后的 HTML。在你的浏览器中点击任意一个网站，你会发现**检查**或**检查元素**或**查看源**或**查看页面源**选项。点击它，你会很容易地看到那个网站的 HTML。您可以通过右键单击特定元素来隐藏特定元素。
