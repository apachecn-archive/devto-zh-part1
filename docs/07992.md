# FMEAddons 对 Magento 片段的扩展

> 原文:[https://dev . to/fmeaddons/magento-snippets-extension-by-fmeaddons](https://dev.to/fmeaddons/magento-snippets-extension-by-fmeaddons)

FMEAddons 的 Magento snippets extension 允许您在 Google 搜索引擎中显示关于您的产品的重要信息。有了这个插件，你可以在 Google snippets 中使用 13 种类型的数据，这有助于吸引访问者访问你的网站。

Snippet extension 有助于在登录页面上获得更多有针对性的流量，并促进销售。您可以获得 100%退款保证的扩展，它支持以下领域

姓名
面包屑
SKU
网址
图片
描述
价格
货币
状态
品牌及其产品属性
产品颜色及其属性
产品重量及其属性
评论
