# [](#exciting-year-to-be-a-web-developer)成为一名 web 开发者的激动人心的一年。

## 201 x 的 App 热潮

让我们回到 2010 年，智能手机疫情的崛起。多亏了 iPhones &便宜的安卓手机，你可以发现每两个人中就有一个在用智能手机。每一秒钟就有一个人在拍摄《愤怒的小鸟》或成为一名水果忍者，whatsapp 正在成为 goto 沟通渠道，人们实际上是在付费下载一个应用程序。突然之间，每个公司都想拥有一个应用程序。但是为什么呢？

**离线访问**:由于应用程序是下载一次就可以永久使用的，一旦下载，用户只需轻点就可以访问它们。
**通过通知重新定位**:你可以通过向用户发送通知、相关产品信息来吸引他们的注意力，从而让他们再次使用这个应用。
**原生特性**:原生应用兼容设备的硬件和特性，如摄像头、位置、传感器、文件系统等。
**更好的性能**:由于应用程序可以访问设备硬件，并且可以自由设计，因此应用程序应该具有更好的性能和速度。

## web 赢在哪里:

自从互联网出现以来，万维网已经成为我们信息消费的主要来源。让我们来看看为什么 web 仍然存在并获胜:

**可发现性**:你在谷歌上搜索某样东西，然后嘭！你有一个你从来不知道存在的网站列表。你很容易在网上发现新的内容和网站。
**更广泛的覆盖范围和即时访问**:从低端功能手机到闪亮的 iphones，每一部手机里都有浏览器。点击一个网址，瞧，无论设备配置如何，你都可以在几秒钟内看到内容。
**可共享性**:即使本地应用需要通过网络共享某些东西，它们也需要一个网址。URL 是将某人指向来源的最佳方式。
**开发**:一个网站不需要针对其他操作系统或设备进行不同的开发。这是一个写一次就能到处跑的东西。
**更新&维护**:你可以一天多次更新你的 webapp 但是频繁更新一个 app？伙计，你的用户会很生气的。

## [](#the-solution-progressive-web-apps-)解？渐进式网络应用程序:

如果你把网络的所有优点和本地应用提供的功能结合起来，会怎么样？你会得到进步的网络应用。让我们看看所有进步的 web 应用程序能做什么。

可安装的(Installable):只需轻轻一点，你的网站就可以安装在手机上，并像本地应用一样添加到主屏幕上。
通知功能:你可以发送推送通知，就像本地应用一样。
**离线**:使用离线缓存 API，你安装的 webapp 也可以离线运行。
**更新是一件乐事**:你只需更新你服务器上的网站，应用程序缓存就会自动更新，用户不会察觉。你想更新多少就更新多少，而用户不会注意到。
**渐进式**:虽然对 PWA 的支持在迅速增加，但可能会有一些浏览器不支持所有的 API。在这种情况下，您的 PWA 将像任何其他网站一样运行。所以，没有更多的支持问题。支持一切。

## [](#what-the-future-holds)未来会怎样:

很明显，下一代 webapps，也称为渐进式 Webapps，将在几年内成为一个明显的标准。随着越来越多的浏览器支持和用户教育，用户将期望网站成为 PWAs。以下是 PWAs 中令人兴奋的事情:

*   从 windows 应用商店在您的计算机上安装 PWAs。微软承诺将 PWAs 带到 edge，然后 PWAs 可以添加到 windows 商店，用户可以在他们的计算机上下载。这不是很棒吗？
*   Firefox for android，增加了对渐进式网络应用程序的支持。
*   苹果在 Safari Tech preview 中增加了对服务人员的支持。([好事来了](https://mobiforge.com/news-comment/safari-service-workers-and-other-pwa-good-news-stories))
*   Tinder 推出了 PWA。Twitter、优步、Pinterest、谷歌地图和许多其他大玩家也是如此。另外，你现在正在超快的 PWA 上阅读这篇文章，试着把它添加到主屏幕上。；)
