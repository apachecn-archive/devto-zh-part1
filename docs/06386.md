# 20 个帮助你学习编码的好资源

> 原文：<https://dev.to/apium_hub/20-great-resources-to-help-you-learn-to-code>

这不是我们科技博客中第一篇讨论如何学习编码的文章，然而，我们还没有讨论学习编码的最佳网站或任何在线编程课程。

比方说，由于某些原因，你决定或刚刚面临一种情况，你需要学习如何编写代码，但更喜欢在网上做？然后在这里，就在介绍部分的下面，你会得到你要找的东西。我分析了这个问题，将在 2017 年列出学习编码的最佳方法。

### 学习如何编码有哪些方法？

为了说明这一点，我想列出学习代码的最有效的方法。我不会说有任何特定的和完全正确的方法，但肯定会有一个对你来说更好的方法。

1.  就你可能是编程行业的新手而言，试着获取更多关于它的[技术趋势](https://apiumhub.com/tech-blog-barcelona/tech-trends-software/)、顶级编程语言的信息。这应该会给你更清晰的想法，在选择正确的[编程语言](https://apiumhub.com/tech-blog-barcelona/new-programming-languages/)时，你应该关注什么。
2.  编程线上和线下课程。这种方式是最主要、最常见、最高效的学习方式。
3.  软件博客是另一个伟大的想法，可能会帮助你取得更好的结果。
4.  跟踪行业中的主要影响者总是一个好主意。例如，我们的移动开发人员关注并阅读了一份[顶级移动影响者](https://apiumhub.com/tech-blog-barcelona/great-mobile-development-influencers/)的列表。
5.  阅读总是一个好主意，这里有一份必读的编程书籍清单。
6.  youtube 上的视频教程。对于一些初学者来说，这可能是查看可视化示例的一个强制选项。

### 学习代码的在线编程课程列表

##### 

##### 1。Coursera

Coursera 被认为是一个大型的在线课程库，在那里你可以从世界上几乎任何一个角落的真正的大学教授那里学到不同的东西。这是一个很大的优势，因为你可能会发现用不同语言教授的课程，从英语到日语。最后，如果你愿意为你的学习过程支付一点费用，你可以得到一个真正的认证证书，证明你完成了课程，并可能在你的简历上增加一分。

[面向所有人的 Python](https://www.coursera.org/specializations/python)(密歇根大学)

##### 

##### 2。edX

EdX 是一个由麻省理工学院和哈佛大学支持的开源平台，在“计算机科学”类别下提供了一百多门课程。它涵盖了 Java、C#、Python、前端开发人员和许多其他主题。你可以在课程结束时选择一个没有证书的免费选项，或者支付一点费用来获得它。

##### 

##### 3。自由代码营

首先通过既定的课程教授编码(大约。总共 800 小时)，然后给你为非营利组织工作的实践经验。可以在那里找到的主题:HTML、CSS、JavaScript、数据库、DevTools、Node.js 和 Angular.js

##### 

##### 4。嘎破折号

大会是一个免费的在线学习平台。完全基于项目。每走一遍，您就构建了一个“项目”。你可以获得免费的在线 Web 技术教程以及其他主题和语言:HTML、CSS、JavaScript 等。

##### 

##### 5。可汗学院

这个在线资源涵盖了许多主题，包括许多计算机编程。内容做得很好，涵盖的主题和科目:JS、HTML/CSS、SQL 等。

##### 

##### 6。麻省理工学院开放式课程

你不需要任何帐号就可以访问麻省理工学院的课程资料。他们的编程部分的主题范围从工程问题解决的介绍到计算机动画中使用的算法。无帐户选项的唯一缺点是，你将看不到一些课程的作业的任何答案，所以你不能确定你是否在正确地做作业。

##### 

##### 7。奥丁计划

Odin 项目是由第一个在线编码训练营 Viking Code School 的创建者完成的。这是他们的免费版本，你可以在上面学习糟糕的编程技术，比如 HTML，CSS，JavaScript & jQuery，Ruby 编程，Ruby on Rails

##### 

##### 8。乌达城

提供单独的课程，以及“纳米学位”,为特定的职业培训你，如前端 web 开发人员或数据分析师。课程材料是免费的，但是纳米学位需要学费。

##### 

##### 9。体贴的

Thinkful 提供各种自定进度的编程课程。Thinkful 有自己的在线导师，所以要记住，你必须为有自己导师在身边的课程付费。通常，一门课程可能需要你 2-3 个月的时间。推荐的课程之一可以是 Python，但是，如果您对任何其他技术感兴趣，您可以在网站上找到它们。

##### 

##### 10。破折号

General Assembly 因其校园课程而出名。但是它的免费在线网络技术教程 Dash 做得相当不错。比起 Codecademy 的我更喜欢它。如果您对学习 HTML、CSS 和 JavaScript 感兴趣，这是一个不错的起点。

##### 

##### 11。微软虚拟学院

微软虚拟学院是一所免费的在线学校，课程涵盖微软相关主题和特定的微软产品。无论你是对初学者的编码感兴趣，还是已经有经验并想创造下一个大游戏，或者希望建立一个可以改变世界的应用程序，MVA 都提供编码培训和课程，可以帮助你实现你的目标。

每个课程都包含一个视频和视频副本的 PDF 下载。每个模块的末尾都有自我评估部分。学生将获得分数，并可以根据他们取得的进步获得不同级别的认可。通过观看视频、下载视频抄本和自我评估测验获得积分；会员赢取的积分越多，她参与其他 MVA 活动的机会就越多。排行榜追踪最活跃的成员。虽然 MVA 提供免费的微软认证培训，但会员不能直接通过学院获得认证。

##### 

##### 12。[采访位](https://www.interviewbit.com/)

这是一个免费的在线平台，有顶级学院的课程；InterviewBit 专用于安置准备。如果你能自信地回答上的所有问题，那么你就为编码面试做好了准备。一旦你完成了采访，你可以继续找另一个有更多实践经验的来源。还有，InterviewBit 里面有很多测试用例。

##### 

##### 13。厨师长

CodeChef 是一个练习编码的好地方。有大量有问题的案例可以帮助你加强你的编码技能。首先，试着自己解决问题，但是如果你没有得到想要的结果，你可以随时查看别人的解决方案并从中学习。而 CodeChef 则专注于竞技编程。

##### 

##### 14。[堆积溢出](https://stackoverflow.com/)

阅读、学习、分享知识和建立职业生涯的最受欢迎的资源之一，毫无疑问，它是世界上最大的开发者社区之一。

##### 

##### **15。沃盖拉**

Vogella 是一个充满了深入教程的网站，从设置的基础，一直到一些更具体和更高级的概念。该网站仍然免费使用，其大部分收入来自广告和用户捐赠。如果你从这些材料中得到一些好处，为什么不表现出一些爱呢？

##### 

##### 16。传授…额外技能

没有任何开始的要求，你也不需要有任何以前的经验来学习 Upskill，这是一个免费的训练营，带你从初学者到高级。一系列视频片段以一种轻松的对话式风格，引导您完成各种投资组合项目。主要焦点是全栈 web 开发，因此您将学习 HTML、CSS、Javascript、Ruby on Rails、Git、Bootstrap 等等。

##### 

##### 17。 [Tutorialsdojo](https://www.google.es/url?sa=t&rct=j&q=&esrc=s&source=web&cd=1&cad=rja&uact=8&ved=0ahUKEwi15s2534XVAhVDuxQKHSjEC20QFggkMAA&url=http%3A%2F%2Ftutorialsdojo.com%2F&usg=AFQjCNG_20_bRrtouhKhEoctKCrYyFddWw)

Tutorials Dojo 是技术相关主题的一站式学习门户。他们涵盖了三大领域:移动开发及其主要技术；Web 开发和包含视频的在线教程。

##### 

##### 18。[苹果开发者页面](https://developer.apple.com/)

特别是对于那些想学习 Swift 代码以及如何在 iOS 平台上创建和实现移动应用程序的人来说，你可以查看苹果开发者页面，那里有足够多的细节、教程和解释。

##### 

##### 19。**代码学院**

Code Academy 是初学者和那些想学习新事物的开发人员中的知名资源。这是一个免费的选择。从这个编码资源中，你可以通过广泛的课程学习技术和编程技能，这些课程涵盖的主题有:PHP、Python、AngularJS、Java、JavaScript、jQuery 和许多其他内容。已经有超过 2000 万人在这个平台上学习编码。不要错过从零开始学习和理解编码的机会。

##### 

##### 20。 [Hackr.io](https://hackr.io/)

这个列表我想以源代码来结束，它可能涵盖了这里没有提到的所有其他源代码。Hackr 是按技术和语言分类的教程、课程和资源的搜索者。
