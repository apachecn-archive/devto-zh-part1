**< pre >** 标签用于显示预先格式化的文本，这意味着它将显示与输入格式相同的输出，并且不会删除多余的空格。

### [](#html-style-attribute%C2%A0)HTML 样式属性

虽然我现在要告诉你的，在你学会 CSS 之后你将永远不会用到。在这一部分中，我们将学习如何使用样式属性来设计 HTML 内容的样式。让我们先来看看样式属性的语法:

现在让我们来看看您可以在样式属性中使用的一些重要属性:

**属性使用**

1.  background-color 定义该元素的背景颜色
2.  颜色定义该元素的文本颜色
3.  font-family 定义给定元素使用的字体
4.  字体大小定义给定元素的文本大小
5.  text-align 定义了给定元素的对齐方式，现在让我们看看您可以为该属性使用哪些值。

对于前两个属性，您应该输入任何支持颜色作为值。现代浏览器支持 ***140 种命名颜色*** 。你可以在 http://htmlcolorcodes.com/的 T4 看到它们。对于字体系列，您应该输入浏览器支持的任何 ***有效字体*** 。对于字体大小，您可以在 ***百分比(%)或*** 像素中输入大小。对于文本对齐，对齐方式可以是 _ **左对齐、右对齐或居中对齐** _。
