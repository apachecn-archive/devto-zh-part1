# Jquery 的死亡

> 原文:[https://dev.to/designfrontier/death-to-jquery-5hm6](https://dev.to/designfrontier/death-to-jquery-5hm6)

昨天我在 Pandamonium 上展示了“jQuery 之死”,这是我们在 Instructure 的内部技术会议，因为说实话是时候了。

jQuery 对网络的发展至关重要，这也是我们大多数人诚实工作的原因。如果没有它，我们永远也不能建立我们今天所知道的网络。但是，现在...是时候向前看了。

DOM APIs 已经标准化，使用 fetch，基于流的 API 的流行通常处理 AJAX 是非常容易的跨浏览器的，无论你选择哪种风格的 API。作为一个平台，网络的力量比以往任何时候都大。

除此之外，像 React 和使用 jQuery 这样的数据驱动 UI 层开始与我们构建应用程序的方式背道而驰。至少在 react 中，这是一种反模式，表明缺乏理解或者某个地方在项目持续期间会有问题。

不管你对 jQuery 有什么看法，这些幻灯片至少是有趣的...制作随机生成的背景真的很有趣。以后可能要写这个了。

来看看吧！
