# 我迷路了吗？

> 原文：<https://dev.to/lumochift/am-i-lost-265a>

我为自己是一名软件工程师、程序员、代码圣人或者其他任何你称之为的人而自豪😄。你知道在我选择计算机科学学位之前我不知道计算机科学到底是什么吗？这里是关于我的代码冒险之路的简短历史。:p

# 初中牌坊

在这个时代，我不知道我在做什么，我讨厌进入初中 1 玛琅，因为我想进入另一所学校。每天只是去上学，学习，玩耍，制造一些麻烦。只是一些麻烦，我想没有那么多。哈哈…我仍然记得我触摸电脑的那一天，是的。像黑客一样使用它，我一直记得当我试图在 MS-DOS 上执行命令`del *.*`时的有趣时刻，然后一切都变暗了。

在那次**伟大的**实验之后，我恐怕要再次探索那次`black thing`。除此之外，我不喜欢电脑，因为太贵了，我买不起。你知道吗，我讨厌数学，喜欢学生物。这个拱门对我来说没有什么美好的回忆。一切都那么平淡，用 MS-DOS 记忆不好。

# 高中牌坊

以前的拱门很无聊吧。我的现实生活从这个拱门开始，这是我人生中第一次热爱学习会计科目。什么？真的吗？会计？是的，除了物理和化学。但这并不是一个很长的故事，在我的班级搬到计算机实验室附近后，一切都发生了 180 度的转变。每天我总是在实验室，是的，只是玩游戏。那个时代的反恐精英是除了星际争霸之外的现象级游戏之一。我在高二的时候有了一台电脑，但事实仍然没有改变，每天都在玩游戏。之后，我选择了 Brawijaya 大学的计算机科学学位，因为学校离我家很近。但是真正上大学的决定是在我没能入选海军学院之后。

# 本科生拱门

是的，最后一个拱门。欢迎来到数学专业。数学？什么？我讨厌数学，但这是我选择的道路，我必须活下去。第一年，我接触了 C++世界，带着指针的东西离开了。我精通 C++？还没掌握只是知道而已。然后第二年接触 java，也接触另一门数学学科，如代数、线性代数、数值方法、运筹学、统计学、自动机等。哈哈…如果你像我一样讨厌数学，你有两个选择:学习如何去热爱它，也许至少努力学习它，或者离开。:p

回到 java 的故事，从 C++开始很容易打印到屏幕上只是`cout<<"bla bla bla";`但是 java 对于 IO 有更复杂的。但这只是我认识 java 的第一个星期的故事，每天都在用 java 度过美好的一天，直到我在大学的最后一年。C#入侵，我无法避免，因为比 java (IMHO)简单又好。我只用了两周就完成了我最后一年项目的申请，但是我花了 6 个月才学会解决这个问题。我的本科生涯以 C#作为主要语言来完成编程工作。

# 考研拱门

`null pointer exception`

# 程序员拱门

还有 bonus arch，作为程序员的短篇小说。从 C#程序员开始，首先是项目接口微控制器，然后是医疗软件。用 C#学习一年，然后集中使用 python 和 tornado 包来构建网站，并使用 backbonejs 作为前端脚本。仅仅一年，现在仍然使用 Golang 作为主要的编程语言。关于我的冒险的冗长乏味的故事。

# 结论

一开始讨厌数学，然后以同样的情况结束。现在我意识到我没有迷失在选择计算机科学的道路上。对于每个决定加入我选择的地狱世界的人。给你两条信息，热爱你的工作，永不放弃。全世界所有的程序员都将是你最好的朋友，别忘了他们。我爱你们所有人。:p
