# 在特殊 Fn 键和标准 Fn 键之间切换[MAC]

> 原文:[https://dev . to/riyaz/toggle-between-special-fn-keys-and-standard-fn-keys-MAC](https://dev.to/riyaz/toggle-between-special-fn-keys-and-standard-fn-keys-mac)

MacBook 键盘隐藏(合并)了 Mac 硬件键下的功能键，如亮度控制和音量控制，所以如果你想使用功能键，我们应该在按任何 fn 键之前按住特殊的`fn`键。

作为一名开发人员，我更喜欢使用 fn 键，而不是按`fn`键，然而当我不开发时，我更喜欢标准的硬件键，如屏幕亮度和音量。

Mac 有一个[设置](https://support.apple.com/en-us/HT204436)允许我们改变这种行为，但在这两者之间定期切换很乏味，所以我创建了一个[阿尔弗雷德脚本](http://www.packal.org/workflow/toggle-fn-keys-standard-keys)来缓解这一点。

有了这个 Alfred 工作流程，无论你在哪里，你都可以在这两个键之间切换，试试吧。
