# 杰基尔样板包

> 原文:[https://dev.to/mariordev/jekyll-boilerpack-4oc9](https://dev.to/mariordev/jekyll-boilerpack-4oc9)

上周我发布了一个 Jekyll 样板文件，我称之为[样板文件](https://github.com/mariordev/jekyll-boilerpack)。在创建了几个 Jekyll 项目后，我厌倦了每次都要从头开始设置一切。通常需要准备的是相同的东西，所以我创建了 Jekyll BoilerPack，它是一个 repo，包含启动一个新的 Jekyll 项目所需的所有常用样板文件。

BoilerPack 预配置了以下工具，因此您可以立即投入使用:

*   引导 3 (Sass 端口)
*   字体真棒(从 CDN 加载)
*   自动贴合
*   咕哝

它还具有以下开箱即用的特性:

*   静态导航条
*   粘性页脚
*   旧浏览器警告通知
*   禁用-javascript 通知
*   站点范围的自定义通知
*   动态转到顶部按钮

您可以在 [GitHub repo](https://github.com/mariordev/jekyll-boilerpack) 上找到详细的安装说明和使用文档。请随意复制和修改。我希望它对你和对我一样有用。

干杯！
