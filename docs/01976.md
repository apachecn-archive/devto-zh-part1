在[上一篇文章](https://thecodingexpress.blogspot.in/2017/12/wds3-html-basic-body-tags.html)中我们了解到:

*   HTML 标题
*   段落
*   链接
*   图片现在我们有 3 条我们应该永远遵守的规则:

1.  虽然 HTML 标签不区分大小写，也就是说你可以使用 T2 P T3 或 T4 P T5 来标记段落，但是你应该总是使用小写标签，因为很多标准在他们的文档中都使用小写标签，这样你会养成一个好习惯。
2.  你永远不应该忘记 HTML 中的结束标签。在某些情况下，如果不使用结束标记，文档会显示得很完美，但请将其视为巧合。在许多情况下，它会产生错误或不希望的输出。
3.  在上节课中，我们学习了由名称和值对组成的属性，请始终将它们用作 name="value "。**引号**(")虽然不是必须的，但是用起来是个好习惯。您可以使用单引号或双引号，但最好是双引号。当值本身包含双引号时，必须使用单引号。
    两个重要的注意事项:

4.  你可以在**嵌套形式**中使用 HTML 标签(事实上你已经在使用了)。在所有的例子中，我们将所有的文档都写在了< html >和< /html >标签中，这意味着我们所有的其他标签都嵌套在这些标签之间。类似地，您可以在段落标记内使用标题标记来创建段落内的标题。

5.  有些元素不需要任何内容。没有内容的 HTML 元素称为**空元素**。< br >是一个没有结束标签的空元素(< br >标签定义了一个换行符)。空元素可以像这样在开始标签中被“封闭”:< br / >。HTML5 不要求您关闭空元素。

### HTML 属性

在上一个教程( [WDS3](https://thecodingexpress.blogspot.in/2017/12/wds3-html-basic-body-tags.html) )中，在学习 HTML 链接和图像标签时，我们学习了一个叫做属性的新概念。现在让我们详细研究一下:

*   属性提供了更多关于元素的控制/信息。
*   属性总是在标签对的起始标签中指定。
*   对于属性的名称和值对，属性被写成 **name="value"** 。
    现在让我们看看一些属性及其用途:

*   HTML 链接是用< a >标签定义的。链接地址在 href 属性中指定。我们已经在 [WDS3](https://thecodingexpress.blogspot.in/2017/12/wds3-html-basic-body-tags.html) 中做了一个例子。

*   **src** -用于显示图像的< img >标签。它代表图像的来源。

*   **宽度**和**高度**——表示一幅图像的宽度和高度。

*   **alt** -代表图像不显示时使用的替代文本。

*   **title** -该属性用于< p >标签内的段落元素。当鼠标悬停在段落上时，title 属性的值将显示为工具提示。
