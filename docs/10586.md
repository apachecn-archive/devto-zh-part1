# 我的第一台开发机器

> 原文:[https://dev.to/ben/my-first-dev-machine](https://dev.to/ben/my-first-dev-machine)

我在学习编码。这次是真的。我不被允许在图书馆的电脑上安装 Rails，所以我花 40 美元从一个朋友那里买了一台 Windows 机器，安装了 Ubuntu。一个不同的朋友给我指出了这个方向，他认为电脑价格的下跌很大程度上是由于软件问题和营销，而不是硬件问题。这是一个非常好的决定。这不是一个伟大的机器，但它完全完成了任务。

我真的没有多余的钱可以花。我同意用 20 美元分期付款购买。到了兑现我的后半部分安排的时候，我仍然没有钱，所以在离开城镇之前，我避免碰到那个朋友。他并不在乎，但我在第二次见到他时还了他一些啤酒。那时我是一名收入颇丰的软件开发人员。

我很高兴我买了那台电脑。
