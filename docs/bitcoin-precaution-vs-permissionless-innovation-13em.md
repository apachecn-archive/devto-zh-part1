# 比特币:预防与未经许可的创新

> 原文：<https://dev.to/michaelsolati/bitcoin-precaution-vs-permissionless-innovation-13em>

作为《哈利·波特》系列的粉丝，《哈利·波特与凤凰社》中有一句话一直让我印象深刻。"*必须阻止为了进步而进步，因为我们久经考验的传统往往不需要修补。这句话只是魔法部部长顾问多洛雷斯·乌姆里奇长篇独白的一个片段。它被主要人物解释为意味着魔法世界的管理机构害怕变化，更喜欢停滞不前的社会，因为他们感觉好像已经实现了一个完美的世界或平衡。这种扼杀创新和进步的欲望最终会对魔法世界造成伤害，因为尽管他们可能不想承认外部力量的崛起可能会损害他们的生活方式，但他们也没有采取措施让他们的社会为不可避免的变化做好准备。*

因为改变是不可阻挡的，无论是在《哈利·波特》中还是在现实世界中，准备好接受甚至拥抱世界上正在发生的变化是很重要的。一个正在慢慢渗透社会的变化是比特币等加密货币的使用。虽然比特币还不像其他创新产品那样流行，但它已经引发了强烈的反应。许多人对人民币作为一种独立于政府监管和监督的贸易体系的潜力感到兴奋，政府监管和监督可能具有侵扰性和政治动机。其他人担心缺乏制度控制会让加密货币助长犯罪交易。我相信比特币和其他类似的货币有能力更好地重新定义我们看待货币和金融的方式，不应该因为它们变得更加流行而被扼杀。

首先，我们来看看现在的钱是怎么运作的。货币主要有三个功能:[作为交换媒介、记账单位和价值储存手段](http://www.shmoop.com/money-banking/economic-definition.html)。作为一种*交换媒介，*社会已经同意在交易中接受它。商人接受金钱以换取货物，雇员接受金钱以换取劳动，如此等等。作为一种*记账单位*，货币是一种简单的价值交流方式。一部 iPhone 值多少钱？与 Jason Chaffetz 的想法相反，这远远达不到医疗保险的年度成本。其建议零售价约为 650 美元。这样，金钱就成了一种方便的方式，可以很容易地理解和交流价值。美国健康保险的年平均保费是 1255 美元。因此，可以有把握地说，以货币为记账单位计算，每年的健康保险大约需要两部 iPhones。最后，作为价值储存手段，金钱让我们能够保持和积累财富，而不用担心它会贬值。举例来说，如果羊死了，嫁妆就会失去价值，需要食物来维持。

货币很大程度上是由它所服务的功能来定义的。当我们看看比特币今天是如何存在和运作的，我们可以很容易地看到它如何通过扩展货币的功能来破坏我们与金融机构的关系。比特币主要是作为交换的*媒介。虽然不像美元或欧元那样被普遍接受，但它正被越来越多的供应商作为一种货币接受。与由管理机构支持的现代货币不同，比特币与国家无关。这既可以被视为一种好处，也可以被视为一种坏处。*

在之前使用 iPhones 和健康保险的比较中，我们使用美元作为记账单位。如果我们将欧洲的 iPhone 与美国的健康保险进行比较，我们不仅需要将服务或产品的价值转换为货币，还需要转换货币的价值。这在货币的功能中增加了一个额外的步骤或过程，作为一个*记账单位*。虽然比特币目前作为一种小众货币的地位，没有任何大型银行或政府的支持或支持，可能会使其[太不稳定](https://99bitcoins.com/price-chart-history/)而不能用作货币，但如果它获得这种支持并在日常使用中增长([日俄两国已经开始使货币合法化](http://www.cnbc.com/2017/04/12/bitcoin-price-rises-japan-russia-regulation.html)，那么比特币最终可能会取代传统货币，成为全球市场的通用记账单位。

除了全球市场的货币兑换，传统上，一个独立的中介是必要的，它位于使用货币作为交易媒介的双方之间。如果你用借记卡在当地的星巴克购物，你的银行会安全地处理付款并更新他们自己的私人账户。然而，有了比特币，交易是在双方之间直接进行的，而[依赖于分发给庞大的比特币用户网络](https://internationalbanker.com/banking/impact-bitcoin-central-banks/)的账本副本。这些比特币用户存在于世界各地，可以使用记录独立认证任何变更，免去了银行验证交易的需要，将交易的记录和验证放在群众手中。

虽然这在像美国这样的国家看起来不太重要，因为银行通常很容易找到，但对于生活在第三世界国家的人来说，这可能有很大的好处。看看像尼日利亚这样的国家，只有 36%的人口可以获得传统的银行服务。虽然有一些服务将小额金融机构聚集在一起，但费用平均为 12%，这使得许多尼日利亚人不得不亲自将钱放在袋子里放到公交车上发送，而不是通过银行转账。可以说，比特币可能是金融机构的一次令人难以置信的转型和创新，让银行的中央系统及其提供的服务变得多余，同时仍然满足货币的需求和功能，并将金融服务扩展到以前负担不起的人。

对于我们世界的任何变化，总有一些人持怀疑态度。*预防原则*迎合了“ [*新的创新应该被限制或禁止，直到它们的开发者能够证明它们不会对个人、群体、特定实体、文化规范或各种现有的法律、规范或传统造成任何伤害。*](https://www.researchgate.net/publication/272156825_The_Connected_World_Examining_the_Internet_of_Things) “然而，那样的信念会导致停滞不前，导致坚持这一信念的国家在全球技术竞赛中落后。例如，韩国通过接受和致力于蓬勃发展的技术，在许多领域超越了美国。他们自诩拥有世界上最快的无线宽带，而美国电信业创新缓慢，提供的服务普遍较差。欧盟已经开始使用国民身份证，其功能类似于美国的社会保障卡，但具有实际的安全措施和照片，使其成为一种更好的身份识别形式。

我认为我们应该对比特币等新技术采取*无许可创新*的方式。鉴于它们所带来的好处，应该允许它们的发展和扩大，直到它们的危险和风险得到令人信服的证明。用美国法律来说，*创新在被证明有罪之前是无辜的*。1997 年，当克林顿政府发布全球电子商务框架时，T4 的无许可创新政策在美国获得了成功。联邦贸易委员会专员 Maureen K. Ohlhausen 在谈到克林顿政府的政策时指出:“[互联网的成功在很大程度上是由试验不同商业模式的自由所推动的，其中最好的模式得以生存并蓬勃发展，即使面对最初的不熟悉和对消费者和竞争对手的影响的不安。当政策是出于对最坏情况的恐惧而制定时，创新可能会被极大地扼杀。虽然风险可能是危险或可怕的，但这些风险带来的经验反过来会成为我们社会的智慧、知识和进步，这是不断变化的世界所必需的。](https://www.ftc.gov/sites/default/files/documents/public_statements/remarks-commissioner-maureen-k.ohlhausen-ftc-internet-things-workshop/131119iotspeech.pdf)

至于比特币作为货币转型催化剂的出现，我觉得总体上应该受到欢迎和推广。通过扩大其作为交易媒介和记账单位的功能，比特币有可能改变货币。虽然可能有一些政策真空需要解决(两个月前，美国国税局向最大的比特币交易公司[比特币基地发出传票，要求其披露三年内所有美国客户的记录)](https://medium.com/@barmstrong/coinbase-and-the-irs-c4e2e386e0cf)，但这些问题可以及时解决，不需要阻碍并完全破坏比特币和未经许可的创新在我们社会的融合。
