# 一点都不后悔

> 原文：<https://dev.to/dmigwi/no-regrets-at-all-g5>

七年前(2010 年)，我正准备参加 11 月 21 日开始的 KCSE(肯尼亚中学考试证书)考试，这标志着肯尼亚中学教育的结束。考试结束后，一个人将被带入真正无情的世界，在那里他们可以选择去做珠雅卡利/非正式工作，或者继续他们的大学教育，并根据他们在 KSCE 考试中的分数选择课程。

在准备 KCSE 考试的时候，我清楚地知道，无论付出什么代价，我都必须去上大学，学习工程学。早些时候，我们有一个职业招聘会，我瞄准了计算机和电子学士学位。为了准备参加学位课程，我保存了几大块废弃的电子零件和其他物品，这些东西可以让我尽可能顺利地登机。偶尔我会和我妈妈争论，因为在我试图用我的废弃电子零件组装东西后，我把其他电子零件扔得满屋都是。

KSCE 考试成功了，遗憾的是，当结果出来时，我无法加入我梦想中的课程，因为我的成绩低于分数线。我向某个讲师寻求建议，他建议我攻读计算机科学学士学位，希望一年后我可以轻松地转到工程学位。讲师忘了告诉我，他的想法只有在我被一所提供两个学位的大学录取的情况下才能实现。

大学的选择是由一个叫做 JAB 的政府半官方机构完成的，这个机构现在叫做 KUCCPS，我被安排到了一所离我最初想去的学校很远的大学。被录取后的几个星期，我开始意识到现实。在新的大学里，我所能做的就是计算机科学，在我完全适应并接受把它作为一门课程和职业之前，我花了很多时间。甚至尝试过做校际转学但是被拒绝了。

第二年，我开始发现了命令一段代码按照指示去做的感觉。我的第一段代码是一个 C 语言的“Hello World”程序，感觉就像上帝把他的一些权力委托给了我，他常说，要有光，就有光。我觉得计算机科学不像工程学，我只是被我的想象力所限制。我的程序编程讲师在他的一次讲座中解释说，qwerty 键盘的变化可能需要几年时间才能成为标准，但在很短时间内构建的软件可能会改变生活。他基本上告诉我们，硬件的改变需要时间才能感受到它的效果，但我们手中有能力通过软件的改变来改变世界。

随着时间的推移，我学会了如何通过谷歌通宵用 Java 写代码，当时在大学的无线接入点，噪音水平最低，WIFI 速度最快。我开发了一个可以帮助传输文件的应用程序，因为当时我买不起高质量的 u 盘。在我最后一年的项目中，我制作了一个迷宫里的捉迷藏游戏。

构建软件给了我最美好的回忆和快乐的时光。我花了几乎一整天调试一段代码，却发现一个分号修复了这个错误。修复漏洞后，我感觉棒极了，我无法用任何东西来交换这种感觉。如果我从来没有做过软件开发，我确信我会在我临死前没有做过的事情清单上多一个遗憾。

自从我用 c 语言编写我的第一个 hello world 程序已经过去五年了，到目前为止，我对没有参加工程课程并不后悔。软件开发给了我更大的生活目标。我职业生涯的高潮将会是当我制造一个既有计算机软件又有硬件组件的产品的时候。我目前精通 Python 和围棋。我将很快参加一个 C 刷新课程，然后冒险进入硬件编程。一旦做到了这一点，我的生活将会变得充实。
