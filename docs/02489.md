# 是什么让你想成为一名开发者？

> 原文：<https://dev.to/jvarness/what-made-you-want-to-become-a-dev-d58>

在我们的职业生涯或教育中，我们都有那些决定性的时刻，让我们意识到我们想成为开发者。

对我来说，那是在我的第一堂编程课上，我写了一些 Python 代码，把几个数字加在一起。它不像计算器或任何东西，它是字面上的:

```
print 3 + 4 
```
