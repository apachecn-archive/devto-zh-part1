# 为什么你应该从你梦想的工作开始

> 原文：<https://dev.to/_patrickgod/why-you-should-start-with-your-dream-job-8f>

为软件开发人员选择合适的行业是他或她职业生涯中最困难的决定之一。幸运的是，你不必坚持你的第一个决定。

想想你的理想工作。真的。对我来说，是成为一名游戏开发者。也许这能引起你的共鸣。许多软件开发者喜欢视频游戏，也想自己制作游戏。

但也可能是应用程序开发。或者你想为一家汽车公司工作，为新的特斯拉系列打造下一个令人惊叹的功能。

不管是什么，你都应该试着做第一件事。我知道说起来容易做起来难，但是如果你真的想在那个特定的位置上做任何事情，你应该努力工作来实现你的目标。你会的。也许这一点也不难。

有哪些选择？也许有可能在你梦想的公司实习，没有任何薪水，或者至少薪水很低。那你应该去做。

关键是，你需要理想工作的经验。不是因为你会学着表现得更好。这是因为也许，仅仅是也许，你梦想中的工作突然变得不那么好了。在职业生涯的早期了解这一事实至关重要。因为如果你不这样做，你会一直想着，有一份完美的工作可以让你从现在悲惨的职业生涯中解脱出来——我知道这听起来有点极端。然而，如果你热爱这份工作，那么**你会竭尽全力在这个行业取得成功**。

我给大家分享一个小故事。

当我还是个孩子的时候，我第一次玩电子游戏，我就知道我想自己制作游戏。我开始看书，用 QBasic 写文字冒险，学 C++，给 Windows 写小游戏，做游戏的梦想越来越大。当我学习计算机科学时，我做过几份工作。在零售业、银行、5 人公司等等。最后，在硕士学习期间，我和我的朋友在一家游戏公司找到了一份工作。为了达到这一点，我们用 XNA 和 Java 小程序完成了游戏，这样我们就可以在我们的网站上展示它们，我们还组织了与来自我们大学游戏行业的实际开发者的会谈。

我们想成为这个行业的一部分。

我一得到这份工作，就热泪盈眶。我终于做到了。我实现了长久以来的梦想。起初，这很神奇。

当我们拿到学位时，他们想让我们全职为他们工作。当然，我们做了，尽管工资极低。如果我们是从银行或网站开发人员开始的，我们可以轻松地多赚 10 万美元，但是我们的爱，我对游戏的爱更大。

我用 C#做了 Unity 里的游戏。主要是写游戏代码和创造一些粒子效果。镜头在主角周围移动的方式，战斗场景(这是一个回合制的地牢爬虫，名为“地牢帝国”)，游戏内的聊天系统是我做的。游戏杂志写了关于这款游戏的报道，游戏发布后，我有生以来第一次可以见到玩我参与的游戏的玩家并与他们交谈。绝对惊艳。

但是，残酷的事实是，最终，兴奋消失了。我想强调的是，这是我的个人经历，你的经历可能会不同。但对我来说，这份工作并没有真的变成“另一份工作”，而是我对其他领域的兴趣增加了。**我想知道软件开发世界还能提供什么**——我想赚更多的钱。

所以我在一家银行找到了一份工作，这是一场灾难，然后我转到了网站开发，这也是我一直在做的事情。

这个故事的寓意是什么？尝试新事物。越早越好。你可能会对所谓的“理想工作”的结果感到惊讶。不要把钱放在第一位，除非它是你绝对的第一要务。如果你想创业，那就去做吧。也许行得通，也许行不通。也许你只是需要更多的经验，然后再开一家公司。无论哪种方式，**你都会获得经验，这是你能得到的最有价值的东西**。没人能把这从你身边夺走。最重要的是，你不会过着梦想一份不存在的工作的生活。
