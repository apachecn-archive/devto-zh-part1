### 资助 PyPI 基础设施和开发的提案

前几天，我在听最新一集的[跟我说说 Python:我们是否未能资助 Python 的核心基础设施？](https://talkpython.fm/episodes/show/84/are-we-failing-to-fund-python-s-core-infrastructure)，由来自 [Python 软件基金会](https://www.python.org/psf/)、 [PyPI](https://pypi.org/) 和[的嘉宾组成的小组宣读文档](https://readthedocs.org/)。作为一个写开源代码的人，可持续发展的话题总是在我脑海中浮动。能够主要致力于让我大脑发痒的事情肯定会很棒，但即使你有一个非常成功的项目——我没有——它仍然非常难以实现。

我一直想知道像 PSF 这样的组织是如何做到这一切的，特别是对于那些拥有我们在 PyPI 中看到的流量水平的基础设施和系统。我能得出的最相似的例子是研究项目，在那里大量的时间被用来寻找合适的资金。
