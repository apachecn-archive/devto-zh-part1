# 机器人在选举中发挥的作用比人们意识到的要大

> 原文:[https://dev . to/Ben/bots-is-being-a-big-role-in-the-election-than-people-awareness](https://dev.to/ben/bots-are-playing-a-bigger-role-in-the-election-than-people-realize)

众所周知，选举越来越受到信息技术的影响。由于[希拉里电子邮件丑闻](http://www.vox.com/cards/hillary-clinton-email-scandal)以及最近[苹果公司和联邦调查局](http://www.nytimes.com/interactive/2016/03/03/technology/apple-iphone-fbi-fight-explained.html)民主党全国委员会黑客之间的争斗，这次选举中“服务器”、“加密”和“网络攻击”等术语进入辩论的次数超过了以往任何一个竞选季。民主党全国委员会和克林顿竞选团队的黑客已经表明，技术不再是一个被遗忘的政策问题，而是美国政治的积极影响者。

没有被广泛认识到的是，底层技术是如何戏剧性地使坏人能够以前所未有的规模篡改这场选举，不是在散布新闻的成功黑客企图中，而是就在我们的鼻子底下进行日常选举话语。有组织的巨魔显然一直在利用网络和社交媒体来获得对特朗普的巨大支持，但我认为，这种操纵的规模可能比人们意识到的要大得多，通过代表大量互联网活动的机器人。黑帽自动化正在扭曲全国的对话。

在第一场总统辩论后， [#TrumpWon](https://twitter.com/hashtag/TrumpWon) 成为推特上最热门的标签。结果很快与亲特朗普的巨魔团伙有组织地行动联系起来。鉴于实际民意调查的结果，很少有人认为这种说法是真实的或重要的，但它仍然转移了话题，在真正的 trollish 结果中，人们通过愤怒反对它来合法化这一概念。当社交网络认为一个话题重要到足以成为趋势时，我们会看到它迅速扩散到其他媒体渠道。

社会证明是人类决策过程中的一个关键因素。在心理学实验的历史中，这是一个被反复测试和证明的概念。你可能听说过臭名昭著的斯坦利·米尔格拉姆实验(Stanley Milgram experiments )( T1)——受试者在所谓权威人物的指导下被哄骗做出残忍的行为。除了权威的影响之外，心理学实验还证实了一个观点，即人们在任何环境下都努力保持一致，并对看似流行的东西投入大量精力，即使它与他们自己的逻辑结论不一致。[强盗洞穴实验](http://www.simplypsychology.org/robbers-cave.html)以一种迷人的方式证明了这一点，其中一些参与者愿意同意大多数人的意见，即使有明确的证据反对。在决定采取何种行动时，社会认可往往比个人想法更重要。

网络话语很重要，即使是大二的。长期以来，民众一直受到推动对话的宣传的摆布。互联网的分布式本质意味着宣传不那么来自中央政府和单一的媒体来源，而是来自每一场争论的每一方的半组织的意识形态蜂巢。

我们总体上可以接受，特朗普有一段时间拥有一群狂热的互联网支持者。但有一种预期是，所有这些支持都表明了与特朗普持相同意识形态的人的汗水和奉献。你是否认同意识形态不应该影响你是否将他的支持规模视为可接受的现实。这是民主的本质。但我认为天平已经倾斜，从根本上破坏了我们的民主理念。由相对较少的人分发的软件是许多这种互联网活动的背后原因。特朗普的支持者中没有一大群人宣称#特朗普赢了。这是一个规模更小、更狂热的组织，伪造这一活动。我没有特权知道到底有多少活动是人类点击转发按钮，有多少是机器人，但我有理由相信，在这个选举周期内，自主活动远远比人类更能代表可测量的互动。

以产生有利可图的社会证明为目的的假账户的存在是一个众所周知的问题。毫无疑问，米特罗姆尼的竞选活动是由虚假的社交媒体活动支撑的。2014 年，Instagram 承诺[清除其服务中明显虚假的账户](http://www.bbc.com/news/technology-30548463)，一些名人受到了残酷的影响。贾斯汀比伯失去了 3，538，228 名粉丝，阿肯失去了超过一半的粉丝，一些看似受欢迎的账户被发现几乎完全是虚假粉丝的产品。Instagram 当天失去了 18，880，211 名自己的粉丝，这说明了这些公司从感知活动中受益的不正当动机。Fiverr 是一个致力于让人们更容易购买价格约为 5 美元的服务的网站，它提供的服务可以让你花 5 美元购买[数万条真正的美国转发。你能想象购买 10，000 条真实转发的想法有多荒谬吗？即使一个真正的人每秒可以转发一条微博，他们也要花将近三个小时来完成这个完全不可能的动作。我想见见有精力](https://www.fiverr.com/smokings/add-1000-real-usa-retweets-on-up-to-5-your-tweets)[转发 10，000 次的候选人。Twitter 自己一度发布了一份声明](http://www.cnn.com/2016/09/27/politics/donald-trump-hillary-clinton-stamina/)[声称他们的用户群中有 8.5%是已知的机器人，他们的测量策略在我看来甚至更保守。Twitter 是一家最近很难增加其活跃用户的公司，可能会在任何一天被卖给另一家科技巨头，它有很多理由淡化机器人的人口统计数据，因为担心他们的股票会比现在跌得更厉害。](http://qz.com/248063/twitter-admits-that-as-many-as-23-million-of-its-active-users-are-actually-bots/)

我描述的这种恶意自动化和网络本身一样古老。受此影响最大的行业是在线广告，其中机器人流量继续构成大量流量。资深广告主管 bob Hoffman[对彭博](http://www.bloomberg.com/features/2015-click-fraud/)说:“没有人知道确切的数字，但可能你在网上消费的 50%都被窃取了。”就像 Instagram 和那些受益于知名度提升的名人一样，受益者也有动机淡化这种行为。像谷歌这样的公司，世界在线广告的领导者，确实采取措施来防止欺诈。他们永远无法完全消除剥削，但他们可以引入障碍，使其计算成本更高，因此无利可图。然而，在政治上，成功欺诈的典型经济学并没有以同样的方式发挥作用。欺诈自动化的成功并不取决于欺诈单位的盈利能力。在这种情况下，成功的衡量标准不太明确，具有相同最终目标的有能力的欺诈者网络会成倍增加。

我不是第一个讨论自动化的人，因为它指的是这次选举，但是大多数人都集中在[发现明显的机器人活动](http://www.theatlantic.com/politics/archive/2016/06/have-twitter-bots-infiltrated-the-2016-election/484964/)上，没有确定明显机器人活动的临界量。然而，我提出这样一个观点，即由于技术的日益成熟，这次选举中的 bot 活动比以往任何时候都更加明显，而且所引用的研究不可能确定有多少活动可以实现自动化。国家安全局，也许还有谷歌和脸书，有资源可以对自动化对这次选举的影响程度做出现实的猜测。

我不从事政治新闻，我是一名软件工程师，对这方面有很多实际的见解。当我年轻的时候，有几次我抓住机会以对现实世界有影响的方式影响在线投票或投票驱动的竞赛。我不想谈论我自己的功绩。我向你保证，他们在思想上是有品味的，从来没有为钓鱼或任何骚扰的目的。然而，对于一个没有预算的单身青少年来说，它们的影响力大得惊人。我认为，正是这种行为，加上政治激励和技术进步，导致了我们所称的更广泛的人类活动。作为一家初创公司的首席技术官和一家副业公司的创始人，在我目前的职位上，我对其中一些主题产生了专业兴趣。在我的公司 [Argo](http://argohq.com) 中，我专注于通信的可扩展性。我们公司对构建技术感兴趣，该技术使利用自动化通信和人与人之间的通信之间的关系更加无缝和可靠，允许更少的人与许多人进行通信，用于诸如消息驱动的服务和商业之类的目的。通过实践开发，我研究了在线交流的细节，以便形成一种声音，将统一的内容传达给其他软件开发人员的国际不同受众。这种一对多的交流不对称让我着迷。也就是说，我相信我对这件事的洞察力是基于相关的经验。

围绕这次选举的讨论正受到算法的影响，其影响方式远远超过了对人类聊天进行分类的方式——它们也产生了人类聊天。有了自然语言处理和卷积神经网络等技术，你可能会称之为专门的人工智能，机器人能够以复杂的方式在网络上无缝交互是完全可能的。像谷歌、微软、亚马逊和苹果这样的公司正在投入数十亿美元专门用于使这成为可能的技术。其他组织，如开放人工智能，也在做同样的事情。这只是在美国，只是我们所知道的努力。显而易见，我们的政府也在进行这种努力，世界上每个国家的组织和政府都可能在做同样的事情，只是进步程度未知。

这些现代软件思想的进步，专注于能够在复杂的棋盘游戏(如围棋)中击败人类的人工智能，为可能的事情奠定了基础，但我相信这对政治对话产生有意义的影响还需要几光年的时间。通过网络传播对话所需的技术并不需要如此复杂的东西。通过 Twitter 放大信息并不需要很多复杂的对话。人类之间的对话足够分散和随机，当人类这样做时，机器人可以无缝地模仿这种行为，并融入人群。很难或不可能区分一个在互联网上随机呼喊支持特朗普并转发所有推文的人和一个参与相同行为的机器人。

一个有趣的例子说明了一个简单的对话机器人可以拥有的力量，那就是一个 [troll-the-troll 机器人](http://www.dailydot.com/unclick/arguebot-twitter-bot-bait-jerks/)。它被开发出来与巨魔进行对话，以浪费他们的时间并挫败他们。机器人 Arguetron 由愤怒的 Twitter 骚扰者的语言触发，并做出简单的反驳，如“不。不对，不对，不对。”它曾经有过一次争论，持续了[几乎连续十个小时](https://twitter.com/srhbutts/status/784172957207670784)。

## [](#scripting-discourse)脚本话语

欺骗互联网活动并不需要复杂的软件架构。一个简单的脚本就可以对一个在线社区产生影响，比如一条充满标签的推文，一次投票，也许是 Reddit 评论或 upvote。考虑下面的脚本，你可以打开你的浏览器控制台，粘贴进去，它会自动导航到一个特定的 tweet，在投票中投票，然后喜欢并转发那个投票。如下所示，这个脚本不是一个精确的实现，但本质上是自动化行为所需要的。

```
function voteInTwitterPoll() {
    window.location.href = "https://twitter.com/user/status/7830811217409";  
    document.getElementById("poll_choice_1").click();  
    document.getElementById("poll_submit").click();  
    document.getElementById("like_button").click();  
    document.getElementById("retweet_button").click();  
    document.getElementById("retweet_confirmation_button").click();  
} 
```

这是一个六步的过程，浏览器被指示导航到一个特定的推文，在投票中投票，然后喜欢并转发它。打包成一个浏览器扩展，人们可以方便地操作他们的浏览器来自动完成大部分人类可以做的事情。对这种行为的任何限制都可以通过支持软件在更高的层次上进行操作。如果实施得当，就不可能检测到自动化。

演示的脚本是这个场景的最简单版本。它本身并不特别强大，但也不远了。想象一下，不是在浏览器中运行一次脚本，而是整晚一遍又一遍地运行。你可以很容易地投票并转发一个投票 10，000 次。因为您只能执行这些操作一次，所以您需要在脚本中添加一个部分，在那里您还创建了一个 Twitter 帐户。这并不难。一次又一次，你创建一个 Twitter 账户，导航到该推文，采取这些行动，重复。但 Twitter 有反垃圾邮件措施来防范这种事情，所以也许你会给自己的行为增加一点多样性，也许你会随机发布几条可信的推文，添加一张个人资料图片，并在投票前喜欢其他几条推文。这将使 Twitter 的措施很难防范你的行为，特别是因为它们有动机尽可能多地消除你第一次行动的障碍。

正如你所料，科技巨头在检测自动化的能力方面非常成熟，但由于用户体验预期和相互冲突的业务目标，这是一个根深蒂固的问题。增长和盈利能力经常与安全措施发生冲突。最近[发生的关于雅虎不顾一切地刺激增长却忽视安全的令人震惊的故事](http://www.nytimes.com/2016/09/29/technology/yahoo-data-breach-hacking.html)就是一个教科书般的例子。认为企业需要简化流程而不是保护流程是非常容易的。强迫新用户进行一个潜在的令人沮丧的过程可能会被否决，因为他们可能永远不会回来。事实上，Twitter 采取了一些相反的措施，比如将“首次发微博的时间”作为一个关键的绩效指标。像 Twitter 和 Reddit 这样的公司，他们的文化影响力远远超过了他们的盈利努力，特别容易做出更难打击恶意自动化的决定。

Twitter 可能仍然能够清除垃圾邮件，通过审计他们看到的行为来贴现明显的垃圾邮件，并应用他们自己的机器学习专业知识来跟踪指示垃圾邮件的模式。他们肯定已经这么做了，但这是一场失败的战斗，因为所有恶意的参与者需要模仿一个看似可信的用户的行为，这并不是一个高标准。这包括欺骗位置信息，创建参与持续各种活动的帐户，例如每隔几天发一次推文，喜欢随机发推文，偶尔转发和直接发消息。人类在 Twitter 和其他网络上参与的活动并不复杂，因此模仿起来并不容易。但是这增加了过程的复杂性，其中不可能像最初描述的那样在一夜之间发送这 10，000 张投票。如果一个恶意演员的自动化脚本每隔几天就进行一次有效的投票以摆脱追踪，这将不会非常有效。这就是并行性发挥作用的地方。

## [](#scaling-up)放大比例

该脚本可以相对容易地跨不同的计算机实例并行运行。我上面演示的脚本可以直接在你电脑的浏览器上运行。这种策略的大规模实际执行将在不同的方式下完成，以确保可伸缩性、隐蔽性和可靠性。它可以在一台强大的机器上以并行的方式完成，通过使用用户拥有和操作的硬件或通过商业云服务(如亚马逊网络服务或谷歌云或微软 Azure)协调分布在几台机器上。通过云计算，这些流程可以以几乎为零的成本大规模扩展。与将要产生的社会影响相比，无论花费多少都是微不足道的。任何买得起食品杂货的人都可以在云中运行强大的自动拖拽引擎。大多数服务甚至提供一个免费层，该层将提供足够的计算能力来进行所描述的活动，并且由于其提供中的隐私的明显必要性，这些服务中的任何一个都没有办法合法地识别这些使用，并且任何以这种方式监管这种行为的尝试本质上都是徒劳的。

为了伪造在线行为而构建的系统的可伸缩性和可靠性由相关人员之间共享的知识来确保，无论他们是以有组织的方式行事，还是仅仅作为志同道合的个人进行交流。面对这些网络的自我修复分布式本质，任何防御措施都会被迅速规避，大多数针对这些群体的尝试都会失败。

事实上，这一切都可以通过一个相当简单的脚本来执行，这对于编码人员保持领先地位的能力来说非常重要。任何时候出现问题，一个简单的修改是可能的，我们犁。任何相当熟练的软件开发人员都能够封装任何复杂的行为，并保持核心应用程序逻辑代码的敏捷和不断变化。这也是允许一个人利用复杂的机器学习软件来将复杂性构建到应用程序中，而不必理解技术本身。所有需要的东西都可以以开放源码的形式免费获得，集成很少只是简单的复制和粘贴。术语“脚本小子”指的是不成熟的青少年利用他人的作品，而没有真正理解它是如何工作的。这类活动的实施者涵盖了从脚本小子到有经验的软件开发人员的所有范围。

我提到的科技公司都意识到了这种行为。他们本身并没有错，因为只要有动机，这种行为就不可避免，但他们确实有一定的动机不提高对这种行为的认识，因为害怕这种言论可能导致的普遍反弹和反技术情绪。这种技术有如此多的潜在好处，很明显，许多技术话语是如此无知和幼稚，例如唐纳德·特朗普(Donald Trump)的“网络”咆哮，以至于有一种动力不去关注网络技术在这一政治进程中带来的破坏。冷静讨论的可能性意味着几乎没有讨论。很难想象哪种结果是最坏的。很少有人在提出担忧的同时不散播恐惧。政治光谱的双方在对他们有利的时候都利用制造恐惧，这是这个等式中不幸的人性的一面。

人们很难将计算机活动的规模和互联活动的全球网络的规模概念化。它让以人为中心的测量单位相形见绌了很多倍，甚至还没有接近。据保守估计，脸书的数据仓库存储超过 300。这是分配给地球上每个人的数百万字节的数据，而且这个数字还在快速增长。存储一个字符需要几个字节，存储一个字符串需要几个字符，比如你的名字。传递的数据量也远远超过了个人存储的数据量。对于这个星球上每一个活着的人来说，在世界计算中的信息和活动都是这个人口的不可估量的倍数。与世界的计算能力相比，社交媒体互动、在线投票甚至选举本身中的数字是如此之小，以至于难以想象所有力量都无法以惊人的轻松操纵人类规模的话语。只需要几千次自动转发，就能把一个无关紧要的话题提升为全国性的现象。社会证明是一件非常强大的事情，它只需要达到人类规模的宏伟。与计算机规模相比，人类人口是一个舍入误差。

## [](#but-who)可是谁？

我希望你相当确信网上的许多活动是自动进行的，以便推动开始对话和影响政治进程，但我还没有讨论谁是这种行为的幕后黑手。我不确定谁是这种行为的幕后黑手，就像我确定这种行为正在发生一样，但我可以推测，有很多团体在做这种事情，他们的行为背后有不同的动机。

自动化在网络上被用于各种不同的目的，因为一个人类活动可以合法地触发许多自动化的活动，甚至很难区分人类活动和机器人活动。但我观察的主题是这场选举和对唐纳德·特朗普的人为支持。值得考虑的是，谁可能是任何旨在提高此人形象、感知影响力和成为美国总统机会的欺骗性策略的幕后黑手。它实际上归结为三个因素的组合:手段、动机和潜在后果。这种结合很难实现，这也是为什么网上的日常活动不太可能受到这类活动的威胁。和其他复杂的不道德行为一样，有很多原因导致大多数人不接受这种行为。如果是的话，事情看起来会比现在糟糕得多。

### [](#donald-trump-and-his-staff)唐纳德·特朗普和他的员工

这是一个有明显动机实施这种策略的团队。如果他们成功了，他们的男人就会成为总统，这在他们的世界里是一件好事。但是这个答案不太可能满足这个场景中的其他需求。特朗普没有表现出任何迹象表明他有能力组织这些复杂的技术交易，或者有能力让自己身边有这样的人。因此，除非他正在玩世界历史上最长的骗局，否则我不认为特朗普是“网络”的一员。如果他被抓了，后果也太多了。因此，尽管特朗普似乎不受他大部分不道德行为后果的影响，但这种行为似乎不符合他的利益。

在论证这种情况的可能性时，特朗普的自我和气质是一张王牌。他对电视收视率等人气指标的痴迷，证明了对风险的非理性偏好。考虑到后果，只有行为不理智的人才会把自己置于这种境地，但这个男人的许多其他行为在本质上可以被认为是独特的自我驱动和情绪化的。

### [](#nation-sponsorship-such-as-russia)国家赞助，如俄罗斯

我们知道很多俄罗斯参与支持特朗普的游说活动。弗拉基米尔·普京拥有大规模部署非自动化的手段，而且该国肯定在某种程度上参与了对美国的计算机攻击，所以我们可以相当肯定，俄罗斯在自动化这种行为方面有一定水平。但我认为，民族国家的参与在一定程度上受到大规模牵连的负面后果的抑制。也就是说，俄罗斯显然参与其中，但我认为我们把太多的注意力放在了他们身上。因为集中活动，分布性和多样性是这个群体无法受益的优势。这类活动最好由不太集中的小组进行，他们不容易被跟踪或离线。俄罗斯更有可能致力于合作，而不是大规模的通用话语自动化。

### [](#distributed-terrorist-groups-such-as-isis)散布 ISIS 等恐怖组织

ISIS 会从这些选举中获益良多。一个被煽动反伊斯兰言论的不稳定团体正是这些组织所需要的，以提高他们的形象，并保持西方敌人美国的叙事。不像俄罗斯这样的国家，他们采取这些行动不会有什么损失。作为一个分布式群体，本质上是一个自我修复网络中分散的节点组合，失策的后果很少，而他们的行动会带来很多好处。

不过，出于几个原因，我对他们实现这一目标的手段表示怀疑。由于其组织的分布式性质，基于对宗教和针对其人民采取的行动的后果的不道德解释，更容易就隐性知识和相互理解的主题进行交流。对美国政治文化叙事的协同攻击似乎超出了这些团体的能力范围。我无法想象这种组织能够完成对美国话语的秘密渗透所需要的大众教育。

### [](#domestic-unorganized-and-semiorganized-groups)国内无组织和半组织的团体

我认为这个团体最有可能在这次选举中对大部分自主的亲特朗普运动负责。如上所述，完成这些特技的技术能力门槛相当低。虽然普通美国人在技术上不太成熟，不太可能是这种行为的实施者，但有数百万人是这样，而且大多数人有能力学习这些策略，即使他们只是在遵循愚蠢的指令。

在西方流行文化中，有一段轻松操纵民意测验的著名历史。在担任《科尔伯特报告》讽刺主持人期间，史蒂芬·科拜尔养成了一个习惯，在网上民意调查中恶心他的粉丝群。你可能听说过英国船只 [Boaty McBoatface](http://www.bbc.com/news/uk-36225652) 的故事，在网上为该船命名的投票以滑稽收场。有一次[谢伊体育场因为投票出错而被关闭](http://deadspin.com/377674/shea-stadium-fans-disagree-with-the-internet)。还有几十个国际知名的此类活动，但就我们的故事而言，最值得注意的是在 [Time 100 在线投票中被选为第一名的时间“模拟辩论赛”](http://content.time.com/time/specials/packages/article/0,28804,1894410_1893837_1894180,00.html)模拟辩论赛(通常全小写，因为模拟辩论赛)是 4chan 创始人克里斯托弗·普尔的网名。4chan 要对网络上的大部分事件负责。通过[在整个社区分发软件](https://techcrunch.com/2009/04/21/4chan-takes-over-the-time-100/)，很可能是一个简单的自动化脚本，该网站的用户能够如此轻松地操纵投票，除了选择模拟法庭作为获胜者之外，整个投票还被操纵为宣读 Marblecake，另一个有几个含义的 4chan 内部参考。

[![Christopher Poole](../Images/bb7eaaf43daca5feeaae14a0cf2728c6.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--nfD6Ws6t--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://i.imgur.com/H7ibaEw.jpg)

4chan 声称每个月有数百万的访问者，但是考虑到一个程序员所拥有的扩展能力，这甚至不是完全必要的。一个人、一个脚本和一些云计算信用可以很容易地完成任何数量的投票操作，只需要几个小时或几天的时间来完成这项任务。但是除了纯粹的技术原因之外，有一个社区是有帮助的。该社区是一个分享想法、提醒其他开发人员潜在漏洞的地方，并且通常会鼓励彼此投入时间来完成此类任务。

4chan 及其用户群的故事既有深度又复杂。它通过网络文化对美国文化的影响被大大低估了。4chan 是 meme 文化、匿名黑客组织(T0)和衍生组织(如 T2)Lulzsec(T3)的发源地。这是“另类右翼”的主要集会。社区和运动之间的关系是复杂的，但它们肯定是交织在一起的。

众所周知，4chan 的另类右翼巨魔军已经被部署来对这场辩论造成严重破坏。这个社区行动背后的动机显然很强，但也相当复杂。虽然 4chan 和相关社区一直是种族主义的出口，但它总是集体标榜自己是“为了 lulz”。也就是说，像“niggerfaggot”这样的煽动性短语的广泛使用被认为是某种自我意识的讽刺。这与特朗普公开的仇外心理和几乎更微妙的种族主义信息的非常严重的影响背道而驰。4 成龙的政治倾向似乎在意识形态上更自由，但政治总是让位于对体制的全面破坏。正是这种逻辑让唐纳德·特朗普和伯尼·桑德斯都受到欢迎，尽管他们持有完全不同的意识形态立场。他们都代表了这个群体认为剥夺了他们权利的引擎中的扳手。但特朗普给这个社区提供了前所未有的机会，让他们选出一个小丑，并在这样做时感到强大。这就是动机，一个分散的自我之旅，来自一群被抛弃的人，他们把它描述为“lulz”。

单靠禅的动机是不够的，我们还需要建立组织和维持某种集体使命的方法。对大多数美国人来说，特朗普的整个候选资格似乎就像是一个“阴阳魔界”插曲。我们已经开始接受一些现实，但这真的没有什么意义。他的崛起与预期大相径庭，以至于整件事看起来像是某人搞砸了的游戏。我认为，从很多方面来说，的确如此。4chan 和它的一般文化，传播到互联网的其他部分，特别是 Reddit，现在比以往任何时候都更是 Twitter，是一个无组织的蜂巢，但它是一个强大的蜂巢。但是，如果没有一定数量的中央集权的支持，像这样的团体是不可能认真对待这些目标的。帕尔默·卢奇出场。

Luckey 最近被揭露为“另类右翼”拉屎运动的部分资金支持者。他是 Oculus Rift 的创始人，该公司以 20 亿美元的价格卖给了脸书，并为他带来了约 7 亿美元的意外之财。那可以买很多东西。彼得·泰尔[向美国展示了当你把激进的意识形态和大量金钱混在一起时会发生什么。Palmer Luckey 是这个国内技术驱动型集团的最后一块拼图。他为成员们提供了机会，让他们真正地把时间投入到制定策略和运用权力上。任何复杂的技术工作都需要大量的工时，而 Luckey 通过投资让这一切变得更加可能。](http://gawker.com/this-is-why-billionaire-peter-thiel-wants-to-end-gawker-1778734026)

帕尔默·勒基，通过偶然地资助那些本身也许并不违法或叛国的行动，并通过不亲自组织这些事业，在某种程度上使自己从这些行动的个人后果中解脱出来。

有了既定的手段、动机和对后果的解脱，这种倾向于技术的另类国内群体最有可能参与这种倾向于自动化的话语行为。这就是他们正在做的。

## [](#a-new-world)新世界

支撑任何选举的社会问题都在发挥作用，但它们正以不同的戏剧性方式表现出来。随着网络技术带来的前所未有的“权力”分配，这次选举，以及所有未来具有全国影响力的事件，都必须通过潜在的大规模操纵的镜头来审视，就像这里正在上演的那样。选举比其他事件受到更大打击的原因是，在大多数情况下，这样做的动机并不强烈，但随着技术的进步，动机的门槛将会降低。

在这些技术的历史上，我们正处于一个独特的实验阶段。政府和知情组织秘密处理此事是不够的。爱德华·斯诺登的美国国家安全局泄密事件以及随后的所有爆料，特别是最近的 T2 雅虎电子邮件间谍事件都说明了这一点。我们需要对塑造我们生活的技术进行透明和公开的集中讨论。这些担忧应该说出来，但为了这个国家的福祉，技术也需要以乐观的态度呈现，因为为了政治利益的耸人听闻和恐惧只会在我们努力领先于国际竞争对手的时候伤害我们。

值得注意的是，这篇文章中没有任何关于希拉里·克林顿如何利用这种行为并从中受益的讨论。我在这次选举中支持克林顿，这使我有偏见，但我真的认为就这种行为的规模和程度而言，故事在特朗普一边。特朗普从这种激进、分散的破坏中获益匪浅。不管是好是坏，技术一直是克林顿竞选的重要组成部分，但我不认为这一论断中指定的动机同样适用于强大的克林顿竞选。

尽管发生了这一切，特朗普的竞选似乎还是失败了，但这个故事不是关于特朗普、Luckey、4chan、普京或 ISIS 的。讨论的是这种自动化的规模和力量，以及它对一个国家的对话可能产生的影响。这些问题在公众意识中并不存在。如果对机器人的影响没有一个共同的理解，我们的世界模型就会偏离现实。
