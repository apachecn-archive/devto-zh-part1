# 纽约市的技术专家敲响了仍能敲响的钟

> 原文:[https://dev . to/mg tei/NYC-technologists-ring-the-bells-than-still-can-ring](https://dev.to/mgtei/nyc-technologists-ring-the-bells-that-still-can-ring)

*   DoSomething.org 的李佳薇和 T2 的首席运营官
*   萨隆·伊特巴雷克([推特](https://twitter.com/saronyitbarek)——[网站，“博客网](http://bloggytoons.com/)”)

组织了一次 [NYC Tech for Good (Meetup)](https://www.meetup.com/Tech-For-Good-NYC/) 会议，讨论当前形势(即 2016 年选举结果)。这是纽约市技术社区的开发者(和项目经理)的一个机会！还有数据科学家！同学们！等)倾吐心声，集思广益。感谢主办方，包括提供披萨和茶点的 [ClarifAI](https://www.clarifai.com/) 和提供场地的[DoSomething.org](https://www.dosomething.org/us)。

我强烈推荐阅读媒体上的[凯文·斯科特](http://www.scottdesignllc.com/)的[笔记(包括编号外卖清单和有用链接)。它会让你对所讲的内容有一个更广泛的了解，而我的笔记则更集中于我感兴趣的内容，可能会稍微深入这个主题。](https://medium.com/@thekevinscott/notes-from-the-tech-for-good-meetup-b35a84e7e781#.8rjo2npx5)

昨晚和今天早上的 Meetup 活动讨论中提出了一些问题，即不要重复美国公民自由联盟、南方贫困法律中心和奥巴马/埃里克·霍尔德的后总统计划等组织已经在做的工作，而是要接触现有的组织，看看我们如何才能提供帮助。

一个人强调了**将技术应用到尚未应用的领域的重要性**(例如，美国公民自由联盟和 SPLC 已经有了供人们报告侵权行为的应用程序)，并提到**环境保护主义是技术尚未覆盖的领域**；值得注意的是，弗林特的一个私人房主首先发现了他们的水中含有铅。

有人提出了选举对儿童已经产生和将要产生的影响。我会附和丹·梅茨关于修复纽约市寄养系统的紧迫性的观点。这个服务于纽约市数千名最弱势儿童和年轻人的系统的状况非常糟糕，我认为它是一个技术尚未涉足但可能产生重大直接影响的领域。

**密码术/加密以及如何在监控状态下组织**被多次提及，其中 [TOR、2FA、Signal(在这篇媒体文章中均有描述)](https://medium.freecodecamp.com/tor-signal-and-beyond-a-law-abiding-citizens-guide-to-privacy-1a593f2104c3#.2ye3fqr7k)和 [VPN (Wiki)](https://en.wikipedia.org/wiki/Virtual_private_network) 被捧为当前标准。**“加密应该*成为*标准。”**

那天晚上我最喜欢的名言是“**给我一个回购**；我不在乎是在 Github 还是 Bitbucket 上。”人们渴望从一个 ***始乱终弃的*** 回购开始编码。也有关于策划黑客之夜/黑客马拉松的讨论。但是我们需要让项目经理参与进来，他们可以将如何满足客户转化为一系列可操作的项目、目标和所需的资源。项目经理，参与进来！

在较小的州和地方政府竞选中变得活跃是多次被收买的，特别是为了让选民为 2018 年的下一次中期选举感到兴奋。2018 年，将对 25 个民主党参议员席位进行投票，而共和党只有 8 个席位。

一个想法是联系那些在以前的选举中投票给奥巴马但在 2016 年没有投票给民主党的人，找出原因(无论是因为不感兴趣还是因为选民压制)，然后在第一个 100 天的邮件列表中瞄准这些观众，告诉他们下一任总统正在做什么，直接影响他们。

共和党人也有亚历克为他们的事业制定千篇一律的立法( [*上周今晚*制作了一个关于州议会的视频&亚历克](https://www.youtube.com/watch?v=aIMgfBZrrZ8))。民主党人是否应该有类似的东西，也看看像茶党这样的组织，以及他们如何动员人们参加地方竞选？

建议参与到你关心的现有组织的“肉空间”中；为此，我想推荐[NewYorkCares.org](https://www.newyorkcares.org/)在当地提供志愿服务机会，并在会上响应呼吁其他人将他们的时间和/或资源给予[皇后联盟(C4Q)](http://www.c4q.nyc/) 的声音，该联盟运行[接入代码](http://www.c4q.nyc/accesscode/)。**作为一名 C4Q 的学生，我可以证明我们对资源和志愿者的需求，以及该计划对学生和整个技术多样性的强大影响。**

[Code.gov](https://code.gov/#/)、[美国数字服务](https://www.whitehouse.gov/participate/united-states-digital-service)和 [CodeCorps](https://www.codecorps.org/) 被提到作为研究使用的资源。还有一位来自见证组织的代表，他公开表示同情和团结优先于陈腐的同理心，并采取行动建立强烈的个人联系感。

这些是我从会议中获得的与技术相关的收获(有更多的情感和个人收获，我很乐意在其他地方讨论)。这个帖子上的图片是 Ben Halpern 在这个小组进行过程中所做的笔记。感谢每一个来过或者想要来的人，感谢他们正在研究发生了什么。

技术不能解决所有问题，但是我们作为技术人员可以“敲响仍然可以敲响的钟。”
