# Android Studio 2.2.3 解决方案中的 IllegalStateException

> 原文:[https://dev . to/arslanark/illegalstateexception-in-Android-studio-223-solution](https://dev.to/arslanark/illegalstateexception-in-android-studio-223-solution)

每个使用 Android Studio 的人在通过活动列表创建新活动时都会遇到这个错误。

Illegalstateexception:向导完成后尝试了无效的操作。

这个问题的解决方法很简单，与其从活动列表中选择，不如使用图库。请等待 Android Studio 开发人员解决这个问题，当您在 Android Studio 或任何开发软件中看到错误时，请务必报告。
