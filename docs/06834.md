# JS 蒙特利尔-反应故事书介绍

关于[反应故事书](https://github.com/kadirahq/react-storybook)的演讲。

这些幻灯片可在[storybookslides.iamdeveloper.com](https://storybookslides.iamdeveloper.com)获得。

# 要在本地查看幻灯片

*   运行`npm i`或`yarn`。
*   运行`npm start`或`yarn start`。
*   打开浏览器，点击 [http://localhost:3000](http://localhost:3000) ，就可以滚了。

注:感谢[眼镜](https://github.com/FormidableLabs/spectacle)项目和[眼镜样板](https://github.com/FormidableLabs/spectacle-boilerplate)便于编写幻灯片。

# 运行工作台(React Storybook)

*   如果您还没有运行`npm i`或`yarn`，请运行。
*   运行`npm run workbench`或`yarn run workbench`。
*   打开浏览器，点击 [http://localhost:9001](http://localhost:9001) ，就可以滚了。
*   工作台在[storybookdemo.iamdeveloper.com](https://storybookdemo.iamdeveloper.com)可用
