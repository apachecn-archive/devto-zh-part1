# 云计算

> 原文:[https://dev.to/funkysi1701/cloud-computing-1jpb](https://dev.to/funkysi1701/cloud-computing-1jpb)

我在一家提供云计算网站的公司工作，但是什么是云计算？

维基百科将其定义为“云计算涉及部署远程服务器和软件网络组，允许集中数据存储和在线访问计算机服务或资源。”

呃。那是什么意思？简单来说，这意味着将你的数据存储在远程服务器上。任何时候你在网站上提交信息，你都是在向“云”提交你的信息

但是云还能为你做更多的事情。搜索巨头谷歌也开发了 android 手机操作系统，并开发了一系列云计算产品。Google Drive 产品允许你上传甚至创建文件，这些文件被保存到云中，一旦进入云中，就可以从任何有互联网连接的设备上访问它们。你再也不用冒你最喜欢的文件在一台打不开的电脑上或者在一个死硬盘上的风险了，把它保存到云端，你就可以一直访问它了。

谷歌并不是唯一这样做的公司，微软已经将 OneDrive 整合到他们的操作系统的核心，苹果和许多其他公司也发布了类似的产品。

这个云计算时代让你的数据和设备之间的分离更加清晰。用不了多久，不管你用什么设备，你的文件都会被处理。你在办公室处理一个电子表格，你把它保存到云端，然后你可以在旅途中用你的平板电脑继续处理它。就在睡觉前，你想检查你是否在那个文档中添加了什么，所以你很快检查了你的手机。这多酷啊！

但是如果我戴上系统管理员的帽子，云计算的意义远不止于此。微软 Azure 允许许多你通常在服务器上运行的服务被保存到云中。这有很多好处，可以让你专注于其他问题，而让云来照看这些服务。
