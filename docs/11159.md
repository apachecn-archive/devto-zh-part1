# 学习贸易

> 原文:[https://dev.to/designfrontier/learning-the-trade-7h](https://dev.to/designfrontier/learning-the-trade-7h)

我想我会把我发现的有趣和有用的资源列表放在一起。这绝不是详尽无遗的，随着时间的推移，我会发现新的有趣的东西。分为以下几个部分。

播客:

[响应式网页设计播客](http://responsivewebdesign.com/podcast/)这是我目前最喜欢的播客之一。它以可靠的方式涵盖了响应式设计和编码的关注点。设计师必备。

如果你对函数式编程感兴趣，尤其是对 Haskell 感兴趣，Haskellcast 会很棒。

我的播客列表中最近增加了一个新内容。有趣的交易调查。有时感觉它的信息有点过时，但很有趣。

Frontside 主要以 Ember 为中心，但他们有一些真正伟大的人类兴趣/过程片段，值得一听。

文章:

Ethan Marcotte 的响应式网页设计。引发这一切的文章。

Eric Elliot 撰写的 JavaScript 第 1 部分的两大支柱

Eric Elliot 的《JavaScript - pt 2 函数式编程的两大支柱》

杰克·富兰克林的《ES6 方式的 JavaScript 模块》

弗兰克·奇梅罗的《网络的纹理》。弗兰克的任何作品都非常优秀。

弗兰克·奇梅罗的《什么样的屏幕需要》

网飞 UI 团队的大型 JavaScript 应用程序的规模和性能。基本上他们放出来的视频都值得一看。

[道格拉斯·克洛克福特的精彩片段](https://www.youtube.com/watch?v=bo36MrBfTk4)

JavaScript 上的克罗克福德是克罗克福德在 JS 上为 YUI 做的演讲的播放列表

书籍:

[JavaScript 好的部分](http://www.amazon.com/JavaScript-Good-Parts-Douglas-Crockford/dp/0596517742/ref=sr_1_1?ie=UTF8&qid=1445366911&sr=8-1&keywords=JavaScript+the+good+parts)。不管你怎么说 Crockford，这仍然是 javascript 开发的基础工作。

雄辩的 Javascript 有一些很棒的部分。特别是作为 js 中函数式编程的初级读本。

Web 表单设计是迄今为止最好的关于表单的书籍。必读。

Frank Chimero 设计的形状。最初，这本书是一个奇妙的设计。

其他材料:

前端电视我的一个朋友做的一个非常棒的项目[特雷弗·米勒](http://www.trevordmiller.com/)。在一口大小的视频中涵盖各种伟大的主题。

exercism.io 是我最近一直在玩的一个很酷的语言学习工具。非常喜欢。

[练习](https://github.com/kolodny/exercises)在范围上类似于练习，但仅侧重于 js。
