# 不编码的宁静

> 原文：<https://dev.to/jonstodle/the-peace-of-not-coding>

*这篇文章最初发表在我的博客上[jonstodle.com](https://blog.jonstodle.com/the-peace-of-not-coding/)T3】*

在过去的几年里，我一直认为我需要在业余时间做一些兼职项目。我需要做些事情。写点代码。我的每一分钟空闲时间。

在很大程度上，这种方式效果很好。没问题。

然而。过去的几个月不太好。我一直在想我应该去编程，但我没有。我坐在笔记本电脑前，不写任何代码。我可以这样做几个小时。

我不再编码，而是看电视节目、有趣的技术讲座或阅读有趣的博客文章。但我没有编码。事实证明，这让我很紧张。直到一个月前，我对自己说:“你知道吗？对于我来说，坐在笔记本电脑前，仅仅为了编码的目的而试图想出一个新的副业是没有意义的。我在做别的事情。”

突然之间，我有了很多空闲时间去做一些我以前没有时间做的事情:我终于在车库里砍了木头；我在车库里收拾了一下；我开始在第二个车库收拾东西。(听起来有点像住在宫殿里)。

最棒的是:这真是一次自由的经历。我没有意识到我有多紧张，因为我感觉不编码是失败的。现在我终于放下了，我注意到我感觉轻松多了。我很惊讶我感觉好多了。它甚至有助于我不去砍柴的坏良心。

当然，一旦我把自己从编码的需要中解放出来，我实际上又开始重新拾起它；但是只有当我喜欢的时候…
