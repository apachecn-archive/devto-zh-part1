# 多仓库库存:Amasty 的 Magento 2 扩展

> 原文:[https://dev . to/all _ Lina 1/multi-warehouse-inventory-magento-2-extension-by-a masty-89c](https://dev.to/all_lina1/multi-warehouse-inventory-magento-2-extension-by-amasty-89c)

Magento 2 的多仓库库存允许在众多仓库中轻松管理库存，并在商店视图级别上跟踪库存项目。有了这个模块，您将能够为每个仓库的所有库存数据运行一个不错的网格。让客户使用智能算法选择最近的仓库。

## [](#features)特性

1.  从 Magento Stock grid 或者从后端的产品页面在不同的仓库之间有效地分配库存:编辑每个仓库的产品数量；跟踪缺货的商品；自动将退回的物品添加到仓库等。；
2.  使用 Magento SOAP API v1、v2 和 Magento REST API 更新/检索股票数据；
3.  将所有产品分为三个库存级别:仓库中的总库存；待装运的存货；可供出售的股票；
4.  隔离来自不同仓库的产品订单，将仓库分配给商店视图和客户群，设置自定义运输方式和费率；
5.  使用仓库选择的智能算法:仓库优先级；位置(最近的仓库)；已下订单的商店视图；仓库分配到的客户组；
6.  通过 CSV/XML 文件手动或自动导入股票数据；通过“导出历史记录”选项卡导出信息。
