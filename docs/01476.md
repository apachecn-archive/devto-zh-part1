# 关于最近的网站的想法

> 原文:[https://dev.to/takimura/web-28a5](https://dev.to/takimura/web-28a5)

在最近的网站上，显示后仍继续显示的话，[Javascript](http://d.hatena.ne.jp/keyword/Javascript) 会以一定的间隔动作，更新数据的类型随处可见。 [虽然多见于 Twitter](http://d.hatena.ne.jp/keyword/Twitter) 等[SNS](http://d.hatena.ne.jp/keyword/SNS) 网站，但这会加重浏览器的重量，无法说[可用性](http://d.hatena.ne.jp/keyword/%A5%E6%A1%BC%A5%B6%A5%D3%A5%EA%A5%C6%A5%A3)很好。

现在，网站和以前有很大的变化，即使是网站，也几乎像[APP 软件](http://d.hatena.ne.jp/keyword/%A5%A2%A5%D7%A5%EA%A5%B1%A1%BC%A5%B7%A5%E7%A5%F3%A5%BD%A5%D5%A5%C8%A5%A6%A5%A7%A5%A2)那样行动的东西在增加。
