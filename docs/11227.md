# 关于 Windows 10 的 10 个事实

> 原文:[https://dev.to/funkysi1701/10-facts-about-windows-10-40ia](https://dev.to/funkysi1701/10-facts-about-windows-10-40ia)

今天微软发布了 Windows 10，我还能说什么呢？我和许多技术人员一样，今天非常兴奋。这个版本的 window 是微软第一次免费发布。

10 你听说正确的 Windows 10 是**免费的**。如果您在 2016 年 7 月 29 日之前将设备升级到 windows 10，它将是免费的，并将永远在该设备上运行。

```
Upgrade to Windows 10 within the first year and it is yours free, forever. 
No subscriptions, no additional costs. 
```

9 与任何升级一样，在做任何事情之前，请务必备份您的计算机和任何重要文件。微软和任何 IT 专家都会告诉你这一点。关于升级的一个很好的指南，请看 Scott Hansleman 的博客。我已经升级了我的笔记本电脑，没有任何问题，但总会有例外，所以要格外小心。

8 Windows 10 是 Windows 的最后一个版本**。我不知道这意味着什么，但我可以猜一猜。**

通常情况下，微软每隔几年就会推出一个新版本的 windows。随着每个新版本的出现，当人们试图让软件和硬件与下一个版本一起工作时，就会出现令人头痛的问题。相反，微软将尝试发布操作系统的更新和补丁，以保持最新。

然而我能看到的只是工作了这么久。最终，微软会推出新的产品，但不太可能被称为 Windows。也许这将是在云中托管的东西，因为他们在 Azure 上有非常强大的存在，这只会随着时间的推移而增长。

7 Windows 10 有 7 种不同的版本。

*   Windows 10 家庭版——相当于我们从 XP 开始就有的家庭版。这个版本是为你的典型消费者设计的。
*   windows 10 Mobile——该版本旨在手机和平板电脑上运行。与以前的手机版本不同，你能在桌面操作系统上找到的一切都应该在手机上可用(但针对较小的屏幕进行了优化)
*   windows 10 Pro——这是我将在工作电脑上测试的版本。它是为小型企业设计的，允许加入域。
*   Windows 10 企业版–该版本仅通过批量许可提供，因此一般用户不容易获得，但主要面向大型企业。
*   Windows 10 教育版——另一个只能通过批量许可获得的版本，面向学校和大学。
*   Windows 10 移动企业版——该版本适用于商业领域的移动设备和平板电脑。标准策略可应用于您所有员工的移动设备。
*   Windows 10 物联网核心——这是一个有趣的版本，旨在运行在硬件和物联网(IoT)设备上。我很想在我的树莓派上玩这个版本

6 Windows 10 推出了一款名为 Edge 的全新网络浏览器。多年来，web 开发人员一直讨厌 Internet Explorer，因为它不支持最新的 web 标准。Internet Explorer 6 导致许多用户迁移到 firefox 和后来的 Chrome 等浏览器，尽管微软尽了最大努力开发更高版本，但人们并没有迁移回来。Edge 是一次重新开始的尝试。

2011 年，苹果推出了个人助理 Siri，现在微软给我们带来了 windows 版的 Cortana。有了 Cortana，你可以让你的电脑做任何事情，设置提醒，搜索互联网，回答问题，任何事情。借助高级语音识别功能，您可以说话，也可以键入问题。Cortana 现在是 Windows 10 的一部分。

4 Windows 10 带回开始菜单。您一直用来访问程序的开始菜单的工作方式是一样的，但 windows 8 开始屏幕的功能已经融入其中。

3 Windows 10 更**安全**。所有操作系统都经常发布安全补丁，但 windows 10 在设计时就考虑到了安全性。对于企业来说，可以选择完全锁定用户环境，只允许“白名单”程序运行。

2 **Continuum** 是 Windows 10 最具革命性的事情之一。Continuum 是一个概念，即无论使用什么设备或在什么位置，计算体验都是持续的。Continuum 会自动调整您的屏幕，以充分利用其大小。平板电脑和手机有平铺的视图和开始屏幕，如果你连接一个键盘和显示器，它会像一个完整的 PC 一样工作。运行 windows 10 的手机能做的事情和你的笔记本电脑一样多，这真是令人难以置信。
