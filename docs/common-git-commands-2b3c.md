# 常见 git 命令

> 原文：<https://dev.to/funkysi1701/common-git-commands-2b3c>

我使用 git 作为我的源代码控制系统。以下是一些最常见的 git 命令，排名不分先后。

*   签出一个 git 存储库**git clone/path/to/Repository**
*   添加要提交的文件 **git add**
*   将文件提交到 **git git commit -m“提交消息”**
*   将更改推送到服务器 **git 推送到源主机**
*   显示状态 **git 状态**
*   创建新分支 **git checkout -b**
*   切换到分支 **git checkout**
*   获取远程更改 **git pull**
*   将不同的分支合并到当前分支 **git merge**
*   查看合并冲突 **git diff**
*   暂时存储未提交的更改 **git stash**
*   撤消提交 **git 重置–硬**
*   显示关于提交的详细信息 **git show**
*   显示当前分支的版本历史 **git 日志**
*   获取所有远程分支 **git 获取原点**

大多数时候我使用 [git 扩展](https://sourceforge.net/projects/gitextensions/)工具来完成我的 git 工作，但是有时候只有命令行可以完成。
