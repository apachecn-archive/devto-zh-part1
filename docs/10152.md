# 我科技生涯的开始

> 原文：<https://dev.to/kaseycodes/the-startup-of-my-career-in-tech>

这个决定如此清晰，发生得如此之快，我几乎不相信。我和招聘人员开完会开车回家，意识到我厌倦的不是我的工作，而是我的职业。我漫不经心地问自己，如果回到大学，重新来过，我会想做什么？编码爆发了，就像是班上的孩子在空中挥舞着手，乞求被叫到名字。

我知道这到底是什么意思吗？这是不是最近“很酷”并且在我脑子里的事情？那是不是意味着我必须回到大学去获得另一个学位？我认识一个“编码”的人吗？女人会编码吗？我是不是太老了，学不到完全不同的东西了？那会浪费我已经拥有的商学学位吗？

当我仔细阅读开始出现的问题时，我想起了我在中学时建立的第一个网站。我的家人因为飓风而撤离，我坐在一个家庭成员的电脑前写 HTML 并创建了一个关于正向我们袭来的飓风的网站。我笑了，因为我创造了一些东西，并立即看到它出现在屏幕上。然后我意识到那是差不多 20 年前的事了！

在接下来的几周里，我做了大量关于转行到 web 开发的研究，发现了编码训练营，使用代码学校和代码学院来看看我是否真的喜欢编码，并参加聚会与我自己社区中通过编码训练营完成职业转变的人交谈。

从那时起不到一个月，我就在车里和 Bloc 一起开办了在线 web 开发训练营，我的科技事业也开始起步了！
