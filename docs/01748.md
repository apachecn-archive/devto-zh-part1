我们列出了一些非常方便的免费应用程序来帮助你开发 iOS。

1.  [看门狗](https://www.cerebralgardens.com/watchdog)监视你生成的文件并自动清理。如果您必须手动清除您的派生数据，这将非常有用。

2.  oh-my-zsh 用更现代的感觉改造你的码头。主题和替身可以很好地改善您的工作流程和节省您的时间。

3.  Zeplin 是一个 UI 设计和分享工具。一个很酷的功能是能够直接从一个模仿 Xcode 所需的所有大小的图标下载。

4.  [阿卡特兹【Xcode 缺失的包管理器，大量的插件可以帮助你的开发，所以看看吧。](http://alcatraz.io/)

5.  [SimSim](https://github.com/dsmelov/simsim) 快速访问模拟器的文件夹。方便获取您的数据库文件等。

6.  简单的数据库浏览器，它非常适合查看你的应用程序的数据库并找出错误。

7.  [Charles Proxy](https://www.charlesproxy.com/) 捕捉应用程序的所有网络请求，确保一切正常。方便快速地将错误归咎于服务器。

8.  Postman 一个向你的 API 发送网络请求的有用工具。非常适合测试发送到服务器或从服务器返回的数据的格式。

9.  Git 北海巨妖如果你不喜欢基于终端的 Git，那么这个 GUI 是最好的。可视化分支图在复杂的项目中很有帮助。

10.  [App Launch Pad](https://theapplaunchpad.com/)Quick a 轻松定制截图。展示你的应用程序的全部潜力，并确保你把它送到需要它的人手中！

11.  [Fabric](https://get.fabric.io/) 更好的分析平台之一，允许您记录崩溃和自定义事件。你对应用程序的使用了解得越多，你就能更好地进行改进。

12.  [Gifable](http://gifable.co.uk/) **把你的屏幕录制成 Gif** 。非常适合向所有人展示你的成果。这不是免费的，但相当便宜。
