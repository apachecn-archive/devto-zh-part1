# 现在服务:过时

> 原文：<https://dev.to/designfrontier/now-serving-obsolescence-2273>

这篇文章是“现在服务”系列的一部分，它着眼于设计和消费文化中正在发生的变化。

“到目前为止，我们在技术上取得了进步，但不知何故，这几乎是某种偏执，我们害怕真正地说:我们生活在第三次技术革命中。我口袋里有一个 iPod，我有一部手机，我有一台笔记本电脑，但不知何故，我最终回到家，像椅子一样坐在木质纺锤形惠滕格尔上。因此，在某种程度上，你可以说我们正在建造所有这些真正庸俗的舞台布景，与我们生活的时代毫无关系。奇怪。我发现它非常...在某种程度上非常反常。

…

为什么我们觉得需要一遍又一遍地重温这个原型？"

—物化中的卡里姆·拉希德(片段)

虽然我经常不同意卡里姆的观点，但他说的有些话是对的。我们确实会重新审视原型，即使我们拥有的技术允许我们完全抛弃它们。卡里姆认为这在某种程度上是有悖常理的。我不同意，我觉得有道理。这是人之常情。

今天，文化正在发生转变，就像 80 年代和 90 年代引进第三世界国家制造的廉价商品一样。它的根源在于我们对廉价商品的可获得性、当前后衰退世界的金融不稳定性以及人类对联系和永恒的渴望的融合反应。它现在正在发生。

让我们从一点历史开始。自 20 世纪 60 年代以来，美国和其他发展中国家一直在将日益复杂的商品生产转移到发展中国家。正如美国政府问责局(Government Accountability Office)所指出的(特指半导体行业)，“尽管较低的劳动力成本最初是吸引企业迁往海外的关键因素，但技术进步、可用的熟练工人和外国政府政策等其他因素也发挥了作用。”随着越来越多的廉价劳动力，发展中国家友好的政府政策，以及航空公司和互联网对世界的缩小，企业将商品生产转移到海外面临越来越低的障碍。

这产生了一些后果。首先，发达国家的制造业岗位开始减少。第二，制造商品的成本下降了，在某些情况下下降幅度很大。我记得在设计学校听过一堂课，讲的是在中国制造锅的成本，以及生产一个分段焊接的空心锅柄比生产一个铸造的实心锅柄更便宜。原因很简单，因为节省的材料成本比焊接、打磨和打磨接缝的额外成本要高。从某种意义上来说，这对发达国家的生活方式来说是一件好事，因为它大幅降低了商品成本。停下来想一想过去二十年来电脑、耳机或厨房用品价格的下降。

这给我们带来了第三个后果。廉价的一次性商品的兴起。为了明确起见，我把任何有计划淘汰的东西都包括在一次性商品的范畴内。甚至电脑、打印机和其他消费电子产品。结合廉价劳动力的兴起和塑料革命，毕业生认为:

麦奎尔先生:我想对你说一句话。就一个词。

本杰明:是的，先生。

麦奎尔先生:你在听吗？

本杰明:是的，我是。

马奎尔先生:塑料制品。

(来源)，现在市场上已经充斥着廉价的小摆设，这些小摆设的更换成本比修理成本更低。

制造业这种转变的一个快速例子可以在我目前的一个兴趣/爱好中找到。版画制作。主导印刷业几十年的活版印刷已经在很大程度上被数字胶印所取代。在我们的家中，它已经被喷墨打印机和激光打印机所取代，这些打印机打印文件的速度足以让本·富兰克林震惊。凸版印刷过去是，现在也是一项昂贵的投资，必须精心呵护和维护。我的喷墨打印机不是。事实上，我至少有一次买了一台全新的打印机，因为它比购买新的墨盒便宜。打印机，不仅仅是用来生产的物品，在很大程度上已经成为昙花一现。

随着这一廉价化过程在 90 年代加速，并进入新世纪的第一个十年，它开始产生有趣的影响。人们对此很不高兴。当然，我们喜欢这些便宜又容易买到的商品。我们享受它为我们提供的改善的生活方式。但是，在很大程度上，你得到了你所付出的，我们知道这一点。我们能买到的大多数商品从任何意义上来说都是便宜的。我们不喜欢这样。

我们不喜欢我们买的所有东西在 6 个月后都会过时或坏掉的想法。人类对计划淘汰的廉价商品的半永久性感到不安。我们需要承诺，老式的持久的承诺。我们可以每天使用的东西，然后传给我们的孩子。
