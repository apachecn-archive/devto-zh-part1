# Adtech 简介

> 原文：<https://dev.to/horia141/introduction-to-adtech-9k1>

这是关于互联网广告技术的系列文章中的第一篇。简称 *AdTech* 。这不是一个特别好的网上话题，所以也许这对人们会有用。

我的目标受众是刚开始接触广告技术的技术人士。这意味着开发者，设计师，产品经理，UX 研究员等。希望这也是对更有知识的人的一个很好的复习。它绝不是完整或完美的。

什么是 [*广告*](https://en.wikipedia.org/wiki/Advertising) 我们都有一个直观的认识。这是一种交流形式，其中一方试图说服另一方。我们称第一方为*广告商*，而第二方为*观众*。绝大多数情况下的目标都是商业性的。广告商试图说服观众购买产品或使用服务。还有非商业广告，如政治广告或公共服务公告。不过，它们远没有这么大。

除了广告商和观众，还有三分之一的网络广告玩家。*发布者*是能够将广告商的广告拿到观众面前的一方。他们这样做是因为他们已经有了自己的受众，他们代表广告商向他们展示广告。在大多数情况下，出版商是某种媒体组织，尽管这不是一个硬性规定。因此，互联网上的发布者包括网站、搜索引擎、社交网络、视频平台、移动应用、播客等。传统媒体包括报纸、广播电台、电视频道等。虽然广告商与受众之间有一些直接的交流，但大部分是通过出版商进行的。

*AdTech* 是互联网上广告**的技术。最后一点至关重要。广告总是涉及一些技术。例如，用于印刷报纸广告或拍摄商业广告。但是，只是随着互联网的兴起，技术成分才起了至关重要的作用。一个保证整个工程学科致力于此的领域。事实上，世界上一些最大的科技公司从 AdTech 中赚钱。**

网络广告的一些特性，使其有别于常规广告:

*   发布器几乎被普遍使用。事实上，大部分直接面向受众的广告都是在出版商发布自己的内部广告时出现的。
*   观众与广告的每次互动都可以被跟踪和测量。这样可以更好地报告和衡量广告活动。可以检查广告是否已经被实际看到，并量化与它的交互水平。传统广告的精确度要低几个数量级。
*   从每个观众的在线行为中可以推断出他们的大量信息。这允许更好地定向广告。传统广告依赖于观看电影或阅读报纸的观众群体的广泛属性。
