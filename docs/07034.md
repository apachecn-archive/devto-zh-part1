# 为什么要在安卓做训练？

### 机构问题

我们都知道 Android Studio 是每个计算机科学的人在今年夏天都要上的一门课，或者在更早的时候上过。众所周知，Android Studio 软件是免费的，甚至大部分插件也是免费的。开发者唯一需要支付的是 google play 商店中的开发者账户 25 美元，用于发布他们的应用程序。

[![alt text](img/507a4511a6f9991099730bedd452ba6f.png "dev account payment")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--wH8qGLli--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/zn4wjjdbu4itudhsj6v5.png)

但是大多数机构像其他课程一样收取 Android 培训课程的费用，但没有为学生提供这一功能。

### 谷歌的新操作系统

[![alt text](img/f3d7bc69aa835d49247b57c0277d5f67.png "fuchsia os")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--OfIC-kyW--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/bmklu7q558x6ls5r9ml1.png)

是的，你没听错。谷歌正在开发一个名为 Fuchsia 的新操作系统，这是一个不基于 linux 的开源操作系统。它是一个基于微内核**品红**的实时操作系统。(c++爱好者在此贡献)。这个操作系统的应用程序是由谷歌的 [flutter SDK](https://flutter.io/getting-started/) 构建的。使用这个 SDK，你可以为 android 和 ios 开发应用程序。

### 言归正传

那么，如果这个新的操作系统最终取代了 android，为什么这些机构不在 flutter SDK 中培训学生，而是在 Android 工作室中培训他们。因此，带着我对 android 操作系统和新的移动架构的未来事件的关注，我希望 Android 的暑期实习生好运，并建议学习一些更常见或在多个平台上受支持的东西。
