# 关于马修

> 原文:[https://dev.to/anotherdevblog/about-matthew](https://dev.to/anotherdevblog/about-matthew)

我叫马修·沃特金斯。我是一名软件工程师、丈夫、父亲、博客作者和摩门教徒。

## [](#what-i-do)我做什么

我目前在北卡罗来纳州莫里斯维尔一家名为 [ChannelAdvisor](http://www.channeladvisor.com) 的电子商务公司工作。我们是一家 SAAS 企业，致力于让在线销售变得更容易。非常棒的地方，非常好的人。

## [](#why-i-do-it)我为什么这么做

当我年轻的时候，我一直想成为一名发明家。我一直梦想成为韦恩·萨林斯基，创造一些新的和创新的产品，让人们的生活变得更容易或不那么乏味。软件开发让我每天都可以这样设计、构建和发明。太牛逼了。
