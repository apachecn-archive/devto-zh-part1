# 当猫不在的时候…

> 原文：<https://dev.to/wiredferret/when-the-cats-away-265c>

…老鼠会自我组织？

我的经理正在度假。假期真的很喜欢。从 Slack 中退出，没有邮件或电话，没有出席会议，什么都没有。这似乎是他正在做的:

[![Hawaiian beach](img/9b8fcbb0d03540720025cc17d563ee09.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--fXLzYiQP--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://www.heidiwaterhouse.com/wp-content/uploads/2018/01/f88b7d5e-0217-44f1-8d1a-40fff5d8050c.jpg)

他已经走了，大概有*周*了。好的，我想是 2 周。但它意义重大，即使是在年底这个怪异的节日里。

对此我有几点看法:

*   我愿意继续在管理层有重大休息时间的地方工作，并把它们当作休息时间。这意味着我也觉得请假没问题，尽管有时我不在的时候，我的同事会觉得有点困难。文化确实来自上层，当你的文化涉及到工作之外的生活时，它就会显现出来。
*   当通常向你提供答案的人不在身边时，你不得不开发替代的信息来源。这在很多方面都很棒。您不会拘泥于您的信息，组织会实践冗余。>仔细想想，真正的假期是团队的混沌工程。
*   我们确实在系统中发现了一些小故障，我们可以提前修复或在下次解决。例如，他是安排我们回顾展的人，不是在固定的日子。我们都不知道参数是什么。但我们只是没有，下次我们可以设置它，所以这不是一个交易。迭代。
*   在一个心理安全的团队中，没有你的经理在身边做决定是可以的。我赶了一个期限。一位同事告诉我她对我工作的优先顺序。我和一个队友一起决定来年的资金分配。这样做感觉很安全，因为我们可以相信，当我们的经理回来时，他会很高兴我们做了我们的工作，而不是等待他。
*   他离开前发的最后一封邮件提醒我们，他信任我们做好自己的工作，我们可以互相寻求帮助，如果我们有需要，可以去找管理团队。一个人还能要求什么呢？自主和信任能让我们变得最好、最快乐。
*   他会是一个更好的经理，因为他花时间盯着太阳和沙滩，而不是看着电脑屏幕或进行情绪劳动。以我们目前的速度招聘是令人疲惫的，因为如果你做得对，招聘是非常情绪化的。他照顾好自己意味着我们这些他团队中的人可以相信当我们需要他的时候他会在我们身边。那是很好的计划。

这些观察结果中有许多可以归结为信任。离开你的团队需要信任。在没有严密监督的情况下，得到信任来完成你的工作是很重要的。感觉到被重视，而不觉得自己被束缚或有义务，这真的非常非常重要。

> 重要并不等同于被重视。

祝你假期愉快，老板！我们会在另一边抓住你。
