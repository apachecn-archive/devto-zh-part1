# ICYMI 是什么意思？

> 原文：<https://dev.to/funkysi1701/icymi-what-does-it-mean-3akg>

一段时间以来，我看到 ICYMI 在互联网上四处传播。我不知道它是什么意思，所以只是忽略它，但我终于受够了，我谷歌了一个定义，我现在知道它的意思。

```
ICYMI – In Case You Missed It 
```

Enter fullscreen mode Exit fullscreen mode

ICYMI 经常在社交媒体网站上被用来表示你可能错过的文章。

软件开发充斥着缩写词和技术术语，我并不总是知道它们的意思。

阅读时，很容易忽略它们，试图理解文章的其余部分。然而，有大量的资源可以用各种不同的方式解释技术术语。

我试图在这个[博客](http://www.funkysi1701.com/)上解释一些不同的术语和概念。在写它们的过程中，我经常做一些研究，阅读各种不同的定义，有些让我困惑，而另一些让我去啊，现在有意义了。

当为了学习而阅读时，不要害怕深入挖掘并理解它，不要像我读 ICYMI 一样一遍又一遍地阅读相同的术语而不理解它们。
