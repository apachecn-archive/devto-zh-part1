# 嗨，我是马杜尔·阿胡佳

> 原文：<https://dev.to/madhur/hi-im-madhur-ahuja>

我是一名程序员/技术人员。我喜欢使用所有标准技术构建 web 应用程序，比如 HTML、CSS、JavaScript、JQuery、ASP.NET 和 SQL Server。我也知道很多不同的编程语言。。NET C#是我的最爱，我非常了解 Java，也非常了解 Javascript。
对于操作系统，我在 Linux 和 Windows 上都有经验，Linux 是我目前的操作系统。
我也有开发安卓应用的经验。

在进入职业世界之前，我在普纳的巴拉蒂·维迪亚佩斯的 T2 工程学院学习信息技术，度过了人生中最有收获的几年。
我对电脑、互联网和 Linux 的兴趣可以追溯到 1997 年，那时我第一次接触到电脑。那台电脑(英特尔奔腾 120 MHz，1.2GB 硬盘，8MB 内存)是我拥有过的最棒的东西之一。
