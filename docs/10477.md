# 为什么我从 Windows 换到了 Mac，而且不会再换了

> 原文：<https://dev.to/alexhyettdev/why-i-switched-from-windows-to-a-mac-and-won-t-go-back-3n7a>

作为一名. NET 开发人员，我的大部分计算生涯都是在微软掌舵下度过的。这是我的旅程，也是我为什么看不到自己回到 Windows 的原因。

我 10 岁的时候开始编程，用的是我爸下班带回家的一台 Windows 3.1 机器(我假设它们在这一点上已经过时了)。从那以后，到目前为止，我几乎使用了所有的 Windows 版本。用 Windows 95 做的开始菜单和 Windows 98 带来的稳定性的启示。Windows ME(:不寒而栗:)，Windows 2000，XP，8，8.1，现在 Windows 10。

老实说，我很喜欢 Windows 10，但之前版本存在的烦恼仍然存在。另外，在过去的几年里，我经历了一些不幸的事情，这让我远离了窗户。

从 Windows 8 开始。在第一版 Windows 中，我不得不在谷歌上搜索如何关机。你可以想象当他们做出天才的决定，把关机按钮放回开始屏幕时，我有多欣慰。然而，像许多人一样，我没有像我希望的那样无缝升级到 Windows 8.1。我有一些信心，在 Windows 安装了 170 个更新后，我会启动并运行新版本。

因此，一个周六，我点击了更新按钮，并希望。几个小时后，我收到一条消息，说安装失败，必须花 3 个小时回滚所有更新。所以周日，我经历了一次安装 3 个更新的痛苦过程，看看我是否能找到罪魁祸首。这一次一切都很好，但我失去了我的周末，并获得了一点点对 Windows 的仇恨。

两个月后，在另一次 Windows 更新后，我的机器无法启动。我得到了一个有用的选项，可以将我的机器恢复到 Windows 的工作版本，我很乐意这样做。成功了，太棒了。然后我看到了它，它的缺失。没有关机键，几个月前浪费了整整一个周末，现在又回到了 Windows 8.0。

FFS！事后看来，我接下来做的事情可能有点鲁莽，但我不能再浪费一个周末升级到 Windows 8.1。所以我做了任何愤怒的开发者都会做的事情。我把笔记本电脑擦了一下，装了 Ubuntu。Linux 满足了我作为开发人员的大部分需求。我可以运行一个不错的 light IDE (Atom，Visual Studio 代码),还可以运行 docker。我用 Ubuntu 做了 6 个月的主 OS。

这些年来，我一直在使用 Linux，并且尝试了许多不同的发行版。我甚至通过涉猎 Arch Linux 完成了整个 Linux 从零开始的过程。我喜欢 Linux，但它总是需要相当多的时间投入来设置一切。其中一些是试图找到 Windows 应用程序的体面替代品，或者试图让 Windows 应用程序在 Wine 中工作。不管怎样，我的生活已经到了这样一个地步，我没有时间在电脑上“瞎忙”(我妻子这么说)，去尝试做一些简单的事情。

当 Windows 10 发布时，我想我会再给微软一次机会。另外，我需要再次使用 Visual Studio 进行一些开发。微软在 Windows 10 上改进了很多。然而，有一个关键因素驱使我加入苹果阵营，那就是 Windows Updates。

当我在上下班的火车上时，我会做我的个人项目。家里有两个年幼的孩子，每天两个小时的通勤是我唯一能做兼职的时间。在有限的时间内，我需要一台能让我充分利用时间的机器。然而，当我几次尝试工作时，Windows 却有其他想法。这是以强制 Windows 更新的形式出现的。我对自动更新没有问题，但是如果它们需要重启，我应该可以选择什么时候重启。

Windows 有一个习惯，在你关机花半个小时进行更新之前，会给你 5 分钟的时间来保存你的工作。至少他们会给你一个警告，但是如果你当时不在电脑前，这个警告就不管用了。我刚开完会，发现我的电脑已经重启，正在安装更新。

我对新开发笔记本电脑的要求如下:

*   快速——我的旧笔记本电脑过去常常要花 10 分钟才能打开 Visual Studio，这令人震惊。我需要一个速度快，有像样的 CPU，内存和固态硬盘。
*   便携——在火车上工作意味着我必须随身携带笔记本电脑，所以它必须轻便，不要太大。分辨率好的话 13”屏幕应该够大了。
*   使用的乐趣——最后一个有点主观。然而，在漫长的一天后，我发现自己不想做兼职项目，因为使用我的笔记本电脑并不有趣。

我相信你可能已经猜到了我选择的笔记本电脑给这封邮件的标题和上述要求。是的，我已经换了一台 [13 寸带 Touch Bar](https://geni.us/gswnaD) (关联 [*](https://www.alexhyett.com/affiliate-disclaimer) ) (16GB RAM)的 MacBook Pro。

这是我的第一个苹果产品，我从来没有欠过 iPod 或 iPhone。事实上，我曾经想知道苹果产品到底有什么魅力。从外面看，与竞争对手相比，他们似乎动力不足，缺乏特色。你可以用比苹果电脑更低的价格买到配有最新英特尔处理器和更好显卡的 Windows 笔记本电脑。溢价的原因是他们只是工作。

这可能是我变老的迹象，但我没有时间花几个小时修补 Linux 或几天更新 Windows。我需要的是能工作的东西，开箱即用，不问问题，苹果一针见血。

有了 Mac，我就拥有了最好的一切，因为我还可以在虚拟机上安装 Windows 或 Linux，并拥有所有这三个操作系统。因此，我修补 Linux 的日子可能还没有结束，但我看不到自己会再次回到 Windows。
