# 馅饼、鲨鱼和红宝石:我在 RubyConf 的一周

> 原文:[https://dev . to/bengreenberg/beignets-sharks-and-ruby-my-week-at ruby conf-c9g](https://dev.to/bengreenberg/beignets-sharks-and-ruby-my-week-at-rubyconf-c9g)

几个月前，我申请参加 RubyConf 2017，作为一名机会学者，并没有真的期望被选中。毕竟，任何一个人获得备受追捧的奖学金的几率都很小。最后，我没有得到第一个名额，继续我的日常计划。然而，就在会议前几周，我收到了一封电子邮件，这封邮件让我几乎要跳起来:有一个职位空缺……我会想要吗？我怎么能拒绝呢？在与我的妻子和我们共享的家庭谷歌日历核对后，我接受了，并开始准备前往新奥尔良。

[![A fair representation of my reaction to the acceptance email](../Images/22631f5b2d3248b5827cace0736e1cb6.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--kAx31rtu--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://cdn-images-1.medium.com/max/1600/1%2AAmI9wRbXrfIWGESx6eEiTw.gif)

作为一个几个月前才从熨斗学校毕业的人，在这个有数百名经验丰富的编码专家参加的会议上，我会发现什么呢？我会受到欢迎吗？我会一直觉得自己完全不知所措吗？我非常幸运的是，我的另一位朋友，熨斗学校的毕业生(特约撰稿人)叶切尔·卡尔门森也被选为机会学者，所以我不会完全一个人下去，但我们会在周围有数百只红宝石(数字海洋)鲨鱼的海洋中。

[![The awesome “Sammy” shark from DigitalOcean](../Images/543cedb45e566c3336dcce5fd376cb20.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--tpfYNbek--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1200/1%2A2Ftf58dlNwaIUV6lfUsoMA.png)

那么，这次经历像我担心的那样残酷吗？绝对不行。开幕辞不是别人，正是 Yukihiro“Matz”Matsumoto，Ruby 的创造者，他强调了 Ruby 社区的包容性。“Ruby 很好，所以我们很好，”或者也可以这么说，“Matz 很好，所以我们很好。”从我抵达会议酒店的那一刻起，我就没有被鲨鱼的海洋包围，而是被穿着 t 恤、拿着贴有贴纸的笔记本电脑、带着欢迎的微笑和渴望认识你的人的海洋包围。我很快就有了家的感觉。

事实上，我意识到我知道的比有时我相信自己的要多。我在熨斗学校接受的教育，我目前在学校兼职技术教练的工作，我在项目上的持续工作，以及定期写博客和痴迷地阅读像 *The Practical Dev* 这样的网站上的帖子，确实让我了解最新的信息。这并不是说我有时不会感到力不从心——我有时会，但那些是我有目的地参加一些我知道会延伸和推动我的知识边界的事情的时刻。
作为谷歌云平台赞助的机会学者计划的一部分(感谢谷歌！)我被指派为会议的“导游”。在我的情况下，我实际上被分配了两个向导，因为在与另一位学者的会议之前有一个时间安排的问题。虽然我花了更多的时间和我原来的导游在一起，但在整个会议期间，两位导游都渴望提供建议和友好的面孔。我最初的向导也是 bootcamp 毕业生，她现在在一家主持许多 bootcamp 组合项目以及大量企业应用程序的公司做得非常好，这也让我很受鼓舞……我想他们的主要业务是那些企业系统，而不是小型学生顶点项目！我们还被分成了几个由导游和学者组成的小组，我的小组主持人以他特有的方式非常热情地欢迎并了解他小组中的每一个人。他的活力(以及他的身高)使他在整个会议中很容易被发现。

对我来说，最重要的亮点是什么？

**会见马茨**

[![“Matz is nice so we are nice”](../Images/972e56ea8e02b463621342b5cc8ab72d.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--SkeUA0Pb--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1200/1%2AgJdXFaHPaJBYLQa-LWQXzw.jpeg)

这怎么可能不是头号亮点呢？Matz 最终在大约 25 年前创造了一种编程语言，这种语言被证明在简单性方面是优雅的，在适用性方面是不可思议的健壮。Ruby 已经成为许多 bootcamp 毕业生的第一语言，因为它把我们带入了编程的世界，这不是偶然的。虽然在我人生的这个阶段，我也花了很多时间在 Javascript 上，花了一些时间在 PHP 上，花了很多时间在 React 框架上，但是当他们听到“Ruby on Rails”这个词时，谁会不笑呢？我有机会走到他面前，感谢他的巨大贡献，当然，还自拍了一张照片。很明显，他比我更擅长自拍艺术，因为他知道不像我那样看着相机！

**结交新朋友**

当你学习编码时，你会很快爱上它。如果你对它没有热情，在调试的第一个晚上之后，直到太阳升起，你可能很难保持这种投入。然而，在那之前，你生命中的人可能无法完全陪你走完那段旅程。你开始说一种大量使用英语的外语，但是你的爱人和现在的朋友却无法理解。他们想要理解，但是很难跟上所有的 git 提交、迭代器和数据结构。你可能认为你是宁静的，但他们只是觉得你很奇怪。因此，结交分享你激情的新朋友是一件非常愉快的事情。可以和你谈论代码的人，这样你就可以和你的老朋友和你的另一半专注于其他话题。

在这次会议上，我结交了许多新朋友，包括机会学者和经验丰富的程序员。我们在会议间隙的走廊上相遇，在演讲时坐在一起，晚上一起去世界咖啡馆，周六晚上一起喝鸡尾酒。这些人中有几个是我之前在 Slack 上认识的网名，但大多数我之前根本不认识。我遇到了一个“开发者福音传道者”，她很快解释说她不是来改变我的，她甚至在 Twitter 上为我大声疾呼，在俄亥俄州从事非常有趣的开源项目的人，纽约同胞(很多人来自布鲁克林)和一对来自哥伦比亚的夫妇，他们刚刚花了一年时间环游世界回来。我遇到了很多来自各地的 bootcamp 毕业生和现在的 bootcamp 学生，包括西雅图、丹佛、奥斯汀等地。

我很高兴能与所有这些人保持联系，我有机会与他们见面并成为朋友。2018 年的下一个 RubyConf 现在不是一个理论上的想法，而是一个与真实的人重新联系的具体机会，一个实际的程序员和 Rubyists 社区。

**学习，学习和更多的学习**

在 RubyConf 要学的东西太多了！从涉及编程、心理学和社会学交叉的大型主题演讲，到每天不同主题的会议，从 TDD 到开源，到机器学习，不断有一个人的思想被扩展和推动的时刻。

[![Augmenting Human Decisions with Data Science](../Images/2cf9aead5f61b429abc1e9d72d5e6767.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--5aNQMsKx--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1200/1%2AFIL2uCTg_RWDuv9ynIwNvQ.jpeg)

我参加的每一次会议都是深思熟虑、引人入胜、准备充分的。在众多突出的演讲中，我会暂时记住那些关于“用数据科学增强人类决策”、“Git 驱动的重构”和“间歇性故障的确定性解决方案”的演讲。我将在接下来的帖子中发布我对这些讲座的笔记和心得。眼下，就两个字:`rspec --bisect`。如果你已经知道了这个命令，给自己一个掌声，但是要记得你第一次学习它的时候，以及测试中的一个范式转变。现在你可以理解我的感受了。

**不要装衣服**

说真的，我认为这些会议应该改名为 SockConf 或 ShirtConf。我希望有人能提前告诉我，第一天你需要的只是一些东西，否则在参观完赞助商的展位后，你的手提箱里就会装满 t 恤和袜子。还有，贴纸。这么多贴纸。我不确定我的整个公寓是否有足够的空间来放置我过去几天收集的所有贴纸。

总之，非常感谢我的指导老师[斯特拉·科顿](https://twitter.com/practice_cactus)、[凯莱布·汤普森](https://twitter.com/calebthompson)、[艾莉森·麦克米兰](https://twitter.com/allie_p)在组织学者项目中发挥了重要作用，感谢[乔南·谢弗勒](https://twitter.com/thejonanshow)是一位令人难以置信的团队主持人，感谢我所有的新朋友，感谢他们的出色表现(包括[卡罗琳](https://twitter.com/CarolineTaymor)、[乔安娜](https://twitter.com/gildedsquirrel)、 [CJ](https://twitter.com/itsradditude) 、[劳伦](https://twitter.com/Laurenelee88)、[克洛伊](https://twitter.com/ChloeCondon)

**新奥尔良**

最后，但肯定不是最不重要的，是新奥尔良市本身。这是一个特殊的地方，有着独特的文化，人们似乎一直都很快乐。我知道最后一部分肯定不是真的，但来自纽约，我觉得每个人都生活在持续的微笑中。虽然我在会议期间没有太多地看到新奥尔良，但我很高兴在会议结束后与路易斯安那州梅泰里的犹太人社区一起享受了一个特殊的安息日。在会议期间，我还抽出时间与人们一起沿着河边散步，或者在城市中漫步，欣赏建筑。在散步的时候，我看到了新奥尔良大屠杀纪念馆和移民纪念碑，两者都位于河边。
