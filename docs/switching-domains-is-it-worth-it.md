# 转换域。值得吗？

> 原文：<https://dev.to/stefanbc/switching-domains-is-it-worth-it>

过去的一个月，我的个人网站一直处于离线状态。域名过期了，我没有做任何事情来防止这种情况发生。

我忽略这一点有多种原因。

最重要的一点是，域名提供商没有一个包括客户关注和满意度的商业计划。换句话说，他们不能确保客户总是被告知和了解他们账户上发生的所有事情。

例如，我没有收到任何预先通知，该域名即将到期。我唯一收到的是一个通知，上面写着:您的域名已经过期，我们试图通过您在帐户中提供的电话号码联系您，您需要支付这 X 笔钱才能取回它。

*/开始咆哮*

他们没有试图打电话给我。我的手机全天候开着，就在我身边。我没有接到他们的任何电话。

其次，仅仅在东西过期后给你的顾客提供账单，没有任何事先通知，不是一个好的商业策略。

根据我的经验，人们不喜欢你不给他们发送关于他们正在购买的东西的更新。

所以，我认为，他们的商业模式根本不是以顾客为中心的。

*/结束咆哮*

在这整个惨败结束后，我仍然有一个急需域名的小水滴。

当我在寻找一个新的域名时，我开始关闭所有让我的网站保持工作状态的服务。

这意味着我关闭了 droplet，清空了域名服务器，关闭了所有我激活的监控服务(出于某种原因，我使用了多个，而不是一个),并暂停了邮件系统。

所以关闭一切都很容易。这影响了很多事情。首先，从 SEO 的角度来看，一切都乱了套。就好像我的网站脱离了网络。

这种情况持续了一个月左右。然后，大约一周前，我偶然发现了这个域名，并跟随 Alphabet 的脚步注册了它。

## 好玩的部分

大约花了 3 个小时才恢复正常。因为这是一个新的域名，而我是第一个所有者，DNS 传播花了大约 3 个小时。从多方面来看，这很好，原因如下。

首先，我要向 Ghost 致敬，它让更改域名变得如此容易。只需替换配置文件中的域名并重启服务就足够了。

Cloudflare 设置只需点击几下鼠标...搞定了。

打开所有的监控服务，花了一点时间，因为域名是不同的，我基本上不得不再次添加每个检查。

由于我使用的是第三方服务，设置邮件服务也只需点击几下鼠标。

现在一切都恢复正常，只剩下一个问题了。SEO 方面。我想...划掉那个...我必须更新我拥有的每个公共账户，以突出新域名。这意味着我花了大约 2 小时来编辑每个账户和修改网站。Twitter，G+，Github 等等等等。不要误会我的意思，我不是在抱怨这方面，因为这是一个机会，让我可以快速了解我拥有的每个帐户，并有一个公共配置文件。

就网站 SEO 而言，我没有太多要做的事情，只需在分析和网站管理员工具中添加必要的东西就足够了。

## 结论

问题依然存在，值得吗？作为一个人，作为一个在他/她周围创造了品牌的人，转换域名对你有影响吗？

最初我会说，是的，但这完全取决于你的情况。从营销角度来看，这可能意味着品牌更新，但从技术角度来看，这却是一场噩梦。

在做出这样的决定之前，你必须非常努力地思考，从 SEO 的角度和技术的角度来看待一切。如果改变域名太难，那么坚持使用它，你应该增加你的知名度，但这是一个完全不同的讨论。
