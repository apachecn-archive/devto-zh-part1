# 错过了一个平板电脑

> 原文:[https://dev.to/tcratius/missed-a-tablet-74o](https://dev.to/tcratius/missed-a-tablet-74o)

夜晚，入睡
醒来，感觉又是新的一天
以我的方式走完这一天
悲伤，感同身受，不知所措
又错过了我的平板电脑

一小时后，快乐，稳定，不再在一个球里
像你一样太干净
从这里看起来容易多了
告诉我！告诉我是真的吗！
告诉我一切都好，没问题

我不再知道是平板电脑的原因还是只有我，没人能告诉我是我疯了才会感到如此深切的悲伤 T2 还是我们已经忘记了感受我希望找出真相 T4 而不是男爵夫人的感受

平板下沉，生活是美好的，是吗？
