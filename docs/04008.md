# 使用 GitHunt 在新的 Chrome 标签中获取热门 GitHub 项目

> 原文:[https://dev . to/fatosmirina/get-trending-github-projects-in-your-new-chrome-tab-with-git hunt-80n](https://dev.to/fatosmorina/get-trending-github-projects-in-your-new-chrome-tab-with-githunt-80n)

GitHub 是广为人知的最著名的版本控制库之一，在这里你可以找到很多编程语言的无数公共项目。它使您可以轻松地与团队成员以及全球其他开发人员协作，开展许多公共项目，您可以从中学习和受益，也可以为之做出贡献。在许多其他事情中，GitHub 显示了*趋势项目*，这让你很容易区分哪些项目比其他项目更受关注。因此，您有能力找到在最近几年获得了其他开发者的大量好评的项目。

Chrome 扩展致力于增加您对开源项目的了解，这使得寻找新趋势项目的过程变得更加容易。GitHunt 可以作为 Chrome 扩展安装在你的浏览器中，然后就可以在你的新标签中看到 GitHub 上的热门项目。您可以查看每日、每周或每月的趋势项目。

您可以选择仅用一种语言或所有语言显示趋势项目。它目前不能选择一种以上的语言，也不能包含所有的语言。

除了项目标题和星级数之外，您还可以查看简短的项目描述(如果之前已添加)、打开的问题数，并且只需单击新选项卡中显示的项目即可访问该项目。

# [](#why-should-you-install-this-extension)为什么要安装这个扩展？

GitHub 被称为软件开发人员的新简历，因为它是向潜在的招聘人员展示你的知识、技能、专长和代码风格的一个非常好的方式，可以让他们对你有一个印象。

因此，许多开发人员在 GitHub 上公开他们的一些最佳作品，这也可以为可信的创新和其他人学习和贡献的机会打开大门。即便如此，总是偶然发现这些非常好的项目确实很难，而且你可能也没有总是直接访问 GitHub 的习惯，只是为了看到趋势项目。因此，将它们放在你的新标签页中可以增加你了解新的有趣项目的机会，你可以从中受益。

看到像脸书、谷歌和 T2、微软这样的大公司从开源项目中获益如此之多，你可能会更愿意在浏览器中使用这个扩展。

一个简单的项目可能会出现在你的新标签中，可能已经解决了困扰你这么长时间的问题。你可能想在谷歌上打开一个新的标签来寻找一个合理的解决方案，但是这个解决方案可能比你想象的还要早。

这些项目也可以启发你新的知识、想法和模式，来构建你下一个流行的开源项目。您还可以确定许多解决方案的模式，这些模式已经在许多趋势项目中展开，并提出您自己的解决方案。

GitHunt 也是一个开源项目，你可以在 GitHub 上找到。如果你对如何改进它有任何想法，作者已经呼吁你的贡献。我已经做了[几个小的错别字修复](https://github.com/kamranahmedse/githunt/pull/15)并邀请你继续改进这个伟大的项目。

你可以通过访问它的 [GitHub 页面](https://github.com/kamranahmedse/githunt)来阅读更多内容并查看这个扩展的源代码。你可以通过 [Chrome 网上商店](https://chrome.google.com/webstore/detail/githunt/khpcnaokfebphakjgdgpinmglconplhp)下载安装。
