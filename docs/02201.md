ReactJS 是 web 开发中的一场革命。其强大而庞大的开发者社区使其成为 web 应用程序开发项目中最受欢迎的技术。你应该为你的下一个大项目考虑这一点。它是由 facebook 等顶级 It 公司及其开发者社区形成的开源库。它与 HTML 标签一起使用，主要用于应用程序的用户界面。

关于 ReactJS 及其对 web 开发的革命性影响，您应该知道以下 5 件事:

## [](#1-what-is-react)1。什么是反应？

React 是一个开源的 JavaScript 库，具有声明性、高效性和灵活性。它由像脸书这样的顶级 It 公司维护。ReactJS 主要做 MVC(模型视图控制器)方面的工作。它用于构建用户界面。Web 开发人员必须在 react 中创建组件，以便使用这些组件并构建面向性能的应用程序。

## [](#2-component-of-react)2。反应成分

React 组件在 web 开发中起着非常重要的作用。如果 web 开发人员在他们的项目中使用 reactjs，他们需要在 react 中创建组件，然后使用它。React 组件使用 render 方法进行数据输入和输出。React 提供了开源 javascript 库所能提供的所有灵活性。

## [](#3-react-elements-and-instances)3。反应元素和实例

React 元素是描述组件实例或 DOM 节点及其所需属性的普通对象。实例仅包含有关组件类型、其属性以及其中任何子元素的信息。元素是一种通信模式，它告诉 react 用户想在屏幕上看到什么。它是一个不可变的描述对象，有两个字段——type:(string | react class)和 props: Object。

## [](#4-dom-elements-on-reactjs)4。反应堆上的 DOM 元素

DOM 节点由字符串中的元素类型表示，这些元素具有特定的名称和定义属性的属性。DOM 元素很轻，因为它们只是 ReactJS 上的对象，不需要解析。

当创建为元素树时，元素可以嵌套为父元素和子元素，但它不是实际的实例，而是描述。

## [](#5-one-way-data-flow)5。单向，数据流

ReactJS 是从属性开始的单向数据流。推荐 Flux 作为 React 的应用架构。Flux 是一种数据流编程风格。组件不能直接修改属性，而必须通过回调函数来修改值。这样，属性向下流动，动作向上流动。ReactJS 是 [web 应用程序开发](http://a2zfame.com/web-development/)中的动态元素，因为它为需要不断更新数据的应用程序提供了广泛的支持。
