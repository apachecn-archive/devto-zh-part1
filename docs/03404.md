我注意到上面的两种行为在新的和“旧的”语言之间非常不同。

Java、C、C++、Objective-C 和 C#用 Java 的方式定义函数和变量。

Kotlin、Swift、Rust、TypeScript 和 Go 都将类型放在右边。
