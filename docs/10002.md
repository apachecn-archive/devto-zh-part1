## [](#nothing-in-life-is-free)生活中没有什么是免费的

你以前听过这句话。通常是一些年长的人，关于一些看起来不相关的话题，但我能说的是这是真的。这该死的说法是真的。

随着年龄的增长，我写这篇文章的时候已经 35 岁了，我一直在脑子里应用和说这句话。我把它当作年轻时毫无根据的“智慧”一笔勾销。或者当我开始理解政治的时候，我认为它是政治废话。但是这句话及其应用适用于你生活中的每一件事。

另一种我对自己或我与之交谈的不幸的人说的方式是:

**以什么代价**

## [](#how-does-it-apply)如何应用？

让我举几个例子来说明这一点如何适用于似乎不适用的领域。

### [](#gaming)博彩

我是一个游戏玩家，我的妻子会说我上瘾了，我很容易被拉去连续玩几个小时的电子游戏。我大半辈子都在玩游戏，准将 64 年到现在，但我在这里付出了什么？

时间。

这里的时间是我可以花在我妻子身上的时间，我可以花在这个博客上，我可以花在修理我总是抱怨的那个堵塞的水槽上。像很多伟大的事情一样，时间是有限的。

### [](#love)爱情

从金钱的角度来看，爱情是免费的。这里没有争论。但是这仍然需要时间，通常是大量的时间。这需要注意力、专注、关怀和对其他情感的一系列探索。但这也需要现金，也许不是很多，但也有一些。

你可能会去拜访新的爱人，因为你们一开始就没有住在一起，对吗？

运输费用。

你们打算一起做什么？

娱乐费用。

你必须吃东西，即使是在家做的，你也增加了+1 的量。

食物费用。

因此，虽然浪漫主义者说爱情是免费的，但事实并非如此。这需要花费一些东西，通常是好几样东西。

### [](#work)工作

接受另一个州 WOOT 的高薪工作。抛开明显的搬家费和交通费，还有什么呢？

你可能会离开已经建立关系的地方(朋友、家人、同事)。这些关系将会受损、退化或至少改变。你不再是格伦先生的隔壁了，所以在照看院子的时候不会再叫了。妈妈不再整天在家听你抱怨乔·鲍勃教授在测验中给你打了低分。

### [](#work)工作

在办公室和家里的时间和精力是不花在其他地方的时间和精力。在家工作加剧了这种情况。当你在考虑实现 API 集成的最佳方式时，你不会去想 Geordi(家猫)的跳蚤药物，也不会去听你所爱的人真正在说什么。你听到了一切，但听它是不同的，并且被翻腾的工作所阻挡。

还不算通勤的时间、努力、压力和金钱成本。

### [](#politics)政治

这是一个相当明显的问题。撇开我的政治信仰不谈，支持或反对某事的立法对一些人有利，对另一些人不利。这种平衡很重要，但看起来非常令人怀念。参议院刚刚通过了一项法案，降低了 T2 和 T3 人群的税收。问问自己付出了什么代价？那些税收的钱被用在了某个地方，现在没有了。有东西被剪掉了，对吧？

## [](#what-to-do)做什么？

这个条目更多的是让我咆哮一些越来越多为我弹出的东西。读者们，这是恳求你们在做决定时，要有更理性的思考。我们一天中会做出很多决定，而这些决定往往是我们要做的决定，是我们的标准。但是试着问问你自己，对于你所做的一些更大的决定，“付出了什么代价”。

最近，一位密友的父亲相当突然地去世了。他们之间有一些问题，他们并不太接近。但和他谈论这件事时，他说的第一件事是他希望在他死前能再见他父亲一面。

我承认，这是最坏的情况，有人从你的生活中永久消失。但是正如我上面指出的，我们采取的一些看似最简单的决定或方法可能会让我们在其他地方付出代价。

记住，

**以什么代价**
