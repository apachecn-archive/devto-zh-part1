**AWS 简介**

亚马逊网络服务是 Amazon.com 公司在 2006 年提供的一个可扩展的云计算平台。由于其相对低廉的价格和多个领域服务的多样性(例如存储、物联网、计算...)和详细的文档，被认为是 Azure 的重要竞争对手。在本指南中，您将了解使用的基本知识:

*   简单通知服务(SNS 短信)
*   简单存储服务(S3)
