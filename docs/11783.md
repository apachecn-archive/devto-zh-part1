# 思科代客-很好，非常好

> 原文:[https://dev.to/briannipper/cisco-valet-尼斯-非常-尼斯-271i](https://dev.to/briannipper/cisco-valet---nice-very-nice-271i)

我刚刚购买了思科代客服务，我不得不说我喜欢它。【T2![](img/6ad9278f6f0aef40734c1750d92f915c.png)

我花了大约 10 分钟关掉电脑，解开我现在的路由器和电缆调制解调器，然后将代客泊车放入循环。一切启动后...我说完了。

贴身男仆赢得了它的名字。它自动分配了一个唯一的名称(至少对我所在的区域是唯一的)和一个随机密码(可能不是无法确定的，但足够唯一)。作为一个额外的奖励，它包括一个客人部分。

另一个额外的好处是漂亮的小自动配置 USB 密钥，它可以为您希望连接到无线网络非访客部分的任何设备提供即时设置。例如，我写这篇文章时使用的内置 WiFi 的有点“旧”的笔记本电脑(Dell Latitude D510)在几分钟内就连接上了。

总的来说，塔吉特百货的 90 美元非常不错，也是我推荐的产品。
