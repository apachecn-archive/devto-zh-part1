# 克服我对编码的恐惧

> 原文：<https://dev.to/ddinardi0/overcoming-my-fear-of-coding>

是的，我曾经害怕编码。仅仅是提到它就足以让我感到脊背发凉，就好像我听到了一些我从来不知道的神秘实践。尽管我从小就沉迷于视频游戏和各种技术，但我无法开始理解晦涩难懂的标签、功能和操作符的神秘海洋。

难怪我会成为一名作家。我已经为巴西的一些设计和技术出版物写了一部高度幻想小说，一部轻松小说和许多特写。令我沮丧的是，我甚至本地化了一些关于编码的文章和教程，就在那时，我开始对这个领域产生了某种，比如说，钦佩。人们可以用那些奇怪的文本字符串做一些漂亮的事情，他们让它听起来很简单。它仍然让我毛骨悚然，你知道，但现在他们稍微少了一些黑暗的毛骨悚然。就像当你看到你死去的奶奶的鬼魂:它是一个鬼魂，但同时，它也是你的奶奶。

我对整个计算机编程实体的感觉在这种奇怪的神秘光谱中持续着，直到我失去了一家小广告公司的文案工作，这促使我的生活中出现了一个非常有趣的篇章。我把我的奇幻小说翻译成英文，尝试了一下作为喜剧 youtuber 的运气，然后我写了一个关于 youtuber 的八卦女风格的故事。我个人对我所做的一切非常满意，但是因为我不擅长推销自己的东西，所以这在金钱上并没有真正的回报。长话短说，我正准备迫降到断背山，需要找一份新的写作工作。我的国家正处于政治和金融危机的漩涡之中，这绝非易事。

感受到我的苦恼，我的一个好程序员朋友提出了一个最荒谬的建议:成为一名程序员！当然，我对这种想法嗤之以鼻。即便如此，他还是给我发了一个可汗学院在线编码课程的链接。“看看它”，疯子告诉我。出于某种原因，我做到了。我点击了那个该死的东西，上了第一个基本的 JavaScript 类，只是为了好玩。

毕竟，这有点酷，也没那么神秘。

我继续下一课，下一课，再下一课。突然，我开始理解那些我认为对我来说深不可测的东西。见鬼，我甚至很享受自己变出了简单的形式，设置了循环，变量和函数，表演了各种难以形容的编码魔法。

一晃三个月过去了，我已经申请了技术学校，现在，在一个周日的下午，我正在做一个愚蠢的项目来测试我初露头角的 HTML 技能。仍然很难相信我能成为一名成熟的程序员，我仍然一无所知的语言和工具的广度非常令人生畏，但我喜欢目前为止的旅程。我想我已经忘记了完全从你的感知元素中学习一些东西是如何扩展你的思维并引导你进入最意想不到的方向的。就在几个月前，我从来没有想过，我会在新的一年里寻找科技行业的实习机会，而不是像秃顶的罗里·吉尔摩那样去找一份写作工作。

但是，嘿，如果我的新工作包括编码，那也算写作，对吗？
