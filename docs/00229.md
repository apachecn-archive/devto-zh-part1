# [](#javascript)Javascript

JavaScript 通常缩写为 JS，是一种高级的、动态的、弱类型的、基于原型的、多范例的解释型编程语言。与 HTML 和 CSS 一起，JavaScript 是万维网内容生产的三大核心技术之一。它用于使网页具有交互性，并提供在线程序，包括视频游戏。大多数网站都采用它，并且所有现代 web 浏览器都支持它，不需要借助内置的 JavaScript 引擎的插件。许多 JavaScript 引擎中的每一个都代表 JavaScript 的不同实现，都基于 ECMAScript 规范，有些引擎不完全支持该规范，而许多引擎支持 ECMA 之外的其他特性。

jQuery 的语法旨在使导航文档、选择 DOM 元素、创建动画、处理事件和开发 Ajax 应用程序变得更加容易。jQuery 还为开发人员提供了在 JavaScript 库之上创建插件的能力。这使得开发人员能够为低级交互和动画、高级效果和高级主题化小部件创建抽象。
