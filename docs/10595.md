# 我们的公开集体运动

> 原文:[https://dev.to/betweenthewires/our-opencollective-campaign](https://dev.to/betweenthewires/our-opencollective-campaign)

今天，the Wires 正在发起一场 [OpenCollective 活动](https://opencollective.com/betweenthewires)，为 2017 年的继续内容筹集**5000**美元。

## **四周内**

自从我们在 Medium 上上线以来已经整整四周了！为了纪念这一里程碑，我们认为分享前三个故事的一些初始数据会很有趣([吉列尔莫·劳赫](https://betweenthewires.org/between-the-wires-guillermo-rauch-2819177beedc#.b4avwspgk)、[尤雨溪](https://betweenthewires.org/between-the-wires-evan-you-cb56660bc8a4#.6y1wpf4gw)和[克里斯·科伊尔](https://betweenthewires.org/between-the-wires-chris-coyier-382a217a0f1#.wjenez7zb)):

*   中等: **68K** 总阅读分钟数， **25K** 浏览量，阅读率在 **35%到 57%** 之间
*   社交媒体关注者:媒体上 391 人，**推特上 255 人**[推特上](http://twitter.com/betweenthewires)
*   超过 **15** 名候选人通过[我们的读者在线表格](https://goo.gl/TSPEqf)推荐

当 Preethi 和我开始这个项目时，我们不知道社区是否会像我们一样热情。我们使用了三次最初的采访作为衡量社区兴趣的方式，鉴于早期的积极反馈，我们决定继续《连线之间》!

## **发起我们的公开集体活动**

为了继续我们的工作，我们需要你的支持。

今天，我们在[open collection](https://opencollective.com/)上发起了一场[连线活动](https://opencollective.com/betweenthewires)。我们的目标是筹集**5000 美元**在 2017 年做**20–25**额外的采访。我们欢迎社区贡献和企业赞助。

开发这些精心制作的采访是一项充满爱心的工作，需要大量的时间和成本。每次面试，我们花 12 个小时左右的时间研究候选人，完成面试，拍照，编辑润色笔录，最后发布推广。

我们的劳动力是免费的，因为这是一个充满激情的项目，但我们确实需要寻找社区贡献来支付费用，如转录，编辑，摄影设备租赁和一些杂费。我们的所有费用都将在 OpenCollective 上提交，以确保 100%的透明度。
