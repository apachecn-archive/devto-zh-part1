# 我的第一台电脑

> 原文:[https://dev.to/yelluw/my-first-computer](https://dev.to/yelluw/my-first-computer)

那是 1984 年。我父母打算给我们买一个任天堂娱乐系统。我的哥哥当时是个超级书呆子(现在也是)，他说服他们买了一台电脑。嘘！

我记得它看起来有多大多笨。它是奶油色的。有巨大的按钮。当你打开它的时候只显示一个愚蠢的蓝屏。我父母给我们买了一辆 C64 准将车。我一度讨厌这东西。当我的朋友在他们的 NES 上玩超级马里奥兄弟的时候，我有一台非常烦人的电脑，它唯一的能力似乎就是在屏幕上(光标)闪烁一个浅蓝色的方块。

我的大书呆子哥哥总是用它做一些事情。他会按下按钮，然后显示器旁边一个看起来很奇怪的东西(1541 drive)开始发出奇怪的声音。他不知怎么搞了个游戏在上面运行。但这是一个奇怪的黑白游戏，主角是一根小棍子。我讨厌电脑，也讨厌游戏。

时间快进到一年后，我弟弟已经不用电脑了。他不再经常使用它了。主要是为了玩 Ultima，但是游戏比他喜欢的难，失去了兴趣。一天，在找事情做的时候，我打开了电脑。想象一下我，七岁左右，坐在这块大塑料面前。我哥哥不知何故教会了我`PRINT`和`GOTO`命令。这将提供无尽的娱乐时间与类似的节目:

```
10 PRINT “My brother smells.”

20 GOTO 10 
```

是的，我真正的第一个节目是我取笑我的兄弟。无论如何，在这一天事情是不同的。电脑的[手册](http://www.lemon64.com/manual/)放在键盘旁边。我打开它，开始浏览。里面有很多教你如何写 BASIC 语言的小程序。我不知道什么是基础，也不关心。驱使我输入程序的原因是手册是英文的，我想知道那些单词的意思。从那天起，我继续输入手册上介绍的任何程序。只是在发现我可以修改它们之后。这是一个非常令人沮丧的过程，但出于某种原因，我不得不弄清楚它是如何工作的。我仍然有一种内在的、有弹性的动力去弄清楚事物是如何运作的。特别是电脑和软件。我的第一台电脑唤醒了那个驱动器。我不知道没有它我会变成什么样。

我的第一台电脑进入了我的生活，我讨厌它。它很大。笨重。哑巴。基本没用。但它成了我过去非常重要的一部分。我希望我留着它。

**PS。我给了我女儿一个非常愚蠢的树莓派。她也讨厌它，但是我抓到她悄悄地把 Python 输入 IDLE…**
