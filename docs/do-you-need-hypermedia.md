# 需要超媒体吗？

> 原文：<https://dev.to/grahamcox82/do-you-need-hypermedia>

TL；博士-不。

近来超媒体风靡一时。新的超媒体 API 和标准一直在涌现。但是你真的需要这么麻烦吗？

超媒体很难。这不仅仅是把 URIs 加到你的回答和你的工作中。如果你想做得正确，会有各种各样的复杂性，从 URL 结构到有效载荷结构，甚至到声明的媒体类型。

如果你要把公共 API 支持添加到一个现有的应用程序中，这个应用程序已经有了一个现有的用户群和一个现有的或者很有潜力的开发者群，那么是的，超媒体可能是一个好主意。

如果你刚刚起步，还没有产品或用户或任何东西，超媒体将是成本大于收益。它会降低您的速度，并使您的最终产品开发和维护更加复杂，而且没有任何好处。当你拥有了所有这些有利的东西时，你会更好地理解你的领域，并且更好地做超媒体。
