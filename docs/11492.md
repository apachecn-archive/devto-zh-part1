# Flickr API Size enum

> 原文：<https://dev.to/adamkdean/flickr-api-size-enum-iaa>

仅供将来参考，这里是作为 JavaScript 枚举的 Flickr API 大小。

```
var flickr_sizes = {
    'square':       '_s',
    'large_square': '_q',
    'thumbnail':    '_t',
    'small':        '_m',
    'small_320':    '_n',
    'medium':       '',
    'medium_640':   '_z',
    'medium_800':   '_c',
    'large':        '_b',
    'large_1600':   '_h',
    'large_2048':   '_k',
    'original':     '_o'
}; 
```
