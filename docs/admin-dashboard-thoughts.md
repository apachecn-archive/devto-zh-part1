# 管理仪表板想法

> 原文：<https://dev.to/ben/admin-dashboard-thoughts>

不知何故，我发现构建良好的管理仪表板仍然很困难，尽管如此多的企业和项目都有相同的需求。在 Rails 领域，我已经使用了 Rails Admin、ActiveAdmin 和 Administrate。到目前为止,*可能是*最好的。这不是一个配置噩梦，但我仍然觉得这些工具没有一个能完全处理数据查询、显示资源、数据可视化和管理可视化。

我有兴趣尝试将 [Forest Admin](http://www.forestadmin.com/) 作为一家公司，试图通过服务来进入这个领域，但在你尝试新的方向之前，很难预见会遇到什么问题。

我还需要为我正在开发的不同应用程序找到一个与 Node/Mongo 集成的好工具。

有人对我可能不知道的服务有什么建议，或者对我如何构建管理仪表板有什么想法吗？
