# 雇佣热情的前端开发人员

> 原文:[https://dev.to/ben/hiring-passionate-front-end-developer](https://dev.to/ben/hiring-passionate-front-end-developer)

使用激动人心的新 JavaScript 技术！还有非常古老、糟糕的 JavaScript 技术，我们无法摆脱它，因为我们害怕接触它。和一些骨干一起工作，jQuery，YUI 的一些图书馆。我们尝试了有角度的，所以这整个有角度的东西都是分部分进行的。我们刚刚听说混音在 React 中是垃圾，这很糟糕，因为这是我们曾经做过的一件大事。当我们试图让我们的测试套件达到任何新成员都可以实际工作的程度时，你可能会先忙上几周，如果不是更长的话。如果你是一名女性，我们可能最终会少付你一点，但会因为聘用了一名多元化员工而表扬自己。

我们的公司文化是非常宽松和灵活的。你可以在任何你觉得最有效率的时间工作，但是我们会私下根据你在这方面的选择来评判你。但是别担心，我们绝不会粗鲁到告诉你我们的担忧。

您应该很好地理解一些我们并没有真正使用的任意模式，并准备在持续的基础上与您的同事假装理解一些编程主题。我们只雇佣 10 倍的工程师，所以我们鼓励胡说八道的文化，让自己看起来更聪明。就招聘过程本身而言，确保为技术面试学习你的二分搜索法树，但是不要担心，我们可能最终会根据你不会与之共事的人的直觉来招聘。

你必须充满激情，我们的意思是你必须始终对所有事情表现出激情，否则你可能会引发创始人的不安全感，他们永远不会承认他们愚蠢的广告平台是互联网上的一种消耗。

我们只雇佣忍者，或者摇滚明星。
