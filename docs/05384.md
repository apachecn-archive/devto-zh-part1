你可能不需要 Redux

我有服务器端开发背景，所以一直在学 DDD。在我看来，Redux 架构似乎包含了事件源和 CQRS。

React 组件不可避免地只包含基本领域问题，

*   发生了什么事？
*   **演示是什么数据？**

我认为 redux 鼓励 DDD 本质。
