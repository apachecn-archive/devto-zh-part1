# 我可以使用的 cli 节点包

> 原文：<https://dev.to/mahmoudelmahdi/cli-node-package-for-caniuse-48b>

**命令行界面**用于[的节点包可以使用](https://caniuse.com)，一个浏览器支持表，用于在您的终端中支持前端 web 技术。

#### 安装

适用于 npm 和纱线的包装:

```
~ npm install caniuse-cli
# or 
~ yarn add caniuse-cli 
```
