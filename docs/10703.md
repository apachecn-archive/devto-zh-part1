# go-of file

> 原文：<https://dev.to/dlion/go-ofile-1cip>

安全评估的第一阶段侧重于收集尽可能多的目标应用程序信息。这样做的一个工具是 [Goofile](http://tools.kali.org/information-gathering/goofile) ，它允许使用谷歌搜索指定域中的指定文件类型…

我听说这个 python 脚本的开发在 2012 年停止了，所以昨天我写了自己的 Golang 版本，名为 Go-ofile。

你想了解更多吗？[读吧！](https://domenicoluciani.com/2016/11/01/go-ofile.html)
