# [](#what-is-a-favicon-)什么是 favicon？

favicon 是一个扩展名为 *ico* 的小图像文件，用作网站用来覆盖浏览器选项卡 favicos 的代表性图像。这是我对 favicon 的定义。

# [](#introducing-favicon-generator)介绍图标生成器。

我是一个即将到来的现代网络开发人员，我过去常常给 wapka 网站编码，那时，我常常从源代码中获得 favicos。我不知道它们是如何产生的。我问了我的一个朋友，他说他在网上调整图片的大小，并把它们用作 favicos。所以我想到制作一个*命令行*工具，可以在我的控制台上从图像中生成图标。哇..超级酷不是吗？。然后我做了 **favico 发电机**。

# [](#what-can-i-do-with-the-tool-)工具可以做什么？

它是一个方便的工具，可以在全球范围内安装，因此，可以在您的终端或控制台的任何角度调用它。该工具主要用于从图像生成 favicon.ico 图像。

# [](#why-should-i-get-this-tool-)我为什么要得到这个工具？

你应该得到它，因为它节省了每次上传和下载的压力。你可以随意创建网站图标。

# [](#where-can-i-get-this-tool-)在哪里可以得到这个工具？

这是一个 npm 模块，我把它开发成一个模块，所以它可以安装没有压力。要做到这一点，只需访问我的 github repo 的完整说明。回购的链接会在本文最后分享。同时，您可以使用命令从 npm 注册表安装它。

```
sudo npm i -g favico-generator 
```

不要忘记查看这里的说明。
