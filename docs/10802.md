## [](#so-let-scala-games-begin)那么让(scala)游戏开始吧。

Scala 是由 Martin Odersky 教授创建的一种编程语言。它是面向对象的，也是一种函数式编程语言。所以你可以用 OOP，FP 或者两者都用来写你的代码。在 Scala 中，一切都是对象，基本类型不存在，操作符是方法。此外，有趣的面向对象的东西是特征接口，案例和对象类。函数在 FP 中是一等公民，所以你可以把它们赋给变量，像其他对象一样传递给其他函数。Scala 中其他有趣的 FP 概念是不可变变量、不可变数据结构和模式匹配。

Scala 是一种静态类型的语言，但感觉像是动态类型的。我把 Scala 称为现代版的 Java，混合了 Ruby 和 Python。Scala 最大限度地减少了不必要的语法(比如变量类型的声明),有了灵活的语法，你就有了用动态类型语言编写代码的感觉。

它是一种 JVM 语言，但有一个. NET 版本的 Scala，也存在 Scala.js 和 Scala-native 处于开发模式。

谁用 Scala？Twitter，LinkedIn，网飞，Tumbler，Foursquare …
