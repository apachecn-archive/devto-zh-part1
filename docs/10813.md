# 苹果事件 TL；面向开发人员的灾难恢复

> 原文:[https://dev.to/ben/apple-event-tldr-for-developers](https://dev.to/ben/apple-event-tldr-for-developers)

从一个开发者的角度来看，这个事件是相当蹩脚的。一些新功能和升级的芯片，性能更好。这很好，新相机看起来很酷，但对我们这些东西的制造商来说没有什么值得报道的。iOS 10 有很多新的东西需要考虑，但这些已经在之前的活动中讨论过了。

随着智能手机的发展，几代人将会越来越分散。支持更高图形和性能的新手机只是前景的一部分，我敦促开发者(或他们的老板)在探索新的图形和处理能力时，从渐进增强的角度来考虑。

苹果的无线耳机显然使用了一种[“类蓝牙技术”](http://www.theverge.com/2016/9/6/12825504/apple-wireless-headphones-custom-chip-iphone-7)，但并不是真正的蓝牙。这是值得注意的。

## 以下是我对这次活动的记录

*   苹果创立了[每个人都可以编码](http://www.apple.com/education/everyone-can-code/)的倡议，与 [Swift Playgrounds](http://www.apple.com/swift/playgrounds/) 合作
*   几个随机的 Apple Watch 功能。加上口袋妖怪 Go。我个人有点讨厌 Pokemon Go。

### 下一代 Apple Watch，《Series 2》

*   双核处理器，速度快 50%
*   新的 GPU 提供更好的图形/渲染性能
*   更亮、更好的显示屏
*   “游泳证明”
*   内置 GPS
*   watchOS 3
*   没有提到电池寿命

### iPhone 7

*   iOS 10
    *   应用程序的 Siri 功能
    *   HomeKit:“人人支持 HomeKit”
    *   更时髦的 iMessage。
    *   丰富的通知
*   “taptic engine”主页按钮，带有应用程序 API
*   升级的摄像头硬件+软件
    *   API 访问更多信息和照片功能，如实时照片
    *   iPhone 7 Plus 有两个摄像头用于光学变焦。
        *   景深
*   更明亮的显示屏
*   立体声扬声器，2 倍音量，更大音域
*   音频 Lightning 端口:附带适配器
*   苹果无线耳机:“Airpods”
*   Apple Pay 将在日本发布，日本有不同的 NFC 协议
*   新芯片:“A10 聚变”
    *   4 个内核:2 个主内核，2 个“高效率”内核
    *   6 核图形芯片
*   延长电池寿命
*   存储容量翻倍，达到 32GB/128GB/256GB
