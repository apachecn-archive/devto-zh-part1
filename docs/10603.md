## [](#favorite-dev-apps)最喜欢的开发应用

[![Imgur](../Images/18fb09de0f263cf969b9396a598f870a.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--K9pw_2st--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://i.imgur.com/JAP2Wn4.jpg)

如你所见，我的 Mac 上有很多应用程序，但我最喜欢的应该是:

1.  阿尔弗雷德:工作流程。我可以创建一个要点，列出我的回购，搜索黑客新闻和堆栈溢出，并使用 Toggl API 的快速键盘快捷键跟踪我的时间。

2.  Jetbrains IDEs :我知道 Visual Studio 现在在 Mac 上，VSCode 有 Intellisense，但是对于大项目来说，WebStorm 和 IntelliJ 是很好的应用。

3.  VSCode :为了构建我的 CRUD 应用程序，我使用了 VSCode，它自首次发布以来已经走过了漫长的道路。我是#TeamAtom，但是因为智能感知，我已经换了。伟大的应用程序和快速。

4.  [ShellRunner](https://itunes.apple.com/us/app/shell-runner/id689441182?mt=12) :一个测试 shell 脚本的小实用程序。

5.  [表情](https://itunes.apple.com/us/app/expressions/id913158085?mt=12):又一个玩 RegEx 的实用程序。伟大的新手和白发苍苍的老兵一样。

6.  [颤](https://itunes.apple.com/us/app/quiver-the-programmers-notebook/id866773894?mt=12):程序员的笔记本。我这里有几个笔记。界面简单但很好看

7.  SnippetsLab :一个保存你以后想用的代码片段的好地方。

最后一点...

1.  [破折号](https://kapeli.com/dash):无需互联网连接，直接在你的 Mac 电脑上操作。多棒啊。你可以通过购买来添加你自己的文档和注释，尽管它是免费的(如果你没有购买，你必须等待 20 秒左右才能在后台使用它)。值 15 美元。
