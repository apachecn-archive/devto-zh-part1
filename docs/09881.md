# 你讨厌 JavaScript 吗？

> 原文:[https://dev.to/reverentgeek/do-you-hate-javascript](https://dev.to/reverentgeek/do-you-hate-javascript)

[![NAAAAAAAN!](../Images/deffcfab1f561eba3c1cef3f7aa09fb7.png)T2】](//cdn-images-1.medium.com/max/1600/0*GRB7pnGaz3Uiz9sr.jpg)

如果你在 Twitter 上关注我，你会知道我很喜欢 JavaScript。最近有人问我，

*“你讨厌 JavaScript 吗？”*

不要！我非常喜欢使用 JavaScript 来构建应用程序。

## [](#but-i-like-a-good-laugh)但是，我喜欢开怀大笑。

我所知道的每一种编程语言都有自己的挑战和特点。JavaScript 以异常古怪而闻名。有时，有点令人沮丧和困惑。

由于我经常使用 JavaScript，我有资格嘲笑它的缺点。对我来说，取笑我不懂的其他语言并不酷。

[![](../Images/22a7a6a398ef56376e150ad7025160a8.png)T2】](//cdn-images-1.medium.com/max/1600/0*-QcXOsYe-ssnF2AO.jpg)

事实上，自 ES2015 (ES6)以来，JavaScript 语言本身已经变得*更好了*。一起工作是一种乐趣。我们今天面临的挑战更多的是语言之外的挑战。诸如可供选择的库的激增、框架的复杂性、打包和构建工具，以及对旧浏览器的支持。

## 有一个 JavaScript 可以做到这一点。

成为 JavaScript 开发人员的最佳时机。

[阿特伍德定律](https://blog.codinghorror.com/the-principle-of-least-power/)指出:

> 任何*可以*用 JavaScript 写的应用，*最终都会*用 JavaScript 写。

现在比以往任何时候都更是如此。你可以用 [React](https://facebook.github.io/react/) 、 [AngularJS](https://angularjs.org/) 、 [Vue.js](https://vuejs.org/) 或者其中一个【插入大量】框架来构建网站。你可以用 [Cordova](https://cordova.apache.org/) ， [Sencha](https://www.sencha.com/products/touch/) ， [Ionic](https://ionicframework.com/) ， [NativeScript](http://www.telerik.com/nativescript) ，或者 [tabris.js](https://tabrisjs.com/) 来构建移动应用。可以用 [Electron](http://electron.atom.io/) 或者 [NW.js](https://nwjs.io/) 构建桌面 app。你可以用[强尼五号](https://www.sparkfun.com/j5ik)或者 [Cylon.js](https://cylonjs.com/) 建造机器人。

## [](#want-to-learn-more-about-javascript)想了解更多 JavaScript 知识？

对编程一窍不通？JavaScript 是一个很好的起点！

天天用 JavaScript？总是有更多的东西要学！这就是我热爱做程序员的原因！

### [](#get-some-tools)获取一些工具。

这里有一些你可能想放在身边的工具。

1.  在你喜欢的浏览器中打开 JavaScript 控制台: [Chrome](https://developers.google.com/web/tools/chrome-devtools/console/) 、 [Firefox](https://developer.mozilla.org/en-US/docs/Tools/Browser_Console) 、  。现代网络浏览器内置了优秀的开发工具。
2.  给自己找个好编辑。 [Visual Studio Code](https://code.visualstudio.com/) 是目前我最喜欢的 JavaScript。它是免费的。它可以在 Mac、Windows 和 Linux 上运行。调试是顶尖的。而且，越来越好了。
3.  安装 [Node.js](https://nodejs.org/) 。这不仅是一个玩 JavaScript 的好方法，你还可以得到 [npm](https://www.npmjs.com/) ，它是安装其他 JS 实用程序和框架的必要工具。
4.  使用 npm 安装[实时服务器](https://www.npmjs.com/package/live-server)。这是一个很棒的小网络服务器，你可以用它来玩 JavaScript，HTML 和 CSS。它会监控您的文件，并立即用最新的更改刷新您的浏览器。

接下来，查阅一些学习资料。这不是一份详尽的清单。如果有喜欢的，请在评论中建议！
