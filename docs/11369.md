实际上，我很惊讶我预期的代码比实际情况差得多，我不是说它好，但是我可以看到不止一个方法，我看不到任何明显重复的代码。我可以看到一些可怕的变量名。以后的自己注意不要用 tmp676_674！

假设我在 2004 年写了这篇文章，那时候我在做什么呢？那是在我开始从事 It 工作之前。(很难想象我知道！)甚至可能是在我开始和 Keith 住在一起之前，在他为我的第一个 It 角色招募我之前。我想我当时可能是 Defra 的管理员。

如果我们用技术术语来描述 2004 年，那是在我加入 facebook 之前，Office 2003 和 XP 可能已经安装在我的个人电脑上，可能感觉是全新的。
