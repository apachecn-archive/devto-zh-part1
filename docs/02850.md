# 没有位置信息的解析器是小孩子的用处

解析器必须带有位置信息(例如文件、行、行、字节)。

没有位置信息，我们无法提供解析数据的适当错误信息。

KISS T1 和 T2 YAGNI T3 也很重要，但是如果你提供一个库，位置信息对解析器来说是必要的。
